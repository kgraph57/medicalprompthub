## 現状

HPの顔となるHeroSectionなんですが、正直なところ主張が弱くてインパクトが足りない感じがしています。普通HPだとここが顔になるので、ここでインパクトを出したいんです。

特に気になっているのは：
- タイトルが小さくて、視覚的なインパクトが不足
- メッセージが控えめで、ユーザーに刺さる表現になっていない
- 全体的にスペーシングが狭くて、余白が少ない
- CTAボタンが目立たなくて、アクションを促す力が弱い

## お願いしたいこと

HPの顔となるHeroSectionを、よりインパクトのあるデザインと言葉に改善していただけますか？

### 改善ポイント

1. **視覚的インパクトの強化**
   - タイトルを大きく、グラデーションなどで目立たせる
   - 背景にグラデーションを追加して、視覚的な階層を作る

2. **メッセージの強化**
   - サブタイトルをより明確で力強い表現に
   - コアメッセージをボックスで強調表示
   - 医療従事者の心に響く言葉選び

3. **インタラクション要素の改善**
   - 検索バーを大きく、より目立つデザインに
   - CTAボタンを魅力的に、ホバーエフェクトを追加
   - ボタンにアニメーション効果を追加

4. **スペーシングの改善**
   - パディングとマージンを拡大して、余白を確保
   - レスポンシブデザインを考慮した適切なサイズ調整

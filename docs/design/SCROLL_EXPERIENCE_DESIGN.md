# スクロール体験改善設計書

## 概要

現在のホームページでは「もっと見る」ボタンをクリックすると、すぐにプロンプト一覧が表示される。しかし、ユーザーがサービスの全体像や各コンテンツの価値を理解する前にプロンプトが表示されるため、サービスの魅力が十分に伝わらない。

本設計では、スクロール時にナラティブな体験を提供し、ユーザーがサービスの各機能（Courses、Tips、Prompts、Guides）を段階的に理解できるようにする。

## 現状の問題点

1. **即座にプロンプト一覧が表示される**：「もっと見る」をクリックすると、すぐにプロンプトカードが表示され、サービスの全体像が見えない
2. **各コンテンツの説明が不足**：Courses、Tips、Guidesなどの各機能が何をするのか、ユーザーにとってどんな価値があるのかが伝わらない
3. **スクロール体験が単調**：企業のホームページのようなナラティブな体験がなく、ユーザーのエンゲージメントが低い

## 改善の方向性

### 1. ナラティブなセクション構成

「もっと見る」をクリックした後、以下の順序でセクションを表示する：

```
Hero Section（現在のまま）
  ↓ スクロール
Feature Overview Section（新規）
  - サービスの3つの主要価値を提示
  ↓ スクロール
Content Showcase Section（新規）
  - Courses、Tips、Guides、Promptsを視覚的に紹介
  ↓ スクロール
Use Case Section（新規）
  - 具体的な使用シーンを3-4個提示
  ↓ スクロール
Prompt Grid Section（既存を改善）
  - プロンプト一覧
```

### 2. 各セクションの詳細設計

#### Feature Overview Section

**目的**：サービスの3つの主要価値を簡潔に伝える

**コンテンツ**：
- **診断支援**：100以上の実践的なプロンプトで、鑑別診断や症例分析をサポート
- **学習支援**：体系的なコースとTipsで、AIの効果的な活用方法を学習
- **業務効率化**：症例報告、論文執筆、統計解析などの業務を効率化

**デザイン**：
- 3カラムのカードレイアウト（モバイルは1カラム）
- アイコン + タイトル + 説明文
- スクロール時にフェードイン

#### Content Showcase Section

**目的**：各コンテンツタイプ（Courses、Tips、Guides、Prompts）を視覚的に紹介

**コンテンツ**：

1. **Courses（コース）**
   - タイトル：「体系的に学ぶAI活用コース」
   - 説明：「AIの基礎から実践まで、ステップバイステップで学べる構造化されたコース」
   - CTA：「コースを見る」→ `/courses`
   - プレビュー：人気コース3つを小さなカードで表示

2. **Tips（活用テクニック）**
   - タイトル：「すぐに使えるAI活用Tips」
   - 説明：「プロンプトの書き方から高度なテクニックまで、実践的なTipsを提供」
   - CTA：「Tipsを見る」→ `/tips`
   - プレビュー：人気Tips 3つを小さなカードで表示

3. **Guides（ワークフロー）**
   - タイトル：「実務に使えるワークフローガイド」
   - 説明：「症例報告、論文執筆、英文校正など、実務に直結するステップバイステップガイド」
   - CTA：「ガイドを見る」→ `/guides`
   - プレビュー：主要ガイド3つを小さなカードで表示

4. **Prompts（プロンプト集）**
   - タイトル：「100以上の実践的プロンプト」
   - 説明：「診断、研究、文書作成など、あらゆる場面で使えるプロンプトライブラリ」
   - CTA：「プロンプトを探す」→ スクロールダウン
   - プレビュー：人気プロンプト3つを小さなカードで表示

**デザイン**：
- 2x2グリッドレイアウト（モバイルは1カラム）
- 各セクションは背景色を微妙に変えて視覚的に区別
- ホバー時にカードが浮き上がるアニメーション

#### Use Case Section

**目的**：具体的な使用シーンを提示し、ユーザーが自分の状況に当てはめられるようにする

**コンテンツ**：

1. **救急外来での診断支援**
   - シーン：「複雑な症状を持つ患者の鑑別診断をAIがサポート」
   - 使用機能：Prompts（鑑別診断）
   - 効果：「診断時間を30%短縮」

2. **症例報告の作成**
   - シーン：「症例報告の構成から執筆までをステップバイステップでガイド」
   - 使用機能：Guides（症例報告ワークフロー）
   - 効果：「作成時間を50%削減」

3. **研究データの統計解析**
   - シーン：「統計解析コードの生成から結果の解釈までをサポート」
   - 使用機能：Prompts（統計解析）
   - 効果：「解析時間を40%短縮」

4. **AI活用スキルの習得**
   - シーン：「AIの基礎から実践まで、体系的に学習」
   - 使用機能：Courses（AI基礎コース）
   - 効果：「2週間で実務レベルに到達」

**デザイン**：
- 4カラムのカードレイアウト（タブレットは2カラム、モバイルは1カラム）
- アイコン + タイトル + 説明 + 効果
- スクロール時に順番にフェードイン

### 3. スクロールアニメーション

**実装方針**：
- Intersection Observer APIを使用
- スクロール位置に応じてセクションをフェードイン
- パフォーマンスを考慮し、アニメーションは軽量に

**アニメーションパターン**：
- フェードイン（opacity: 0 → 1）
- スライドアップ（transform: translateY(20px) → translateY(0)）
- 段階的表示（子要素を順番にアニメーション）

## 技術実装

### 新規コンポーネント

1. `FeatureOverviewSection.tsx`
   - サービスの3つの主要価値を表示

2. `ContentShowcaseSection.tsx`
   - Courses、Tips、Guides、Promptsを紹介

3. `UseCaseSection.tsx`
   - 具体的な使用シーンを表示

4. `ScrollAnimation.tsx`（または hooks/useScrollAnimation.ts）
   - スクロールアニメーションのロジック

### 既存コンポーネントの修正

1. `Home.tsx`
   - 新規セクションを追加
   - セクションの順序を調整

2. `HeroSection.tsx`
   - 「もっと見る」ボタンのスクロール先を調整

3. `PromptGridSection.tsx`
   - セクションタイトルを調整
   - 初期表示数を調整（6 → 9）

## デザインシステム

### カラーパレット

既存のTailwind設定を使用：
- Primary: `primary-600`, `primary-700`
- Neutral: `neutral-900`, `neutral-600`, `neutral-500`
- Background: `bg-white`, `bg-gray-50`

### タイポグラフィ

既存のクラスを使用：
- Section Title: `text-h2 font-semibold`
- Card Title: `text-h3 font-semibold`
- Body: `text-body`

### スペーシング

- Section間: `py-16 md:py-24`
- Card間: `gap-6 md:gap-8`
- Container: `max-w-7xl mx-auto px-4 md:px-6`

## 実装順序

1. **Phase 1**：新規セクションのコンポーネント作成
   - FeatureOverviewSection
   - ContentShowcaseSection
   - UseCaseSection

2. **Phase 2**：スクロールアニメーションの実装
   - useScrollAnimation hook作成
   - 各セクションにアニメーション適用

3. **Phase 3**：Home.tsxへの統合
   - 新規セクションを追加
   - セクションの順序を調整

4. **Phase 4**：レスポンシブ対応とテスト
   - モバイル、タブレット、デスクトップでの表示確認
   - パフォーマンステスト

## 期待される効果

1. **ユーザーエンゲージメントの向上**
   - スクロール時のナラティブな体験により、サービスの理解が深まる
   - 各コンテンツの価値が明確になり、探索意欲が高まる

2. **コンバージョン率の向上**
   - 具体的な使用シーンを提示することで、ユーザーが自分の状況に当てはめやすくなる
   - CTAが明確になり、次のアクションが取りやすくなる

3. **ブランドイメージの向上**
   - モダンでプロフェッショナルなデザインにより、信頼性が向上
   - 企業ホームページのような洗練された体験を提供

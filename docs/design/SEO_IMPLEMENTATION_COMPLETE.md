## 📊 実装内容

### 各ページのSEO要素

1. **タイトルタグ**
   - 50-60文字
   - キーワード含む
   - ページごとに一意

2. **メタディスクリプション**
   - 150-160文字
   - CTA含む
   - 魅力的な説明

3. **Open Graph（OGP）**
   - og:title
   - og:description
   - og:image
   - og:url
   - og:type
   - og:locale

4. **Twitter Card**
   - twitter:card
   - twitter:title
   - twitter:description
   - twitter:image

5. **構造化データ（JSON-LD）**
   - WebSite（ホーム）
   - Organization（ホーム）
   - Article（プロンプト、Tips）
   - Course（コース詳細）
   - LearningResource（レッスン詳細）
   - HowTo（ガイド詳細）
   - FAQPage（FAQ）
   - CollectionPage（Changelog）
   - Periodical（ジャーナル詳細）

6. **カノニカルURL**
   - 全ページに設定

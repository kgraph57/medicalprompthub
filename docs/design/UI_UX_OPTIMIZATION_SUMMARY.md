## 🎯 最適化の原則

### 1. 8pxグリッドシステム
- すべてのサイズは8pxの倍数
- パディング: 8px, 12px, 16px, 24px
- マージン: 8px, 12px, 16px, 24px
- ギャップ: 8px, 12px, 16px, 24px

### 2. 情報密度
- **高密度**: ダッシュボード、一覧ページ
- **中密度**: 詳細ページ、コンテンツ表示
- **低密度**: ヒーローセクション

### 3. 視覚的階層
- **Primary**: タイトル、主要な数値（text-xl, text-2xl）
- **Secondary**: 説明、統計（text-base, text-sm）
- **Tertiary**: ラベル、メタ情報（text-xs, text-[10px]）

### 4. カードサイズ
- **コンパクトカード**: 80-100px（統計、メトリクス）
- **標準カード**: 120-160px（一覧表示）
- **詳細カード**: 200-300px（詳細情報）

### 5. アニメーション
- **Duration**: 200-300ms（素早い反応）
- **Easing**: `ease-out`（自然な動き）
- **Scale**: 1.01-1.02（控えめな拡大）

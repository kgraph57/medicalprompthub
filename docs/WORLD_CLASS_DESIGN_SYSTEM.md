# 世界最高水準のデザインシステム

## 🎯 設計原則

### 1. 8pxグリッドシステム
すべてのサイズは8pxの倍数で統一
- **パディング**: 8px, 12px, 16px, 24px, 32px
- **マージン**: 8px, 12px, 16px, 24px, 32px, 48px
- **ギャップ**: 8px, 12px, 16px, 24px

### 2. 情報密度の最適化
- **高密度**: ダッシュボード、一覧ページ（1画面に多くの情報）
- **中密度**: 詳細ページ、コンテンツ表示（読みやすさと情報量のバランス）
- **低密度**: ヒーローセクション（視覚的インパクト重視）

### 3. 視覚的階層
- **Primary**: 最も重要な情報（タイトル、主要な数値）
- **Secondary**: 補足情報（説明、統計）
- **Tertiary**: メタ情報（ラベル、日付）

### 4. タイポグラフィスケール
- **H1**: 32-40px（font-bold）
- **H2**: 24-28px（font-bold）
- **H3**: 18-20px（font-semibold）
- **Body Large**: 16px（font-normal）
- **Body**: 14px（font-normal）
- **Small**: 12px（font-normal）
- **Tiny**: 10px（font-normal）

### 5. カードサイズの標準化
- **コンパクトカード**: 高さ 80-100px（統計、メトリクス）
- **標準カード**: 高さ 120-160px（一覧表示）
- **詳細カード**: 高さ 200-300px（詳細情報）
- **フルカード**: 高さ 400px+（コンテンツ表示）

### 6. スペーシングシステム
- **セクション間**: 48px, 64px, 80px
- **カード間**: 12px, 16px, 24px
- **要素間**: 8px, 12px, 16px
- **内部余白**: 12px, 16px, 24px

## 🎨 カラーパレット

### Primary Colors
- **Primary**: 医療の信頼感を表すブルー系
- **Success**: 緑系（完了、成功）
- **Warning**: オレンジ系（注意）
- **Error**: 赤系（エラー）

### Neutral Colors
- **Background**: 白/ダークグレー
- **Foreground**: ダークグレー/白
- **Muted**: グレー系（補足情報）

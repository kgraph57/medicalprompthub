## 📊 優先順位

### 🔴 最優先（フェーズ1）

#### 1. クイズ機能の実装
**目的**: 理解度を確認し、定着を促進

**実装内容**:
- 各レッスンにクイズセクションを追加

**技術**:
- クイズコンポーネントの作成
- 回答状態の管理
- 正解率の計算

#### 2. 実践演習の追加
**目的**: 実際にAIを使う経験を提供

**実装内容**:
- 「AIに質問してみる」演習
- 「AIの回答を評価する」演習
- 「自分の業務での活用を考える」演習

**技術**:
- 外部AIツールへのリンク
- 演習用のフォーム
- 回答の保存（ローカルストレージ）

#### 3. 医療現場の具体例の強化
**目的**: 実践的な理解を促進

**実装内容**:
- ケーススタディの追加
- 実際の活用事例
- 自分の専門分野での応用を考える演習

## 📊 評価システム

### 理解度の測定

1. **クイズスコア**
   - 各セクションのクイズ: 80%以上で合格
   - 総合クイズ: 80%以上で合格

2. **実践演習の完了**
   - 演習1: AIに質問してみる ✅
   - 演習2: 回答を評価する ✅
   - 演習3: 活用計画を作成 ✅

3. **実践課題の提出**
   - AI体験レポート
   - 活用計画

### XP獲得

- レッスン完了: +10 XP
- クイズ80%以上: +5 XP（ボーナス）
- 実践演習完了: +3 XP（各演習）
- 実践課題提出: +5 XP

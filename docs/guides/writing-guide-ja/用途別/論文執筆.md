# 論文執筆の書式ガイドライン

学術論文の執筆に関する書式ガイドラインをまとめています。

## 論文構成

### 基本構成（順序）

1. **表紙**
2. **目次**
3. **本編**
   - 序論
   - 本論
   - 結論
4. **註**
5. **参考文献**（年代順、URLは閲覧日を記載）
6. **図表一覧**
7. **図表**
8. **発表論文リスト**（任意）

## タイトル・副題

- **副題**は前1字空け + 全角ダッシュで挟む
- 例：タイトル ―副題―

## 章・節の表記

- **章**：全角算用数字 + 後1字空け（例：1 章）
- **節**：全角算用数字 + 後1字空け（例：1.1 節）
- 章末・節末は**1行空ける**

## 註・引用

### 註の形式

- 脚注方式または末尾方式のいずれかを選択し、統一する
- 和文文献、英文文献、欧文文献、その他言語の文献で適切な形式を使用する
- 同上・前掲・op.cit./Ibid.などの略記法を適切に運用する

### 引用の表記

- 直接引用は引用符で囲む、または字下げする
- 間接引用（要約・言い換え）も出典を明記する
- URLを含む場合は閲覧日を記載する（全角括弧+半角年月日）
  - 例：（閲覧日：2025年12月10日）

## 参考文献

### 並び順

- **年代順**に並べる
- URLを含む場合は閲覧日を記載する（全角括弧+半角年月日）

### 表記形式

#### 和書

- 著者名（出版年）『書名』出版社
- 例：山田太郎（2025）『論文執筆の基礎』東京出版

#### 洋書

- 著者名（出版年）*書名* 出版社
- 例：Smith, J. (2025) *Academic Writing Guide* Oxford University Press

#### Webサイト

- 著者名（公開年）「ページタイトル」サイト名、URL（閲覧日：全角括弧+半角年月日）
- 例：田中花子（2025）「論文執筆のコツ」学術サイト、https://example.com（閲覧日：2025年12月10日）

#### 電子資料

- 適切な形式で記載する（データベース名、アクセス日など）

#### ジャパンナレッジ

- 適切な形式で記載する

## 図表

### 図版

- **通し番号**を付ける（図1、図2など）
- キャプションは**全角丸括弧+半角数字**（例：（図1））
- **出典を必ず明記する**（執筆者作成・撮影の場合も明記）

### 表

- **通し番号**を付ける（表1、表2など）
- キャプションは**全角丸括弧+半角数字**（例：（表1））
- **出典を必ず明記する**

### 図表一覧

- 論文の最後に図表一覧を配置する
- 図表の番号、タイトル、ページ数を記載する

## フォント・サイズ

### 基本設定

- **和文**：MS明朝
- **欧文**：Times

### サイズ指定

- **章**：10.5pt MSゴシック太字
- **節**：10pt MSゴシック
- **本文**：10pt MS明朝
- **タイトル等**：指定に従う

## 数字の表記

- 数字は**全て半角**で表記する
  - 年号・日付・世紀・単位など
  - 例：2025年、12月10日、21世紀、100kg

## 本文表記の統一

- 用語の表記を統一する
- 略語を使用する場合は、初出時に正式名称を併記する
- 専門用語は適切に使用する

## 論文執筆の基本原則

### 型を守る

- 論文には「型」がある
- 読者やエディターはそれを期待している
- 型通りに書いていく

### 対になる構成

- **Introductionの最後 ↔ Conclusion**
  - Introductionの最後とConclusionは同時に書く
- **Methods ↔ Results**
  - MethodsとResultsはペアで書く
  - Methodsは再現性と正確性
  - Resultsは淡々と結果だけ（考察は書かない）

### 執筆の優先順位

1. **MethodsとResultsを最優先**
2. **Introductionの最後とConclusionを同時に書く**
3. **Abstractはその後**
4. **Titleは最後**

### 論文の構造と流れ

```
Abstract
  ↑ ↓
Introduction → Methods → Results → Discussion
  ↑                                            ↓
Conclusions ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

### MethodsとResultsの対応

- Methodsで書いた内容に対応するResultsを書く
- ResultにはReferenceを使わない
- Methodsは再現性があるかどうかが重要
  - 読者が研究を再現できるように詳細に書く

## チェックリスト

論文提出前に以下を確認する：

- [ ] 論文構成が適切か（表紙→目次→本編→註→参考文献→図表一覧→図表）
- [ ] タイトル・副題の表記が正しいか
- [ ] 章・節の表記が統一されているか
- [ ] 註と参考文献が適切に記載されているか
- [ ] 参考文献は年代順に並んでいるか
- [ ] 図表の出典がすべて明記されているか
- [ ] フォント・サイズが指定通りか
- [ ] 数字は半角で統一されているか
- [ ] 用語・表記が統一されているか
- [ ] MethodsとResultsが対応しているか
- [ ] Introductionの最後とConclusionが対応しているか
- [ ] 剽窃がないか確認した

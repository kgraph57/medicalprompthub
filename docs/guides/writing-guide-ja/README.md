# 文章術ガイド

医師としての信頼性を保ちながら、多様なジャンルで効果的な日本語文章を作成するための完全ガイドです。

## 📚 フォルダ構成

```
文章術/
├── README.md                    # このファイル（全体のガイド）
├── 普遍的なルール.md            # 全ジャンル共通の基本ルール
└── 用途別/                      # ジャンル別の詳細ガイド
    ├── 日本語文章作成・完全指示書.md  # 総合ガイド（最重要）
    │
    ├── 【ビジネス系】
    │   ├── ビジネス文書の書き方.md
    │   └── セールスコピーの書き方.md
    │
    ├── 【Web・メディア系】
    │   ├── ブログ記事の書き方.md
    │   ├── note記事の書き方.md
    │   └── Web記事の書き方.md
    │
    ├── 【教育・技術系】
    │   ├── 教育コンテンツの書き方.md
    │   └── 技術解説の書き方.md
    │
    ├── 【学術・研究系】
    │   ├── レポート執筆.md
    │   └── 論文執筆.md
    │
    ├── 【創作系】
    │   └── エッセイの書き方.md
    │
    └── 参考文献.md              # 各ジャンルの参考資料
```

## 🚀 使い方

### 初めて使う場合

1. **普遍的なルール.md** を最初に読む
   - 全ジャンル共通の基本原則を理解

2. **用途別/日本語文章作成・完全指示書.md** を読む
   - 日本語文章の本質構造と理論的背景を理解
   - AIプロンプトも含む実践的なガイド

3. 執筆するジャンルに応じて、**用途別/** フォルダ内の該当ファイルを参照

### 執筆時のワークフロー

```
1. 普遍的なルール.md で基本原則を確認
   ↓
2. 用途別/[ジャンル名].md で詳細ルールを確認
   ↓
3. 執筆
   ↓
4. 各ファイルのチェックリストで最終確認
```

## 📖 各ファイルの特徴

### 普遍的なルール.md
- **対象**：全ジャンル共通
- **内容**：剽窃回避、表現の自由、著作権、数字表記、基本原則、文体統一、段落構成、引用ルール
- **使用タイミング**：執筆前の確認

### 日本語文章作成・完全指示書.md
- **対象**：全ジャンル（特に医療・教育・技術解説・note/書籍）
- **内容**：
  - 日本語文章の本質構造（可読性の3層構造、後出し言語特性）
  - ジャンル別の正解の文章設計
  - 文レベル完全ルール
  - 視覚設計ルール
  - AIプロンプト
- **使用タイミング**：深く理解したい場合、AIで文章を書き直す場合

### ジャンル別ファイル
各ファイルには以下の項目が含まれています：
- **構成**：基本的な文章構造
- **語彙の選び方**：適切な言葉選び
- **トーン**：文章の調子・雰囲気
- **文体**：です・ます調/だ・である調の使い分け
- **改行・段落**：視覚的な読みやすさ
- **見出し設計**：見出しの付け方
- **想定読者との距離感**：読者との関係性
- **可読性向上のテクニック**：読みやすくする工夫
- **よくある誤り**：避けるべき失敗例
- **文例・テンプレート**：実践例

## 🎯 医師としての信頼性を保つために

全ジャンル共通で以下の点に注意：

- ❌ **避けるべき表現**
  - YouTuber的表現（「めちゃくちゃ」「やばい」など）
  - 断定的すぎる表現（「必ず」「絶対に」など）
  - 過度な感情表現

- ✅ **推奨される表現**
  - 客観的な表現（事実に基づく記述、数値やエビデンスの提示）
  - 適度な謙遜（「と考えられる」「可能性がある」など）
  - 専門用語の適切な使用と初出時の説明

## 💡 よくある質問

### Q: どのファイルから始めればいいですか？
A: 初めての場合は「普遍的なルール.md」→「日本語文章作成・完全指示書.md」の順で読むことをおすすめします。

### Q: 複数のジャンルが混在する場合は？
A: 主要なジャンルに合わせて、必要に応じて他のジャンルのファイルも参照してください。

### Q: AIで文章を書き直したい場合は？
A: 「日本語文章作成・完全指示書.md」の第6章にAIプロンプトが含まれています。

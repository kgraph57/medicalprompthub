## 📚 コース一覧

### レベル1: AI基礎編（初心者向け）

   - 1980年代: エキスパートシステム
   - 2000年代: 機械学習の台頭
   - 2010年代: ディープラーニング革命
   - 2020年代: 生成AIの登場

3. **AIの現状と未来**
   - 現在のAIの能力
   - 医療分野での活用例
   - 今後の展望

## 📊 コース構造

```
コース（Course）
  ├─ レッスン（Lesson）
  │    ├─ スライド（Slide）
  │    │    ├─ テキストコンテンツ
  │    │    ├─ 画像・図表
  │    │    └─ インタラクティブ要素
  │    └─ クイズ（Quiz）
  │         ├─ 選択問題
  │         ├─ 正誤問題
  │         └─ 記述問題
  └─ 最終テスト
       └─ コース完了でバッジ獲得
```

---

## 🎮 ゲーミフィケーション要素

### XP獲得
- レッスン完了: +10 XP
- クイズ正解: +5 XP
- コース完了: +20 XP（ボーナス）

### バッジ
- 各コース完了で専用バッジ
- 全コース完了で特別バッジ

### レベルシステム
- レベル1: 初心者（0-200 XP）
- レベル2: 初級者（201-500 XP）
- レベル3: 中級者（501-900 XP）
- レベル4: 上級者（901-1400 XP）
- レベル5: エキスパート（1401+ XP）

## 📝 実装方針

### データ構造
- コース情報はデータベースに保存
- レッスンコンテンツはMarkdownファイルで管理
- クイズはJSON形式で定義

### UI/UX
- スライド形式の学習画面
- 進捗バーの表示
- 完了時のアニメーション
- バッジ獲得通知

### 段階的実装
1. コース1-3（基礎編）を優先実装
2. コース4-5（技術理解編）を追加
3. コース6-7（実践編）を追加

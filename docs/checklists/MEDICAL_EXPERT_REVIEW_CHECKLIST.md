# 医療専門家レビューチェックリスト

> **このチェックリストは、医療専門家がプロンプトとガイドをレビューする際に使用します。最低3名の医療専門家によるレビューが必須です。**

**レビュー対象**: 全プロンプト、全ガイド  
**レビュー期限**: リリース前1週間  
**承認条件**: 最低3名の医療専門家による承認

---

## 📋 レビュー項目

### 1. 医学的正確性

- [ ] **プロンプトの医学的内容が正確か**
  - [ ] 診断支援プロンプトの内容が正確か
  - [ ] 治療計画プロンプトの内容が正確か
  - [ ] 薬剤・処方プロンプトの内容が正確か
  - [ ] その他のプロンプトの内容が正確か

- [ ] **医療情報の最新性**
  - [ ] 2024年以降のガイドラインに準拠しているか
  - [ ] 古い情報や廃止された治療法が含まれていないか
  - [ ] 最新の研究結果が反映されているか

### 2. 使用場面の適切性

- [ ] **プロンプトの使用場面が適切か**
  - [ ] 実際の医療現場で使用可能か
  - [ ] 使用場面の説明が明確か
  - [ ] 適切な専門分野・職種で使用できるか

- [ ] **リスク評価**
  - [ ] プロンプトの使用に伴うリスクが適切に評価されているか
  - [ ] 重篤な疾患を見逃すリスクがないか
  - [ ] 誤診・誤治療のリスクがないか

### 3. 免責事項の適切性

- [ ] **免責事項が適切か**
  - [ ] 「医療行為の代替ではない」旨が明記されているか
  - [ ] AIの限界について説明されているか
  - [ ] ファクトチェックの重要性が強調されているか
  - [ ] 緊急時の対応について記載されているか

### 4. ガイドの品質

- [ ] **症例報告ガイド**
  - [ ] 医学的に正確か
  - [ ] 実際の症例報告作成に役立つか
  - [ ] 各ステップが明確か

- [ ] **論文執筆ガイド**
  - [ ] 医学的に正確か
  - [ ] 実際の論文執筆に役立つか
  - [ ] 各ステップが明確か

- [ ] **その他のガイド**
  - [ ] 医学的に正確か
  - [ ] 実際の業務に役立つか

### 5. 改善提案

- [ ] **プロンプトの改善提案**
  - [ ] より正確にするための改善点
  - [ ] より使いやすくするための改善点
  - [ ] 追加すべきプロンプト

- [ ] **ガイドの改善提案**
  - [ ] より正確にするための改善点
  - [ ] より使いやすくするための改善点
  - [ ] 追加すべきガイド

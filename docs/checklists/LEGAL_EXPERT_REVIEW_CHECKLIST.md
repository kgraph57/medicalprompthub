# 法務専門家レビューチェックリスト

> **このチェックリストは、法務専門家がプライバシーポリシーと利用規約をレビューする際に使用します。法務専門家によるレビューが必須です。**

**レビュー対象**: プライバシーポリシー、利用規約、免責事項  
**レビュー期限**: リリース前2週間  
**承認条件**: 法務専門家による承認

---

## 📋 レビュー項目

### 1. プライバシーポリシー

- [ ] **法的要件の完全性**
  - [ ] 個人情報保護法（日本）に完全準拠しているか
  - [ ] GDPR（EU）に完全準拠しているか（該当する場合）
  - [ ] その他の関連法規に準拠しているか

- [ ] **データ主体の権利**
  - [ ] アクセス権が明記されているか
  - [ ] 訂正権が明記されているか
  - [ ] 削除権（忘れられる権利）が明記されているか
  - [ ] データポータビリティ権が明記されているか
  - [ ] 異議申立権が明記されているか
  - [ ] 権利行使の方法が明確か

- [ ] **データの取り扱い**
  - [ ] 個人情報の収集について説明されているか
  - [ ] データの利用目的が明確か
  - [ ] データの保存期間が明記されているか
  - [ ] 第三者への提供について説明されているか
  - [ ] データのセキュリティ対策について説明されているか

- [ ] **Cookie・トラッキング**
  - [ ] Cookieの使用について説明されているか
  - [ ] Cookieの種類と目的が明確か
  - [ ] Cookieの無効化方法が説明されているか
  - [ ] 同意管理について説明されているか

- [ ] **連絡先**
  - [ ] プライバシーに関する問い合わせ先が明記されているか
  - [ ] データ保護責任者（DPO）が指定されているか（必要に応じて）

### 2. 利用規約

- [ ] **法的有効性**
  - [ ] 利用規約が法的に有効か
  - [ ] 準拠法・管轄裁判所が明記されているか
  - [ ] 契約の成立要件が明確か

- [ ] **免責事項**
  - [ ] 免責事項が適切に記載されているか
  - [ ] 法的に有効な免責事項か
  - [ ] 過失責任の免責が適切か

- [ ] **知的財産権**
  - [ ] 著作権について説明されているか
  - [ ] 商標権について説明されているか
  - [ ] ライセンスについて説明されているか

- [ ] **利用者の責任**
  - [ ] 利用者の責任が明確に記載されているか
  - [ ] 禁止事項が明確か
  - [ ] 違反時の対応が明記されているか

- [ ] **サービスの変更・終了**
  - [ ] サービスの変更について説明されているか
  - [ ] サービスの終了について説明されているか
  - [ ] 通知方法が明確か

### 3. 医療情報に関する免責

- [ ] **免責事項の明確性**
  - [ ] 「医療行為の代替ではない」旨が明確に記載されているか
  - [ ] 複数箇所に記載されているか
  - [ ] 強調表示されているか

- [ ] **リスクの明示**
  - [ ] AIの限界について説明されているか
  - [ ] ハルシネーション（誤情報生成）について説明されているか
  - [ ] ファクトチェックの重要性が強調されているか

### 4. 国際的コンプライアンス

- [ ] **GDPR準拠**
  - [ ] データ主体の権利が実装されているか
  - [ ] データ処理記録が整備されているか（必要に応じて）
  - [ ] プライバシー影響評価（DPIA）が実施されているか（必要に応じて）

- [ ] **個人情報保護法（日本）準拠**
  - [ ] 個人情報の適切な取り扱いが記載されているか
  - [ ] 同意の取得について説明されているか
  - [ ] 第三者提供の制限について説明されているか

- [ ] **その他の規制**
  - [ ] HIPAA（米国、該当する場合）に準拠しているか
  - [ ] その他の関連規制に準拠しているか

# 緩和ケア計画立案ガイド：実践編 ステップ3

## 計画の実行と調整：多職種連携と情報提供の最適化

緩和ケア計画は、医師、看護師、薬剤師、ソーシャルワーカー、理学療法士、栄養士など、多職種チームによって実行されます。AIは、チーム間の情報共有を円滑にし、患者・家族への情報提供の質を高めることで、計画の実行と継続的な調整を支援します。

### 1. 多職種連携のためのAI活用

多職種カンファレンスや情報共有の場で、AIは以下の点で貢献します。

| 連携タスク | AIの具体的な貢献 | メリット |
| :--- | :--- | :--- |
| カンファレンス準備 | 各職種の記録（看護記録、リハビリ記録など）から、現在の問題点と介入効果を抽出し、カンファレンス用の簡潔なサマリーを作成。 | 準備時間の短縮、議論の焦点化、全職種が最新情報を共有。 |
| ケア目標の共有 | 患者のACPで明確になった価値観と目標を、全職種が理解しやすい「共通言語」に変換し、ケア計画の各項目と関連付けて提示。 | ケアの一貫性の確保、職種間の認識のズレの防止。 |
| 経過記録の標準化 | 職種特有の専門用語を、他の職種にも理解しやすい表現に変換したり、記録の抜け漏れをチェックしたりする。 | 記録の質の向上、情報伝達の効率化。 |

### 2. 実用的なプロンプト例：患者・家族向け説明資料の作成

専門的な内容を含む緩和ケア計画を、患者や家族が理解しやすい言葉で説明するための資料を作成するプロンプトです。

#### プロンプト：専門用語を避けた患者向け説明資料の作成

\`\`\`
あなたは、患者とその家族に寄り添う医療コミュニケーターです。以下の専門的な緩和ケア計画の内容を、専門用語を避け、共感的で分かりやすい言葉で説明するための資料（A4サイズ1枚程度）のドラフトを作成してください。

対象読者: 医療知識のない患者本人とその家族。
トーン: 暖かく、希望を失わせない、しかし現実的な情報を提供する。

緩和ケア計画の専門的な内容:
*   現在の主要な問題: [例：難治性疼痛、呼吸困難、食欲不振]
*   目標: [例：NRS 4以下への疼痛コントロール、自宅での生活の継続]
*   症状緩和のための介入: [例：オピオイドの持続皮下注射、酸素療法、抗不安薬の頓用]
*   今後の見通し: [例：病状の進行に伴い起こりうること、その際の対応]
*   相談窓口: [例：24時間対応の訪問看護、担当ソーシャルワーカー]

出力形式:
以下の見出しを用いて、患者・家族が安心できるような説明資料の構成案と本文を作成してください。
1.  現在の状況と目標
2.  痛みを和らげるための工夫（薬とそれ以外の方法）
3.  もしもの時のために
4.  いつでも相談できる窓口
\`\`\`

### 3. 注意点とTips

*   双方向性の確保: AIが作成した説明資料は、必ず患者・家族の理解度に合わせて調整し、質問や懸念を自由に表現できる双方向の対話のきっかけとして使用してください。
*   文化・宗教的配慮: 患者の文化や宗教的背景によっては、特定の言葉や表現が不適切である場合があります。AIにその情報を入力し、配慮した表現を提案させることで、より個別化されたコミュニケーションが可能です。

# 緩和ケア計画立案ガイド：基本編 ステップ1

## AIを活用した緩和ケア計画立案の概要とメリット

緩和ケア計画の立案は、患者の身体的、精神的、社会的、スピリチュアルなニーズを包括的に評価し、患者とその家族の価値観と目標に沿ったケアを提供するための複雑なプロセスです。AI、特に大規模言語モデル（LLM）は、このプロセスを効率化し、より患者中心の意思決定を支援するための強力なツールとなり得ます。

### 1. AIが貢献できる主要な領域

AIは、緩和ケア計画立案の初期段階において、以下の3つの主要な領域で医療従事者を支援します。

| 貢献領域 | 具体的なAIの役割 | メリット |
| :--- | :--- | :--- |
| 情報統合と要約 | 膨大な電子カルテ、検査結果、経過記録から、緩和ケアに関連する重要な情報を迅速に抽出し、要約します。 | 評価時間の短縮、見落としの防止、多職種連携の円滑化。 |
| リスク評価と予測 | 症状の悪化リスク、特定の介入に対する反応予測、予後予測に関連するエビデンスを提示します。 | 客観的なデータに基づいた意思決定の支援、早期介入の機会創出。 |
| コミュニケーション支援 | 患者や家族への説明資料の作成、困難な会話のための対話スクリプトの提案、倫理的ジレンマの整理を支援します。 | 患者・家族との質の高いコミュニケーションの実現、医療従事者の精神的負担軽減。 |

### 2. 実用的なプロンプト例：緩和ケアニーズの抽出

患者の医療記録（匿名化されたもの）から、緩和ケアの観点で最も重要な情報を抽出するためのプロンプトです。

#### プロンプト：医療記録からの重要情報抽出

\`\`\`
あなたは経験豊富な緩和ケア専門医です。以下の患者の匿名化された医療記録のサマリーを読み、緩和ケア計画立案に不可欠な情報を抽出してください。

抽出対象情報:
1.  主要な診断名と現在の病状: 進行度、治療歴、現在の治療目標。
2.  身体的苦痛: 疼痛、呼吸困難、倦怠感などの主要な症状とその重症度（NRS/VAS）。
3.  精神的・社会的苦痛: 不安、抑うつ、経済的な問題、家族構成、介護状況。
4.  患者の価値観と希望: 治療に関する意向、人生の目標、重要視していること（もし記録にあれば）。
5.  予後に関する情報: 医師による予後予測（もし記録にあれば）。

医療記録サマリー:
[ここに匿名化された医療記録のサマリーを貼り付け]

出力形式:
各項目について箇条書きで簡潔にまとめてください。
\`\`\`

### 3. 注意点とTips

*   最終判断は人間が: AIの出力は、あくまで情報整理と意思決定を支援するためのものです。患者の個別性、感情、価値観はAIには完全に理解できません。最終的な緩和ケア計画は、必ず医療従事者の専門的な判断と患者・家族との対話に基づいて決定してください。
*   匿名化の徹底: 医療記録をAIに入力する際は、個人情報（氏名、生年月日、住所など）を完全に匿名化し、医療機関のセキュリティポリシーを厳守してください。

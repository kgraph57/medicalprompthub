# 緩和ケア計画立案ガイド：応用編 ステップ1

## 倫理的ジレンマの整理と予後予測の支援

緩和ケアの現場では、治療の差し控え・中止、人工的な栄養・水分補給の是非など、しばしば倫理的ジレンマに直面します。また、予後に関する情報は、患者・家族の意思決定に不可欠です。AIは、これらの複雑でデリケートな状況において、客観的な情報と倫理的枠組みを提供することで、医療従事者の意思決定を支援します。

### 1. AIによる倫理的ジレンマの整理

AIは、特定の臨床状況における倫理的な論点を整理し、関連する法的・倫理的ガイドラインを提示することで、医療従事者の思考プロセスをサポートします。

| 支援タスク | AIの具体的な役割 | 医療従事者の役割 |
| :--- | :--- | :--- |
| 論点の明確化 | 提示された事例（匿名化済み）から、倫理的ジレンマの核心（例：自律性の尊重 vs. 生命の維持）を抽出し、対立する価値観を整理。 | AIの整理に基づき、関係者（患者、家族、チーム）との対話の焦点を定める。 |
| ガイドラインの提示 | 関連する国の法律、医療倫理指針、専門学会のガイドライン（例：終末期医療に関するガイドライン）を検索し、要点を提示。 | 提示されたガイドラインを、個別の患者状況に照らし合わせて適用可能性を判断。 |
| 代替案の検討 | 倫理的ジレンマを解決するための複数の代替案（例：段階的な治療の縮小、緩和的鎮静の検討）と、それぞれのメリット・デメリットを提示。 | 患者・家族の価値観に最も合致する代替案を選択し、合意形成を図る。 |

### 2. 実用的なプロンプト例：倫理的ジレンマの構造化分析

治療の継続・中止に関する倫理的ジレンマを、体系的に分析するためのプロンプトです。

#### プロンプト：倫理的ジレンマの四分割法による分析

\`\`\`
あなたは、医療倫理の専門家です。以下の匿名化された臨床事例について、Jonsenらの四分割法（医学的適応、患者の意向、QOL、周囲の状況）に基づき、倫理的ジレンマを構造化して分析してください。

臨床事例:
[ここに、治療の継続・中止に関する匿名化された事例（患者の病状、予後、家族の意見、医療チームの懸念など）を貼り付け]

分析の要件:
1.  医学的適応: 治療の目的、効果、負担、予後に関する客観的な事実を整理。
2.  患者の意向: 患者の自律性、ACPの有無、意思決定能力、過去の発言を整理。
3.  QOL（生活の質）: 患者の現在の苦痛、将来のQOLの可能性、治療によるQOLへの影響を整理。
4.  周囲の状況: 家族の意見、経済的・社会的要因、医療チームの負担、文化・宗教的背景を整理。

結論:
上記の分析に基づき、この事例における最も重要な倫理的論点と、推奨される対話の方向性を簡潔に述べてください。
\`\`\`

### 3. 注意点とTips

*   AIは判断を下さない: AIは倫理的判断を下すことはできません。その役割は、論点を整理し、客観的な情報を提供することに限定されます。最終的な倫理的判断は、医療従事者チームと患者・家族との対話を通じて行われるべきです。
*   予後予測モデルの限界: AIによる予後予測モデル（例：特定の疾患における生存期間予測）は、あくまで統計的な情報です。個々の患者の予後は、予測モデルの結果だけでなく、臨床的な経過、患者の回復力、そして何よりも患者の希望を考慮して慎重に伝える必要があります。

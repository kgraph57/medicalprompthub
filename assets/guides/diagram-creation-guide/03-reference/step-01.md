# 応用編 - ステップ1: プレゼンテーション用図解の作成

## はじめに

このステップでは、学会発表や教育用のプレゼンテーション図解を作成する方法を学びます。
プレゼンテーション用の図解は、視覚的にわかりやすく、簡潔であることが重要です。

---

## プレゼンテーション図解の特徴

プレゼンテーション用の図解には以下の特徴があります：

- **簡潔性**: 複雑な内容を簡潔に表現
- **視覚的インパクト**: 一目で理解できる視覚的なデザイン
- **明確性**: メッセージが明確に伝わる
- **統一性**: スライド全体で統一されたデザイン

---

## 効果的なプロンプト例

### 例1: 学会発表用の病態生理図

```
以下の内容を学会発表用の図解として作成してください：

【タイトル】
心筋梗塞の病態生理

【主要なメッセージ】
冠動脈閉塞による心筋虚血と壊死

【主要な要素】
1. 正常な冠動脈と心筋
2. 冠動脈の閉塞
3. 心筋の虚血と壊死
4. 心機能の低下

【視覚的な要件】
- シンプルでわかりやすいデザイン
- 色分けで正常と異常を区別
- 矢印でプロセスの流れを示す
- プレゼンテーション用の高解像度
- 背景は白または薄い色
```

### 例2: 教育用の診断フロー図

```
以下の内容を教育用の図解として作成してください：

【タイトル】
胸痛の診断アプローチ

【主要なメッセージ】
系統的な診断アプローチの重要性

【主要な要素】
1. 症状の評価
2. 緊急性の判断
3. 診断検査の選択
4. 治療方針の決定

【視覚的な要件】
- 教育用にわかりやすいデザイン
- 各ステップを明確に区別
- 色分けで重要度を示す
- フローチャート形式
- 印刷可能な高品質
```

---

## プロンプトのポイント

### 1. メッセージを明確にする

図解で伝えたい主要なメッセージを明確にします。

### 2. 簡潔性を重視する

複雑な内容を簡潔に表現します。

### 3. 視覚的インパクトを考慮する

一目で理解できる視覚的なデザインを指定します。

### 4. 用途に応じた形式を指定する

学会発表用、教育用など、用途に応じた形式を指定します。

---

## 実践例

実際にNanobananaでプレゼンテーション用図解を作成してみましょう：

1. 上記のプロンプト例をコピー
2. NotebookLMのNanobananaに貼り付け
3. 「生成」ボタンをクリック
4. 生成された図解を確認
5. 必要に応じてプロンプトを修正して再生成
6. スライドに組み込む

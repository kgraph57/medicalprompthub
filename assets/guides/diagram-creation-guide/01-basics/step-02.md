# 基本編 - ステップ2: 病態生理の図解化

## はじめに

このステップでは、病態生理を図解化する方法を学びます。
病態生理は複雑なメカニズムを視覚的に説明する必要があるため、適切なプロンプト設計が重要です。

---

## 病態生理図解の基本構造

病態生理図解には以下の要素が含まれることが多いです：

- **正常状態**: 正常な生理的プロセス
- **異常状態**: 疾患による変化
- **プロセスの流れ**: 時系列での変化
- **関連要素**: 関連する臓器、細胞、分子など

---

## 効果的なプロンプト例

### 例1: 心筋梗塞の病態生理

```
以下の心筋梗塞の病態生理を図解化してください：

【正常状態】
- 冠動脈から心筋への正常な血流
- 心筋細胞の正常な機能

【異常状態】
- 冠動脈の閉塞（血栓、プラーク）
- 心筋への血流途絶
- 心筋細胞の虚血と壊死
- 心機能の低下

【プロセスの流れ】
1. 冠動脈の狭窄・閉塞
2. 心筋への血流減少
3. 心筋細胞の虚血
4. 心筋細胞の壊死
5. 心機能の低下

【視覚的な要素】
- 正常な状態は青色で表示
- 異常な状態は赤色で表示
- 矢印でプロセスの流れを示す
- 各段階にラベルを付ける
```

### 例2: 糖尿病の病態生理

```
以下の2型糖尿病の病態生理を図解化してください：

【主要な要素】
1. インスリン抵抗性
2. 膵臓のβ細胞機能低下
3. 高血糖状態

【プロセスの流れ】
1. インスリン抵抗性の発生
2. インスリン分泌の増加
3. β細胞の疲弊
4. インスリン分泌の低下
5. 高血糖の持続

【視覚的な要素】
- インスリンを黄色で表示
- 血糖を赤色で表示
- プロセスの流れを矢印で示す
- 各臓器（膵臓、肝臓、筋肉）を明確に区別
```

---

## プロンプトのポイント

### 1. 構造を明確にする

病態生理の各段階を明確に分けて記述します。

### 2. 視覚的な要素を指定

色、矢印、ラベルなどの視覚的な要素を指定します。

### 3. 関連要素を含める

関連する臓器、細胞、分子などを含めます。

### 4. 時系列を明確にする

プロセスの流れを時系列で明確にします。

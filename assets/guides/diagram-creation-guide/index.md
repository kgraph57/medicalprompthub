# 【初心者向け】Nanobanana活用ガイド:実践プロンプト集

## はじめに

**Nanobanana Pro**は、Googleの生成AI「Gemini」に搭載された最新の画像生成モデルです。医学研究のスライド、グラフィカルアブストラクト、教育資料など、専門的な図解を簡単に作成できます。

このガイドでは、Nanobananaの基本的な使い方と、医学図解向けの実践的なプロンプト例を紹介します。また、コミュニティで共有されているベストプラクティスも参照できます。

---

## Nanobanana Proの特徴

Nanobanana Proは、従来の画像生成AIの課題を解決し、医学・科学分野での活用に最適な5つの特徴を持っています。

![Nanobanana Proの5つの特徴](/Helix/assets/guides/diagram-creation-guide/five_features.png)

### 1. 驚異の「文字描写力」

日本語を含む多言語のテキストを、画像内に**正確に描写**できます。ロゴデザイン、ポスタータイトル、図表のラベルなど、文字が重要な医学図解に最適です。

**活用例:**
- スライドタイトルや見出しの作成
- 図表内のラベリング
- 教育用ポスターのデザイン

### 2. 最大2Kの高解像度

印刷用途や大画面での表示にも耐えうる**最大2K解像度**を実現。学会発表や論文投稿にも使用できる品質です。

### 3. 広範囲な知識（Enhanced World Knowledge）

医学・生物学の専門知識を理解し、解剖学的構造、病態生理、診断フローなどを**正確に描画**します。

### 4. 洗練されたスタイル変換

写真をイラスト調に変換したり、スケッチを精密な図解に変換したりと、**画風を自在にコントロール**できます。

### 5. 進化した「写真合成」

複数の画像を自然に統合し、光の向き、影の落ち方、質感まで調整して**一枚の新しい画像**として生成します。

## Nanobanana Proの使い方

### 基本的な操作手順

1. **Geminiアプリを開く**
   - [Geminiアプリ](https://gemini.google.com/)にアクセス

2. **「思考モード」を選択**
   - 画面上部のモード切替で「思考モード」を選択

3. **「🍌画像を生成」ツールを選択**
   - ツールメニューから「🍌画像を生成」を選択

4. **プロンプトを入力**
   - 作成したい図解の内容を具体的に記述

5. **画像を編集・統合する場合**
   - 「＋」ボタンから画像を最大6枚まで追加可能

### 利用可能なプラン

- **無料プラン**: 1日数回まで試用可能
- **Google AI Pro / Ultra プラン**: 無制限利用
- **Google Workspace with Gemini**: 企業向けプラン
- **Vertex AI**: 開発者向けAPI

## プロンプトのコツ

![効果的なプロンプトの構造](/Helix/assets/guides/diagram-creation-guide/prompt_structure.png)

### ✅ 効果的なプロンプトの書き方

1. **具体的に記述する**
   - ❌ 「心臓の図を描いて」
   - ✅ 「心臓の冠状断面を医学教科書風のイラストで描いてください。主要な構造（心房、心室、弁、大血管）にラベルを付けて」

2. **スタイルを指定する**
   - 「医学教科書風」「Nature風」「ポップなイラスト風」など

3. **含める要素を列挙する**
   - 「矢印で血流の方向を示す」「色分けで機能を区別する」など

4. **アスペクト比を指定する**
   - 「16:9の横長で」「正方形で」「縦長のポスター形式で」

5. **テキストを含める場合は明示する**
   - 「タイトルに『心不全の病態生理』と入れて」

### 💡 医学図解に特化したTips

![BioRenderスタイルの活用](/Helix/assets/guides/diagram-creation-guide/biorender_style.png)

- **「テイスト: BioRender」と付ける**: 医学分野でよく使われる作図サービス「BioRender」のスタイルを指定すると、プロフェッショナルな医学図解が生成されやすくなります
- **素材画像を活用する**: BioRenderなどの素材ライブラリから画像をダウンロードし、Nanobananaで組み合わせることで、より高度な図解が作成できます

## 実践例: 医学スライド作成

![医学図解作成のワークフロー](/Helix/assets/guides/diagram-creation-guide/workflow.png)

### ケース1: 学会発表用スライド

**目的**: 研究成果を視覚的に伝えるスライドを作成

**プロンプト例**:
```
学会発表用のスライドを作成してください。
タイトル: 「新規糖尿病治療薬の作用機序」
スタイル: Nature Medicine風のモダンデザイン
含める要素:
- 薬剤の分子構造
- 細胞内シグナル伝達経路
- 血糖値の変化グラフ
- 主要な効果を3つの箇条書きで
アスペクト比: 16:9
```

### ケース2: 教育用資料

**目的**: 医学生向けの教材を作成

**プロンプト例**:
```
医学生向けの教育資料を作成してください。
テーマ: 「心電図の基本波形」
スタイル: 教科書風のわかりやすいイラスト
含める要素:
- 正常心電図の波形（P波、QRS波、T波）
- 各波形の意味を簡潔に説明
- 時間軸と電位軸のラベル
背景色: 白
```

### ケース3: グラフィカルアブストラクト

**目的**: 論文投稿用のグラフィカルアブストラクトを作成

**プロンプト例**:
```

スタイル: Cell誌風のモダンでカラフルなデザイン
含める要素:
- 腸管の断面図
- 腸内細菌のイラスト
- 免疫細胞の活性化
- 矢印で相互作用を示す
テイスト: BioRender
アスペクト比: 正方形
```

## よくある質問（FAQ）

### Q1: 無料プランでどれくらい使えますか？

A: 無料プランでは、1日数回までNanobanana Proを試用できます。頻繁に使用する場合は、Google AI Proプラン（月額2,900円）への加入をおすすめします。

### Q2: 生成された画像の著作権は？

A: Geminiで生成された画像の権利については、[Googleの利用規約](https://policies.google.com/terms/generative-ai)をご確認ください。学術利用の場合は、生成AIを使用したことを明記することが推奨されます。

### Q3: 日本語の精度はどうですか？

A: Nanobanana Proは、日本語のテキストレンダリング精度が大幅に向上しています。ただし、複雑な医学用語や長文の場合は、生成後に確認・修正が必要な場合があります。

### Q4: BioRenderスタイルとは？

A: BioRenderは、医学・生物学分野で広く使われている作図サービスです。プロンプトに「テイスト: BioRender」と付けることで、プロフェッショナルな医学図解スタイルを指定できます。

### Q5: 複数の画像を統合できますか？

A: はい。「＋」ボタンから最大6枚の画像をアップロードし、Nanobanana Proが自然に統合します。既存の素材画像を組み合わせて、オリジナルの図解を作成できます。

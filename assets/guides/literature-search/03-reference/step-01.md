
# ステップ1: 診療ガイドラインを検索・要約する

![Evidence Hierarchy Pyramid](/Helix/assets/guides/literature-search/evidence_pyramid.png)

## エビデンスの集大成を使いこなす

診療ガイドラインは、特定の疾患に対する診断、治療、管理に関する推奨事項を、専門家がシステマティックレビューに基づいてまとめたものです。これらは、日々の臨床における意思決定の礎となる、最も質の高いエビデンスの集大成と言えます。

しかし、ガイドラインは数百ページに及ぶこともあり、その全てを読み通し、必要な情報を探し出すのは容易ではありません。AIを活用することで、特定の臨床疑問に関連する推奨事項を、ピンポイントで、かつ迅速に抽出することが可能になります。

---

## ガイドライン検索・要約プロンプト

このプロンプトは、特定のテーマに関する最新の主要な診療ガイドラインを同定し、その中から知りたい推奨事項を抜き出して要約させるためのものです。

### 📋 ガイドライン要約プロンプト（コピペ用）

```
あなたは、最新の診療ガイドラインに精通した専門医です。以下の臨床疑問について、関連する最新の主要な診療ガイドラインを挙げ、その中での推奨事項を要約してください。

【臨床疑問】
[例: 脂質異常症の一次予防において、LDLコレステロールの管理目標は？]

【タスク】
1.  この疑問に答える上で最も権威のある最新の診療ガイドライン（例: ESC, AHA/ACC, 日本の学会など）を2-3つ同定してください。
2.  そのうち最も重要と思われるガイドライン1つについて、上記疑問に関連する推奨事項を、推奨クラスとエビデンスレベルが分かるように、表形式でまとめてください。
3.  主要な推奨事項の背景となる考え方や、注意点を簡潔に解説してください。
```

## 実践例: 脂質異常症の管理目標

### 入力プロンプト

```
あなたは、最新の診療ガイドラインに精通した循環器専門医です。以下の臨床疑問について、関連する最新の主要な診療ガイドラインを挙げ、その中での推奨事項を要約してください。

【臨床疑問】
心血管疾患の一次予防において、脂質異常症（特にLDLコレステロール）はどのように管理すべきか？リスク層別化と管理目標について知りたい。

【タスク】
1.  この疑問に答える上で最も権威のある最新の診療ガイドライン（欧米および日本）を2つ同定してください。
2.  そのうち欧州のガイドラインについて、リスクカテゴリー別のLDL-C管理目標値を、推奨クラスとエビデンスレベルが分かるように、表形式でまとめてください。
3.  主要な推奨事項の背景となる考え方を簡潔に解説してください。
```

### AIの出力例

```
承知いたしました。脂質異常症の一次予防におけるLDL-C管理目標に関するガイドラインの推奨事項をまとめます。

### 1. 主要な関連ガイドライン

1.  2019 ESC/EAS Guidelines for the management of dyslipidaemias (欧州心臓病学会/欧州アテローム性動脈硬化学会 ガイドライン)
2.  動脈硬化性疾患予防ガイドライン 2022年版 (日本動脈硬化学会)

### 2. ESC/EAS 2019 ガイドラインの推奨事項

以下に、心血管疾患リスクカテゴリー別のLDL-C管理目標値を示します。

| リスクカテゴリー | 定義（例） | LDL-C管理目標値 | 推奨クラス | エビデンスレベル |
| :--- | :--- | :--- | :---: | :---: |
| 超高リスク | 確定したASCVD (心筋梗塞後など)<br>SCORE ≥10% | < 55 mg/dL かつ 50%以上低下 | I | A |
| 高リスク | 家族性高コレステロール血症<br>重症なCKD (eGFR <30)<br>SCORE 5-10% | < 70 mg/dL かつ 50%以上低下 | I | A |
| 中リスク | 若年の糖尿病患者 (1型<35歳, 2型<50歳) で罹病期間<10年<br>SCORE 1-5% | < 100 mg/dL | IIa | A |
| 低リスク | SCORE <1% | < 116 mg/dL | IIb | A |

*ASCVD: 動脈硬化性心血管疾患, SCORE: 10年間の致死的心血管イベントリスクスコア*

### 3. 推奨事項の背景解説

2019年のESC/EASガイドラインは、それまでのガイドラインよりもさらに踏み込み、特に高リスクおよび超高リスク群に対して、より厳格なLDL-C管理目標値を設定したことが大きな特徴です。これは、IMPROVE-IT試験や複数のPCSK9阻害薬の臨床試験から、「LDL-Cは低ければ低いほど良い（The lower, the better）」というエビデンスが確立されたことを反映しています。

治療の基本はスタチンですが、目標値を達成できない場合は、エゼチミブの追加、さらにはPCSK9阻害薬の追加が推奨されています。このように、個々の患者のリスクを正確に評価し、それに応じた強力な脂質低下療法を行うことが、心血管イベントを予防する上で極めて重要であると強調されています。
```

## まとめ

このステップで学んだこと:

✅ 臨床疑問に関連する最新の診療ガイドラインをAIに同定させる方法
✅ ガイドラインの核心である「推奨事項」を、推奨クラスやエビデンスレベルと共に抽出・要約させるプロンプト
✅ 推奨事項の背景にあるエビデンスや考え方を解説させ、理解を深めるアプローチ

AIを活用することで、分厚いガイドラインの中から、日々の診療で本当に必要な情報を、迅速かつ正確に手に入れることができます。

次のステップでは、このガイドの最後として、自身の研究テーマの立案や論文執筆に役立つ、網羅的な文献レビューをAIに支援させる方法を探ります。

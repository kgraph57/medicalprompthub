# 03-reference/step-01.md: 最新エビデンスの検索・要約と処方提案への活用

## 1. エビデンス検索の効率化

処方提案の説得力を高めるためには、最新かつ信頼性の高いエビデンス（科学的根拠）が必要です。AIは、膨大な医学文献の中から、必要な情報を迅速に抽出し、要約する能力に優れています。

### エビデンス検索・要約プロンプト例

特定の薬剤やDRPに関する最新の知見を検索し、処方提案の根拠として活用します。

```
# 役割設定
あなたは、医学文献の検索と要約に特化した医療情報専門家です。

# 検索テーマ
「高齢者におけるベンゾジアゼピン系薬の減薬が転倒リスクに与える影響」

# 依頼
1.  過去5年間のランダム化比較試験（RCT）またはシステマティックレビューから、最も関連性の高い論文を3つ特定してください。
2.  それぞれの論文について、研究デザイン、主要な結果（転倒リスクの相対リスク減少率など）、結論を日本語で簡潔に要約してください（各100字以内）。
3.  要約した情報を基に、医師への処方提案に含めるべき最も重要なエビデンスを1つ選定してください。
```

### AIによる要約の活用

AIが要約したエビデンスは、そのまま提案文の「O (Objective)」や「A (Assessment)」の根拠として組み込むことができます。

提案文への組み込み例:

> 「最新のシステマティックレビュー[1]によると、高齢者におけるベンゾジアゼピン系薬の漸減は、転倒リスクを平均25%減少させることが示されています。このエビデンスに基づき、今回の漸減を強く推奨いたします。」

## 2. 複雑な相互作用の評価

多剤併用患者では、予期せぬ薬物相互作用（DDI: Drug-Drug Interaction）のリスクが高まります。AIは、複数の薬剤間の複雑な相互作用を網羅的にチェックし、臨床的意義を評価するのに役立ちます。

### 相互作用評価プロンプト例

```
# 役割設定
あなたは、薬物動態学および薬力学に精通した臨床薬理学者です。

# 患者情報
- 腎機能: eGFR 35 mL/min/1.73m²
- 処方薬: 薬剤A（CYP3A4阻害作用あり）、薬剤B（CYP3A4基質）、薬剤C（腎排泄型）

# 依頼
1.  処方薬AとBの間の薬物相互作用（DDI）のメカニズム（例：代謝酵素阻害）と、その結果生じる臨床的影響（例：Bの血中濃度上昇による有害事象リスク）を評価してください。
2.  薬剤Cについて、患者の腎機能低下を考慮した場合の用量調整の必要性を評価し、具体的な推奨用量を提示してください。
3.  これらの相互作用と用量調整の必要性について、医師への報告書として簡潔にまとめてください。
```

## 3. 💡 Tips: 参照情報の明記

AIにエビデンスの要約を依頼する際は、「参照した論文のタイトル、著者、ジャーナル名、DOIまたはURLを必ず含めること」を指示することで、情報の信頼性を担保できます。

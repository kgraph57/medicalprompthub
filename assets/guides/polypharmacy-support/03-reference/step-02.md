# 03-reference/step-02.md: 患者・家族への説明支援と継続的な薬物治療管理

## 1. 患者・家族への説明資料作成支援

ポリファーマシー対策を成功させるには、患者や家族の理解と同意（アドヒアランス）が不可欠です。AIは、専門的な内容を平易な言葉に変換し、患者教育用の資料作成を支援します。

### 患者説明資料作成プロンプト例

減薬の必要性や、減薬によって期待される効果、注意すべき症状などを、非専門家向けに作成します。

```
# 役割設定
あなたは、患者教育に長けた医療コミュニケーターです。

# 対象者
85歳の患者A氏とそのご家族（非医療従事者）

# 説明テーマ
「抗不安薬D（ベンゾジアゼピン系）の減薬の必要性とその効果」

# 依頼
1.  「ポリファーマシー」という言葉を使わず、多剤併用のリスク（例：ふらつき、転倒）を分かりやすく説明してください。
2.  今回の減薬の目的（例：転倒リスクを減らし、安心して生活できるようにすること）を明確に伝えてください。
3.  減薬中に起こりうる一時的な症状（例：寝つきが悪くなる、不安を感じる）と、その際の対処法を具体的に記述してください。
4.  全体を優しい口調で、A4用紙1枚に収まるようにまとめてください。
```

### 💡 Tips: 専門用語の言い換え

AIに「専門用語を使わない」と指示するだけでなく、具体的な言い換え例（例：ポリファーマシー → 薬の飲みすぎ、PIM → 体に合わない薬）を提示することで、より適切な資料が作成されます。

## 2. 継続的な薬物治療管理への応用

ポリファーマシー対策は一度きりの介入ではなく、継続的なプロセスです。AIは、患者の長期的な薬物治療管理において、以下の点で貢献します。

### 継続管理プロンプト例：服薬アドヒアランスの評価

患者の服薬状況に関する情報をAIに入力し、アドヒアランス低下のリスクを評価します。

```
# 役割設定
あなたは、服薬アドヒアランス評価の専門家です。

# 患者情報
- 85歳、独居、認知機能低下あり
- 処方薬: 8剤（朝4錠、昼1錠、夕2錠、寝前1錠）
- 家族からの情報: 「時々、薬を飲んだかどうかわからなくなると言っている」「PTPシートから薬を出して保管していることがある」

# 依頼
1.  この患者が抱える服薬アドヒアランス低下の主要な要因を3つ特定してください。
2.  それぞれの要因に対する具体的な介入策（例：一包化、服薬カレンダー、訪問看護との連携）を提案してください。
3.  介入策を提案する際の優先順位を決定し、その理由を述べてください。
```

## 3. AIを「臨床支援ツール」として活用する

AIは、医療従事者の判断を代替するものではなく、判断を支援するツールです。AIの出力結果を鵜呑みにせず、必ず自身の臨床経験と照らし合わせ、最終的な判断を下すことが、安全で質の高い医療を提供するための鍵となります。

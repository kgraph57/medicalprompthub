# 01-basics/step-01.md: AI活用の基礎: ポリファーマシー対策におけるAIの役割

## 1. ポリファーマシー対策におけるAIの役割

ポリファーマシー（多剤併用）は、単に服用薬剤数が多いことだけでなく、それによって有害事象のリスクが高まる状態を指します。医療従事者にとって、膨大な患者情報と複雑な処方内容から潜在的なリスクを特定することは、時間と労力を要する作業です。

AI（特に大規模言語モデル, LLM）は、この課題に対して以下の役割を果たします。

1.  リスク患者の迅速なスクリーニング: 処方データや検査値、病歴などの構造化・非構造化データを解析し、有害事象（DRP: Drug-Related Problems）のリスクが高い患者を瞬時に抽出します。
2.  処方適正化の提案支援: STOPP-JやBeers Criteriaなどのガイドラインに基づき、不適切な処方（PIM: Potentially Inappropriate Medication）の候補を提示し、減薬や代替薬の提案をサポートします。
3.  情報収集と要約の効率化: 処方提案の根拠となる最新のエビデンス（論文、ガイドライン）を検索・要約し、医師への提案資料作成を支援します。

## 2. AI活用によるメリット

| メリット | 具体的な効果 | 臨床現場での活用例 |
| :--- | :--- | :--- |
| 業務効率の向上 | スクリーニング時間の短縮、情報検索の自動化 | 1日あたりのチェック可能患者数の増加、対人業務への時間配分増加 |
| リスクの見落とし防止 | ガイドラインや最新知見に基づいた網羅的なチェック | 複雑な相互作用や稀な副作用の早期発見、潜在的なDRPの特定 |
| 提案の質の向上 | エビデンスに基づいた論理的な提案文の自動生成 | 医師への処方提案の説得力向上、チーム医療における円滑なコミュニケーション |

## 3. AI活用のための準備

AIを効果的に活用するためには、以下の情報を用意し、AIに正確に伝えることが重要です。

1.  患者の基本情報: 年齢、性別、体重、腎機能（eGFR/Cr）、肝機能。
2.  現在の処方内容: 全ての薬剤名、用量、用法、処方理由。
3.  既往歴・合併症: 診断名、アレルギー歴、特に認知機能障害や転倒歴の有無。
4.  検査値: 直近の腎機能、電解質、血糖値など、薬剤評価に必要なデータ。

## 💡 Tips: AIに「医療従事者」として振る舞わせる

AIに専門的なタスクを依頼する際は、最初にペルソナ（役割）を設定することで、回答の質が格段に向上します。

プロンプト例（ペルソナ設定）:

```
あなたは、高齢者薬物療法に精通した認定薬剤師です。
以下のタスクを実行するにあたり、常に最新のガイドライン（STOPP-J、高齢者の安全な薬物療法ガイドラインなど）に基づき、論理的かつ実践的な視点から回答してください。
```

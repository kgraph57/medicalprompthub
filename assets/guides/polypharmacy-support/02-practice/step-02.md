# 02-practice/step-02.md: 医師への提案文作成とコミュニケーション支援

## 1. 提案の論理構築：SOAP形式の活用

AIによるスクリーニング結果を、医師に理解されやすく、かつ行動を促す提案文に変換することが重要です。医療現場で広く用いられるSOAP形式（Subjective, Objective, Assessment, Plan）を応用し、提案の論理を構築します。

### SOAP形式を応用した提案文の構成

| 項目 | 提案文での役割 | AIへの指示例 |
| :--- | :--- | :--- |
| S (Subjective) | 患者の主観的な訴えや、薬剤関連の症状。 | 「患者の訴え（例：ふらつき、眠気）を簡潔にまとめてください。」 |
| O (Objective) | 客観的なデータ（処方内容、検査値、既往歴、AIのスクリーニング結果）。 | 「PIMの指摘と、その根拠となるガイドライン（STOPP-Jなど）を明記してください。」 |
| A (Assessment) | DRP（薬剤関連問題）の特定と、それが患者に与えるリスクの評価。 | 「特定されたDRP（例：転倒リスク増大）が、この患者にとってなぜ重大かを論理的に説明してください。」 |
| P (Plan) | 具体的な減薬・代替薬の提案と、モニタリング計画。 | 「減薬の優先順位と、具体的な減薬スケジュール（例：〇〇を半量に）を提案してください。」 |

## 2. 説得力のある提案文のプロンプト例

ステップ01の症例（A氏）に基づき、最も優先度の高い「抗不安薬D（ベンゾジアゼピン系）」の減薬提案文を作成します。

```
# 役割設定
あなたは、高齢者薬物療法に精通した認定薬剤師です。
以下のSOAP形式の構成に基づき、医師への処方変更提案文を作成してください。

# 患者情報（再掲）
- 85歳、女性、転倒歴あり（過去6ヶ月で2回）
- 処方薬に抗不安薬D（ベンゾジアゼピン系）が含まれる。

# 提案の構成要素
- S: 患者は最近、日中の眠気とふらつきを訴えています。
- O: 処方中の抗不安薬Dは、高齢者における長期使用がSTOPP-J基準でPIMと指摘されています。また、過去6ヶ月で2回の転倒歴があります。
- A: 抗不安薬Dの中枢神経抑制作用により、転倒リスクが著しく増大していると評価します。
- P: 抗不安薬Dを〇〇mgから〇〇mgへ漸減することを提案します。代替として、非薬物療法（睡眠衛生指導）の強化を検討します。

# 依頼
上記の構成要素を統合し、医師が臨床的意義を理解しやすいよう、丁寧かつ論理的な提案文（200字程度）を作成してください。
```

### AIによる提案文の出力例

> A先生、お疲れ様です。A氏（85歳）の処方についてご提案です。
> 患者様は日中の眠気とふらつきを訴えており、過去6ヶ月で2回の転倒歴があります。現在処方中の抗不安薬D（ベンゾジアゼピン系）は、STOPP-J基準で高齢者の転倒リスクを増大させるPIMとされています。この薬剤が転倒の主因となっている可能性が高いと評価します。つきましては、転倒リスク軽減のため、抗不安薬Dを〇〇mgへ漸減し、並行して睡眠衛生指導を強化することを提案いたします。ご検討をお願いいたします。

## 3. 💡 Tips: 根拠の明示と代替案の提示

提案文には、必ず客観的な根拠（STOPP-J、Beers Criteria、ガイドライン名）を明記し、単なる減薬だけでなく代替案（非薬物療法、他剤への変更）をセットで提示することで、提案の受容性が高まります。

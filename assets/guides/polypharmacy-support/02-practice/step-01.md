# 02-practice/step-01.md: PIMスクリーニングとDRPリスク評価の実践

## 1. 症例設定とプロンプトの適用

ここでは、具体的な症例を用いて、AIによる潜在的に不適切な処方（PIM）のスクリーニングと薬剤関連問題（DRP）のリスク評価を実践します。

### 症例：A氏（85歳、女性）

| 項目 | 詳細 |
| :--- | :--- |
| 年齢/性別 | 85歳、女性 |
| 体重 | 45kg |
| 腎機能 | eGFR 35 mL/min/1.73m² (CKD G3b) |
| 既往歴 | 高血圧、変形性膝関節症、不眠症、うつ状態、転倒歴あり（過去6ヶ月で2回） |
| 現在の処方 | 8剤 |

### 現在の処方内容（8剤）

1.  降圧薬A: 5mg 1日1回 朝食後 (高血圧)
2.  利尿薬B: 2.5mg 1日1回 朝食後 (高血圧)
3.  NSAIDs C: 60mg 1日3回 毎食後 (膝関節痛)
4.  抗不安薬D（ベンゾジアゼピン系）: 0.5mg 1日1回 就寝前 (不眠症)
5.  抗うつ薬E: 10mg 1日1回 朝食後 (うつ状態)
6.  胃薬F: 20mg 1日1回 朝食後 (胃粘膜保護)
7.  骨粗鬆症薬G: 10mg 1日1回 朝食後 (骨粗鬆症)
8.  便秘薬H: 1錠 1日1回 就寝前 (便秘)

### 実践プロンプト例

基本プロンプトテンプレートに症例情報を入力します。

```
# 役割設定
あなたは、高齢者薬物療法に精通した認定薬剤師です。
以下の患者情報と処方内容に基づき、ポリファーマシー対策の観点から評価を行ってください。

# 患者情報
- 年齢: 85歳
- 性別: 女性
- 体重: 45kg
- 腎機能: eGFR 35 mL/min/1.73m² (CKD G3b)
- 既往歴: 高血圧、変形性膝関節症、不眠症、うつ状態、転倒歴あり（過去6ヶ月で2回）

# 現在の処方内容（全8剤）
1. 降圧薬A: 5mg 1日1回 朝食後 (高血圧)
2. 利尿薬B: 2.5mg 1日1回 朝食後 (高血圧)
3. NSAIDs C: 60mg 1日3回 毎食後 (膝関節痛)
4. 抗不安薬D（ベンゾジアゼピン系）: 0.5mg 1日1回 就寝前 (不眠症)
5. 抗うつ薬E: 10mg 1日1回 朝食後 (うつ状態)
6. 胃薬F: 20mg 1日1回 朝食後 (胃粘膜保護)
7. 骨粗鬆症薬G: 10mg 1日1回 朝食後 (骨粗鬆症)
8. 便秘薬H: 1錠 1日1回 就寝前 (便秘)

# 評価依頼
1. 潜在的に不適切な処方（PIM）の候補を特定し、その根拠（STOPP-J基準など）を提示してください。
2. 薬剤関連問題（DRP）のリスク（特に転倒、認知機能低下、腎機能悪化）を指摘してください。
3. 減薬または代替薬の提案を、具体的な薬剤名と用量を含めて行ってください。
4. 提案の優先順位を決定し、その理由を述べてください。
```

## 2. AIによるスクリーニング結果の解釈

AIは、このプロンプトに対し、以下のようなPIMとDRPリスクを指摘するはずです。

| 指摘されたPIM/DRP | 根拠（AIの出力例） | 臨床的リスク |
| :--- | :--- | :--- |
| 抗不安薬D | STOPP-J基準: 高齢者におけるベンゾジアゼピン系薬の長期使用（転倒リスク増大）。 | 転倒、認知機能低下、せん妄のリスク。転倒歴があるため最優先で介入が必要。 |
| NSAIDs C | 腎機能低下（eGFR 35）患者へのNSAIDs使用。利尿薬Bとの併用による腎機能悪化リスク。 | 腎機能のさらなる悪化、心血管系イベントのリスク。 |
| 抗うつ薬E | 抗コリン作用による認知機能低下、転倒リスク。抗不安薬Dとの併用による中枢神経抑制作用の増強。 | 転倒、認知機能低下。 |

## 3. 💡 Tips: 評価の深掘り

AIの初期回答でPIMが特定されたら、さらに具体的な情報を引き出すためにプロンプトを連鎖させます。

連鎖プロンプト例（NSAIDs Cについて深掘り）:

```
指摘されたNSAIDs Cについて、代替薬として非薬物療法（温熱療法、運動療法など）の検討と、薬物療法を行う場合の推奨される代替薬（例：アセトアミノフェン）の最大用量と、その際の腎機能への影響について具体的に教えてください。
```

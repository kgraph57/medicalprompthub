# 02-practice/step-03.md: 減薬後のモニタリングとチーム医療での情報共有

## 1. 減薬後のモニタリング計画策定

処方変更（減薬・中止）はゴールではなく、新たなスタートです。AIは、減薬後のモニタリング計画の策定においても、その精度と効率を高めることができます。

### モニタリング計画策定プロンプト例

減薬によって生じる可能性のある離脱症状や原疾患の再燃、新たなDRPを予測し、モニタリングすべき項目と期間を明確にします。

```
# 役割設定
あなたは、高齢者薬物療法に精通した認定薬剤師です。
以下の処方変更案に基づき、減薬後の患者モニタリング計画を策定してください。

# 処方変更案
- 薬剤: 抗不安薬D（ベンゾジアゼピン系）
- 変更内容: 〇〇mgから〇〇mgへ漸減（2週間かけて）
- 目的: 転倒リスクの軽減

# 依頼
1.  モニタリングすべき主要な症状（離脱症状、原疾患再燃など）を3つ挙げ、その症状が出現した場合の対応策を簡潔に示してください。
2.  モニタリング期間（例：減薬後1週間、1ヶ月）と、その期間中に確認すべき客観的指標（例：転倒回数、睡眠時間、不安スコア）を提案してください。
3.  患者や介護者への指導内容（特に注意すべき点）をまとめてください。
```

### AIによるモニタリング項目提案例

| モニタリング項目 | 予測されるリスク | 対応策（AI提案） |
| :--- | :--- | :--- |
| 不眠・不安の増悪 | 離脱症状、原疾患（不眠症）の再燃 | 漸減ペースの調整、非薬物療法の強化、必要に応じて代替薬の検討。 |
| 転倒回数 | 減薬効果の評価 | 転倒回数の減少を確認。変化がない場合は他の原因を検討。 |
| 認知機能 | 認知機能の改善（減薬効果） | MMSEやHDS-Rなどの簡易評価スケールを定期的に実施。 |

## 2. チーム医療での情報共有とAIの活用

ポリファーマシー対策は、医師、薬剤師、看護師、管理栄養士、ケアマネジャーなど多職種連携（チーム医療）が不可欠です。AIは、職種間の情報共有を円滑にするためのサマリー作成に役立ちます。

### チーム共有用サマリー作成プロンプト例

```
# 役割設定
あなたは、多職種連携を円滑にするための医療情報コーディネーターです。
以下の情報を基に、チームメンバー（医師、看護師、ケアマネジャー）向けに、患者のポリファーマシー対策に関する要点サマリーを作成してください。

# 含むべき情報
1.  患者の主要な問題点（転倒リスク、腎機能低下など）
2.  今回の処方変更の目的と内容（例：抗不安薬Dの漸減による転倒リスク軽減）
3.  各職種が特に注意すべきモニタリング項目（例：看護師はふらつきの有無、ケアマネジャーは生活環境での転倒状況）

# 依頼
専門用語を避け、簡潔で行動につながるサマリー（150字程度）を作成してください。
```

### AIによるサマリー出力例

> A氏（85歳）は転倒リスクが高いため、本日より睡眠薬（抗不安薬D）を漸減します。
> 【看護師・介護者へ】 ふらつきや夜間の不眠・不安の増悪がないか観察をお願いします。
> 【医師へ】 2週間後の再診時に、転倒回数と睡眠状況の評価をお願いします。目標は転倒ゼロとQOLの維持です。

## 3. 💡 Tips: 職種ごとの視点

AIにサマリーを作成させる際、「誰に伝えるか」を明確にすることで、その職種が必要とする情報に焦点を当てた、実践的なアウトプットが得られます。

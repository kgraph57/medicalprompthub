## 4. 推奨の相違点におけるエビデンスレベルと対象集団の分析

### 4.1. 相違点の背景にある要因の特定

ステップ3で特定された推奨の相違点は、臨床現場での判断に最も影響を与える部分です。この相違がなぜ生じているのか、その背景にある要因をAIに分析させます。

主な分析要因:

1.  エビデンスレベルの違い: 一方のガイドラインが採用したエビデンスが、もう一方では採用されていない、あるいは解釈が異なっている可能性があります。
2.  対象集団の違い: ガイドラインの対象とする患者集団（年齢、併存疾患、人種など）が異なると、推奨も変わってきます。
3.  経済的・文化的背景: 医療制度や経済状況の違い（例：日本と欧米）が、推奨される治療法や検査に影響を与えている場合があります。

### 4.2. エビデンスレベルと対象集団の深掘り分析

AIに、相違点に関連する推奨事項の原典テキスト（ステップ2で抽出したテキスト）を再入力し、エビデンスの根拠に関する記述を抽出させます。

### 実用的なプロンプト例：相違点の背景にあるエビデンスの深掘り

```prompt
あなたは、ガイドラインの推奨根拠を分析する専門家です。
以下の「主要な相違点」の表で特定された臨床的テーマについて、その相違の背景にある要因を分析してください。

# 入力データ
[ステップ3で特定された「主要な相違点」の表を貼り付け]

# ガイドラインの推奨事項原文（再入力）
[相違点に関連する推奨事項の原文テキストを、ガイドラインごとに明記して貼り付け]

---
## 分析指示
各テーマについて、以下の項目を抽出・分析し、Markdown形式の表で出力してください。

1.  エビデンスの根拠: 各ガイドラインが推奨の根拠として言及している主要な研究（例：RCT名、メタアナリシス）やエビデンスレベルの記述を抽出。
2.  対象集団の限定: 推奨が適用される患者集団（例：高齢者、特定の併存疾患を持つ患者）に関する限定的な記述を抽出。
3.  相違の推定要因: 上記の分析に基づき、推奨が異なる主な推定要因（例：採用したエビデンスの違い、対象集団の定義の違い）を考察する。

---
## 出力形式
| 臨床的テーマ | ガイドライン | エビデンスの根拠（抽出） | 対象集団の限定（抽出） | 相違の推定要因 |
| :--- | :--- | :--- | :--- | :--- |
| [テーマ1] | A | | | |
| | B | | | |
| [テーマ2] | A | | | |
| | B | | | |
```

### 注意点とTips

*   Tips: AIに「批判的吟味（Critical Appraisal）」の視点を持つよう指示することで、単なる情報抽出に留まらない、より深い分析結果を得られることがあります。
*   注意点: AIはエビデンスの質を評価することはできません。AIが抽出した研究名やエビデンスレベルは、必ず医療従事者自身が原典で確認し、その質を評価する必要があります。

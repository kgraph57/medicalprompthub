# 03-reference/step-01.md: 応用編：再入院リスク予測モデルの活用とAIによる介入計画

## 1. 再入院リスク予測モデル（RRP）の活用

多くの医療機関では、再入院リスクを予測するためのスコアリングシステム（例：LACE Index, HOSPITAL Scoreなど）が導入されています。AI（LLM）は、これらのモデルが算出したリスクスコアを、単なる数値としてではなく、具体的な介入計画に落とし込むための橋渡し役として機能します。

### 実用的なプロンプト例：リスクスコアに基づく介入計画の具体化

```
あなたは、リスクマネジメントの専門家です。
以下の[患者情報]と[再入院リスクスコア]に基づき、スコアを10%低下させることを目標とした、具体的な3つの介入戦略を提案してください。

介入戦略の要件:
1.  多職種連携を前提とした、実行可能な内容であること。
2.  患者の生活背景（例：一人暮らし、経済的困難）を考慮していること。
3.  介入戦略ごとに、担当職種と期待される効果を明記すること。

[患者情報]
（例: 75歳女性、慢性閉塞性肺疾患（COPD）で入院。退院時処方薬が5種類以上。喫煙歴あり。娘は遠方に居住。）

[再入院リスクスコア]
LACE Index: 12点（高リスク）

[LACE Indexの内訳]
L: Length of stay (入院期間): 7日
A: Acuity of admission (入院時の緊急度): 救急搬送
C: Comorbidity (併存疾患): COPD、高血圧
E: Emergency department visits (退院前6ヶ月の救急外来受診回数): 1回
```

## 2. AIによる患者教育資料のパーソナライズ

標準的な患者教育資料は、すべての患者に一律で提供されますが、患者のリテラシー、認知機能、関心に応じて内容をパーソナライズすることで、理解度とアドヒアランス（治療への積極的な取り組み）が劇的に向上します。

### 実用的なプロンプト例：パーソナライズされた教育資料の生成

```
あなたは、医療情報デザインの専門家です。
以下の[標準的な患者教育資料のテキスト]を、[患者の特性]に合わせて、最も効果的に情報が伝わるようにリライトしてください。

リライトの要件:
1.  トーン: [患者の特性]に記載されたトーンを厳守する。
2.  キーワード: [標準的な患者教育資料のテキスト]に含まれる重要なキーワード（例：塩分制限、利尿薬、体重測定）は保持する。
3.  形式: 箇条書きを多用し、視覚的に分かりやすい形式にする。

[標準的な患者教育資料のテキスト]
（例: 心不全患者向け「水分・塩分制限の重要性」に関する一般的な説明文）

[患者の特性]
（例: 70代男性、漁師。専門用語を嫌い、「男らしい、簡潔で力強い言葉」を好む。読解力は高くないが、具体的な指示には従う。）
```

### 注意点とTips

*   Tips: AIに「この患者が最も関心を持つであろう事柄（例：漁師なら「体力維持」）」を考慮に入れるよう指示することで、教育資料への関心を高めることができます。
*   注意点: パーソナライズは理解度を高めますが、医療情報の正確性を損なわないよう、リライト後の内容を必ず確認してください。

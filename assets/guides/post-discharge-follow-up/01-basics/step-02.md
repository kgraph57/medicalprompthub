# 01-basics/step-02.md: 診療記録からフォローアップの「核」となる重要情報を抽出する

## 1. フォローアップ計画に必要な情報の特定

効果的な退院後フォローアップ計画を作成するためには、以下の3つのカテゴリーの情報を診療記録から抽出する必要があります。

| カテゴリー | 抽出対象となる情報 | AIに期待する役割 |
| :--- | :--- | :--- |
| 臨床情報 | 診断名、手術/処置内容、入院中の経過、退院時処方、今後の治療方針、再入院リスク因子。 | 膨大なテキストからの要約とリスク因子の特定。 |
| 生活・社会情報 | ADL（日常生活動作）レベル、認知機能、介護状況、家族構成、居住環境、経済状況、利用可能な社会資源。 | 非構造化データ（看護記録、MSW記録）からの関連情報の抽出と整理。 |
| 教育・指導情報 | 患者・家族への指導内容（服薬、食事、運動、創処置など）、理解度、未解決の疑問点。 | 指導記録からの指導内容の確認と、理解度に応じた文案の提案。 |

## 2. AIを活用した重要情報の抽出プロセス

AIに診療記録（要約・抜粋）を入力し、必要な情報を構造化された形式で抽出させます。

### 実用的なプロンプト例：構造化された情報抽出

以下のプロンプトは、AIに特定の情報項目を抽出し、表形式で出力させるためのものです。

```
あなたは、退院後フォローアップ計画の作成を支援する情報整理の専門家です。
以下の[診療記録の抜粋]を読み込み、退院後のケアに最も重要となる以下の情報を抽出し、Markdown形式の表で出力してください。

抽出項目:
1.  主たる退院時問題点 (Primary Discharge Problem): (例: 心不全の増悪リスク、褥瘡管理の継続)
2.  退院時ADLレベル (Discharge ADL Level): (例: 自立、一部介助、全介助)
3.  キーパーソン (Key Person): (例: 妻、長女、訪問看護師)
4.  退院時処方の変更点 (Key Medication Changes): (例: 新規導入薬、増量薬、中止薬)
5.  緊急時の連絡基準 (Emergency Contact Criteria): (例: 体重増加が2kg/日以上、呼吸困難感の増強)
6.  必要とされる社会資源 (Required Social Resources): (例: 訪問看護、訪問リハビリ、配食サービス)

[診療記録の抜粋]
（ここに、匿名化・抽象化した診療記録のテキストを貼り付けます。例：60代男性、心不全で入院。退院時EF 35%。自宅は2階建てで階段昇降に不安あり。妻が主介護者だが、日中はパートで不在。新規にSGLT2阻害薬を導入。…）
```

### 注意点とTips

*   Tips: 抽出項目を具体的に指定することで、AIの「幻覚（ハルシネーション）」を抑制し、必要な情報のみを正確に引き出すことができます。
*   注意点: 抽出した情報が最新かつ正確であることを、必ず元の診療記録と照合して確認してください。AIは文脈を理解しますが、誤った情報源からは誤った情報を抽出します。

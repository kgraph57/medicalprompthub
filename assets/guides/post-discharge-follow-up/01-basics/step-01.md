# 01-basics/step-01.md: 退院後フォローアップ計画におけるAI活用の意義と基本原則

## 1. AIが解決できる課題：なぜ今、AIが必要なのか

退院後フォローアップ計画（Discharge Planning）は、患者の再入院防止とQOL（生活の質）維持に不可欠です。しかし、多忙な医療現場において、この計画作成は以下の課題を抱えています。

1.  情報過多と時間不足: 診療録、看護記録、検査結果など、膨大な情報から必要な要素を短時間で抽出する必要がある。
2.  個別化の難しさ: 標準的なテンプレートでは不十分であり、患者の生活環境、理解度、家族構成といった非構造化データを考慮した真に個別化された計画作成が求められる。
3.  多職種連携の複雑性: 医師、看護師、薬剤師、管理栄養士、医療ソーシャルワーカー（MSW）など、関わる職種が多く、それぞれのタスクと連携を明確にする必要がある。

AI（特に大規模言語モデル, LLM）は、これらの課題に対し、情報の要約・抽出、個別化された文案の生成、タスクの明確化という形で貢献します。

## 2. AI活用の基本原則：倫理と安全性を確保する

AIを医療現場で活用する際は、以下の基本原則を厳守してください。

| 原則 | 内容 | 医療従事者としての行動 |
| :--- | :--- | :--- |
| データ保護 | 患者の個人情報（PII）や機密性の高い医療情報（PHI）をAIモデルに直接入力しない。 | 匿名化、抽象化、または架空のケーススタディとして情報を整形してからプロンプトに入力する。 |
| 最終責任 | AIが生成した内容はあくまで「ドラフト」であり、最終的な判断と責任は医療従事者が負う。 | AIの出力を鵜呑みにせず、必ず臨床的妥当性と患者の状況に基づき検証・修正する。 |
| 透明性 | AIの活用は、患者や関係者に対して透明性をもって行うことが望ましい。 | 計画作成の効率化にAIを利用したことを、必要に応じてチーム内で共有する。 |

## 3. 使用するAIツールの選定と準備

本ガイドでは、汎用的なLLM（例：GPT-4, Claude 3, Geminiなど）を想定していますが、機密性の高い情報を扱う場合は、セキュリティが確保された院内環境のAI、またはデータが学習に利用されない設定の商用AIサービスを利用してください。

### 実用的なプロンプト例：AIへの役割付与

AIに「退院後フォローアップ計画の専門家」としての役割を与えることで、出力の質が向上します。

```
あなたは、高度な専門知識を持つ退院調整看護師です。
これから提供する患者の診療サマリーと生活背景情報に基づき、再入院リスクを最小限に抑えるための包括的かつ個別化されたフォローアップ計画の骨子を作成してください。
あなたの役割は、単なる情報整理ではなく、患者と家族が自宅で安全かつ円滑に療養生活を送るための実行可能なロードマップを提供することです。
```

### 注意点とTips

*   Tips: 最初のプロンプトでAIに「ペルソナ（役割）」と「目的」を明確に与えることで、その後の対話全体が一貫性のあるものになります。
*   注意点: 決して患者の氏名、住所、電話番号などの特定可能な情報を入力しないでください。

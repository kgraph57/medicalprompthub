# 02-practice/step-02.md: 多職種連携のためのタスクリストと情報共有の最適化

## 1. AIによる多職種タスクの明確化

退院後フォローアップ計画の成功は、病院スタッフ、地域医療・介護スタッフ間のシームレスな連携にかかっています。AIは、計画書の内容に基づき、各職種が退院前後に実行すべき具体的なタスクを自動で分類・明確化できます。

### 実用的なプロンプト例：職種別タスクリストの生成

```
あなたは、退院調整におけるプロジェクトマネージャーです。
以下の[退院後フォローアップ計画書ドラフト]の内容に基づき、退院前（残り3日間）と退院後（最初の1週間）に、以下の職種が実行すべき具体的なタスクを抽出し、Markdown形式のチェックリストとして出力してください。

対象職種:
1.  病棟看護師 (退院前)
2.  医師 (退院前)
3.  医療ソーシャルワーカー (MSW) (退院前)
4.  訪問看護師 (退院後)
5.  ケアマネジャー (退院後)

タスクの具体例:
*   医師: 「退院後1週間以内の外来予約日時を確定し、患者に伝える。」
*   病棟看護師: 「退院時内服薬を1回分ずつにセットし、服薬指導を最終確認する。」
*   訪問看護師: 「退院翌日に初回訪問し、バイタルサインと創部の状態を確認する。」

[退院後フォローアップ計画書ドラフト]
（02-practice/step-01でAIが生成した計画書ドラフトのテキストを貼り付けます）
```

## 2. 情報共有の最適化：サマリーの自動生成

多職種間の情報共有において、長文の診療情報提供書や退院サマリーは、多忙な現場では読み飛ばされがちです。AIを活用し、「この患者のフォローアップで最も重要な3つのポイント」に絞ったサマリーを自動生成することで、情報伝達の効率と精度を向上させます。

### 実用的なプロンプト例：連携用キーポイントサマリーの生成

```
あなたは、地域連携室のベテラン職員です。
以下の[退院後フォローアップ計画書ドラフト]と[職種別タスクリスト]に基づき、訪問看護ステーションとケアマネジャー向けに、この患者のフォローアップで絶対に押さえておくべきキーポイントを3点に絞って、簡潔な箇条書きで出力してください。

出力形式:
- 患者名: [匿名化された患者情報]
- 主病名: [主病名]
- 最重要キーポイント 1: [具体的な内容と理由]
- 最重要キーポイント 2: [具体的な内容と理由]
- 最重要キーポイント 3: [具体的な内容と理由]

[退院後フォローアップ計画書ドラフト]
（02-practice/step-01でAIが生成した計画書ドラフトのテキストを貼り付けます）

[職種別タスクリスト]
（上記でAIが生成した職種別タスクリストのテキストを貼り付けます）
```

### 注意点とTips

*   Tips: 職種別タスクリストは、電子カルテのタスク管理機能や、連携ツール（例：LINE WORKS, Slackなど）に直接入力できる形式（CSVなど）で出力するようAIに指示することも可能です。
*   注意点: サマリーはあくまで「注意喚起」であり、正式な情報提供は必ず公的な文書（診療情報提供書など）で行う必要があります。

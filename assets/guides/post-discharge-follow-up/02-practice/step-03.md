# 02-practice/step-03.md: フォローアップの評価と計画の動的修正

## 1. AIによる進捗報告の構造化と評価

退院後のフォローアップ期間中、訪問看護師やケアマネジャーから定期的に進捗報告が上がってきます。これらの報告は、多くの場合、自由記述のテキスト形式であり、重要な情報を迅速に把握するのが困難です。AIは、これらの報告を構造化し、計画の達成度を評価するのに役立ちます。

### 実用的なプロンプト例：進捗報告の構造化とリスク評価

```
あなたは、退院後フォローアップ計画の評価担当者です。
以下の[進捗報告テキスト]と[元の計画書]を比較し、以下の項目について分析し、Markdown形式で出力してください。

分析項目:
1.  計画達成度: [元の計画書]の主要目標（3つ）に対する現在の達成度を「高」「中」「低」で評価し、その理由を記述。
2.  新たなリスク要因: 報告テキストから、退院時には見られなかった新たな問題点やリスク要因（例：食欲不振、内服アドヒアランスの低下、家族の疲弊）を抽出。
3.  計画の修正提案: 新たなリスク要因を考慮し、[元の計画書]のどの部分をどのように修正すべきか、具体的な提案を3点記述。

[進捗報告テキスト]
（例: 訪問看護師からの報告。バイタルは安定しているが、体重が2日で1kg増加。患者は「少しだるい」と訴え、散歩を控えている。娘さんは「薬の管理が大変」と疲労を訴えている。）

[元の計画書]
（02-practice/step-01でAIが生成した計画書ドラフトのテキストを貼り付けます）
```

## 2. 患者・家族からの質問への迅速な対応支援

退院後、患者や家族から電話やメッセージで質問が寄せられることがあります。AIは、これらの質問に対し、元の計画書と患者の病態に基づいた、一貫性のある、かつ迅速な回答ドラフトを作成するのに役立ちます。

### 実用的なプロンプト例：質問への回答ドラフト生成

```
あなたは、患者の不安を解消する専門の医療相談員です。
以下の[患者からの質問]に対し、[元の計画書]と[患者の病態]に基づき、専門的でありながらも、共感と安心感を与える回答ドラフトを作成してください。

回答の制約:
*   必ず「まずは落ち着いて」という言葉で始める。
*   回答の最後に「もし不安が続くようでしたら、すぐに病院にご連絡ください」という導線を設ける。
*   回答は200字以内にまとめる。

[患者からの質問]
（例: 「昨日から足が少しむくんでいる気がします。計画書には体重増加に注意とありますが、どうしたらいいですか？」）

[元の計画書]
（02-practice/step-01でAIが生成した計画書ドラフトのテキストを貼り付けます）

[患者の病態]
（例: 慢性心不全、退院時BNP 300pg/ml、利尿薬内服中）
```

### 注意点とTips

*   Tips: AIに「回答のトーン」や「文字数制限」といった制約を設けることで、医療現場で即座に利用できる、質の高いコミュニケーション文案が得られます。
*   注意点: AIが生成した回答は、緊急性の判断を伴うものではありません。緊急性の高い症状（例：胸痛、呼吸困難）に関する質問に対しては、AIの回答を参考にしつつも、必ず医療従事者が直接対応し、適切な指示を出してください。

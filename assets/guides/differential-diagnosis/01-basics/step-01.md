# ステップ1: 鑑別診断の基本原則を理解する

## 鑑別診断とは？

鑑別診断とは、患者の症状や所見から考えられる疾患を列挙し、優先順位をつけて診断を絞り込むプロセスです。

---

## 鑑別診断の基本原則

### 1. VINDICATE法

鑑別診断を網羅的に考えるためのフレームワークです。

| 頭文字 | カテゴリー | 例 |
| :---: | :--- | :--- |
| V | Vascular（血管性） | 脳梗塞、心筋梗塞、大動脈解離 |
| I | Inflammatory/Infectious（炎症・感染） | 肺炎、髄膜炎、心内膜炎 |
| N | Neoplastic（腫瘍） | 肺癌、脳腫瘍、悪性リンパ腫 |
| D | Degenerative/Drugs（変性・薬剤） | 変形性関節症、薬剤性肝障害 |
| I | Iatrogenic/Idiopathic（医原性・特発性） | 術後合併症、特発性血小板減少性紫斑病 |
| C | Congenital（先天性） | 先天性心疾患、遺伝性疾患 |
| A | Autoimmune/Allergic（自己免疫・アレルギー） | 関節リウマチ、SLE、アナフィラキシー |
| T | Trauma（外傷） | 骨折、頭部外傷 |
| E | Endocrine/Metabolic（内分泌・代謝） | 糖尿病、甲状腺機能亢進症 |

---

### 2. Common things are common

頻度の高い疾患を優先的に考える

> 「馬の蹄の音が聞こえたら、シマウマではなく馬を考えよ」

#### 例: 発熱の鑑別診断

頻度の高い疾患:
- 上気道炎（風邪）
- インフルエンザ
- 尿路感染症
- 肺炎

頻度の低い疾患:
- 心内膜炎
- 結核
- 悪性腫瘍

→ まず頻度の高い疾患を除外してから、稀な疾患を考える

---

### 3. Don't miss diagnosis（見逃してはいけない疾患）

致死的または緊急治療が必要な疾患を優先的に考える

![見逃してはいけない疾患](/Helix/assets/guides/differential-diagnosis/dont_miss.png)

#### 主訴別のDon't miss diagnosis

| 主訴 | Don't miss diagnosis |
| :--- | :--- |
| 胸痛 | 急性冠症候群、大動脈解離、肺塞栓症、気胸 |
| 呼吸困難 | 肺塞栓症、急性心不全、気胸、アナフィラキシー |
| 腹痛 | 腸管穿孔、腸管虚血、大動脈瘤破裂、異所性妊娠 |
| 頭痛 | くも膜下出血、髄膜炎、脳腫瘍、側頭動脈炎 |
| 意識障害 | 低血糖、脳卒中、髄膜炎、中毒 |

---

### 4. 優先順位付けの3つの基準

鑑別診断をリストアップしたら、以下の基準で優先順位をつけます。

#### 基準1: 緊急性（Urgency）

致死的または緊急治療が必要か？

- 高: 大動脈解離、肺塞栓症、くも膜下出血
- 中: 肺炎、尿路感染症
- 低: 風邪、筋肉痛

---

#### 基準2: 頻度（Frequency）

どのくらい頻度が高いか？

- 高頻度: 風邪、胃腸炎、筋肉痛
- 中頻度: 肺炎、尿路感染症
- 低頻度: 心内膜炎、結核、悪性腫瘍

---

#### 基準3: 所見の一致度（Fit）

患者の症状・所見とどのくらい一致するか？

- 高一致: 支持する所見が多い
- 中一致: 一部の所見が一致
- 低一致: 否定的な所見が多い

---

## 鑑別診断の思考プロセス

### ステップ1: 主訴を明確にする

例: 「胸痛」

---

### ステップ2: 解剖学的アプローチ

胸痛の原因を解剖学的に分類

| 部位 | 疾患 |
| :--- | :--- |
| 心臓 | 急性冠症候群、心膜炎、心筋炎 |
| 大血管 | 大動脈解離 |
| 肺 | 肺塞栓症、気胸、肺炎、胸膜炎 |
| 食道 | 胃食道逆流症、食道痙攣、食道破裂 |
| 胸壁 | 肋骨骨折、肋間神経痛、帯状疱疹 |
| その他 | パニック障害、筋肉痛 |

---

### ステップ3: 病態生理学的アプローチ

VINDICATE法を使って分類

| カテゴリー | 疾患 |
| :--- | :--- |
| Vascular | 急性冠症候群、大動脈解離、肺塞栓症 |
| Inflammatory | 心膜炎、胸膜炎、肺炎 |
| Neoplastic | 肺癌、縦隔腫瘍 |
| Degenerative | 変形性脊椎症 |
| Trauma | 肋骨骨折 |
| Others | 胃食道逆流症、パニック障害 |

---

### ステップ4: 優先順位付け

Don't miss diagnosis を優先

1. 急性冠症候群 ★★★（緊急性: 高、頻度: 中、一致度: 高）
2. 大動脈解離 ★★★（緊急性: 高、頻度: 低、一致度: 中）
3. 肺塞栓症 ★★★（緊急性: 高、頻度: 低、一致度: 中）
4. 肺炎 ★★（緊急性: 中、頻度: 高、一致度: 高）
5. 胃食道逆流症 ★（緊急性: 低、頻度: 高、一致度: 中）

## 鑑別診断の落とし穴

### 落とし穴1: アンカリングバイアス

最初に思いついた診断に固執してしまう

例:
- 胸痛 → 急性冠症候群と決めつける
- 心電図正常 → 「急性冠症候群ではない」と判断
- 大動脈解離を見逃す

対策:
- 必ず複数の鑑別診断を挙げる
- 否定的な所見があっても、他の鑑別を考える

---

### 落とし穴2: 確証バイアス

自分の仮説を支持する情報ばかり集めてしまう

例:
- 「肺炎だろう」と思い込む
- 発熱、咳、肺野の浸潤影に注目
- 肺塞栓症の可能性を見落とす

対策:
- 否定的な所見も積極的に探す
- 「この診断を否定する所見は何か？」と自問する

---

### 落とし穴3: 可用性バイアス

最近経験した症例や印象的な症例を優先してしまう

例:
- 最近、大動脈解離の症例を経験
- 胸痛患者を見ると、すぐに大動脈解離を疑う
- 頻度の高い急性冠症候群を見落とす

対策:
- 頻度を考慮する（Common things are common）
- 稀な疾患を除外する前に、頻度の高い疾患を除外

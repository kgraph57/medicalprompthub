# Step 4: 論文を「疑う」技術〜探偵のように批判的吟味〜

### 📋 このステップの目的

論文は、ただ読むだけでは不十分です。
書かれていることを鵜呑みにせず、探偵のように鋭い目で「本当にそうなの？」と疑うことが、プロの作法です。
このステップでは、論文の質を見抜き、結果を正しく解釈するための「批判的吟味」という名の調査スキルを学びます。

### ⏱️ 所要時間

15〜20分

### 🎯 達成基準

- 論文の「研究デザイン」が、その研究の目的に合っているか評価できる。
- 論文に潜む「バイアス」という名の犯人（選択、情報、交絡）を見つけ出せる。
- 著者の「結果の解釈」が、客観的で妥当なものか判断できる。

### 📝 詳細手順

![批判的吟味](/assets/guides/paper-reading/images/critical_appraisal.png)

論文という「事件現場」を調査するポイントは、大きう3つです。

![批判的吟味のチェックポイント](/assets/guides/paper-reading/04-critical-appraisal/images/critical_appraisal_checklist_refined.png)


#### 調査ポイント1: 研究デザインは適切か？

研究の「問い」に対して、選ばれた「研究デザイン」がベストなものか、まず確認します。

| 問いの種類 | ベストな研究デザイン | 調査のヒント |
|---|---|---|
| 治療の効果は？ | RCT | ランダム化、盲検化は？ |
| 診断の精度は？ | 診断精度研究 | ゴールドスタンダードとの比較は？ |
| 予後はどうなる？ | コホート研究 | フォローアップ期間は十分？ |
| 原因は何か？ | コホート研究 | 交絡因子は調整されてる？ |

#### 調査ポイント2: バイアスはないか？

バイアスとは、結果を歪める可能性のある「偏り」のこと。
主な犯人はこの3人です。

- 選択バイアス: そもそも研究に参加した人たちに偏りがあるケース。
- 情報バイアス: データの集め方や測定方法に偏りがあるケース。
- 交絡: 結果に影響する別の要因（交絡因子）が考慮されていないケース。

#### 調査ポイント3: 結果の解釈は妥当か？

著者の解釈が、客観的でフェアなものかを見極めます。

- 因果関係の言い過ぎ: 観察研究なのに「〜が原因」と断定していないか？
- 過度な一般化: 特定の患者さんだけの結果を、全員に当てはまるかのように語っていないか？
- 臨床的意義の評価: 統計的に差があっても、臨床的に意味のある差なのか？

### 💡 AIへの「話し方」

#### プロンプト1: 「この研究デザイン、どう思う？」

```
# 指示
この論文の研究デザインについて、プロの探偵として評価してくれ。

# 論文の情報
- 研究の問い (PICO): 
- 研究デザイン: 
- Methodsの要約: 

# 評価ポイント
1. このデザインは、問いに答えるのにベストか？
2. やり方（ランダム化とか）に穴はないか？
3. もっと良くするなら、どうする？
```

#### プロンプト2: 「この論文の弱点を3つ教えて」

```
# 指示
この論文を読んで、弱点やツッコミどころを3つ、鋭く指摘してくれ。

# 論文
[論文のPDFをアップロード]
```

### ⚠️ よくある「落とし穴」

- 失敗: p値が0.05未満だからと、結果を信じ込む。
  - 対処法: p値はあくまで参考情報。
バイアスの有無や臨床的意義を総合的に判断しましょう。

- 失敗: 観察研究なのに、因果関係を信じてしまう。
  - 対処法: 「関連がある」と「原因である」は全くの別物。
観察研究からは因果関係は言えません。

### 💪 上級者への道

- CASPツールを使いこなす: 研究デザインごとに用意されたチェックリストを使えば、吟味のモレがなくなります。
- 複数の論文を比較する: 同じテーマの論文を複数比較することで、結果の一貫性や、研究ごとの質の差が見えてきます。

## 読解支援のための効果的なプロンプト設計

AIに画像診断レポートの読解支援を依頼する際、その効果はプロンプトの質に大きく左右されます。単にレポートを貼り付けるだけでなく、AIが最大限の能力を発揮できるように、役割、情報、形式の3要素を明確に定義することが重要です。

### 3つの要素で構成するプロンプト設計

効果的なプロンプトは、以下の3つの要素を構造的に含みます。

| 要素 | 目的 | 具体的な指示内容 |
| :--- | :--- | :--- |
| 役割（Role） | AIの視点と専門性を設定する。 | 「あなたは経験豊富な放射線科医です」「あなたは患者に説明する看護師です」など。 |
| 情報（Context & Data） | 処理対象のデータと、その背景情報を提供する。 | 匿名化されたレポート全文、患者の臨床情報（年齢、性別、主訴）、具体的な質問。 |
| 形式（Format） | 期待する出力の構造と粒度を指定する。 | 「3つの箇条書きで要約」「表形式で」「小学生にもわかる言葉で」など。 |

### 実用的なプロンプト例：レポートの要約と重要所見の抽出

レポート全体を迅速に把握し、クリティカルな情報を見逃さないためのプロンプトです。

プロンプト例 2.1：レポートの要約と重要所見の抽出

```
あなたは、多忙な臨床医を支援するAIアシスタントです。
以下の画像診断レポート（匿名化済み）を読み、以下の指示に従って情報を整理してください。

レポート:
---
[画像診断レポートの所見と結論の全文]
---

指示:
1.  要約: レポート全体を3つの箇条書きで簡潔に要約してください。
2.  重要所見: 治療方針に直結する可能性のある、最もクリティカルな所見を1つだけ抽出してください。
3.  フォローアップ推奨: 読影医が推奨しているフォローアップ検査や期間があれば、そのまま引用してください。
4.  形式: 上記1〜3を、以下のMarkdown形式の表にまとめてください。

| 項目 | 内容 |
| :--- | :--- |
| 全体要約 | [要約1] [要約2] [要約3] |
| 最重要所見 | [抽出した所見] |
| フォローアップ | [推奨内容] |
```

Tips:
*   段階的な質問: 一度に全てを尋ねるのではなく、「まず要約」「次に重要所見」のように段階的に質問することで、AIの思考プロセスを整理させ、精度の高い回答を引き出せます。
*   匿名化の徹底: レポートを貼り付ける前に、必ず個人情報が完全に削除されているかを再確認してください。

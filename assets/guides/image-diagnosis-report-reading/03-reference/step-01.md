## レポートの記述品質評価と教育的活用

AIは、レポートの読解支援だけでなく、レポート自体の記述品質の評価や、医学生・研修医への教育ツールとしても応用できます。これにより、医療機関全体のレポート作成レベルの向上と、若手医師の読影能力の育成に貢献します。

### レポートの記述品質評価

読影医が作成したレポートの記述が、必要な情報を含んでいるか、曖昧な表現を避けているか、推奨されるガイドライン（例：BI-RADS, PI-RADSなど）に準拠しているかをAIに評価させます。

プロンプト例 6.1：レポートの記述品質評価

```
あなたは、画像診断レポートの品質管理を行うAI専門家です。
以下の画像診断レポート（匿名化済み）を、[評価基準]に基づき評価し、改善点を提案してください。

画像診断レポート（匿名化済み）:
---
[画像診断レポートの所見と結論の全文]
---

評価基準:
1.  網羅性: 臨床情報（検査目的、既往歴など）との関連付けが十分か。
2.  明確性: 曖昧な表現（例：「〜の可能性が否定できない」）が多用されていないか。
3.  構造化: 所見、結論、推奨事項が明確に分離されているか。
4.  ガイドライン準拠: [特定のガイドライン名、例: PI-RADS v2.1]に準拠した記述があるか。

指示:
1.  上記の各評価基準について、10点満点で点数を付け、その理由を簡潔に述べてください。
2.  レポートの記述を改善するための具体的な修正案を3点提案してください。
```

### 教育ツールとしてのAI活用

医学生や研修医が、レポートから病態を正確に把握する能力を養うために、AIを仮想の指導医として活用します。

プロンプト例 6.2：教育的な質問と解説

```
あなたは、画像診断レポートの読解を指導する熱心な指導医です。
以下の画像診断レポート（匿名化済み）を読んだ医学生に対して、理解度を確認するための質問を3つ作成し、それぞれの質問に対する模範的な回答と解説を提供してください。

画像診断レポート（匿名化済み）:
---
[画像診断レポートの所見と結論の全文]
---

指示:
1.  レポートの最も重要な所見に関する質問を1つ作成してください。
2.  レポートで言及されている病態の鑑別診断に関する質問を1つ作成してください。
3.  レポートで推奨されているフォローアップに関する質問を1つ作成してください。
4.  各質問に対し、模範的な回答と、その回答に至るための思考プロセスを解説してください。
```

### Tips：フィードバックループの構築

AIによる評価結果を読影医にフィードバックするシステムを構築することで、レポートの品質を継続的に向上させることができます。AIは客観的な評価者として機能し、人間では見落としがちな記述の癖や偏りを指摘できます。

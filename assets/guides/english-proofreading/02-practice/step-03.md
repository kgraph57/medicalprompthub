# Step 3: AIツールを活用した一次校正

## Grammarly、DeepL Write、ChatGPTの使い分け

AIツールは、基本的な文法ミスやスペルミス、不自然な表現を素早く修正するのに最適です。ただし、ツールごとに得意分野が異なるため、使い分けが重要です。

## 主要なAIツール

### Grammarly

- 用途: 文法・スペル・句読点の自動検出
- 優先度: ⭐⭐⭐⭐
- 特徴: 無料版でも基本的なミスは十分に検出できます。ブラウザ拡張機能として使用すると便利です。

### DeepL Write

- 用途: 文章をより自然で洗練された表現にリライト
- 優先度: ⭐⭐⭐⭐⭐
- 特徴: DeepL翻訳の姉妹サービス。文脈を理解した自然な表現への書き換えが得意です。

### ChatGPT / Claude

- 用途: 特定の文章のパラフレーズ、アカデミックな表現の提案
- 優先度: ⭐⭐⭐⭐
- 特徴: プロンプトを使って、よりアカデミックな表現を提案させることができます。

## 推奨ワークフロー

1. Grammarlyで基本的な文法・スペルミスを修正
2. DeepL Writeで不自然な表現を自然な表現にリライト
3. ChatGPTで特定のパラグラフをよりアカデミックな表現に改善
4. 全体を通読して、文脈の一貫性を確認

## AI校正プロンプトの活用

以下のプロンプトを使って、ChatGPTやClaudeに校正を依頼できます。

```
以下の英語の文章を、医学論文（Academic Medical Writing）として自然で洗練された表現に校正してください。
以下の点に注意して修正してください：
1. 簡潔で明確な表現（Concise and Clear）: 冗長な語句を避け、1文1思考を心がける。
2. 能動態の使用（Active Voice）: 受動態よりも能動態を優先する（約70%を目安に）。
3. 否定文を避ける: 肯定文の方が理解しやすいため。
4. "be"動詞を避ける: 動作動詞を使用する。
5. "The first report"や"Unique case"などの表現は避ける。

# 原文
[ここに英文を貼り付ける]
```

# 01-basics/step-02.md: AIを活用した研究計画の骨子作成と制約条件の設定

## 研究計画の骨子を倫理審査の視点で詳細化する

ステップ1で倫理審査の主要な構成要素を把握しました。次に、AIにあなたの研究計画の骨子を与え、倫理審査の3大要素である「研究の意義」「科学的妥当性」「倫理的配慮」の観点から、申請書の下書きに使えるレベルまで詳細化させます。

### 倫理審査の3大要素とAIの役割

| 倫理審査の要素 | 審査員が求めること | AIへの指示のポイント |
| :--- | :--- | :--- |
| 研究の意義 | なぜこの研究が必要で、社会や医療にどのような貢献をするのか。 | 「社会貢献性」と「新規性」を強調した文章構成を指示。 |
| 科学的妥当性 | 研究デザイン、対象者選定、統計手法が目的に対して適切で、結果が信頼できるか。 | 「方法論の厳密性」と「実現可能性」を論理的に記述するよう指示。 |
| 倫理的配慮 | 被験者の人権、安全、プライバシーが最大限に保護されているか。リスクと利益のバランスは適切か。 | 「リスクの最小化」と「インフォームド・コンセントの手順」を具体的に記述するよう指示。 |

### 具体的な手順

1.  役割の再設定: AIに「あなたは倫理審査委員会の委員であり、研究計画の論理的な欠陥と倫理的リスクを指摘する」という役割を与えます。
2.  骨子案の提示: ステップ1で整理した研究の基本情報と、あなたが考えた研究の目的、方法の簡単な骨子を提示します。
3.  3大要素に基づく詳細化の指示: 上記の3大要素それぞれについて、審査員が納得できるレベルの詳細な記述を生成するよう指示します。

### 実用的なプロンプト例

以下のプロンプトは、AIに倫理審査委員の視点を持たせ、研究計画の骨子を詳細な申請書の下書きに変換させるためのものです。

```prompt
あなたは、日本の医療機関の研究倫理審査委員会の委員です。
以下の[研究計画の骨子案]に基づき、倫理審査申請書の「研究の意義」「科学的妥当性」「倫理的配慮」の3つのセクションについて、それぞれ500字程度で詳細な記述を作成してください。

制約条件:
1.  記述は、審査員が「この研究は実施する価値があり、かつ倫理的に問題がない」と判断できる論理的な構成であること。
2.  「倫理的配慮」のセクションでは、想定される最大のリスクを明記し、それに対する具体的な回避・対処方法を記述すること。
3.  専門用語（例：[あなたの研究分野の専門用語]）は、申請書全体で統一された表現を使用すること。

[研究計画の骨子案]:
-   研究目的: [簡潔に記述]
-   研究方法: [簡潔に記述]
-   対象者: [人数、選定基準を簡潔に記述]
-   実施期間: [簡潔に記述]
```

### 注意点とTips

*   プロンプトの具体性: AIに与える骨子案は、具体的であればあるほど、生成される文章の質が向上します。曖昧な表現は避けましょう。
*   専門用語の統一: 倫理審査では、用語の定義や使用方法の統一性が求められます。AIが生成した文章で、専門用語がブレていないか確認し、必要であればプロンプトで修正を指示しましょう。

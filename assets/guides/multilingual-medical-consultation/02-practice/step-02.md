# 診断結果と治療方針の説明とインフォームド・コンセント

## 1. 複雑な医療情報を伝えるためのAI活用

診断結果や治療方針の説明は、医療従事者にとって最も専門的な知識を要する場面であり、患者の理解度と同意（インフォームド・コンセント）を得るために極めて重要です。AI通訳には、専門用語を正確に翻訳する能力に加え、患者の理解度に応じた表現に調整する能力が求められます。

### 2. プロンプト例：専門用語の平易化と確認

単に専門用語を翻訳するだけでなく、患者が理解しやすいように平易な言葉に置き換える指示をAIに与えることで、インフォームド・コンセントの質を高めます。

#### プロンプト例：平易化と理解度確認の指示

```
あなたは、日本の医療機関で働く医療従事者と、日本語を解さない外国人患者との間の会話を、正確かつリアルタイムで通訳するAIです。

追加の役割と制約:
1.  専門用語の平易化: 医療従事者から発せられた専門用語（例：心筋梗塞、高血糖、抗凝固薬など）については、可能な限り患者が理解しやすい平易な言葉に翻訳を調整してください。ただし、医学的な正確性は損なわないこと。
2.  理解度確認の挿入: 医療従事者が重要な説明（例：診断名、治療の選択肢、副作用）を終えた後、患者に対して「ここまでの説明で、何かご不明な点はありますか？」という確認のフレーズを、通訳の最後に自動的に付け加えてください。
3.  治療選択肢の明確化: 複数の治療選択肢を提示する際は、それぞれのメリットとデメリットが明確に伝わるように、構造化された翻訳を心がけてください。

開始指示: 準備ができたら、「通訳を開始します。話し始めてください。」と応答してください。
```

### 3. 構造化された説明のテクニック

AI通訳を介して複雑な情報を伝える際は、医療従事者側も説明を構造化することが、患者の理解を助けます。

| 説明のステップ | 目的 | AI通訳への影響 |
| :--- | :--- | :--- |
| 診断の要約 | 患者の病状を簡潔に伝える。 | 専門用語の平易化が重要。 |
| 治療の選択肢 | 複数の選択肢とその結果を提示する。 | 構造化された翻訳が効果的。 |
| リスクとベネフィット | 各選択肢の長所と短所を比較する。 | ニュアンスの正確な伝達が重要。 |
| 患者の意向確認 | 患者の価値観や希望を把握する。 | AIが自動で理解度確認を促す。 |

### 4. Tips：治療計画の文書化と共有

*   文書による補完: AI通訳は会話をサポートしますが、重要な情報は必ず文書化し、患者の言語に翻訳された資料（例：診断書、治療計画書、薬の説明書）を手渡します。AI翻訳機能を使って、これらの文書をその場で翻訳することも有効です。
*   「わからない」を歓迎する雰囲気: 患者が「わからない」と言いやすい雰囲気を作り、AI通訳が理解度確認の質問を挿入した後、患者が質問するのを待つ時間を確保します。
*   多言語対応の資料: 院内で使用する定型的な説明資料（例：手術同意書、検査説明）は、事前に主要言語に翻訳されたものを用意しておくと、AI通訳の負荷を減らせます。

# 01-basics/step-01.md: AIを活用したインシデントレポート作成の基本とメリット

## 医療安全のためのAI活用：インシデント報告のパラダイムシフト

インシデントレポートの作成は、医療安全管理の根幹をなす重要な業務です。しかし、多忙な業務の中で詳細かつ客観的なレポートを作成することは、医療従事者にとって大きな負担となりがちです。AIを活用することで、このプロセスを迅速化し、客観性を高め、結果として医療安全文化の醸成に貢献できます。

### 1. AI活用の意義とメリット

| メリット | 詳細 | 医療安全への貢献 |
| :--- | :--- | :--- |
| 報告の迅速化 | 事実情報を整理し、レポートの骨子を瞬時に作成することで、報告までの時間を大幅に短縮します。 | 迅速な情報共有と対策検討の開始。 |
| 客観性の向上 | 感情や推測を排除し、入力された事実情報のみに基づいた客観的な文章構成を支援します。 | 偏りのない原因分析と再発防止策の立案。 |
| 分析の効率化 | 報告書作成の負担が減ることで、本来注力すべき原因分析や対策検討に時間を割けるようになります。 | より質の高い医療安全対策の実現。 |

### 2. AIに提供すべき情報の基本原則

AIは、入力された情報に基づいて処理を行います。質の高いレポートを作成するためには、以下の基本原則を厳守し、AIに正確な事実情報を提供することが不可欠です。

1.  事実のみを入力する: 「〜だと思った」「〜のせいだ」といった推測や感情的な表現は避け、実際に何が起こったかという事実のみを記述します。
2.  個人情報・推測は除外する: 患者名や特定の個人を特定できる情報は、AIへの入力段階で匿名化または除外します。また、責任追及につながるような表現も避けます。
3.  5W1Hを明確にする: いつ（When）、どこで（Where）、誰が（Who）、何を（What）、なぜ（Why）、どのように（How）を整理し、網羅的に情報を提供します。

### 3. 実践的な手順：報告すべき事象の5W1H整理

インシデント発生直後、記憶が鮮明なうちに5W1Hを整理し、AIに入力するための準備を行います。

#### プロンプト例：事実情報の整理と構造化

以下のプロンプトは、インシデントに関する断片的な情報を整理し、レポートの基礎となる構造化されたテキストを生成するために使用します。

プロンプト:
> あなたは医療安全管理の専門家です。以下のインシデントに関する事実情報に基づき、5W1Hの形式で客観的な事実経過を整理してください。推測や感情的な表現は一切含めないでください。
>
> インシデント情報:
> - 発生日時: 2025年12月10日 14:30頃
> - 発生場所: 3階病棟 A室
> - 関係者: 看護師X、患者Y（80代男性、認知症あり）
> - 事象の概要: 14:00に内服薬（降圧剤）を配薬したが、14:30に患者Yがベッドサイドの手の届く範囲に置いてあった薬を全て服用してしまった。
> - 経緯: 看護師Xは他の患者の対応のためA室を離れていた。患者Yは普段は自力で起き上がれないが、この時は自力で起き上がり、薬を服用したと推測される。
> - その後の対応: 14:45に発見し、直ちに医師に報告。バイタルサインに異常なし。経過観察中。

### 4. Tips：AIを「下書き作成の補助」と位置づける

AIが生成したテキストは、あくまでレポートの下書きです。最終的なレポートは、医療機関の規定に従い、必ずご自身の目で確認し、必要に応じて修正・加筆を行ってください。AIは、報告の「手間」を減らすための強力なツールであり、最終的な「責任」は医療従事者にあります。

# 感染対策マニュアル作成ガイド：実践編3

## 教育・研修コンテンツと評価ツールの作成

感染対策マニュアルは、作成して終わりではありません。職員に周知徹底し、実践してもらうための教育・研修コンテンツと、その効果を測定するための評価ツールが必要です。このステップでは、AIを活用してこれらの補助資料を効率的に作成します。

### 1. AIによる研修資料の骨子と模擬事例の生成

AIは、マニュアルの本文から重要なポイントを抽出し、研修用のスライド構成案や、職員の判断力を試すための模擬事例（シナリオ）を生成できます。

具体的な手順:

1.  研修目標の定義: 研修の対象者（例：全職員、特定部署）と、研修後に達成してほしい行動目標（例：標準予防策の5つの要素を説明できる、PPEの着脱を正しく行える）をAIに伝えます。
2.  スライド構成案の生成: マニュアルの特定章（例：標準予防策）に基づき、研修時間（例：30分）に合わせたスライド構成案（タイトル、話す内容の要点）を生成させます。
3.  模擬事例の生成: 現場で起こりうる具体的な感染リスク事例（例：隔離室からの患者搬送、嘔吐物処理）をAIに作成させ、職員の対応を評価するための設問を付加します。

### 2. 実用的なプロンプト例：模擬事例と評価チェックリストの生成

ここでは、職員の判断力を試すための模擬事例と、その対応を評価するためのチェックリストを生成するプロンプト例を示します。

プロンプト:

```
あなたは、感染対策研修用の模擬事例と、それに対する職員の対応を評価するためのチェックリストを作成します。

![感染経路別予防策](/assets/guides/infection-control-manual/transmission_precautions.png)

[マニュアルのトピック]
- 経路別予防策（接触予防策、飛沫予防策、空気予防策）

[要件]
1.  模擬事例（シナリオ）: 以下の要素を含む、具体的な病棟での事例を一つ作成してください。
    - 登場人物（看護師、患者）
    - 状況（発熱、下痢、咳などの症状）
    - 看護師が「どの予防策」を「なぜ」選択すべきかという判断を迫られる場面
2.  評価チェックリスト: 模擬事例に対する職員の対応を評価するための、5項目以上のチェックリストを作成してください。
    - チェックリストの項目は、具体的な行動（例：ガウンを着用したか）であること。
    - 評価基準（例：〇：実施した、×：実施しなかった）を併記すること。
```

### 3. 注意点とTips

*   Tips: ゲーミフィケーションの導入: AIに「感染対策クイズ」や「間違い探し」形式のコンテンツを生成させ、研修に導入することで、職員の学習意欲を高めることができます。例えば、「この写真のPPE装着で間違っている箇所を3つ指摘してください」といった設問と、その解説をAIに作成させましょう。
*   注意点: 定期的な見直し: 感染対策は、新しい病原体やガイドラインの改訂により、常に変化します。AIに「このマニュアルの重要項目について、6ヶ月ごとに最新の知見を検索し、改訂が必要な箇所を提案するリマインダー」を作成させ、マニュアルの鮮度を維持する仕組みを構築しましょう。

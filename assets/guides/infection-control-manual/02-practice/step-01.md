# 感染対策マニュアル作成ガイド：実践編1

## 現場で「使える」本文コンテンツの生成

基本編で作成したアウトラインに基づき、いよいよマニュアルの本文コンテンツをAIに生成させます。このステップの目標は、単なる情報の羅列ではなく、現場の医療従事者が直感的に理解し、行動に移せる「使える」文章を作成することです。

### 1. 現場の状況を反映したコンテンツの生成

AIに具体的なコンテンツを生成させる際、最も重要なのは「コンテキスト（文脈）」を与えることです。マニュアルの読者（看護師、清掃スタッフなど）の視点に立ち、彼らが直面する具体的な状況をプロンプトに盛り込みます。

具体的な手順:

1.  トピックの選定: アウトラインから、最も重要かつ現場での行動に直結するトピック（例：「PPEの着脱手順」「針刺し事故発生時の初期対応」）を選びます。
2.  読者と状況の指定: プロンプト内で、そのトピックの主な読者（例：新人看護師）と、その状況（例：COVID-19患者のケア後）を明確に指定します。
3.  形式の指定: 「箇条書き」「ステップ形式」「チェックリスト形式」など、現場での視認性と実行性を高めるための形式を指定します。

### 2. 実用的なプロンプト例：手順書の生成

ここでは、マニュアルの核となる「個人防護具（PPE）の着脱手順」を、新人看護師向けに生成するプロンプト例を示します。

プロンプト:

```
あなたは、新人看護師向けの感染対策マニュアルの作成者です。
「標準予防策」の章に記載する「個人防護具（PPE）の着脱手順」について、以下の要件を満たす手順書を作成してください。

[要件]
1.  手順は「装着」と「脱衣」に分け、それぞれを番号付きのステップ形式（1., 2., 3. ...）で記述すること。
2.  各ステップには、なぜその行動が必要なのかの理由（例：汚染拡大を防ぐため）を簡潔に併記すること。
3.  特に汚染されやすい箇所や、手順を間違えやすいポイント（例：手袋の脱ぎ方）には、太字で注意喚起の言葉を入れること。
4.  手順の最後に、「次のステップ」として手指衛生の徹底を促す導線を設けること。
5.  文体は、専門的でありながらも、新人でも理解しやすい平易な言葉遣いを用いること。
```

### 3. 注意点とTips

*   Tips: 専門用語の定義をAIに依頼する: マニュアルには専門用語（例：エアロゾル、Fomite、標準予防策）が頻出します。AIに「このマニュアルで使用する専門用語の定義リストを、平易な言葉で作成してください」と依頼し、巻末や用語集として活用することで、マニュアル全体の理解度を高めることができます。
*   注意点: 視覚的な要素の指示: AIはテキストを生成しますが、マニュアルは視覚的な要素（図、写真）が非常に重要です。生成された手順書の各ステップに対し、「このステップには、手袋を裏返して脱ぐ様子の写真を挿入する」といった具体的な指示をメモとして残し、後の作業で必ず視覚資料を準備する計画を立ててください。

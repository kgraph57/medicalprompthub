
# ステップ1: 介護保険主治医意見書を作成する

## 多角的な視点が求められる意見書

介護保険主治医意見書は、市区町村が要介護認定を行う際に、申請者の心身の状態について主治医が医学的な見地から意見を述べるための、非常に重要な文書です。単なる診断書とは異なり、病気や怪我そのものだけでなく、それが日常生活動作（ADL）や認知機能にどのような影響を与えているかという、より生活に密着した視点が求められます。

記載項目が多岐にわたり、作成に手間がかかるため、多くの医師が負担に感じています。AIを活用することで、患者の情報を基に、各項目に対応する記述のドラフトを効率的に作成することができます。

---

## 主治医意見書作成支援プロンプト

このプロンプトは、患者の病状、ADL、認知機能などに関する断片的な情報をAIに与え、主治医意見書の各項目を埋めるための文章を生成させるものです。

### 📋 主治医意見書プロンプト（コピペ用）

```
あなたは、高齢者医療と介護保険制度に精通した、経験豊富な主治医です。以下の患者情報に基づき、「介護保険主治医意見書」の主要項目のドラフトを作成してください。

【患者情報】
*   氏名・年齢・性別: 
*   主要な傷病名: 
    1. [例: アルツハイマー型認知症]
    2. [例: 脳梗塞後遺症（右片麻痺）]
    3. [例: 骨粗鬆症]
    4. [例: 高血圧症]
*   ADLの状態: [例: 食事は自立。更衣・入浴・トイレに一部介助が必要。移動は室内は杖歩行、屋外は車椅子。]
*   認知機能の状態: [例: 長谷川式スケール 12/30点。短期記憶の障害が顕著。時間や場所の見当識障害あり。物盗られ妄想が時々見られる。]
*   特別な医療の必要性: [例: 褥瘡の処置（仙骨部）、インスリン注射]

【タスク】
上記の情報を基に、以下の主治医意見書の項目について、具体的な記述のドラフトを作成してください。

1.  傷病に関する意見
    *   診断名、症状の安定性、治療内容、今後の見通しなど
2.  生活機能とサービスに関する意見
    *   移動、栄養・食事、排泄、整容などのADLに関する具体的な状況
    *   認知機能（問題行動の有無など）
    *   医学的管理の必要性（訪問看護の必要性など）
3.  特記すべき事項
    *   [介護者に特に伝えたい注意点など]
```

## 実践例: アルツハイマー型認知症患者の意見書

### 入力プロンプト

```
あなたは、高齢者医療と介護保険制度に精通した、経験豊富な主治医です。以下の患者情報に基づき、「介護保険主治医意見書」の主要項目のドラフトを作成してください。

【患者情報】
*   氏名・年齢・性別: 渡辺 花子 様 (88歳・女性)
*   主要な傷病名: 1. アルツハイマー型認知症, 2. 脳梗塞後遺症（軽度右片麻痺）, 3. 骨粗鬆症, 4. 高血圧症
*   ADLの状態: 食事は自立。更衣・入浴・トイレに一部介助が必要。移動は室内は伝い歩き、屋外は車椅子。
*   認知機能の状態: 長谷川式スケール 12/30点。短期記憶の障害が顕著。時間や場所の見当識障害あり。夕方になると「家に帰る」と言って外に出ようとすることがある（帰宅願望）。
*   特別な医療の必要性: なし

【タスク】
上記の情報を基に、主治医意見書の主要項目のドラフトを作成してください。
```

### AIの出力例

```
承知いたしました。渡辺花子様の介護保険主治医意見書のドラフトを作成します。

### 1. 傷病に関する意見

診断名: アルツハイマー型認知症、脳梗塞後遺症、骨粗鬆症、高血圧症

症状の安定性:
アルツハイマー型認知症は、薬物療法（ドネペジル）により進行抑制に努めていますが、記憶障害、見当識障害は緩徐に進行しています。脳梗塞後遺症による右片麻痺、および高血圧症については、現在の内服治療により状態は比較的安定しています。

治療内容:
ドネペジル、アムロジピン、エディロール等を処方し、月1回の外来で経過観察を行っています。

今後の見通し:
認知機能や身体機能の回復は困難であり、現状の能力を維持し、安全な在宅生活を継続することが治療の目標となります。今後、認知症の進行に伴い、BPSD（行動・心理症状）の変動や、ADLのさらなる低下が予測されるため、介護環境の調整が重要です。転倒による骨折のリスクも高い状態です。

### 2. 生活機能とサービスに関する意見

移動: 室内は壁や家具を支えに伝い歩きが可能ですが、ふらつきが強く、転倒のリスクが高いです。屋外での移動には車椅子の利用が必須です。

栄養・食事: 自身で箸を使って食事を摂取することは可能ですが、時々むせ込みが見られます。食事の準備や後片付けはできません。

認知機能: 中核症状として、短期記憶障害と見当識障害が顕著です。BPSDとして、夕方になると不安が強まり、「家に帰る」と言って外出しようとする帰宅願望が週に数回認められます。介護者によるなだめや説得が必要な状況です。

医学的管理の必要性:
定期的な内服薬の管理が不可欠ですが、ご自身での管理は困難であり、家族または訪問看護師による服薬支援が必要です。また、転倒リスクが高いため、訪問リハビリテーションによる身体機能の維持訓練や、住環境の整備（手すりの設置など）が望ましいと考えます。

### 3. 特記すべき事項

認知症の進行に伴うBPSD（特に帰宅願望）に対し、介護者が一人で対応することは、精神的・身体的負担が非常に大きいと拝察します。デイサービスの利用や、ショートステイの活用など、介護者の負担を軽減するためのレスパイトケアの導入を積極的にご検討ください。また、転倒予防のための見守りも重要です。
```

## まとめ

このステップで学んだこと:

✅ 介護保険主治医意見書が求める、生活機能や認知機能に関する多角的な視点
✅ 患者の断片的な情報を基に、意見書の各項目を埋めるための文章をAIに生成させるプロンプト

【重要】
主治医意見書は、その後の介護サービスの内容を左右する重要な文書です。AIの生成結果は、あくまで客観的な情報を整理したものに過ぎません。患者の日々の生活の様子や、介護する家族の状況などを考慮した、あなた自身の総合的な医学的判断を必ず加えてください。

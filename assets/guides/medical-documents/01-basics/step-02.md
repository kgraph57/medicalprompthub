
# ステップ2: 紹介状・返信状を作成する

## 医療連携の要、SOAP形式

紹介状（診療情報提供書）やその返信状は、患者の情報を医療機関の間で正確に共有し、継続的で質の高い医療を提供するための、非常に重要なコミュニケーションツールです。

これらの文書を作成する上で、情報を構造化し、抜け漏れなく伝えるための世界標準のフレームワークがSOAP（ソープ）です。

SOAPの構成要素:

| 要素 | 名称 | 内容 |
| :---: | :--- | :--- |
| S | Subjective (主観的情報) | 患者の訴え（主訴）、現病歴、既往歴、アレルギー歴など |
| O | Objective (客観的情報) | 身体所見、バイタルサイン、検査結果、画像所見など |
| A | Assessment (評価) | SとOから導き出される、医師による評価、考察、アセスメント、診断 |
| P | Plan (計画) | Aに基づいて立案される、今後の治療計画、検査計画、紹介の目的など |

このSOAP形式を用いることで、受け取った医師は患者の状態を迅速かつ正確に把握することができます。

---

## 紹介状・返信状作成の基本プロンプト

このプロンプトは、SOAPの各項目に関する情報をAIに与え、構造化された質の高い紹介状や返信状のドラフトを生成させるためのものです。

### 📋 紹介状基本プロンプト（コピペ用）

```
あなたは、経験豊富な総合内科医です。以下の患者情報（SOAP形式）に基づき、指定された宛先の専門医への紹介状のドラフトを作成してください。

【宛先】
[例: 〇〇大学病院 呼吸器内科 教授 〇〇先生 御侍史]

【紹介目的】
[例: 精査・加療のため]

【患者情報】
*   患者: [氏名、年齢、性別]
*   紹介元: [あなたの病院・クリニック名、あなたの名前]

【SOAP】
*   S (Subjective):
    *   主訴: 
    *   現病歴: 
    *   既往歴: 
    *   生活歴・家族歴: 
    *   内服薬: 

*   O (Objective):
    *   身体所見: 
    *   検査データ: 
    *   画像所見: 

*   A (Assessment):
    *   [SとOを統合したアセスメント、考えられる鑑別診断など]

*   P (Plan):
    *   [紹介に至った経緯、依頼したい内容など]

【出力形式】
一般的な紹介状のフォーマットに従い、専門医に対する敬意を払った丁寧な言葉遣いで記述してください。特に、AssessmentとPlanを明確に記述してください。
```

## 実践例: 肺癌疑いの患者紹介

### 入力プロンプト

```
あなたは、経験豊富な総合内科医です。以下の患者情報（SOAP形式）に基づき、指定された宛先の専門医への紹介状のドラフトを作成してください。

【宛先】
メディカル大学病院 呼吸器内科 教授 佐藤 一郎 先生 御侍史

【紹介目的】
肺癌の疑い。確定診断および治療方針決定のため。

【患者情報】
*   患者: 鈴木 次郎 様 (72歳・男性)
*   紹介元: まなぶ内科クリニック 院長 まなぶ 太郎

【SOAP】
*   S: 2ヶ月前からの乾性咳嗽と、最近出現した血痰を主訴に来院。40年来の喫煙歴（20本/日）。
*   O: 胸部聴診にて右下肺野の呼吸音減弱。胸部X線写真で右下葉に3cm大の境界不明瞭な腫瘤影を認める。
*   A: 喫煙歴のある高齢男性の血痰と胸部X線写真の異常影であり、肺癌（特に扁平上皮癌など）を強く疑う。
*   P: 確定診断（気管支鏡検査など）および、その後の治療（手術、化学療法、放射線療法）が必要と考え、貴科へご紹介させていただく。

【出力形式】
一般的な紹介状のフォーマットに従い、専門医に対する敬意を払った丁寧な言葉遣いで記述してください。
```

### AIの出力例

```
診療情報提供書

メディカル大学病院
呼吸器内科 教授 佐藤 一郎 先生 御侍史

拝啓

時下、先生におかれましては益々ご清祥のこととお慶び申し上げます。
平素より大変お世話になっております。

さて、この度は下記の患者様をご紹介させていただきますので、何卒ご高診賜りますようお願い申し上げます。

患者氏名: 鈴木 次郎 様 (72歳・男性)

【紹介目的】
肺癌疑いの精査・加療

【臨床経過・症状】
S (Subjective):
主訴は2ヶ月前からの乾性咳嗽と血痰です。40年来の喫煙歴（20本/日）があります。

O (Objective):
身体所見では、胸部聴診にて右下肺野の呼吸音減弱を認めます。
本日施行した胸部X線検査におきまして、右下葉に約3cm大の境界不明瞭な腫瘤様陰影を認めました。

A (Assessment):
長期の喫煙歴を有する高齢男性に認められた血痰と胸部異常陰影であり、臨床的に肺癌、特に扁平上皮癌などを強く疑っております。

P (Plan):
つきましては、気管支鏡検査等による確定診断、および病期診断、その後の治療方針の決定につきまして、先生にご高配を賜りたく、ご紹介申し上げる次第です。

ご多忙の折、誠に恐縮ではございますが、何卒よろしくお願い申し上げます。

敬具

2025年12月13日

まなぶ内科クリニック
院長 まなぶ 太郎
（連絡先: XXX-XXX-XXXX）
```

## まとめ

このステップで学んだこと:

✅ 医療連携におけるSOAP形式の重要性
✅ SOAP形式の情報を基に、AIに構造化された紹介状のドラフトを生成させるプロンプト

【重要】
紹介状は、あなたと紹介先医師との専門家同士の対話です。AIが生成したドラフトを基に、あなた自身の臨床的思考、アセスメント、そして紹介先に最も伝えたい依頼事項を、あなた自身の言葉で加筆・修正することが不可欠です。

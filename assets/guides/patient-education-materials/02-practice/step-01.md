## 既存資料の平易化と再構成

### 1. 難解な専門文書の「翻訳」プロセス

医療現場には、診療ガイドライン、治験プロトコル、添付文書など、専門性の高い文書が多数存在します。これらを患者教育資料の原稿として活用する際、AIに「平易化」のタスクを任せることで、大幅な時間短縮が可能です。

平易化のステップ:

1.  原文の入力: 平易化したい専門文書のテキストをAIに入力します。
2.  ターゲット設定: ターゲット層（例：中学生レベルの知識、70代の高齢者）を明確に指定します。
3.  制約条件の追加: 「専門用語の使用禁止」「比喩の使用」「文の長さの制限」などの制約を加えます。
4.  出力の検証: AIの出力に対し、「医学的正確性」と「ターゲット層へのわかりやすさ」の二つの観点から医療従事者が必ず検証します。

### 2. 実用的なプロンプト例：専門用語の平易化

以下のプロンプトは、専門用語を自動的に検出し、ターゲット層に合わせた表現に置き換えることを指示しています。

```prompt
あなたは、医療情報を一般の人々に伝えるための「平易化専門家」です。
以下の[専門文書]を読み、[ターゲット読者]向けに、医学的正確性を保ちつつ、極めてわかりやすい文章に書き直してください。

# ターゲット読者
[例: 糖尿病と診断されたばかりの、60代の会社員]

# 制約条件
1.  専門用語の禁止: 「インスリン抵抗性」「ヘモグロビンA1c」「網膜症」などの専門用語は、必ず日常的な言葉に置き換えてください。
2.  文の長さ: 一文は最大でも40文字以内にしてください。
3.  トーン: 励ましと安心感を与える、前向きなトーンで記述してください。
4.  比喩の使用: 必要に応じて、理解を助けるための比喩（例：「インスリンは鍵」「血糖値はエネルギーの量」）を使用してください。

# 専門文書
[ここに、平易化したい専門文書のテキストを貼り付ける]

# 出力形式
書き直した文章の後に、置き換えた専門用語とその平易化表現のリストを提示してください。
```

### 3. 資料の再構成：Q&A形式への変換

患者教育資料は、一方的な情報提供よりも、患者が抱く疑問に答えるQ&A形式の方が理解されやすい場合があります。AIは、長文の資料を分析し、想定される質問と回答のペアに再構成する作業を得意とします。

#### 実用的なプロンプト例：Q&A形式への変換

```prompt
以下の[説明文]を分析し、患者が抱くであろう疑問を想定したQ&A形式に再構成してください。

# ターゲット読者
[例: 胃潰瘍の治療を受ける20代の学生]

# 質問の選定基準
- 患者が最も不安に感じるであろう点（例：痛み、食事制限、再発）を優先的に質問として抽出してください。
- 質問は、患者が実際に口にするような、素朴で具体的な表現にしてください。

# 回答の作成
- 回答は、簡潔かつ明確に、箇条書きを交えて記述してください。
- 回答の最後に、次の行動（例：医師に相談、食事記録をつける）を促す一言を加えてください。

# 説明文
[ここに、Q&Aに変換したい説明文のテキストを貼り付ける]
```

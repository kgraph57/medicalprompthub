## 多言語対応とアクセシビリティ向上

### 1. AIによる医療文書の多言語翻訳

多様な文化的背景を持つ患者に対応するため、患者教育資料の多言語化は必須です。AI翻訳は、専門用語の統一性や文脈の正確性を保ちながら、迅速かつ低コストで資料を多言語化する強力な手段となります。

AI翻訳の注意点とTips:

*   専門用語の辞書機能: 多くのAI翻訳ツールには、特定の専門用語（例：疾患名、薬剤名）を登録する機能があります。これにより、翻訳後も用語の揺れを防ぎ、正確性を向上させることができます。
*   文脈の保持: 医療文書は文脈が重要です。可能であれば、単語やフレーズ単位ではなく、段落やセクション全体を一度に翻訳させることで、文脈を考慮した自然な翻訳結果を得やすくなります。
*   ネイティブチェック: AI翻訳の結果は、必ず対象言語のネイティブスピーカーまたは専門家による最終チェック（バックトランスレーションを含む）を実施し、誤解を招く表現がないことを確認してください。

### 2. 実用的なプロンプト例：専門用語を考慮した多言語翻訳

以下のプロンプトは、特定の専門用語を保持しつつ、ターゲット言語の文化に合わせた自然な表現で翻訳することを指示しています。

```prompt
あなたは、医療文書の多言語翻訳を専門とするトランスレーターです。
以下の[日本語の患者教育資料]を[ターゲット言語]に翻訳してください。

# ターゲット言語
[例: 英語 (English)]

# 翻訳の制約
1.  専門用語の保持: 以下の[専門用語リスト]にある用語は、翻訳せずそのまま[ターゲット言語]の表記にしてください。
2.  トーンの維持: 原文の「優しく、安心感を与える」トーンを維持してください。
3.  文化的配慮: [ターゲット言語]の文化圏で不適切または誤解を招く可能性のある比喩や表現は、より一般的な表現に置き換えてください。

# 専門用語リスト
[例: インスリン, ヘモグロビンA1c, 〇〇病院]

# 日本語の患者教育資料
[ここに、翻訳したい日本語のテキストを貼り付ける]
```

### 3. アクセシビリティ向上のためのAI活用

AIは、視覚障害や聴覚障害を持つ患者、あるいは認知機能に課題を持つ患者のために、資料のアクセシビリティを向上させる作業にも貢献します。

| アクセシビリティの課題 | AIによる支援内容 | プロンプトのヒント |
| :--- | :--- | :--- |
| 視覚障害 | 画像や図表に対する代替テキスト（Alt Text）の生成 | 「この画像の内容を、視覚障害者向けに簡潔かつ正確に伝える代替テキストを生成してください。」 |
| 認知機能の課題 | 複雑な文章の要約、重要な情報の強調 | 「この文章の最も重要なポイントを3つに絞り、箇条書きで強調してください。」 |
| 聴覚障害 | 動画資料のスクリプト作成、字幕のタイミング調整（外部ツール連携） | 「この動画のスクリプトから、話者の意図を正確に伝えるための簡潔な字幕テキストを作成してください。」 |

#### 実用的なプロンプト例：代替テキストの生成

```prompt
以下の[画像の説明]に基づき、ウェブアクセシビリティ基準を満たす、簡潔で情報量の多い代替テキスト（Alt Text）を生成してください。

# 画像の説明
[例: 糖尿病患者が、食事、運動、服薬の3つの要素をバランス良く管理している様子を示すインフォグラフィック。中央に「血糖コントロール」と書かれている。]

# 代替テキストの要件
- 画像の目的と内容を正確に伝えること。
- 50文字程度に収めること。
- 装飾的な表現は避けること。
```

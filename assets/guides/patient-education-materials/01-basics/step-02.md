## ターゲット層に合わせた「トーン＆マナー」の設定

### 1. ターゲット層の分析とAIへのインプット

患者教育資料の質は、「誰に」伝えるかをどれだけ深く理解しているかにかかっています。AIに効果的な資料を作成させるには、ターゲット層の特性を詳細に分析し、プロンプトに反映させる必要があります。

| ターゲット特性 | 分析のポイント | AIへの具体的な指示例 |
| :--- | :--- | :--- |
| 年齢層 | 漢字の多用、文字サイズ、表現の親しみやすさ | 「対象は70代の高齢者です。漢字には必ずふりがなを振り、ひらがなを多用してください。」 |
| 疾患の知識レベル | 専門用語の許容度、説明の深さ | 「対象は疾患について初めて知る方です。専門用語は一切使わず、比喩を用いて説明してください。」 |
| 感情状態 | 不安、希望、諦めなど、資料を読む際の心理状態 | 「治療への不安が大きい患者向けです。共感的な言葉を多く使い、安心感を最優先したトーンにしてください。」 |
| 文化的背景 | 特定の表現や比喩が適切か | 「特定の宗教的・文化的背景を持つ人々を不快にさせないよう、一般的な表現に留めてください。」 |

### 2. トーン＆マナーを決定づけるプロンプトの要素

AIにトーン＆マナーを正確に理解させるためには、以下の要素をプロンプトに含めることが有効です。

1.  ペルソナ設定: AIに「どのような役割」を演じさせるか（例：ベテランの看護師、優しい医師、親しみやすい栄養士）。
2.  口調の指定: 文末表現（例：です・ます調、だ・である調、〜しましょう）。
3.  禁止事項の指定: 避けるべき表現や専門用語（例：ネガティブな言葉、医学的専門用語）。

### 実用的なプロンプト例：ターゲット層別トーン設定

#### 例1：小児の保護者向け（不安軽減と行動促進）

```prompt
あなたは、小児科病棟で働く経験豊富な看護師です。
[疾患名]の治療を受けるお子さんの保護者向けに、治療の必要性と自宅でのケア方法を説明する文章を作成してください。

# ペルソナ
経験豊富で優しい看護師。

# トーン＆マナー
- 保護者の不安に寄り添い、共感を示す言葉を多用してください。
- 専門用語は避け、日常の言葉でわかりやすく説明してください。
- 文末は「〜しましょう」「〜してくださいね」といった、行動を優しく促す表現で統一してください。
- 治療のメリットを強調し、前向きな気持ちになれるように配慮してください。

# インプット情報
[治療の具体的な内容、自宅での服薬・食事・活動制限に関する指示]
```

#### 例2：高度な知識を持つ慢性疾患患者向け（情報提供と自己管理支援）

```prompt
あなたは、慢性疾患の自己管理を支援する専門医です。
[疾患名]を長年患い、ある程度の医学知識を持つ患者向けに、最新の治療オプション[治療法名]について解説する文章を作成してください。

# ペルソナ
最新の知見に詳しい専門医。

# トーン＆マナー
- 専門用語の使用を許可しますが、必ず簡潔な定義を併記してください。
- 文体は「だ・である」調で、客観的かつ論理的に情報を提示してください。
- 患者が自己判断できるよう、メリットだけでなく、リスクや限界についても公平に記述してください。
- 最終的な意思決定は患者自身にあることを強調してください。

# インプット情報
[最新治療法の作用機序、臨床試験データ、既存治療との比較]
```

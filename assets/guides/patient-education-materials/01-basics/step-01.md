## AIを活用した患者教育資料作成の基本原則

### 1. AI導入のメリットと役割

患者教育資料の作成は、専門的な正確さと患者へのわかりやすさという二つの側面を両立させる必要があり、多くの時間と労力を要します。AI（大規模言語モデル）を導入することで、以下の役割を担わせ、作業を効率化できます。

*   専門情報の平易化: 難解な医学用語や複雑な治療プロセスを、患者の理解度に合わせた平易な言葉に変換する作業を高速化します。
*   ドラフト作成の高速化: 既存の診療ガイドラインや専門文献をインプットとして、資料の骨子や初稿を瞬時に生成し、ゼロベースでの作成負荷を軽減します。
*   一貫性の確保: 複数の資料間で用語の定義や説明のトーンがぶれないよう、スタイルガイドに基づいた一貫した文章作成を支援します。

### 2. 倫理的配慮と最終責任の所在

AIは強力なツールですが、生成された情報には常に誤りや不適切な表現が含まれるリスクがあります。医療現場でAIを活用する際は、以下の倫理的配慮を徹底してください。

*   情報のダブルチェック: AIが生成した医学的情報、特に用量、副作用、治療法に関する記述は、必ず医療従事者が一次情報源（ガイドライン、添付文書など）と照合し、正確性を確認してください。
*   最終責任の認識: AIはあくまで「アシスタント」であり、患者に提供する資料の内容に関する最終的な責任は、資料を作成・提供した医療従事者または医療機関にあります。
*   個人情報保護: 患者のカルテ情報など、機密性の高い個人情報をAIに入力する際は、利用するAIモデルのデータ利用規約を確認し、情報漏洩のリスクがないことを確認してください。

### 3. 資料作成のゴール設定とAIへの明確な指示

AIに質の高い資料を作成させるためには、資料の「目的」と「ターゲット」を明確に定義し、プロンプトで具体的に指示することが不可欠です。

| 項目 | 定義のポイント | AIへの指示例 |
| :--- | :--- | :--- |
| ゴール | 資料を通じて患者にどのような行動変容や理解を促したいか（例：服薬遵守率の向上、検査同意率の向上） | 「この資料の最終目的は、患者が治療の必要性を理解し、毎日欠かさず服薬することです。」 |
| ターゲット | 患者の年齢層、疾患の知識レベル、教育背景（例：70代の高齢者、糖尿病と診断されたばかりの患者） | 「対象読者は、医学知識のない中学生でも理解できるレベルの文章にしてください。」 |
| トーン | 資料の雰囲気（例：安心感を与える、専門的だが親しみやすい、緊急性を伝える） | 「文章のトーンは、不安を煽らず、優しく寄り添うような口調で統一してください。」 |

### 実用的なプロンプト例：基本構成の作成

```prompt
あなたは、医療情報を一般の患者にわかりやすく伝えるための専門ライターです。
以下の情報を基に、患者教育資料の構成案と各セクションの要点を箇条書きで作成してください。

# 資料の目的
[例: 新しい抗がん剤治療の概要と副作用、治療期間中の生活上の注意点を伝える]

# ターゲット読者
[例: 60代、初めてがん治療を受ける患者とその家族]

# 資料のトーン
[例: 専門的な正確さを保ちつつ、患者の不安を和らげる、希望を持てるような前向きなトーン]

# インプット情報
[例: 治療プロトコルの概要、主要な副作用とその対処法、連絡先情報]

# 出力形式の指示
- 構成案は、導入、治療の概要、副作用と対処法、生活の注意点、Q&A、緊急連絡先の6つのセクションに分けてください。
- 各セクションの要点は、ターゲット読者が最も知りたい情報に絞り込んでください。
```

# 医療AIワークフローガイド：患者教育資料作成

## イントロダクション

### 目的

本ガイドは、医療AI（特に大規模言語モデル, LLM）を活用し、患者の理解度と治療への参加意欲を最大化するための教育資料を効率的かつ効果的に作成するワークフローを提供します。複雑な医療情報を平易な言葉に変換し、個々の患者のニーズに合わせたカスタマイズを迅速に行うことで、医療従事者の負担を軽減し、質の高いインフォームド・コンセントを支援します。

### AIの役割
## 患者教育資料作成の全体フロー

AIを活用した患者教育資料作成のワークフローを視覚的に理解しましょう。

![患者教育資料作成フロー](/assets/guides/patient-education-materials/images/patient_education_materials_workflow.png)

---


AIは、以下の点で患者教育資料作成プロセスに革新をもたらします。

1.  情報変換: 専門的な医学用語や複雑な治療プロトコルを、患者のリテラシーレベルに合わせて平易な言葉に変換します。
2.  構成案作成: 視覚的に分かりやすいインフォグラフィックや図解の構成案を迅速に生成し、デザインの骨子を提供します。
3.  個別化: 患者の背景情報（年齢、既往歴、生活習慣など）に基づき、個別化されたアドバイスやFAQを生成します。
4.  多言語対応: 資料の多言語翻訳を瞬時に行い、多様な文化的背景を持つ患者への対応を可能にします。

### まとめ

AIは、単なる情報提供ツールではなく、個別化、視覚化、行動変容促進の3つの要素を統合した、質の高い患者教育資料を迅速に作成するための強力なコパイロットとなります。

## 基本編：情報の平易化と構造化（2ステップ）

### Step 1: 医療情報の平易化と要約

#### 具体的なステップ

1.  専門情報の入力: 診断名、治療方針、検査結果など、患者に伝えたい専門的な医療情報をAIに入力します。
2.  平易化の指示: 対象とする患者層（例：中学生レベル、高齢者、特定の疾患の知識がない人）を指定し、専門用語を避け、比喩や日常的な言葉を用いて説明するように指示します。
3.  強調点の指定: 患者にとって特に重要な情報（例：服薬の重要性、合併症のリスク）を明確に強調するように求めます。

#### プロンプト例

```
「特発性肺線維症の診断を受け、抗線維化薬であるピルフェニドンによる治療を開始します。この疾患と治療法について、専門用語を避け、中学生でも理解できるように説明してください。特に、薬を毎日欠かさず飲むことの重要性を、具体的な比喩（例：火事を食い止める消火活動）を用いて強調してください。」
```

#### AIの出力例

> 特発性肺線維症とピルフェニドン治療について
>
> 特発性肺線維症とは？
>
> あなたの肺は、酸素を取り込むための柔らかいスポンジのような組織でできています。この病気は、そのスポンジが何らかの原因で硬く、厚くなってしまう病気です。一度硬くなった肺は元には戻りません。例えるなら、柔らかい布が、時間が経つにつれて硬い革のように変化してしまうようなものです。
>
> ピルフェニドンによる治療（消火活動）
>
> ピルフェニドンというお薬は、硬くなるスピードをできるだけ遅くするためのものです。これは、肺の中で起こっている「硬くなる火事」を食い止める「消火活動」のようなものです。この火事は一度つくと完全に消すのは難しいですが、消火活動を続けることで、火の勢いを弱め、燃え広がるのを遅らせることができます。
>
> 最も大切なこと
>
> 消火活動は、毎日続けることが大切です。薬を飲み忘れると、消火活動が中断され、火事が再び勢いを増してしまうかもしれません。毎日決まった時間に、忘れずに薬を飲むことが、あなたの肺を守るための最も重要なステップです。

#### まとめ

AIは、情報の正確性を保ちつつ、患者の認知レベルに合わせた平易な言葉と効果的な比喩を生成することで、医療情報の理解の第一歩を強力にサポートします。

### Step 2: 質問と回答のFAQ形式化

#### 具体的なステップ

1.  患者の不安予測: 疾患や治療に関する患者の一般的な不安や疑問（痛み、費用、生活制限、予後など）を予測します。
2.  FAQの生成指示: 予測した疑問に基づき、簡潔で分かりやすい回答を生成するようにAIに指示します。
3.  トーンの調整: 回答が共感的で安心感を与えるトーンになるように指定します。

#### プロンプト例

```
「特発性肺線維症の治療を開始する患者が、特に不安に感じやすい以下の5つの質問に対して、共感的で安心感を与えるトーンで、簡潔な回答を作成してください。
1. この病気は治りますか？
2. 薬の副作用はありますか？
3. 今まで通りの生活はできますか？
4. 家族にうつることはありますか？
5. 治療費はどのくらいかかりますか？」
```

#### AIの出力例

| 質問 (Q) | 回答 (A) |
| :--- | :--- |
| Q1. この病気は治りますか？ | A. 残念ながら、一度硬くなった肺を元に戻す治療法はまだありません。しかし、今から始めるお薬は、病気の進行を遅らせることが科学的に証明されています。私たちは、あなたの生活の質を維持するために最善を尽くします。 |
| Q2. 薬の副作用はありますか？ | A. お薬には、吐き気や下痢などの副作用が出ることがあります。しかし、ほとんどの場合、飲み続けるうちに体が慣れてきます。もしつらい症状があれば、我慢せずにすぐに教えてください。症状を和らげるためのお薬もあります。 |
| Q3. 今まで通りの生活はできますか？ | A. 治療を始めると、少し生活に変化があるかもしれません。しかし、無理のない範囲で、今まで通り趣味や活動を続けることをお勧めします。息切れが気になる場合は、リハビリテーションの専門家がサポートしますのでご安心ください。 |
| Q4. 家族にうつることはありますか？ | A. 特発性肺線維症は、人にうつる病気ではありませんのでご安心ください。ご家族は、あなたを支える大切な存在です。 |
| Q5. 治療費はどのくらいかかりますか？ | A. 高額な治療になる可能性がありますが、高額療養費制度など、医療費の負担を軽減する公的な制度があります。受付や相談窓口で詳しくご説明しますので、ご心配なくお尋ねください。 |

#### まとめ

AIによるFAQの生成は、患者目線で疑問を先回りし、信頼感と安心感を高めるための効率的なツールです。

## 実践編：視覚化と個別化の強化（3ステップ）

### Step 3: 視覚資料（インフォグラフィック）の構成案作成

#### 具体的なステップ

1.  視覚化の対象決定: 治療プロセス、疾患のメカニズム、服薬スケジュールなど、視覚化すべき複雑な情報を決定します。
2.  構成要素の指示: インフォグラフィックのタイトル、主要なセクション、使用する色、アイコンのアイデア、情報の流れをAIに指示します。
3.  デザイン原則の組み込み: 「シンプルに」「ステップを追って」「安心感を与えるトーンで」など、デザインの原則をプロンプトに含めます。

#### プロンプト例

```
「特発性肺線維症の治療プロセスを説明するためのインフォグラフィックの構成案を作成してください。以下の要素を含めてください。
1. タイトル: 「あなたの肺を守るための3つのステップ」
2. 主要な3つのステップ: (1) 正しい診断と理解、(2) 薬による進行の抑制、(3) 呼吸リハビリと生活改善
3. 各ステップで用いるべきアイコンのアイデア（例：虫眼鏡、盾、葉っぱ）
4. 全体のカラートーンは、安心感と清潔感を与える青と緑のグラデーションで。」
```

#### AIの出力例

| 要素 | 内容 | 目的 |
| :--- | :--- | :--- |
| タイトル | あなたの肺を守るための3つのステップ | 患者の行動を促すキャッチーな表現 |
| ステップ 1 | 正しい診断と理解 | アイコン: 虫眼鏡（🔍）またはパズルのピース。説明: 病気の名前と、それが肺で何を引き起こしているのかを理解する。 |
| ステップ 2 | 薬による進行の抑制 | アイコン: 盾（🛡️）またはストップウォッチ。説明: 毎日のお薬で、病気の進行を遅らせる「防御」の役割を果たす。 |
| ステップ 3 | 呼吸リハビリと生活改善 | アイコン: 葉っぱ（🌿）または歩いている人。説明: 息切れを和らげ、活動的な生活を送るための運動と工夫。 |
| カラートーン | メインカラー: 信頼感のある青。サブカラー: 健康と再生を示す緑。 | 視覚的な安心感の提供 |
| フォント | 大きく、読みやすいゴシック体。 | 高齢者にも配慮したアクセシビリティ |

#### まとめ

AIは、視覚化の骨子を論理的に整理し、デザインの方向性を具体的に提案することで、デザイナーや医療従事者が資料作成にかける時間を大幅に短縮します。

### Step 4: 多様なリテラシーレベルへの対応（リーダビリティ調整）

#### 具体的なステップ

1.  ベーステキストの準備: Step 1で作成した平易化された説明文を用意します。
2.  対象層の指定: 高齢者、外国語話者、低リテラシー層など、特定の対象層を指定します。
3.  調整指示: 文の長さ、使用する漢字の頻度、文法構造、多言語翻訳などをAIに指示します。

#### プロンプト例

```
「以下の説明文を、(A)75歳以上の高齢者向け（漢字を極力減らし、文を短く区切り、ひらがなを多用）と、(B)英語を母国語とする患者向け（専門用語を避け、自然な英語に翻訳）の2つのバージョンに書き直してください。

[Step 1で作成したAI出力例のテキストを挿入]
」
```

#### AIの出力例（抜粋）

> (A) 高齢者向けバージョン（日本語）
>
> あなたの はい は、やわらかい すぽんじ のようなものです。この びょうき は、その すぽんじ が かたく なってしまうものです。お くすり は、この かたく なるのを おそく するためのものです。まいにち、わすれずに のむ ことが、いちばん たいせつ です。
>
> (B) 英語バージョン (English)
>
> What is Idiopathic Pulmonary Fibrosis (IPF)?
>
> Your lungs are like soft sponges that help you breathe. IPF is a disease where this sponge tissue becomes stiff and thick. The medicine, Pirfenidone, acts like a fire extinguisher to slow down this stiffening process. It is crucial that you take your medicine every day to keep the "fire" from spreading quickly.

#### まとめ

AIのリーダビリティ調整機能は、アクセシビリティを劇的に向上させ、インクルーシブな医療を実現するための基盤となります。

### Step 5: 行動変容を促す具体的なアクションプランの組み込み

#### 具体的なステップ

1.  行動目標の設定: 患者に期待する具体的な行動（例：服薬遵守、禁煙、運動、食事管理）を設定します。
2.  行動計画の生成指示: 設定した目標を達成するための、具体的で実行可能なスモールステップの行動計画をAIに作成させます。
3.  動機付けの要素追加: 計画に自己効力感を高めるためのポジティブな言葉や、進捗を記録する要素を追加します。

#### プロンプト例

```
「特発性肺線維症の患者が、服薬遵守と呼吸リハビリを習慣化するための、具体的な1週間の行動計画を作成してください。計画には、チェックリスト形式で進捗を記録できる要素を含め、ポジティブな動機付けの言葉を添えてください。」
```

## 応用編：高度な個別化と統合（2ステップ）

### Step 6: 患者のパーソナリティと学習スタイルに合わせたトーン調整

#### 具体的なステップ

1.  パーソナリティの特定: 患者の性格や学習スタイル（例：論理的、感情的、視覚的、聴覚的）に関する情報をAIに入力します。
2.  トーンと形式の調整指示: 特定のパーソナリティに響くような言葉遣い、トーン、情報の提示形式（例：データ重視、ストーリー重視）をAIに指示します。

#### プロンプト例

```
「以下の説明文を、(A)論理的でデータ重視の患者向け（具体的な数値、統計、メカニズムを強調）と、(B)感情的で共感を求める患者向け（体験談、安心感、サポート体制を強調）の2つのトーンで書き直してください。

[Step 1で作成したAI出力例のテキストを挿入]
」
```

#### AIの出力例（抜粋）

> (A) 論理的・データ重視トーン
>
> ピルフェニドンは、線維化に関わるTGF-βなどのサイトカインを抑制することで、肺機能低下の年間速度を約50%遅延させることが大規模臨床試験で示されています[1]。この治療は、科学的根拠（Evidence-Based Medicine）に基づいた標準治療であり、継続的な服薬が統計的に有意な予後改善に直結します。
>
> (B) 感情的・共感トーン
>
> この病気と向き合うことは、とても不安なことだと思います。あなたは一人ではありません。私たち医療チームは、あなたのつらい気持ちに寄り添い、最善のサポートを提供します。お薬は、あなたの未来の安心を守るためのものです。もし心が折れそうになったら、いつでも私たちに話してください。

#### まとめ

AIは、心理学的なアプローチを組み込み、患者のパーソナリティに合わせたコミュニケーションを実現することで、教育資料の受容度と効果を最大化します。

### Step 7: 外部リソースとの統合と最新情報の自動更新

#### 具体的なステップ

1.  外部リソースの指定: 信頼できる患者会、公的機関のウェブサイト、最新の臨床試験情報などのURLをAIに提供します。
2.  統合指示: 資料の最後に、これらのリソースへの簡潔な紹介文とリンクを組み込むように指示します。
3.  更新性の確保: AIに、資料の定期的な見直しと、最新の治療ガイドラインに基づく自動更新の仕組みを提案させます。

#### プロンプト例

```
「作成した患者教育資料の最後に、以下の信頼できる外部リソースへのリンクと、それぞれの簡潔な紹介文を追記してください。また、この資料を最新の状態に保つための、AIを活用した更新ワークフローのアイデアを提案してください。
- リソース1: [URL: https://www.mhlw.go.jp/](厚生労働省 難病情報センター)
- リソース2: [URL: https://www.resp-reha.jp/](日本呼吸ケア・リハビリテーション学会)
」
```

#### AIの出力例（抜粋）

> 外部リソースとサポート
>
> 治療に関するさらに詳しい情報や、同じ病気と向き合う仲間との交流を求める方のために、信頼できるリソースをご紹介します。
>
> - 厚生労働省 難病情報センター [1]: 難病に関する公的な情報、医療費助成制度など、信頼できる情報を確認できます。
> - 日本呼吸ケア・リハビリテーション学会 [2]: 呼吸リハビリテーションに関する専門的な情報や、自宅でできる運動のヒントを得られます。
>
> AIを活用した資料更新ワークフローの提案
>
> この資料の正確性と最新性を維持するため、以下のワークフローを推奨します。
>
> 1.  ガイドライン監視: 3ヶ月に一度、AIが「特発性肺線維症 治療ガイドライン 最新」などのキーワードで自動検索を実行します。
> 2.  差分抽出: 検索結果と現行資料を比較し、治療推奨度の変更点や新規薬剤の承認情報などの差分を抽出します。
> 3.  改訂案の自動生成: 抽出された差分に基づき、AIが資料の改訂が必要なセクションの候補と改訂案を自動生成します。
> 4.  最終確認: 医療従事者がAIの改訂案を最終確認し、承認することで、常に最新かつ正確な資料を患者に提供します。

#### まとめ

AIは、情報提供の枠を超え、最新性・信頼性を担保するための持続可能な運用ワークフローを設計することで、患者教育の質を長期的に向上させます。

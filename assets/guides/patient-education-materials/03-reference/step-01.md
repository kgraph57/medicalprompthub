## AIによる資料の自動評価と改善

### 1. 患者教育資料の評価基準とAIの役割

質の高い患者教育資料は、単に情報が正確であるだけでなく、「読みやすさ」「理解しやすさ」「行動へのつながりやすさ」が重要です。AIは、これらの評価基準に基づき、資料を客観的に分析し、改善点を提案する「仮想の編集者」として機能します。

AIによる評価の観点:

1.  リーダビリティ（読みやすさ）: 文の長さ、専門用語の頻度、漢字の使用率などを分析し、Flesch-Kincaidなどの指標に基づいた評価を行います。
2.  理解度: ターゲット層の知識レベルを想定し、説明の論理的な流れや、重要な情報の強調が適切かを評価します。
3.  行動変容への影響: 資料が患者に具体的な行動（例：服薬、運動、受診）を促すための明確な指示や動機付けを含んでいるかを評価します。

### 2. 実用的なプロンプト例：リーダビリティと改善点の評価

以下のプロンプトは、作成した資料をAIに評価させ、具体的な改善提案を引き出すためのものです。

```prompt
あなたは、患者教育資料の専門的な編集者です。
以下の[患者教育資料]を、[ターゲット読者]の視点から厳しく評価し、改善点を提案してください。

# ターゲット読者
[例: 75歳の心不全患者。小学校卒業程度の学力]

# 評価基準
1.  リーダビリティ: 文章がターゲット読者にとって平易で、一文が長すぎないか（理想は30文字以内）。
2.  専門用語: 専門用語が適切に言い換えられているか、またはふりがなが振られているか。
3.  動機付け: 治療の継続や生活習慣の改善につながるような、前向きな動機付けが含まれているか。
4.  レイアウト: 箇条書きや見出しが効果的に使われ、視覚的に読みやすいか。

# 患者教育資料
[ここに、評価させたい資料のテキストを貼り付ける]

# 出力形式
- 総合評価（A, B, Cの3段階）
- 改善が必要な点（箇条書きで3点以上）
- 改善提案（具体的な文章の修正例を含む）
```

### 3. A/Bテスト用バリエーションの自動生成

患者の反応は多様であり、一つの資料が全ての人に最適とは限りません。AIを活用して、同じ内容でもトーンや表現を変えた複数のバリエーション（A/Bテスト用）を迅速に生成し、実際に患者の反応を比較することで、より効果的な資料を特定できます。

#### 実用的なプロンプト例：トーン違いのバリエーション生成

```prompt
以下の[資料の核となる情報]に基づき、トーンが異なる2つのバリエーション（A案、B案）を作成してください。

# 資料の核となる情報
[例: 降圧剤の服薬を継続することの重要性]

# ターゲット読者
[例: 40代の働き盛りの高血圧患者]

# A案のトーン
- 危機感の強調: 服薬を怠った場合の具体的なリスク（脳卒中、心筋梗塞）を明確に伝え、行動を促す。
- 文体: 専門的で論理的な「だ・である」調。

# B案のトーン
- 安心感と共感: 服薬の負担に理解を示しつつ、服薬継続による将来の健康的な生活をイメージさせる。
- 文体: 親しみやすい「です・ます」調。

# 出力形式
A案とB案をそれぞれ明確な見出しを付けて提示してください。
```

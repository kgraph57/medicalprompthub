## 症状から考えられる希少疾患をリストアップする

### 1. 鑑別診断におけるAIの活用

希少疾患の診断は、非典型的な症状や所見から、膨大な疾患リストの中から可能性のあるものを絞り込むプロセスであり、極めて困難です。AIは、その広範な知識ベースとパターン認識能力により、医療従事者の鑑別診断プロセスを強力にサポートします。

AIに鑑別診断の提案を求める際は、単なる症状の羅列ではなく、患者の背景情報、具体的な臨床所見、検査結果などを正確かつ網羅的に記述することが、質の高い提案を得るための鍵となります。

### 2. 実践的なプロンプト例：鑑別診断リストの作成

以下のプロンプトは、AIに臨床医の視点を持たせ、具体的な症例情報に基づいて鑑別診断リストを作成させるためのものです。

#### プロンプト例

```
あなたは、診断困難な症例を専門とするベテランの臨床医です。
以下の患者情報に基づき、鑑別診断のリストに含めるべき希少疾患を、可能性の高い順に5つ挙げてください。
それぞれの疾患について、この症例のどの所見が鑑別根拠となるかを具体的に示してください。

【患者情報】
- 年齢: [年齢]歳
- 性別: [性別]
- 主訴: [主訴]
- 臨床所見: [具体的な症状1]、[具体的な症状2]、[非典型的な身体所見]
- 検査所見: [特異的な検査値異常]、[画像所見]
- 家族歴: [特記事項]
```

プロンプトのポイント:
- 詳細な情報提供: 鑑別診断の精度は、提供する情報の質に比例します。非典型的な所見や特異的な検査所見を具体的に記述しましょう。
- 根拠の要求: 単に疾患名を挙げるだけでなく、「鑑別の根拠」を求めることで、AIの思考プロセスを可視化し、臨床的な妥当性を評価しやすくします。

### 3. Tips：AIの回答は「仮説」として扱う

AIが提示する鑑別診断リストは、あくまで「仮説」であり、診断の確定ではありません。

| AIの役割 | 臨床医の役割 |
| :--- | :--- |
| 膨大な情報からの迅速な候補リストアップ | 提示された候補の臨床的妥当性の評価 |
| 既知のパターンに基づく関連性の指摘 | 患者の個別性、地域性、最新の知見との照合 |
| 診断の「ヒント」の提供 | 最終的な診断の責任と判断 |

AIの提案を参考に、次の検査や専門医へのコンサルトの方向性を定めるなど、臨床判断の補助として活用してください。

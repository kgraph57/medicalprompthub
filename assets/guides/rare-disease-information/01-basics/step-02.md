## 信頼できる情報源をAIに尋ねる

### 1. 信頼性の高い情報源の特定

AIは非常に便利ですが、その回答は学習データに依存しており、常に最新かつ正確であるとは限りません。特に医療情報においては、情報の信頼性が極めて重要です。希少疾患に関する情報を収集する際、AIの回答を鵜呑みにするのではなく、情報源（ソース）を特定し、その信頼性を確認することが必須となります。

AIは、特定の疾患に関する信頼性の高いデータベースやガイドラインを迅速にリストアップするのに役立ちます。

| 希少疾患の主要な情報源の例 | 特徴 |
| :--- | :--- |
| Orphanet | 欧州を中心とした希少疾患と孤児薬に関するポータルサイト。疾患の定義、疫学、遺伝子、診断、管理に関する情報を提供。 |
| GARD (Genetic and Rare Diseases Information Center) | 米国NIHの一部門。患者と医療従事者向けに、希少疾患に関する情報を提供。 |
| OMIM (Online Mendelian Inheritance in Man) | 遺伝性疾患と遺伝子に関する包括的なデータベース。 |
| 難病情報センター | 日本の難病に関する公的な情報提供サイト。 |

### 2. 実践的なプロンプト例：情報源の特定と特徴の比較

AIに、特定の疾患に関する信頼性の高い情報源を挙げさせ、それぞれの特徴を比較させることで、どの情報源から詳細な情報を得るべきかを判断できます。

#### プロンプト例

```
希少疾患「[疾患名]」に関する、最も信頼性の高い情報源やデータベースを3つ挙げてください。
それぞれの情報源について、以下の項目を簡潔に説明してください。
1. 運営主体（例：公的機関、学会）
2. 提供情報の種類（例：疫学、診断基準、治療法、遺伝子情報）
3. 特徴（例：欧州中心、遺伝性疾患に特化、日本の公的情報）
```

プロンプトのポイント:
- 比較項目の指定: 運営主体や提供情報の種類など、比較に必要な項目を具体的に指定することで、整理された回答が得られます。
- 「最も信頼性の高い」: AIに情報のキュレーション（選別）を促し、質の低い情報源を除外させます。

### 3. Tips：情報源のクロスチェックと検証

AIが提示した情報源は、必ずご自身で公式サイトにアクセスし、以下の点を検証してください。

1.  情報の更新頻度: 最終更新日が新しいか。
2.  運営主体: 公的機関、信頼できる学会、大学など、中立的で専門性の高い組織が運営しているか。
3.  情報の一次性: 論文やガイドラインの原文（一次情報）にアクセスできるか。

AIの回答は「情報源へのナビゲーター」として活用し、最終的な情報の確認は、医療従事者自身の責任において、信頼できる情報源で行う習慣をつけましょう。

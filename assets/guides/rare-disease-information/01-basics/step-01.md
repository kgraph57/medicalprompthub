## AIを活用した希少疾患情報収集の第一歩

### 1. 希少疾患情報収集におけるAIの役割

希少疾患（Rare Disease）に関する情報は、その名の通り情報量が少なく、特定の専門機関やデータベースに散在していることが多くあります。また、最新の研究や治療法は海外の論文で発表されることが多いため、情報の断片化、言語の壁、最新情報の追跡の困難さが、医療従事者にとって大きな課題となっています。

大規模言語モデル（LLM）をはじめとするAIは、これらの課題を解決するための強力なツールとなります。AIの主な特性として、大量のテキストデータの高速処理、要約・抽出能力、そして高精度な翻訳能力が挙げられます。

| 課題 | AIによる解決策 | AIの活用例 |
| :--- | :--- | :--- |
| 情報の断片化 | 関連情報の統合と整理 | 複数の情報源から得られた情報を一貫性のあるレポートにまとめる |
| 言語の壁 | 専門性の高い医療翻訳 | 海外の最新論文やガイドラインを迅速かつ正確に日本語に翻訳・要約する |
| 最新情報の追跡 | 重要な情報の抽出と要約 | 特定の疾患に関する最新の研究動向を定期的にチェックし、要点のみを抽出する |

### 2. 実践的なプロンプト例：基本情報の迅速な把握

まずは、特定の希少疾患の基本情報を迅速かつ正確に把握するためのプロンプトから始めましょう。AIに具体的な疾患名を指定することで、情報収集の焦点を絞り、より質の高い回答を得ることができます。

#### プロンプト例

```
あなたは経験豊富な希少疾患の専門医です。
希少疾患である「[疾患名]」について、現在知られている主な症状と診断基準を、日本の医療従事者向けに簡潔かつ網羅的にまとめてください。
情報源として、信頼性の高い公的機関（例：Orphanet, 難病情報センター）の情報を参照してください。
```

プロンプトのポイント:
- 役割指定（ペルソナ）: 「経験豊富な希少疾患の専門医」と指定することで、回答の質と専門性を高めます。
- 対象読者の指定: 「日本の医療従事者向けに」とすることで、日本の医療環境に合わせた情報（例：日本の難病指定の有無など）を考慮させます。
- 情報源の指定: 信頼性の高い情報源を参照するよう指示することで、回答の正確性を担保します。

### 3. Tips：具体的な疾患名の指定

AIは一般的な情報には強いですが、希少疾患に関する情報は特定のキーワードに依存します。プロンプトに具体的な疾患名（例：「ハンチントン病」ではなく「ハンチントン病」）や関連する遺伝子名、臨床的なサブタイプを含めることで、AIはより深く、専門的なデータベースから情報を引き出すことができます。

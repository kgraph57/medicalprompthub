## プロンプトテンプレート集と応用例

### 1. 希少疾患情報収集のためのプロンプトテンプレート集

これまでのステップで紹介したプロンプトを、目的に応じてカスタマイズしやすいテンプレートとしてまとめます。これらのテンプレートをベースに、個々の症例や目的に合わせて、必要な情報を追加・変更して活用してください。

| 目的 | テンプレート |
| :--- | :--- |
| 基本情報収集 | `希少疾患「[疾患名]」について、主な症状と診断基準を、信頼できる情報源を参照して簡潔にまとめてください。` |
| 情報源の特定 | `希少疾患「[疾患名]」に関する最も信頼性の高いデータベースを3つ挙げ、それぞれの特徴（運営主体、提供情報）を比較してください。` |
| 鑑別診断支援 | `[年齢]歳、[性別]の患者で、[症状1]、[症状2]、[検査所見]が見られます。鑑別診断のリストに含めるべき希少疾患を5つ挙げ、根拠を示してください。` |
| 最新治療法調査 | `希少疾患「[疾患名]」に対する過去3年以内の最新の治療アプローチと、現在進行中の第2相/第3相臨床試験の概要をまとめてください。` |
| 海外情報要約 | `以下の英文ガイドラインの要点を、日本の医療従事者向けに分かりやすく日本語で要約してください。[英文テキスト]` |
| 患者向け説明文 | `希少疾患「[疾患名]」について、[対象者の属性]に理解できるように、病気の原因、症状、治療法を、共感的なトーンで説明する文章を作成してください。` |

### 2. 複数のプロンプトを組み合わせた応用例

AIの真価は、複数のステップを組み合わせたワークフローとして活用することで発揮されます。

応用ワークフロー例：診断から説明まで

1.  鑑別診断プロンプトで、非典型的な症状から可能性の高い希少疾患（例：疾患A）をリストアップする。
2.  最新治療法調査プロンプトで、疾患Aの最新の治療法と治験情報を収集する。
3.  海外情報要約プロンプトで、疾患Aに関する海外の最新ガイドラインの要点を把握する。
4.  患者向け説明文プロンプトで、収集した情報を基に、患者とその家族に説明するための資料を作成する。

この一連の流れをAIのサポートのもとで迅速に行うことで、診断から治療方針の決定、患者へのインフォームド・コンセントまでの時間を大幅に短縮できます。

### 3. 医療現場でAIを利用する上での倫理的配慮

AIは強力なツールですが、医療現場での利用には、以下の倫理的・実務的な配慮が不可欠です。

| 配慮事項 | 具体的な行動 |
| :--- | :--- |
| 個人情報の保護 | 絶対に患者の氏名、生年月日、ID、詳細な住所など、個人を特定できる情報をAIに入力しないでください。症例情報は匿名化・抽象化して利用します。 |
| 情報の正確性 | AIの回答は、必ず信頼できる情報源（一次情報）と照合し、医療従事者自身がその正確性を検証してから利用します。AIは診断を下すものではありません。 |
| AIの限界の認識 | AIは、最新の知見や未発表の研究、個々の患者の複雑な背景を完全に理解することはできません。AIの回答は参考情報として扱い、最終的な臨床判断は医療従事者が行います。 |
| 責任の所在 | AIの利用によって生じた結果に対する最終的な責任は、利用した医療従事者にあります。 |

AIを「優秀なアシスタント」として最大限に活用しつつ、医療の根幹である人間的な判断と責任を常に持ち続けることが、これからの医療従事者に求められます。

# ステップ2: AIプロンプトの基本を理解する

## AIプロンプトとは？

AIプロンプトとは、AIに指示を出すための「質問文」や「依頼文」のことです。プロンプトの質が、AIの出力の質を大きく左右します。

### 良いプロンプトの条件

1. 具体的: 何をしてほしいか明確に
2. 構造化: 情報を整理して提示
3. 文脈を提供: 背景情報を含める
4. 出力形式を指定: どのような形式で出力してほしいか

---

## 基本的なプロンプトテンプレート

### テンプレート1: 症例発表スライド生成

```
あなたは経験豊富な臨床医です。以下の症例情報から、カンファレンス発表用のスライド構成を提案してください。

【症例情報】
- 年齢・性別: [情報を入力]
- 主訴: [情報を入力]
- 現病歴: [情報を入力]
- 身体所見: [情報を入力]
- 検査所見: [情報を入力]
- 診断: [情報を入力]
- 治療: [情報を入力]
- 経過: [情報を入力]

【出力形式】
以下の構成でスライドの内容を提案してください：
1. タイトルスライド
2. 症例サマリー
3. 現病歴
4. 身体所見
5. 検査所見
6. 鑑別診断（優先順位付き）
7. 診断根拠
8. 治療方針
9. 経過
10. Take-home message

各スライドには、箇条書きで具体的な内容を記載してください。
```

## 実践例：プロンプトの使い方

### 例1: 上腸間膜動脈血栓症の症例

入力するプロンプト:

```
あなたは経験豊富な臨床医です。以下の症例情報から、カンファレンス発表用のスライド構成を提案してください。

【症例情報】
- 年齢・性別: 80歳、女性
- 主訴: 急性腹痛
- 現病歴: 来院3時間前から突然の腹痛が出現。心窩部から全腹部に広がる。嘔吐2回。下痢なし。
- 既往歴: 心房細動（ワルファリン内服中断中）、高血圧、脂質異常症
- 身体所見: 
  - バイタル: 体温37.8°C、血圧100/60mmHg、脈拍110/分（不整）
  - 腹部: 全腹部に圧痛あり、筋性防御なし、腸蠕動音減弱
- 検査所見:
  - 血液: WBC 15,000、CRP 8.5、Lactate 4.2、D-dimer 5.8
  - CT: 上腸間膜動脈に血栓、小腸壁の造影不良、腹水あり
- 診断: 上腸間膜動脈血栓症
- 治療: 緊急開腹術（血栓除去 + 腸管切除）
- 経過: 術後経過良好、術後14日目に退院

【出力形式】
以下の構成でスライドの内容を提案してください：
1. タイトルスライド
2. 症例サマリー
3. 現病歴
4. 身体所見
5. 検査所見
6. 鑑別診断（優先順位付き）
7. 診断根拠
8. 治療方針
9. 経過
10. Take-home message

各スライドには、箇条書きで具体的な内容を記載してください。
```

### AIの出力例

```
【スライド1: タイトルスライド】
- タイトル: 80歳女性、急性腹痛で来院
- サブタイトル: 診断: 上腸間膜動脈血栓症
- 発表者: [あなたの名前]
- 発表日: 2025年12月13日

【スライド2: 症例サマリー】
- 年齢・性別: 80歳、女性
- 主訴: 急性腹痛
- 現病歴: 来院3時間前から突然の腹痛が出現
- 診断: 上腸間膜動脈血栓症
- 転帰: 緊急手術施行、術後経過良好

【スライド3: 現病歴】
- X日 15:00: 突然の腹痛が出現（心窩部→全腹部）
- X日 16:00: 嘔吐2回、下痢なし
- X日 18:00: 症状増悪し救急搬送
- 既往歴: 心房細動（ワルファリン内服中断中）、高血圧、脂質異常症

【スライド4: 身体所見】
バイタルサイン:
- 体温: 37.8°C
- 血圧: 100/60 mmHg
- 脈拍: 110回/分（不整）
- SpO2: 96%（室内気）

腹部所見:
- 全腹部に圧痛（+）、筋性防御（-）
- 腸蠕動音: 減弱
- 直腸診: 血便（-）

【スライド5: 検査所見】
血液検査:
- WBC: 15,000 /μL ↑
- CRP: 8.5 mg/dL ↑
- Lactate: 4.2 mmol/L ↑
- D-dimer: 5.8 μg/mL ↑

CT所見:
- 上腸間膜動脈に血栓（+）
- 小腸壁の造影不良
- 腹水（+）

【スライド6: 鑑別診断】
1. 上腸間膜動脈血栓症 ★
   - 心房細動、急性腹痛、Lactate上昇
2. 腸閉塞
   - 腸蠕動音減弱、嘔吐
   → CT で血栓を確認し除外
3. 急性膵炎
   - 腹痛、嘔吐
   → アミラーゼ正常で除外
4. 腸管穿孔
   - 腹痛、炎症反応
   → 筋性防御なし、CTで除外

【スライド7: 診断根拠】
臨床所見:
- 心房細動（血栓リスク）
- 急性腹痛（腹部所見に乏しい）

検査所見:
- Lactate上昇（腸管虚血）
- D-dimer上昇（血栓形成）
- CT: 上腸間膜動脈血栓

確定診断:
- 造影CT で上腸間膜動脈血栓を確認

【スライド8: 治療方針】
治療選択肢:
1. 保存的治療（抗凝固療法）
2. 血管内治療（血栓溶解療法）
3. 外科的治療（血栓除去術）

選択した治療:
- 緊急開腹術（血栓除去 + 腸管切除）

理由:
- 腸管壊死の可能性が高い
- 発症から時間が経過
- 保存的治療では改善困難

【スライド9: 経過】
術後経過:
- 術後1日目: ICU管理、バイタル安定
- 術後3日目: 一般病棟へ転棟
- 術後7日目: 経口摂取開始
- 術後14日目: 退院

合併症:
- なし

【スライド10: Take-home message】
1. 心房細動患者の急性腹痛では、上腸間膜動脈血栓症を疑う

2. 腹部所見に乏しくても、Lactate上昇があれば腸管虚血を考慮

3. 早期診断・早期治療が予後を左右する
```

## プロンプトのカスタマイズ

### カスタマイズ1: 発表時間を指定する

```
【追加の指示】
発表時間は10分です。各スライドの説明時間の目安も提案してください。
```

### カスタマイズ2: 聴衆のレベルを指定する

```
【追加の指示】
聴衆は内科研修医です。専門用語は使用しますが、基本的な説明も含めてください。
```

### カスタマイズ3: 特定のポイントを強調する

```
【追加の指示】
この症例では、早期診断の重要性を強調したいです。診断の遅れによるリスクについても言及してください。
```

## プロンプトエンジニアリングのコツ

### 1. 役割を明確にする

❌ 悪い例:
```
症例発表のスライドを作って
```

✅ 良い例:
```
あなたは経験豊富な臨床医です。以下の症例情報から、カンファレンス発表用のスライド構成を提案してください。
```

---

### 2. 情報を構造化する

❌ 悪い例:
```
80歳女性、腹痛、心房細動あり、CTで血栓、手術した
```

✅ 良い例:
```
【症例情報】
- 年齢・性別: 80歳、女性
- 主訴: 急性腹痛
- 既往歴: 心房細動
- 検査所見: CTで上腸間膜動脈血栓
- 治療: 緊急手術
```

---

### 3. 出力形式を指定する

❌ 悪い例:
```
スライドを作って
```

✅ 良い例:
```
以下の構成でスライドの内容を提案してください：
1. タイトルスライド
2. 症例サマリー
...

各スライドには、箇条書きで具体的な内容を記載してください。
```

---

## まとめ

良いプロンプトの条件：

✅ 役割を明確にする（「あなたは経験豊富な臨床医です」）  
✅ 情報を構造化する（見出しと箇条書き）  
✅ 出力形式を指定する（スライド構成を明示）  
✅ 文脈を提供する（発表時間、聴衆のレベル）

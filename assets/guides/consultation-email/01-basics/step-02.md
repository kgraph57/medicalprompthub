# 01-basics/step-02.md: AIに効果的なコンサルテーションメールを作成させる「型」

## AIに効果的なコンサルテーションメールを作成させる「型」

AIに質の高いコンサルテーションメールを作成させるためには、単に情報を羅列するだけでなく、「理想的なメールの構成（型）」をAIに理解させることが重要です。この「型」をプロンプトに組み込むことで、AIは論理的で抜けのない、プロフェッショナルなメールを生成できるようになります。

### 1. コンサルテーションメールの理想的な構成要素

専門医が迅速に状況を把握し、的確なフィードバックを行うために、メールは以下の要素で構成されていることが望ましいです。

| 構成要素 | 目的 | AIへの指示のポイント |
| :--- | :--- | :--- |
| 件名 | 内容の要約と緊急度の明示 | 「【至急】」「【ご相談】」など、目的と緊急度を明確に含める。 |
| 挨拶・自己紹介 | 誰からの依頼かを明確にする | 依頼者の氏名、所属、専門医との関係性を明記させる。 |
| 患者の基本情報 | 患者の全体像を把握 | 年齢、性別、主訴、既往歴、ADLなどを簡潔に。 |
| 現病歴・経過 | 相談に至るまでの経緯 | 発症から現在までの重要なイベント、治療介入、反応を時系列で。 |
| 客観的データ | 診断・治療の根拠 | 重要な検査結果（画像、血液、病理など）の特筆すべき点を記載。 |
| 相談内容・質問 | 最も重要な部分 | 具体的な質問を箇条書きにし、何を求めているかを明確にする。 |
| 結び | 感謝と返信期限の依頼 | 協力への感謝と、もしあれば返信希望時期を伝える。 |

### 2. AIに「型」を認識させるためのプロンプト構造

AIに上記の「型」に従ったメールを作成させるには、プロンプトを以下の4つの要素で構造化します。

#### 構造化プロンプトの4要素

1.  役割 (Role): AIに「誰として」振る舞うかを指示します。（例: 経験豊富な臨床医、医療秘書）
2.  タスク (Task): 達成すべき具体的な行動を指示します。（例: 以下の情報に基づき、専門医へのコンサルテーションメールを作成せよ）
3.  制約・形式 (Constraint/Format): 守るべきルールや出力形式を指示します。（例: 敬語を適切に用いること、件名、本文、署名の形式で出力すること）
4.  入力データ (Input Data): メール作成に必要な具体的な情報を提供します。（例: 患者情報、経過、質問事項）

### 実用的なプロンプト例：構造化の適用

```
役割:
あなたは、大学病院の救急科に勤務する経験豊富な若手医師です。

タスク:
以下の[入力データ]に基づき、消化器外科の[専門医の氏名]先生宛に、患者の緊急コンサルテーションメールを作成してください。

制約・形式:
1.  件名、本文、署名の3部構成とすること。
2.  件名には【緊急コンサルト】と明記し、患者の主訴と氏名を含めること。
3.  本文は、挨拶、患者情報、経過、相談内容、結びの順で構成すること。
4.  敬語を適切に用い、プロフェッショナルなトーンを維持すること。

入力データ:
-   患者: 78歳、男性、既往歴: 糖尿病、高血圧
-   主訴: 3日前からの腹痛、嘔吐
-   経過: 来院時より腹部全体に圧痛と筋性防御あり。CTにて小腸の拡張とfree airを認め、消化管穿孔を強く疑う。
-   現在の対応: 輸液、抗生剤投与中。緊急手術の適応と判断。
-   相談内容: 緊急手術の可否と、手術術式に関するご意見を伺いたい。
```

### 注意点とTips

*   制約の具体性: 「丁寧なメール」といった抽象的な指示ではなく、「件名には【緊急コンサルト】と明記すること」のように、具体的な形式を指示することで、AIの出力精度が向上します。
*   情報の粒度: 入力データは、AIが解釈しやすいように箇条書きで提供し、冗長な表現を避けてください。

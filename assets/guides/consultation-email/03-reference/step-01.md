## コンサルテーション後のフィードバックメール作成と感謝の伝達

コンサルテーションメールのやり取りは、専門医からの回答を得て終わりではありません。回答に基づいたフィードバックメールを送ることで、専門医との信頼関係を構築し、今後の連携を円滑にする上で非常に重要です。このステップでは、AIを活用して、専門医からのアドバイスを反映したフィードバックメールを迅速かつ適切に作成する方法を学びます。

### 1. フィードバックメールの重要性と構成要素

フィードバックメールは、専門医の貴重な時間を割いていただいたことへの感謝を伝えるとともに、そのアドバイスをどのように臨床に活かしたかを報告する役割があります。

| 構成要素 | 目的 | AIへの指示のポイント |
| :--- | :--- | :--- |
| 件名 | 相談内容の明確化と感謝 | 「〇〇の件：ご回答ありがとうございました」など、元の件名を含めて返信であることを明確にする。 |
| 感謝の表明 | 協力への謝意 | 迅速かつ的確なアドバイスへの感謝を丁寧に伝える。 |
| アドバイスの要約と実行 | 臨床的行動の報告 | 専門医からのアドバイスを要約し、それに基づき実際に行った行動（例：MTXの中止、追加検査の実施）を報告する。 |
| 患者の経過報告 | 結果の共有 | アドバイス実行後の患者の最新の経過（例：解熱傾向、症状の改善）を簡潔に報告する。 |
| 結び | 今後の連携への期待 | 今後の経過で変化があれば再度相談する旨を伝え、引き続きの指導を仰ぐ。 |

### 2. フィードバックメール作成のためのプロンプト例

専門医からの回答メールの内容をプロンプトに含めることで、AIはそれを参照し、適切なフィードバックメールを作成します。

```
役割:
あなたは、専門医からのコンサルテーション回答に基づき、感謝と経過報告を行う医師です。

タスク:
以下の[専門医からの回答]と[患者の最新経過]に基づき、[専門医の氏名]先生宛に、フィードバックメールを作成してください。

制約・形式:
1.  件名は「Re: [元の件名]：ご回答ありがとうございました」とすること。
2.  本文には、先生のアドバイスを要約し、そのアドバイスに従って行った具体的な治療介入を明記すること。
3.  感謝の意を最大限に表し、今後の連携を円滑にするプロフェッショナルなトーンとすること。

入力データ:
-   専門医からの回答（要約）: 「MTXは一時中止し、抗生剤を広域のものに変更。同時に骨髄検査を考慮すべき。成人Still病の可能性も念頭に置くこと。」
-   患者の最新経過: MTX中止と抗生剤変更後、48時間以内に解熱傾向。CRPは15.0から8.0に低下。骨髄検査は現在準備中。
```

### 注意点とTips

*   「感謝」の具体性: 単に「ありがとうございました」と伝えるだけでなく、「先生のアドバイスに従いMTXを中止したところ、すぐに解熱傾向となり、大変助かりました」のように、具体的な効果を伝えることで、感謝の気持ちがより深く伝わります。
*   経過の簡潔化: 経過報告は、専門医が知りたい「アドバイス実行後の変化」に焦点を絞り、簡潔に記述してください。

## AIが生成したドラフトのレビューと洗練

AIは迅速にメールのドラフトを作成しますが、最終的な送信前に臨床的な正確性、論理的な構成、そして適切なトーンを人間がレビューし、必要に応じて洗練させる必要があります。このステップでは、AIを「校正者」や「表現の調整役」として活用し、メールの質をさらに高めます。

### 1. レビューの3つのチェックポイント

AIが生成したメールのドラフトに対し、以下の3つの観点からチェックを行います。

| チェックポイント | 詳細 | AIによる支援の可能性 |
| :--- | :--- | :--- |
| 臨床的正確性 | 記載された情報（検査値、経過、治療内容）に誤りや矛盾がないか。 | AIは事実確認はできないため、人間が必ず確認。 |
| 論理的構成 | 相談に至るまでの経緯が時系列で分かりやすいか。最も重要な「相談内容」が明確か。 | AIに「相談内容を箇条書きでより明確にせよ」と指示し、構成を改善させる。 |
| トーンと表現 | 専門医への依頼として適切な敬語と丁寧さか。緊急度が適切に伝わるか。 | AIに「より丁寧な表現に修正せよ」または「緊急性を強調せよ」と指示し、表現を調整させる。 |

### 2. AIによる論理構成の洗練：質問の明確化

コンサルテーションメールで最も重要なのは、「何を尋ねたいのか」を明確にすることです。AIが生成したドラフトの質問部分が抽象的であった場合、以下のプロンプトで具体的な質問に洗練させます。

#### 洗練のためのプロンプト例

```
以下のメールの「相談内容」のセクションを、専門医がYes/Noまたは具体的な治療方針で回答できる、より明確で具体的な質問に修正してください。

元の相談内容:
「鑑別診断の進め方、MTXの一時中止の是非、追加で考慮すべき特殊検査（例：骨髄検査）についてご意見を伺いたいです。」

AIへの指示:
「MTXの一時中止」については、リスクとベネフィットを考慮した上で、先生の推奨される具体的な対応を尋ねる形式にしてください。特殊検査については、当院で実施可能かどうかも含めて尋ねる形にしてください。
```

#### AIによる修正例（想定）

```
修正後の相談内容:
先生方のご意見を特に伺いたい点は以下の3点です。
1.  MTXの一時中止について: 現在、リウマチの活動性は安定していますが、不明熱の原因として薬剤熱を完全に否定できません。先生は、この状況でMTXを一時中止することを推奨されますか？推奨される場合、代替薬や中止期間についてご教示ください。
2.  鑑別診断の進め方: 潜在性感染症や成人Still病などの鑑別を進めるにあたり、先生が特に重要視される検査や、当院で未実施の検査（例：骨髄検査、特殊な血清学的検査）で推奨されるものがあればご教示ください。
3.  今後の治療方針: もし先生の専門的な見地から、当科での対応が困難と判断される場合、貴科への転科や転院のタイミングについてご意見を伺えますでしょうか。
```

### 3. AIによる表現の調整：トーンの統一

メール全体のトーンが不統一であったり、専門医への依頼としてややカジュアルすぎたりする場合、AIに全体のトーンを調整させます。

#### トーン調整のためのプロンプト例

```
以下のメール全体を、大学病院の教授職にある専門医へのコンサルテーションとして、より一層丁寧で、かつ緊急性を損なわないプロフェッショナルなトーンに統一してください。特に、依頼の表現を謙譲語を用いて修正してください。

[AIが生成したメールの全文をここに貼り付け]
```

### 注意点とTips

*   段階的な修正: 一度に全てをAIに任せるのではなく、「質問の明確化」「トーンの調整」のように、段階的にプロンプトを分けて修正を依頼することで、より意図に沿った結果が得られます。
*   最終確認は人間: AIはあくまで文章作成の補助です。最終的なメールの文面は、送信前に必ずご自身の目で確認し、責任を持って送信してください。

## 緊急度・関係性に応じたプロンプトの調整

コンサルテーションメールは、その緊急度や、依頼する専門医との関係性によって、求められるトーンや情報量が異なります。AIを活用する際も、これらの状況に応じてプロンプトを微調整することで、より適切で効果的なメールを作成できます。

### 1. 緊急度に応じたプロンプト調整

緊急度の高いコンサルテーションでは、迅速な対応を促すための明確な指示と、必要な情報への即時アクセスを可能にする構成が求められます。

| 緊急度 | 目的 | プロンプトへの追加指示 |
| :--- | :--- | :--- |
| 緊急（数時間以内） | 迅速な判断と対応を依頼 | 「件名に【至急】または【緊急コンサルト】と明記すること。」「本文の冒頭で、現在の患者のバイタルサインと緊急性を要する理由を簡潔に述べること。」 |
| 準緊急（24時間以内） | 翌日までの返信を依頼 | 「結びに、返信希望時期として『本日中または明日午前中までにご返信いただけると幸いです』と含めること。」「重要な検査結果を箇条書きの冒頭に配置すること。」 |
| 非緊急（通常） | 丁寧な情報提供と意見聴取 | 「丁寧な敬語を最優先すること。」「経過は時系列で詳細に記述すること。」 |

#### 緊急時のプロンプト例

```
制約・形式（追加・変更）:
1.  件名には【緊急コンサルト：〇〇】と明記し、電話での連絡を優先する旨を追記すること。
2.  本文の最上部に、現在のバイタルサインと緊急性を要する理由（例：血圧低下、意識レベルの低下）を太字で記載すること。
3.  結びに、「このメールと並行して、〇〇（内線番号）にてお電話でもご連絡差し上げます」という文言を含めること。
```

### 2. 関係性に応じたプロンプト調整

依頼する専門医が、日頃から連携している親しい間柄か、あるいは初めてコンタクトを取る相手かによって、メールのトーンを調整します。

| 関係性 | 目的 | プロンプトへの追加指示 |
| :--- | :--- | :--- |
| 親しい連携医 | 効率的な情報交換 | 「丁寧さを保ちつつ、冗長な挨拶は避け、要点を簡潔にまとめること。」「専門用語を多用しても問題ない。」 |
| 初対面の専門医 | 信頼感の構築 | 「自己紹介と所属を丁寧に行うこと。」「依頼のトーンは最大限に謙虚に、かつ、情報提供は抜けがないように詳細に行うこと。」 |

#### 初対面の専門医へのプロンプト例

```
役割（追加）:
あなたは、相手の専門医との連携をこれから構築したいと考えている、礼儀正しい医師です。

制約・形式（追加）:
1.  メールの冒頭で、「突然のご連絡失礼いたします」という挨拶と、「〇〇先生の〇〇に関するご高名を知り、今回ご相談させていただきました」という、依頼に至った経緯を丁寧に含めること。
2.  患者情報や経過は、専門外の医師にも理解できるよう、専門用語を使いすぎず、分かりやすい表現を心がけること。
```

### 注意点とTips

*   プロンプトの重ねがけ: 複数の調整（例：緊急かつ初対面）が必要な場合は、それぞれの指示をプロンプトに重ねて記述することで、AIはそれらを統合したメールを作成します。
*   電話の併用: 緊急性の高いコンサルテーションでは、メールは情報整理のツールとして活用し、必ず電話での連絡を併用してください。メールはその後の記録として機能します。

## Step 2: AIによる発表構成案（骨子）の作成と最適化

学会発表の成功は、聴衆がスムーズに理解できる論理的な構成にかかっています。AIは、研究データや目的を入力するだけで、発表の骨子を瞬時に提案し、最適化する強力なツールとなります。

### 1. 発表テーマと目的の明確化

まず、AIに発表の核となる情報を提供します。

実用的なプロンプト例：テーマと目的の入力

```
[役割設定]（Step 1で設定した役割を再利用）

以下の研究結果に基づき、国際学会での10分間の口頭発表の構成案（骨子）を作成してください。

研究テーマ: 慢性心不全患者におけるSGLT2阻害薬の早期導入が再入院率に与える影響
主要な結果: SGLT2阻害薬の早期導入群は、標準治療群と比較して、6ヶ月以内の心不全による再入院率が有意に低かった（ハザード比 0.65, 95% CI 0.52-0.81）。
発表の目的: 循環器内科医に対し、慢性心不全治療におけるSGLT2阻害薬の早期導入の重要性と、そのエビデンスを提示すること。
聴衆: 循環器内科の専門医（約80%）、研修医（約20%）。
スライド枚数: 15枚以内。
```

### 2. AIによる構成案の提案とブラッシュアップ

AIは、提供された情報と制約条件に基づき、一般的な医学会発表の構成（背景、目的、方法、結果、考察、結論）に沿った骨子を提案します。

#### 2.1. 提案された骨子の確認と修正

AIが提案した骨子に対し、以下の観点から修正指示を出します。

| 観点 | 修正指示の例 | 目的 |
| :--- | :--- | :--- |
| メッセージの強調 | 「結果」セクションで、主要アウトカムのグラフをより強調するよう指示してください。 | 最も伝えたいメッセージを際立たせる |
| 聴衆への配慮 | 研修医にも理解できるよう、「背景」セクションに心不全治療の標準的な流れを1枚追加してください。 | 聴衆の知識レベルに合わせた調整 |
| 論理の流れ | 「考察」の前に、結果の臨床的意義を明確にするスライドを挿入してください。 | 発表全体の論理的なつながりを強化 |

実用的なプロンプト例：骨子のブラッシュアップ

```
提案された骨子について、以下の点を修正してください。
1. スライド5枚目の「方法」について、統計解析手法（Cox比例ハザードモデル）を具体的に記載するよう修正してください。
2. スライド12枚目の「考察」の前に、「主要結果の視覚的ハイライト」として、ハザード比のフォレストプロットを挿入するスライドを追加してください。
3. 結論のスライドに、「今後の展望」として、この結果を実臨床にどう応用すべきかという提言を追記してください。
```

### 3. 発表時間とスライド枚数の最適化

AIに、提案された骨子とスライド枚数に基づき、各セクションに割り当てるべき理想的な発表時間をシミュレーションさせます。

実用的なプロンプト例：時間配分の最適化

```
全15枚のスライド構成案に基づき、10分間の発表時間に対する各セクションの推奨時間配分を提案してください。
特に「結果」と「考察」に重点を置いた配分としてください。

出力形式:
| セクション | スライド枚数 | 推奨時間（秒） |
| :--- | :--- | :--- |
| 背景 | 2 | 60 |
| 目的 | 1 | 30 |
| ... | ... | ... |
| 合計 | 15 | 600 |
```

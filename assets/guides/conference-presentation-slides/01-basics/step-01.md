## Step 1: AIを活用したスライド作成の基本と準備

学会発表スライドの作成は、研究成果を正確かつ魅力的に伝えるための重要なプロセスです。AIを活用することで、このプロセスを劇的に効率化し、よりインパクトのある発表資料を作成することが可能になります。

### 1. AI活用のメリットと役割

AIは、スライド作成における「優秀なアシスタント」として機能します。特に医療従事者が直面する以下の課題に対して、大きな効果を発揮します。

| 課題 | AIの役割 | 具体的なメリット |
| :--- | :--- | :--- |
| 構成案の検討 | 発表テーマに基づいた論理的な骨子の提案 | 構成検討時間の短縮、論理破綻の防止 |
| 文章の作成・校正 | 専門用語を用いた正確な文章の生成、冗長な表現の修正 | スライドのテキスト品質向上、誤字脱字の削減 |
| 要約・平易化 | 複雑な研究結果や統計データを簡潔に要約 | 聴衆の理解度向上、メッセージの明確化 |
| デザイン補助 | キーメッセージに合わせた視覚的な表現の提案 | 視覚的に魅力的なスライドデザインのヒント |

### 2. AI活用のための心構え：プロンプトエンジニアリングの基礎

AIの出力品質は、入力するプロンプト（指示）の質に依存します。AIを最大限に活用するための基本的な心構えを理解しましょう。

#### 2.1. 役割（ペルソナ）の明確化

AIに、どのような専門家として振る舞ってほしいかを明確に指示します。これにより、出力される情報の専門性と信頼性が向上します。

実用的なプロンプト例：役割設定

```
あなたは、国際的な医学会での発表経験が豊富で、聴衆を惹きつけるスライド構成に精通した医療コミュニケーションの専門家です。
以下のタスクを実行する際は、この役割に基づき、専門的かつ実践的な視点から回答してください。
```

#### 2.2. 制約条件の明示

発表時間、聴衆の専門性、使用言語、スライドの枚数など、具体的な制約条件を明確に伝えます。

実用的なプロンプト例：制約条件の明示

```
発表時間は10分、質疑応答は5分です。聴衆は主に循環器内科の専門医ですが、一部研修医も含まれます。
スライドは最大15枚とし、専門的な内容を分かりやすく伝えることを最優先としてください。
```

### 3. AI利用時の最重要注意点：機密情報とファクトチェック

AIは強力なツールですが、利用にあたっては以下の2点を厳守してください。

1.  機密情報の絶対的な除外: 患者の個人情報、未公開の機密性の高い研究データなど、いかなる機密情報もAIに入力してはいけません。必ず匿名化・一般化された情報のみを使用してください。
2.  出力内容のファクトチェック: AIが生成した文章やデータ解釈は、必ず一次情報（原著論文、生データ）と照らし合わせて、その正確性を確認してください。AIは誤った情報（ハルシネーション）を出力する可能性があります。

## Step 4: スライドテキストの最適化と視覚化の検討

AIが生成したテキストドラフトは、情報量が多く、そのままスライドに貼り付けると「文字だらけのスライド」になりがちです。このステップでは、AIの力を借りて、「話すための資料」から「見せるための資料」へとテキストを最適化します。

### 1. スライド原則に基づいたテキストの圧縮

スライドのテキストは、「1スライド1メッセージ」の原則に基づき、極限まで圧縮する必要があります。AIに、冗長な表現を削ぎ落とし、箇条書きを短くする作業を依頼します。

実用的なプロンプト例：テキストの圧縮と最適化

```
[役割設定]（Step 1で設定した役割を再利用）

以下のスライドテキストドラフトを、学会発表スライドの原則（1スライド5行以内、1行15文字以内）に則って、最も重要なキーワードのみを残すように圧縮・最適化してください。

元のドラフト（スライド9：結果）:
- 主要評価項目は、心不全による再入院または心血管死の複合エンドポイントと定義された。
- SGLT2阻害薬の早期導入群は、標準治療群と比較して、主要評価項目の発生リスクが有意に低かった（ハザード比 0.65, 95% CI 0.52-0.81）。
- この結果は統計学的に極めて有意であった（P < 0.001）。
- Kaplan-Meier曲線では、早期導入群と標準治療群のイベント発生率の乖離が、介入後早期から認められた。

最適化後のテキスト:
- 主要評価項目：心不全再入院 or 心血管死
- HR 0.65 (95% CI 0.52-0.81)
- P < 0.001 (統計学的有意差)
- KM曲線：早期からイベント抑制効果
```

### 2. 視覚化（図表）のアイデア生成

テキストを圧縮した結果、スライドには余白が生まれます。この余白を効果的に活用するため、AIに「このメッセージを伝えるのに最適な視覚的表現」を提案させます。

実用的なプロンプト例：視覚化の提案

```
スライド12（タイトル: 主要結果の臨床的意義、キーメッセージ: SGLT2阻害薬の早期導入は、心不全治療のパラダイムシフトを促す。）の視覚化について、以下の観点からアイデアを提案してください。

1.  図の種類: どのようなグラフ、図、イラストが最も効果的か。
2.  図の構成要素: 図に含めるべき主要な要素（例：矢印、比較対象、数値）。
3.  デザインの方向性: クリーン、ミニマル、またはダイナミックなど、どのようなデザインが適切か。

提案例:
- 図の種類: 治療の変遷を示すタイムライン図
- 構成要素: 過去の標準治療 → 現在の標準治療 → SGLT2阻害薬早期導入（未来の標準治療）への矢印。各段階に再入院率の低下を示す数値を併記。
- デザインの方向性: 未来志向で、明るい青色を基調としたダイナミックなデザイン。
```

### 3. 図表のキャプションと凡例の作成

研究結果を示す図表（グラフなど）には、正確なキャプションと凡例が不可欠です。AIに、データに基づいた正確で簡潔なキャプションを作成させます。

実用的なプロンプト例：図表キャプションの作成

```
以下のデータと図表の種類に基づき、学会発表スライド用のキャプション（図のタイトル）と凡例を作成してください。

図表の種類: Kaplan-Meier曲線
データ: SGLT2阻害薬早期導入群（n=500）と標準治療群（n=500）の6ヶ月間の心不全再入院または心血管死の累積発生率。ハザード比 0.65 (95% CI 0.52-0.81), P < 0.001。

キャプション:
Figure 1. SGLT2阻害薬の早期導入が心不全再入院/心血管死の複合エンドポイントに与える影響

凡例:
- 実線: SGLT2阻害薬早期導入群
- 破線: 標準治療群
- グラフ右下にハザード比とP値を明記。
```

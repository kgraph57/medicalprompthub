## Step 3: 各スライドのタイトルとキーメッセージの生成

Step 2で作成した骨子に基づき、各スライドが持つべき「タイトル」と、そのスライドで聴衆に伝えるべき「キーメッセージ」をAIに生成させます。これにより、スライド作成の方向性が明確になり、メッセージが一貫したものになります。

### 1. スライドごとのメッセージの明確化

AIに骨子全体を渡し、各スライドのタイトルと、そのスライドの目的を簡潔に表現したキーメッセージを生成させます。

実用的なプロンプト例：タイトルとキーメッセージの生成

```
[役割設定]（Step 1で設定した役割を再利用）

以下の骨子に基づき、各スライドのタイトルと、そのスライドで聴衆に最も伝えたいキーメッセージ（1文）を生成してください。

骨子:
1. タイトルスライド
2. COI開示
3. 背景：心不全治療における課題
4. 背景：SGLT2阻害薬の登場と既存エビデンス
5. 目的：本研究の問い
6. 方法：対象患者とデザイン
7. 方法：主要評価項目と統計解析
8. 結果：患者背景
9. 結果：主要アウトカム（再入院率）
10. 結果：サブグループ解析
11. 結果：安全性評価
12. 考察：主要結果の臨床的意義
13. 考察：本研究の限界
14. 結論と提言
15. 謝辞

出力形式:
| スライド番号 | セクション | タイトル | キーメッセージ（1文） |
| :--- | :--- | :--- | :--- |
| 3 | 背景 | 慢性心不全治療の現状と残された課題 | 既存治療では心不全による再入院を十分に抑制できていない。 |
| 9 | 結果 | SGLT2阻害薬の早期導入は再入院リスクを35%低減 | 早期介入により、心不全の予後が有意に改善した。 |
| ... | ... | ... | ... |
```

### 2. キーメッセージのブラッシュアップ：インパクトの最大化

生成されたキーメッセージは、よりインパクトがあり、記憶に残りやすい表現にブラッシュアップします。特に「結果」や「結論」のスライドでは、具体的な数値や強い言葉を用いることが重要です。

実用的なプロンプト例：メッセージの強調

```
スライド9のキーメッセージ「早期介入により、心不全の予後が有意に改善した。」を、より聴衆の記憶に残る、インパクトのある表現に修正してください。
ただし、科学的な正確性は維持してください。

修正案の例:
「SGLT2阻害薬の早期導入は、心不全再入院のゲームチェンジャーとなる。」
「再入院リスクを劇的に低減：早期導入の強力なエビデンス。」
```

### 3. スライドテキストのドラフト作成

タイトルとキーメッセージが確定したら、そのスライドに記載すべき本文（箇条書き）のドラフトをAIに作成させます。この段階では、冗長でも必要な情報を全て含めることを優先します。

実用的なプロンプト例：本文のドラフト作成

```
スライド9（タイトル: SGLT2阻害薬の早期導入は再入院リスクを35%低減、キーメッセージ: 早期介入により、心不全の予後が有意に改善した。）の本文（箇条書き）を作成してください。

含めるべき情報:
- 主要評価項目（心不全による再入院または心血管死）の定義
- ハザード比（0.65）と95%信頼区間（0.52-0.81）
- P値（P < 0.001）
- Kaplan-Meier曲線から読み取れる主要な傾向（早期のイベント発生率の乖離）

注意点:
- 箇条書きは最大5点までとし、簡潔に記述してください。
- 専門用語は正確に使用してください。
```

## Step 7: デザインとレイアウトのAI活用と最終チェックリスト

コンテンツが完成したら、最後に視覚的な魅力を高めるデザインとレイアウトの調整が必要です。AIは、デザインの専門家として、スライドの統一感、視認性、そしてメッセージの伝達力を高めるための具体的な提案を行います。

### 1. デザイン原則に基づいたレイアウトの提案

AIに、スライドのテキストと図表の配置に基づき、最も視認性が高く、メッセージが伝わりやすいレイアウトを提案させます。

実用的なプロンプト例：レイアウトの提案

```
[役割設定]（Step 1で設定した役割を再利用）

以下のスライド内容に基づき、スライドデザインの専門家として、最も効果的なレイアウトを提案してください。

スライド内容:
- タイトル: SGLT2阻害薬の早期導入は再入院リスクを35%低減
- キーメッセージ: 早期介入により、心不全の予後が有意に改善した。
- 本文（箇条書き4点）
- 図表: Kaplan-Meier曲線（主要結果）

提案レイアウトの例:
1.  タイトルとキーメッセージ: スライド上部中央に大きく配置。
2.  Kaplan-Meier曲線: スライドの約60%を占めるように、右側に大きく配置。
3.  本文（箇条書き）: 図表の左側に、縦に並べて配置。各箇条書きの前に、図表の重要なポイントを指し示すアイコン（例：矢印）を配置。
4.  配色: 図表の主要な線（早期導入群）を、背景色（白）と対比する鮮やかな色（例：青）にし、視線を誘導する。
```

### 2. 配色とフォントの統一感のチェック

学会発表では、視覚的な統一感がプロフェッショナリズムを印象づけます。AIに、使用されている配色とフォントが、発表のテーマや学会の雰囲気に合っているかをチェックさせます。

実用的なプロンプト例：デザインの統一感チェック

```
以下のデザイン要素について、統一感と視認性の観点から評価し、改善点を提案してください。

デザイン要素:
- メインカラー: 濃い青（#003366）
- アクセントカラー: 明るいオレンジ（#FF9900）
- フォント: タイトルは游ゴシック Bold、本文は游ゴシック Regular
- 図表の背景: 薄いグレー（#F0F0F0）

評価と改善点の例:
- 評価: メインカラーとアクセントカラーのコントラストは良好で、視認性が高い。しかし、医療分野の発表としては、オレンジがややカジュアルに見える可能性がある。
- 改善点: アクセントカラーを、信頼感のあるターコイズブルー（#00CED1）に変更することを推奨します。フォントは統一感があり、視認性も問題ありません。
```

### 3. 最終チェックリスト：AIによる抜け漏れ確認

発表直前に、AIに最終的なチェックリストを提示させ、重要な抜け漏れがないかを確認します。

実用的なプロンプト例：最終チェックリストの生成

```
学会発表スライドの最終チェックリストを、以下の3つのカテゴリに分けて生成してください。

1.  内容の正確性
2.  デザインと視認性
3.  倫理・コンプライアンス

最終チェックリストの例:
| カテゴリ | チェック項目 | 状態 |
| :--- | :--- | :--- |
| 内容の正確性 | 全てのデータ（数値、P値、CI）は原著論文と一致しているか？ | [ ] |
| | キーメッセージは、スライドのタイトルと一貫しているか？ | [ ] |
| デザインと視認性 | 1スライドあたりの情報量は適切か（文字だらけになっていないか）？ | [ ] |
| | 図表の軸ラベル、凡例は十分な大きさで表示されているか？ | [ ] |
| 倫理・コンプライアンス | COI（利益相反）開示スライドは適切に挿入されているか？ | [ ] |
| | 患者の個人情報（イニシャル、IDなど）は完全に削除されているか？ | [ ] |
```

## 📖 セクション2: APIの基本的な使い方

### APIキーの取得

APIを使用するには、APIキーが必要です。

**取得手順**：
1. Anthropicのウェブサイトにアクセス
2. APIキーのページに移動
3. 新しいAPIキーを生成
4. APIキーを安全に保管

### 基本的なAPI呼び出し

APIの基本的な呼び出し方法：

**例：Pythonでの呼び出し**：
```python
import anthropic

client = anthropic.Anthropic(api_key="your-api-key")

message = client.messages.create(
    model="claude-3-opus-20240229",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "診療記録を要約してください：[記録]"}
    ]
)

print(message.content[0].text)
```

# 統計学の基礎：医療研究に必要な統計

## このレッスンで学ぶこと

このレッスンを完了すると、統計学の基本概念を理解し、医療研究で使用される統計手法の基礎を把握できるようになります。
AIと医療データ分析に必要な統計の知識を習得します。

---

## セクション1: 統計学とは何か

### 統計学の定義

統計学とは、データを収集、整理、分析、解釈するための学問です。

医療での重要性：
- 研究データの分析
- 診断の精度評価
- 治療効果の評価
- 疫学研究

### 記述統計と推測統計

統計学は、大きく2つに分けられます。

#### 記述統計

データの特徴を要約・記述します。

- 平均値、中央値、最頻値
- 標準偏差、分散
- グラフや表での可視化

#### 推測統計

サンプルから母集団について推測します。

- 仮説検定
- 信頼区間
- 回帰分析

## セクション2: データの種類

### 量的データ（連続データ）

数値で測定できるデータです。

例：
- 年齢、体重、身長
- 血圧、血糖値、体温
- 検査値

特徴：
- 平均値、標準偏差を計算できる
- 連続的な値

### 質的データ（カテゴリカルデータ）

カテゴリに分類されるデータです。

種類：
- **名義尺度**: 順序がない（性別、血液型）
- **順序尺度**: 順序がある（重症度、グレード）

例：
- 性別、診断名
- 治療法、合併症の有無

---

## セクション3: 基本統計量

### 中心傾向の指標

データの中心的な値を表します。

#### 平均値（Mean）

すべての値の合計を個数で割った値です。

公式：
```
平均値 = 合計 / 個数
```

特徴：
- 外れ値の影響を受けやすい
- 連続データに適している

#### 中央値（Median）

データを小さい順に並べたとき、中央の値です。

特徴：
- 外れ値の影響を受けにくい
- ロバストな指標

#### 最頻値（Mode）

最も頻繁に出現する値です。

特徴：
- カテゴリカルデータに適している

### ばらつきの指標

データの散らばり具合を表します。

#### 標準偏差（Standard Deviation）

データの平均からの散らばり具合です。

公式：
```
標準偏差 = √(分散)
```

解釈：
- 小さいほどデータが集中している
- 大きいほどデータが散らばっている

#### 分散（Variance）

標準偏差の2乗です。

## セクション4: 分布

### 正規分布

最も重要な分布の一つです。

特徴：
- 左右対称の釣鐘型
- 平均と標準偏差で特徴づけられる
- 多くの統計手法の基礎

医療での例：
- 身長、体重の分布
- 検査値の分布（多くの場合）

### その他の分布

#### 二項分布

成功・失敗の2つの結果がある場合。

例：
- 治療の成功・失敗
- 疾患の有無

#### ポアソン分布

稀な事象の発生回数。

例：
- 副作用の発生回数
- 感染症の発生数

## セクション5: 医療研究での統計の役割

### 研究デザイン

統計は、研究デザインの段階から重要です。

- サンプルサイズの計算
- 研究の設計
- データ収集方法の決定

### データ分析

統計は、データ分析の核心です。

- 記述統計による要約
- 推測統計による検定
- 結果の解釈

### 結果の報告

統計は、結果の報告に不可欠です。

- 統計量の報告
- グラフや表の作成
- 解釈の説明

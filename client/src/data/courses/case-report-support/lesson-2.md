# Case Presentationの作成

## 🎯 このレッスンで学ぶこと

このレッスンを完了すると、AIを活用した症例の要約、時系列の整理、重要な情報の抽出、Case Presentationの作成ができるようになります。

---

## 📖 セクション1: 症例の要約

### 基本的な要約

AIを活用して、症例を要約できます。

**実践例**：
「以下の詳細な症例情報を要約してください。

症例情報：[詳細な症例情報]

以下の項目を含めてください：
- 患者情報（年齢、性別、主訴）
- 現病歴（時系列で整理）
- 重要な検査結果
- 診断と治療方針

各項目を簡潔に、重要な情報のみを含めてください。」

### 構造化された要約

症例を構造化された形式で要約することもできます。

**実践例**：
「先ほどの症例情報を、以下の形式で構造化してください。

形式：
- 患者情報
- 主訴と現病歴
- 身体所見
- 検査結果
- 診断と治療
- 経過

各項目を明確に分けて、要約してください。」

---

## 📖 セクション2: 時系列の整理

### 時系列の整理方法

症例の時系列を整理できます。

**実践例**：
「以下の症例の時系列を整理してください。

症例情報：[症例情報]

以下の形式で整理してください：
- 日付と時間
- 症状の変化
- 検査結果の変化
- 治療の変更

時系列で整理し、重要な変化を強調してください。」

### 重要なイベントの抽出

時系列から重要なイベントを抽出することもできます。

**実践例**：
「先ほどの時系列から、重要なイベントを抽出してください。

以下の項目について抽出してください：
1. 症状の出現
2. 診断の確定
3. 治療の開始
4. 症状の改善
5. 退院

各イベントについて、日付と内容を含めてください。」

---

## 📖 セクション3: 重要な情報の抽出

### 診断に重要な情報の抽出

診断に重要な情報を抽出できます。

**実践例**：
「以下の症例情報から、診断に重要な情報を抽出してください。

症例情報：[症例情報]

以下の項目について抽出してください：
1. 主訴と現病歴の要点
2. 身体所見の重要なポイント
3. 検査結果の異常値
4. 診断の根拠となる所見」

### Case Presentationの作成

抽出した情報から、Case Presentationセクションを作成できます。

**実践例**：
「先ほど抽出した情報から、Case Presentationセクションを作成してください。

以下の形式で作成してください：
1. 患者情報
2. 主訴と現病歴
3. 身体所見
4. 検査結果
5. 診断と治療
6. 経過

各項目について、専門用語を適切に使用し、時系列で整理してください。」

## 💡 重要な洞察：Case Presentationの重要性

- **重要な情報の抽出**：診断に重要な情報を抽出

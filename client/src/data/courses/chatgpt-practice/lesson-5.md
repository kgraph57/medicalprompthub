## 📖 セクション1: Code Interpreterとは

### Code Interpreterの概要

Code Interpreterは、ChatGPT Plusで利用できる機能で、コードを実行してデータ分析やファイル処理ができます。

**主な機能**：
- **データ分析**：CSV、Excelファイルの分析
- **統計解析**：統計的な分析の実行
- **グラフ作成**：データの可視化
- **ファイル処理**：様々な形式のファイルの処理

### 医療現場での活用

**活用例**：
- **検査データの分析**：血液検査データの統計解析
- **研究データの処理**：研究データの分析と可視化
- **レポートの作成**：データに基づいたレポートの作成

## 📖 セクション2: CSVファイルの処理

### データの読み込みと基本分析

CSVファイルを読み込んで、基本的な分析ができます。

**実践例**：
「以下のCSVファイルを読み込んで、基本的な統計情報を表示してください。

ファイル：[CSVファイルをアップロード]

以下の情報を表示してください：
- データの行数と列数
- 各列の基本統計（平均、標準偏差、最小値、最大値）
- 欠損値の有無」

### データのフィルタリング

データをフィルタリングして、特定の条件に合うデータを抽出できます。

**実践例**：
「先ほどのデータから、以下の条件に合うデータを抽出してください。

条件：
- 年齢が60歳以上
- 診断名が『急性心筋梗塞』
- 検査値が正常範囲外

抽出したデータの統計情報も表示してください。」

## 📖 セクション3: 統計解析

### 記述統計

基本的な記述統計を実行できます。

**実践例**：
「以下のデータについて、記述統計を実行してください。

データ：[データを指定]

以下の統計量を計算してください：
- 平均値
- 中央値
- 標準偏差
- 四分位範囲
- 最小値と最大値」

### 推論統計

推論統計も実行できます。

**実践例**：
「以下の2群のデータについて、t検定を実行してください。

群1：[データ1]
群2：[データ2]

有意水準は5%とし、結果を解釈してください。」

## 📖 セクション4: グラフの作成

### 基本的なグラフ

データを可視化するグラフを作成できます。

**実践例**：
「以下のデータを可視化してください。

データ：[データを指定]

以下のグラフを作成してください：
- ヒストグラム：検査値の分布
- 箱ひげ図：群間の比較
- 散布図：2変数の関係

各グラフに適切なタイトルとラベルを付けてください。」

### 医療データの可視化

医療データに特化した可視化も可能です。

**実践例**：
「以下の時系列データを可視化してください。

データ：患者のバイタルサイン（時間、血圧、脈拍、体温）

時系列グラフを作成し、異常値があれば強調表示してください。」

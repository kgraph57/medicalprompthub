# レッスン4: 教育資料の作成 - 医学生・研修医向け

## 1. はじめに

医学生や研修医の教育では、分かりやすく視覚的な教育資料が重要です。本レッスンでは、画像生成AI（Midjourney、DALL-E）を使って教育資料を作成する方法について詳しく学びます。

特に、基礎的な医学知識の視覚化、複雑な概念の視覚化、段階的な説明図などの実践例を通じて理解します。

## 2. 教育用イラストの生成

### 基礎的な医学知識の視覚化

基礎的な医学知識を視覚化するイラストを生成できます。

**プロンプトのポイント：**

- **基礎的な内容**: 「基礎的な」「入門的な」などの指示を追加
- **分かりやすさ**: 「分かりやすい」「シンプルな」などの指示を追加
- **教育的**: 「教育用」「学習用」などの指示を追加

**実際の例：基礎的な医学知識の視覚化**

```
プロンプト例1（心臓の構造）：

「基礎的な心臓の構造図、教育用、分かりやすい、医療、
解剖、シンプル、カラー、イラスト、学習用」

生成される画像：
- 心臓の基本的な構造を説明するイラスト
- 教育用に適したシンプルなデザイン
- 医学生向けの学習資料に適している

プロンプト例2（呼吸器系の構造）：

「入門的な呼吸器系の図、学習用、分かりやすい、
医療、解剖、肺、気管、シンプル、教育用」

生成される画像：
- 呼吸器系の基本的な構造を説明するイラスト
- 学習用に適したシンプルなデザイン
- 基礎的な医学知識の学習に適している
```

### 解剖図の作成

人体の構造を説明する解剖図を作成できます。

**プロンプトのポイント：**

- **解剖部位**: 何の解剖図を作成するか明確に指定
- **詳細度**: 「基礎的な」「詳細な」などの詳細度を指定
- **教育用**: 教育用に適したデザインを指定

**実際の例：解剖図の生成**

```
プロンプト例1（基礎的な解剖図）：

「基礎的な心臓の解剖図、教育用、分かりやすい、
医療、解剖、心臓、心房、心室、シンプル、カラー」

生成される画像：
- 心臓の基本的な解剖図
- 心房と心室を区別できるデザイン
- 教育用に適したシンプルなデザイン

プロンプト例2（詳細な解剖図）：

「詳細な心臓の解剖図、医師向け、精密、医療、
解剖、心臓、血管、弁、専門的、医学教育用」

生成される画像：
- 心臓の詳細な解剖図
- 血管や弁も含む精密なデザイン
- 医師向けの専門的なデザイン
```

## 6. プロンプトの作成テクニック

### 教育用プロンプトの書き方

教育用のプロンプトを書くことで、より適切な画像が生成されます。

**プロンプトの構成：**

1. **主題**: 何を生成するか（解剖図、概念図など）
2. **対象者**: 対象者（医学生向け、研修医向け、患者向けなど）
3. **詳細度**: 詳細度（基礎的な、詳細ななど）
4. **スタイル**: デザインスタイル（シンプル、分かりやすいなど）
5. **用途**: 用途（教育用、学習用など）

**実際の例：教育用プロンプト**

```
プロンプトの構成例：

基本構成：
「[主題] + [対象者] + [詳細度] + [スタイル] + [用途]」

例：
「心臓の解剖図 + 医学生向け + 基礎的な + 分かりやすい + 教育用」

詳細版：
「基礎的な心臓の解剖図、医学生向け、分かりやすい、
 シンプル、カラー、イラスト、教育用、学習用」

改善版：
「基礎的な心臓の解剖図、医学生向け、入門レベル、
 分かりやすい、シンプルデザイン、カラーイラスト、
 教育用、学習用、心房と心室を区別できる」
```

画像生成AIを適切に活用することで、分かりやすく視覚的な教育資料を作成できます。特に、基礎的な内容から詳細な内容まで、対象者に応じて適切な詳細度で視覚化できることが、画像生成AIの強みです。

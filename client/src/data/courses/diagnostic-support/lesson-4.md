# 臨床判断への統合

## 🎯 このレッスンで学ぶこと

このレッスンを完了すると、AI診断支援を臨床判断に統合する方法、医師の判断との組み合わせ、患者への説明ができるようになります。

---

## 📖 セクション1: 診断支援の統合方法

### 診断プロセスへの統合

AI診断支援を診断プロセスに統合する方法：

**統合のポイント**：
- **情報収集段階**：症状入力と情報整理
- **診断検討段階**：鑑別診断のリストアップ
- **診断確定段階**：診断の確認と検証

**医療現場での実践**：
- **段階的な統合**：診断プロセスの各段階でAIを活用
- **医師の判断との組み合わせ**：AIの結果と医師の判断を組み合わせ
- **最終判断は医師**：診断の最終判断は、必ず医師が行う

---

## 📖 セクション2: 医師の判断との組み合わせ

### 協働的な判断

AI診断支援と医師の判断を組み合わせる方法：

**実践例**：
「AI診断支援の結果と、医師の判断を組み合わせて、最終的な診断を決定してください。

AI診断支援結果：
- 診断1：可能性70%
- 診断2：可能性20%
- 診断3：可能性10%

医師の判断：
- 患者の臨床症状から、診断1の可能性が高いと判断
- ただし、追加検査が必要

以下の項目について検討してください：
1. AI結果と医師の判断の整合性
2. 追加で必要な検査
3. 最終的な診断
4. 診断の根拠」

## 📖 セクション3: 患者への説明

### 診断の説明

AI診断支援を活用した診断を、患者に説明する方法：

**説明のポイント**：
- **透明性**：AIの使用を透明にする
- **分かりやすさ**：患者が理解しやすい言葉で説明
- **根拠の説明**：診断の根拠を説明

**医療現場での実践**：
- **AIの役割の説明**：AIが診断を支援したことを説明
- **最終判断は医師**：最終的な判断は医師が行ったことを説明
- **患者の理解**：患者が理解できる言葉で説明

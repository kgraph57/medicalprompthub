## 📖 セクション1: 実践的な画像分析

### 症例1：胸部X線の分析

**実践例**：
「この胸部X線画像を分析してください。

[画像をアップロード]

以下の項目について評価してください：
1. 異常所見の有無
2. 異常所見の位置（右肺、左肺、両側など）
3. 異常所見の特徴（浸潤影、胸水、気胸など）
4. 考えられる診断（可能性の高い順に3つ）
5. 追加で必要な検査

各項目について、具体的に説明してください。」

## 📖 セクション2: 診断支援としての活用

### 画像とテキストの統合分析

画像とテキスト情報を組み合わせて、診断を支援できます。

**実践例**：
「このX線画像と、以下の患者情報を基に、診断の可能性を検討してください。

患者情報：
- 65歳、男性
- 主訴：胸痛、発汗、呼吸困難
- 現病歴：3時間前から症状出現
- 既往歴：高血圧、糖尿病
- 検査結果：心電図でST上昇、トロポニン陽性

[画像をアップロード]

以下の項目について分析してください：
1. 画像所見と患者情報の整合性
2. 最も可能性の高い診断
3. 診断の根拠
4. 追加で必要な検査」

### 複数画像の比較

複数の画像を比較して分析することもできます。

**実践例**：
「以下の2つのX線画像を比較してください。

[画像1をアップロード]
[画像2をアップロード]

以下の項目について比較してください：
1. 画像間の違い
2. 変化の意味
3. 考えられる原因
4. 今後の対応」

## 📖 セクション3: 注意点と限界

### 画像解析の限界

Geminiの画像解析には、以下の限界があります：

**限界**：
- **精度の限界**：人間の専門医ほど正確ではない
- **専門性の限界**：非常に専門的な内容では限界がある
- **画像品質への依存**：画像の品質に依存する

### 安全な活用方法

画像解析を安全に活用する方法：

**実践的な原則**：
1. **補助的なツール**：Geminiは診断支援ツールとして使用
2. **最終判断は医師**：診断の最終判断は、必ず医師が行う
3. **品質の確認**：解析結果の品質を必ず確認
4. **複数の情報源と照合**：重要な情報は、複数の情報源で確認

## 📖 セクション4: 実践的な活用例

### 教育用の活用

Geminiの画像解析を教育用に活用できます。

**実践例**：
「このX線画像について、医学教育用の説明を作成してください。

[画像をアップロード]

以下の項目を含めてください：
1. 正常所見の説明
2. 異常所見の説明
3. 診断の根拠
4. 教育的なポイント

医学生が理解しやすい言葉で説明してください。」

### レポート作成の支援

画像解析結果からレポートを作成することもできます。

**実践例**：
「この画像解析結果から、診断レポートを作成してください。

解析結果：[解析結果]

以下の形式でレポートを作成してください：
- 画像所見
- 診断の可能性
- 診断の根拠
- 推奨される追加検査」

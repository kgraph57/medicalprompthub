# p値と信頼区間 - 統計的推論の基礎

## このレッスンで学ぶこと

このレッスンを完了すると、p値と信頼区間という統計学の基本的な概念を理解し、医療研究で使用される統計的推論の基礎を把握できるようになります。
医学論文を読む際に必要な、統計学の専門用語の基礎を身につけます。

---

## セクション1: p値とは何か

### p値の定義：統計的有意性の指標

p値（p-value）は、統計的検定で得られる結果が偶然によるものである確率を表す値です。

**p値の意味：**
- **小さいp値**：偶然による可能性が低い（統計的に有意）
- **大きいp値**：偶然による可能性が高い（統計的に有意でない）
- **一般的な基準**：p < 0.05が統計的有意性の基準として使われることが多い

**医療での重要性：**
- **治療効果の評価**：新しい治療法が効果的かどうかを判断
- **リスク因子の特定**：疾患のリスク因子を特定
- **診断精度の評価**：診断検査の精度を評価

### p値の誤解：よくある誤解

p値について、よくある誤解があります。

**誤解1：p値が小さいほど効果が大きい**
- p値は効果の大きさを表すものではない
- 効果の大きさは、効果量（effect size）で評価する

**誤解2：p < 0.05なら確実に真実**
- p < 0.05でも、偶然による可能性は5%ある
- 複数の検定を繰り返すと、偶然でも有意になる可能性が高まる

**医療での重要性：**
- p値を正しく解釈することが重要
- 効果量も考慮する必要がある
- 複数の研究結果を総合的に評価する

## セクション2: 信頼区間とは何か

### 信頼区間の定義：推定値の範囲

信頼区間（Confidence Interval：CI）は、母集団の真の値を一定の確率で含むと推定される範囲です。

**信頼区間の意味：**
- **95%信頼区間**：95%の確率で真の値を含む範囲
- **範囲が狭い**：推定の精度が高い
- **範囲が広い**：推定の精度が低い

**医療での例：**
- **治療効果の推定**：新しい治療法の効果を95%信頼区間で推定
- **リスク比の推定**：疾患のリスク比を95%信頼区間で推定
- **診断精度の推定**：診断検査の精度を95%信頼区間で推定

### 信頼区間の解釈：実践的な活用

信頼区間を実践的に解釈する方法：

**解釈のポイント：**
- **範囲を確認**：信頼区間の範囲を確認
- **臨床的有意性**：範囲が臨床的に意味があるか確認
- **統計的有意性**：信頼区間が1（または0）を含まない場合、統計的に有意

**医療での重要性：**
- p値だけでなく、信頼区間も確認することが重要
- 効果の大きさと不確実性を理解できる
- 臨床的な判断に役立つ

## セクション3: p値と信頼区間の関係

### 統計的有意性：p値と信頼区間の対応

p値と信頼区間は、統計的有意性を判断する上で対応関係があります。

**対応関係：**
- **p < 0.05**：95%信頼区間が1（または0）を含まない
- **p >= 0.05**：95%信頼区間が1（または0）を含む

**医療での重要性：**
- p値と信頼区間を組み合わせて判断することが重要
- より包括的な評価が可能

### 実践的な活用：医学論文を読む際のポイント

医学論文を読む際のポイント：

**確認すべき項目：**
- **p値**：統計的に有意かどうか
- **信頼区間**：効果の大きさと不確実性
- **効果量**：臨床的に意味があるかどうか

**医療での重要性：**
- 論文の結果を正しく解釈するために重要
- 診療に役立つ情報を抽出するために重要

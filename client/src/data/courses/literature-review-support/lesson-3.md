# 論文の要約と整理

## 🎯 このレッスンで学ぶこと

このレッスンを完了すると、AIを活用した論文の要約、重要な情報の抽出、論文の分類と整理ができるようになります。

---

## 📖 セクション1: 論文の要約

### 基本的な要約

AIを活用して、論文を要約できます。

**実践例**：
「以下の論文を要約してください。

論文：[論文のテキスト]

以下の項目を含めて要約してください：
1. 研究の目的
2. 研究方法
3. 主要な結果
4. 結論
5. 臨床的意義

各項目を簡潔に、重要な情報のみを含めてください。」

### 構造化された要約

論文を構造化された形式で要約することもできます。

**実践例**：
「先ほどの論文を、以下の形式で構造化してください。

形式：
- 背景
- 方法
- 結果
- 結論
- 限界

各項目を明確に分けて、要約してください。」

---

## 📖 セクション2: 重要な情報の抽出

### 重要な情報の特定

論文から重要な情報を特定できます。

**実践例**：
「以下の論文から、重要な情報を抽出してください。

論文：[論文のテキスト]

以下の項目について抽出してください：
1. 研究の新規性
2. 主要な発見
3. 臨床的意義
4. 研究の限界
5. 今後の課題

各項目について、具体的に示してください。」

### 情報の整理

抽出した情報を整理することもできます。

**実践例**：
「先ほど抽出した情報を整理してください。

以下の形式で整理してください：
- 研究の概要
- 主要な発見
- 臨床的意義
- 研究の限界
- 今後の課題

各項目を明確に分けて、整理してください。」

---

## 📖 セクション3: 論文の分類と整理

### 論文の分類

複数の論文を分類できます。

**実践例**：
「以下の論文を分類してください。

論文1-10：[各論文の情報]

以下の基準で分類してください：
1. 研究デザイン（ランダム化比較試験、コホート研究など）
2. 研究テーマ
3. 研究の質
4. 臨床的意義

各論文について、分類とその理由を示してください。」

### 論文の整理

分類した論文を整理することもできます。

**実践例**：
「先ほど分類した論文を整理してください。

以下の形式で整理してください：
- 研究デザイン別
- 研究テーマ別
- 研究の質別
- 臨床的意義別

各分類について、論文のリストとその特徴を示してください。」

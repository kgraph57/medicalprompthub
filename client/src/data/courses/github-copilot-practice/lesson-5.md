# プロンプトエンジニアリング - 効果的なコメント

## このレッスンで学ぶこと

このレッスンを完了すると、GitHub Copilotに効果的なコメントを書く方法を理解できるようになります。
適切なコメントにより、より良いコードを生成する方法を学びます。

---

## セクション1: 効果的なコメントの書き方

### コメントの重要性：AIへの指示

GitHub Copilotは、コメントからコードを生成するため、コメントの書き方が重要です。

**効果的なコメント：**
- **具体的**：何をしたいか具体的に記述
- **明確**：意図が明確に伝わる
- **コンテキスト**：必要な情報を含める

**医療での例：**
- **良い例**：「患者データのCSVファイルを読み込み、年齢でフィルタリングする」
- **悪い例**：「データを処理する」

---

## セクション2: 段階的な指示

### 複雑な処理：段階的に指示

複雑な処理は、段階的に指示することで、より良いコードが生成されます。

**段階的な指示：**
- **ステップ1**：データの読み込み
- **ステップ2**：データの前処理
- **ステップ3**：モデルの学習
- **ステップ4**：評価

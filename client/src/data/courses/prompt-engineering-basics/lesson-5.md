# 実践的なプロンプト例

## 🎯 このレッスンで学ぶこと

このレッスンを完了すると、実際の医療業務でのプロンプト例を理解し、症例分析、文献レビュー、診断支援などの実践的なプロンプトを作成できるようになります。

---

## 📖 セクション1: 症例分析のプロンプト

### 症例分析プロンプトの例

**実践例1：基本的な症例分析**：
「あなたは経験豊富な内科専門医です。以下の症例について、詳細に分析してください。

症例情報：
- 65歳、男性
- 主訴：胸痛、発汗、呼吸困難
- 現病歴：3時間前から症状出現
- 既往歴：高血圧、糖尿病
- 検査結果：心電図でST上昇、トロポニン陽性

以下の項目について分析してください：
1. 鑑別診断（可能性の高い順に5つ）
2. 各診断の根拠
3. 追加で必要な検査
4. 初期治療方針」

### 複雑な症例の分析

**実践例2：複雑な症例の分析**：
「あなたは経験豊富な内科専門医です。以下の複雑な症例について、段階的に分析してください。

症例情報：[詳細な症例情報]

以下のステップで分析してください：
ステップ1：主訴と現病歴から、可能性のある疾患群を挙げる
ステップ2：検査結果を踏まえて、診断を絞り込む
ステップ3：最も可能性の高い診断と、その根拠を示す
ステップ4：推奨される治療方針を提示する

各ステップで、その理由も説明してください。」

## 📖 セクション2: 文献レビューのプロンプト

### 単一論文のレビュー

**実践例1：単一論文のレビュー**：
「以下の論文を読み込んで、批判的にレビューしてください。

論文：[論文のテキスト]

以下の項目についてレビューしてください：
1. 研究の目的と背景
2. 研究方法の適切性
3. 結果の解釈の妥当性
4. 結論の妥当性
5. 研究の限界と今後の課題

各項目について、具体的な評価と改善案を示してください。」

### 複数論文の比較

**実践例2：複数論文の比較**：
「以下の3つの論文を読み込んで、比較分析してください。

論文1：[論文1のテキスト]
論文2：[論文2のテキスト]
論文3：[論文3のテキスト]

以下の項目について比較してください：
1. 研究方法の違い
2. 結果の違い
3. 結論の違い
4. 研究の質の違い
5. 臨床的意義の違い

各論文の強みと弱みも示してください。」

## 📖 セクション3: 診断支援のプロンプト

### 鑑別診断のリストアップ

**実践例1：鑑別診断のリストアップ**：
「あなたは経験豊富な内科専門医です。以下の患者情報に基づいて、鑑別診断をリストアップしてください。

患者情報：
- 65歳、男性
- 主訴：胸痛、発汗、呼吸困難
- 心電図：ST上昇
- トロポニン：陽性

鑑別診断を、可能性の高い順に5つ挙げてください。各診断について、以下の情報を含めてください：
- 診断名
- 可能性（高/中/低）
- 根拠となる所見
- 追加で必要な検査」

### 診断の根拠の整理

**実践例2：診断の根拠の整理**：
「以下の診断について、診断の根拠を整理してください。

診断：急性心筋梗塞
患者情報：[患者情報]

以下の形式で診断の根拠を整理してください：
- 症状からの根拠
- 検査結果からの根拠
- 画像所見からの根拠
- その他の根拠

各根拠について、具体的な所見を示してください。」

## 📖 セクション4: 論文執筆のプロンプト

### 症例報告書の作成

**実践例1：症例報告書の作成**：
「あなたは経験豊富な医学研究者です。以下の症例情報に基づいて、症例報告書を作成してください。

症例情報：[症例情報]

以下のセクションを含めてください：
1. Introduction：症例の新規性と教育的意義
2. Case Presentation：症例の詳細な提示
3. Discussion：症例の考察と意義

各セクションについて、専門用語を適切に使用し、簡潔に記述してください。」

### 研究論文の作成

**実践例2：研究論文の作成**：
「あなたは経験豊富な医学研究者です。以下の研究結果に基づいて、研究論文のAbstractを作成してください。

研究結果：[研究結果]

以下の項目を含めてください：
- 背景
- 方法
- 結果
- 結論

250文字以内で、専門用語を適切に使用してください。」


# ステップ2: 目的別の診断書を作成する

## 文書の「受け取り手」を意識する

診断書は、その目的によって記載すべき内容の重点や、求められる情報の種類が異なります。文書の「受け取り手」が誰で、その人が何を知りたいのかを意識することが、質の高い診断書を作成する上での鍵となります。

AIに指示を出す際も、この「目的」と「受け取り手」を明確に伝えることで、より適切で実用的なドラフトを生成させることができます。

---

## 1. 身体疾患による休職診断書

**目的:** 身体的な病気の治療に専念するため、一定期間の休職が必要であることを会社に証明する。
**受け取り手:** 会社の人事・労務担当者、産業医
**ポイント:** なぜ就労が困難なのか、その医学的根拠と、必要な休養期間の見込みを明確に記述する。

### 📋 プロンプト（コピペ用）

```
あなたは経験豊富な外科医です。以下の患者情報に基づき、勤務先に提出するための「休職診断書」のドラフトを作成してください。

【患者情報】
*   氏名: 
*   主病名: [例: 胃がん]
*   手術日: [例: 2025年12月1日]
*   術式: [例: 腹腔鏡下幽門側胃切除術]
*   術後経過: 良好。合併症なし。
*   現在の状況: 食事摂取量はまだ少なく、体力も十分に回復していない。創部の痛みは軽度残存。

【診断書の目的】
胃がん術後の体力回復と、補助化学療法に備えるため、術後約2ヶ月間の自宅療養が必要であることを証明する。

【出力形式】
一般的な診断書のフォーマットで、就労が困難である理由と、必要な休養期間を具体的に記述してください。
```

## 2. 労災（労働災害）申請用の診断書（意見書）

**目的:** 発生した傷病が、業務に起因するものであるという医学的意見を証明し、労働基準監督署に提出する。
**受け取り手:** 労働基準監督署の担当官
**ポイント:** 傷病と業務との間の**因果関係**について、客観的な事実に基づいて医学的な意見を述べることが最も重要。

### 📋 プロンプト（コピペ用）

```
あなたは経験豊富な整形外科医です。以下の患者情報に基づき、労働基準監督署に提出するための「労災申請用の診断書（意見書）」のドラフトを作成してください。

【患者情報】
*   氏名: 
*   職業: [例: 建設作業員]
*   受傷日・時刻: [例: 2025年12月10日 午前10時頃]
*   受傷機転: [例: 建設現場で脚立（高さ2m）から転落し、右足首を強く捻った。]
*   主病名: [例: 右足関節外果骨折]
*   治療内容: [例: 観血的整復固定術を施行]

【診断書の目的】
上記の傷病が、業務遂行中に発生したものであるという医学的意見を証明する。

【出力形式】
労災申請用の意見書のフォーマットを参考に、「傷病の発生状況」と「業務との因果関係についての意見」を明確に区別して記述してください。
```

### AIの出力例（労災申請用）

```
労働者災害補償保険 診断書（意見書）

【患者氏名】 〇〇 〇〇 様

【傷病名】 右足関節外果骨折

【傷病の発生状況】
本人からの聴取によれば、上記の傷病は、2025年12月10日午前10時頃、〇〇建設の業務として建設現場で作業中、高さ約2mの脚立から転落した際に右足首を強く捻り発生したものである。

【初診年月日】 2025年12月10日

【症状、所見及び治療の経過】
初診時、右足関節に著明な腫脹、疼痛、皮下出血を認めた。X線検査にて、右腓骨遠位端の骨折線を明瞭に認め、右足関節外果骨折と診断した。転位が大きいため、同日、観血的整復固定術を施行した。

【業務との因果関係に関する意見】
患者の申告する受傷機転（高所からの転落）は、本件骨折を生じさせる原因として医学的に十分考えられます。また、診察時の所見と受傷機転との間に矛盾は認められません。
以上のことから、本件傷病は、業務に起因して発生したものと判断するのが妥当であると考えます。

【今後の見通し】
治癒（症状固定）までには、術後約6ヶ月を要する見込みです。その間、リハビリテーションによる機能回復訓練が必要となります。

以上

2025年12月13日

医療機関名: 〇〇整形外科
医師氏名: (あなたの名前) ㊞
```

## まとめ

このステップで学んだこと:

✅ 診断書の目的と受け取り手を意識した情報提供の重要性
✅ 休職診断書や労災申請用診断書など、特定の目的に特化した診断書作成のポイントとプロンプト

**【重要】**
特に労災や公的給付に関する診断書は、患者のその後の人生に大きな影響を与えます。AIのドラフトを参考にしつつも、**客観的な事実と、ご自身の専門家としての誠実な意見**を、責任を持って記述してください。

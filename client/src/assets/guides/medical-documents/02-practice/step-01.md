
# ステップ1: 退院サマリーを作成する

## 継続的なケアのための情報集約

退院サマリー（退院時要約）は、一人の患者の入院中の情報を集約し、退院後のケアを担当するかかりつけ医、訪問看護師、介護スタッフなど、関係者全員が患者の状態を正確に把握するための、極めて重要な文書です。

質の高い退院サマリーは、退院後の再入院を防ぎ、シームレスな地域連携を実現するための鍵となります。しかし、入院期間が長くなればなるほど、その情報を簡潔かつ網羅的にまとめるのは大変な作業です。AIは、この複雑な情報整理と文書化のプロセスを大幅に効率化します。

---

## 退院サマリー作成プロンプト

このプロンプトは、入院中の様々な情報をAIに与え、構造化された退院サマリーのドラフトを生成させるためのものです。

### 📋 退院サマリープロンプト（コピペ用）

```
あなたは、経験豊富な病棟主治医です。以下の患者の入院経過に関する情報を基に、退院後のケアを担当するかかりつけ医向けの、詳細な退院サマリーのドラフトを作成してください。

【患者情報】
*   氏名: 
*   年齢・性別: 
*   入院日: 
*   退院日: 

【入院情報】
*   **主病名（入院の主たる原因）:** 
*   **副病名（併存疾患）:** 
*   **入院までの経緯:** 
*   **入院中の主要な経過と治療:**
    *   [例: 循環器系: 心不全に対して利尿薬の調整を行い、体重は〇kg減少。症状は安定。]
    *   [例: 呼吸器系: 誤嚥性肺炎に対し、抗菌薬〇〇を〇日間投与し、炎症反応は改善。]
    *   [例: リハビリテーション: ADL（日常生活動作）向上のため、理学療法・作業療法を施行。退院時のADLは〇〇。]
*   **主要な検査結果:**
    *   [退院前の血液検査、画像検査の要点など]
*   **退院時処方:**
    *   [薬剤名、用法・用量をリスト形式で]
*   **退院後の生活指導・注意点:**
    *   [食事（塩分制限など）、運動、次回の外来受診日など]

【出力形式】
一般的な退院サマリーのフォーマットに従い、専門的かつ分かりやすい言葉で記述してください。特に、退院後のケアを担当するかかりつけ医が注意すべき点や、今後のフォローアップでお願いしたい点を明確にしてください。
```

## 実践例: 心不全増悪による入院患者の退院サマリー

### 入力プロンプト

```
あなたは、経験豊富な循環器内科医です。以下の患者の入院経過に関する情報を基に、退院後のケアを担当するかかりつけ医向けの、詳細な退院サマリーのドラフトを作成してください。

【患者情報】
*   氏名: 高橋 清 様
*   年齢・性別: 82歳・女性
*   入院日: 2025年11月28日
*   退院日: 2025年12月13日

【入院情報】
*   主病名: 急性心不全（HFrEFの急性増悪）
*   副病名: 慢性心房細動、2型糖尿病、慢性腎臓病（CKD G3b）
*   入院までの経緯: 1週間前からの呼吸困難と下腿浮腫の増悪を主訴に救急外来を受診。胸部X線でうっ血所見、BNP 1200 pg/mLと高値のため、心不全増悪と診断し緊急入院。
*   入院中の経過: フロセミド静注にて利尿を図り、体重は5kg減少。呼吸困難および浮腫は著明に改善。心不全治療薬（ARNI、β遮断薬、MRA）を再調整し、SGLT2阻害薬を新規導入。糖尿病・腎機能については大きな変動なく経過。
*   退院時処方: サクビトリルバルサルタン、ビソプロロール、スピロノラクトン、ダパグリフロジン、アピキサバン、インスリングラルギン
*   退院後の注意点: 厳格な塩分制限（6g/日未満）と水分管理。毎日体重を測定し、2kg以上の増加があれば連絡するよう指導。次回の当科外来は1ヶ月後。
```

### AIの出力例

```
退院時要約

かかりつけ医 〇〇先生 御机下

いつも大変お世話になっております。
貴院よりご紹介いただきました下記患者様が、この度、当院を退院されましたので、入院中の経過をご報告申し上げます。

**患者氏名:** 高橋 清 様 (82歳・女性)

**入院期間:** 2025年11月28日 ～ 2025年12月13日

**退院時診断**
*   主病名: 急性心不全 (HFrEFの急性増悪)
*   副病名: 慢性心房細動、2型糖尿病、慢性腎臓病（CKD G3b）

**入院経過**
1週間前からの呼吸困難と下腿浮腫の増悪を主訴に当院救急外来を受診されました。BNP 1200 pg/mLと著明高値を認め、急性心不全の増悪と診断し、同日入院となりました。

入院後、フロセミド静注による利尿薬治療を開始し、うっ血の解除に努めました。治療は奏功し、体重は入院時より5kg減少し、呼吸困難および下腿浮腫は著明に改善いたしました。

うっ血解除後、心不全の予後改善薬であるARNI、β遮断薬、MRAの再調整を行うとともに、新たな標準治療薬であるSGLT2阻害薬（ダパグリフロジン）を新規に導入いたしました。導入後も腎機能の悪化や有害事象なく、循環動態は安定しております。

**退院時処方**
*   サクビトリルバルサルタン錠 100mg 1回1錠 1日2回
*   ビソプロロールフマル酸塩錠 2.5mg 1回1錠 1日1回
*   スピロノラクトン錠 25mg 1回1錠 1日1回
*   ダパグリフロジン錠 10mg 1回1錠 1日1回
*   アピキサバン錠 2.5mg 1回1錠 1日2回
*   インスリン グラルギン注 10単位 1日1回

**退院後のフォローアップについて**
退院後は、再び貴院にて内科的フォローアップを賜りますようお願い申し上げます。
特に、以下の点にご留意いただけますと幸いです。

1.  **うっ血の再燃モニタリング:** 体重増加、浮腫の再燃、労作時呼吸困難の出現にご注意ください。
2.  **腎機能・電解質のフォロー:** SGLT2阻害薬、ARNI、MRAを併用しておりますので、定期的な採血によるフォローをお願いいたします。
3.  **服薬アドヒアランスの維持:** 多くの薬剤が予後改善に重要であることを、引き続きご指導いただければと存じます。

次回の当科外来は、1ヶ月後を予定しております。

今後とも何卒よろしくお願い申し上げます。

2025年12月13日

〇〇病院 循環器内科
主治医: (あなたの名前)
```

## まとめ

このステップで学んだこと:

✅ 退院サマリーが継続的なケアにおいて果たす重要な役割
✅ 入院中の複雑な情報を整理し、AIに退院サマリーのドラフトを作成させるためのプロンプト

**【重要】**
退院サマリーは、退院後の患者の安全を守るための命綱です。AIの出力を鵜呑みにせず、**カルテと照合し、全ての情報が正確であることを徹底的に確認**してください。特に、**薬剤の用量**には細心の注意が必要です。

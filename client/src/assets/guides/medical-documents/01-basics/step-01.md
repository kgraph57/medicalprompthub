
# ステップ1: 診断書のドラフトを生成する

## 診断書の基本構成

診断書は、患者の健康状態について、医師が専門的な見地から証明する公的な文書です。その目的は、休職、保険請求、公的支援の申請など多岐にわたります。どのような目的であれ、診断書には共通して記載すべき基本要素があります。

**診断書の基本構成要素:**

1.  **患者情報:** 氏名、生年月日、性別、住所
2.  **診断名:** 主病名、および関連する副病名（ICD-10コードを併記することもある）
3.  **発症・診断日:** 症状が始まった日、および診断が確定した日
4.  **診断の根拠:** 症状、身体所見、検査結果など、診断に至った客観的な根拠
5.  **治療内容と経過:** これまで行ってきた治療（投薬、手術など）とその効果
6.  **現在の状況と今後の見通し:** 現在の症状の程度、治療状況、今後の回復の見込みなど
7.  **特記事項:** （目的に応じて）就労に関する意見、必要な配慮、後遺症の有無など
8.  **作成日、病院名、医師名、押印**

これらの要素を網羅し、目的に応じて記載の詳しさを調整することが重要です。

---

## 診断書作成の基本プロンプト

このプロンプトは、患者の基本情報と診断書の目的をAIに伝えることで、上記の構成要素を網羅した診断書のドラフトを生成させるためのものです。

### 📋 診断書基本プロンプト（コピペ用）

```
あなたは、様々な診療科の経験が豊富な医師です。以下の患者情報に基づき、指定された目的の診断書のドラフト（下書き）を作成してください。

【患者情報】
*   氏名: [氏名]
*   生年月日: [生年月日]
*   主病名: [主病名]
*   診断日: [診断日]
*   診断根拠: [症状、所見、検査結果などを簡潔に]
*   治療経過: [これまでの治療内容を簡潔に]
*   現在の状況: [現在の症状や状態を簡潔に]

【診断書の目的】
[例: 会社に提出し、病気療養のための休職を申請する]

【出力形式】
一般的な診断書のフォーマットに従って、丁寧な言葉遣いで記述してください。特に、診断書の目的に関連する「今後の見通し」や「必要な配慮」については、具体的に記述してください。
```

## 実践例: うつ病による休職診断書

### 入力プロンプト

```
あなたは、経験豊富な精神科医です。以下の患者情報に基づき、指定された目的の診断書のドラフト（下書き）を作成してください。

【患者情報】
*   氏名: 山田 太郎
*   生年月日: 1985年4月10日
*   主病名: うつ病 (F32.1)
*   診断日: 2025年11月15日
*   診断根拠: 抑うつ気分、興味・喜びの喪失、不眠、食欲不振、思考力・集中力の低下が持続。心理検査（SDS）でも高得点。
*   治療経過: 抗うつ薬（SSRI）による薬物療法と、支持的精神療法を開始。
*   現在の状況: 症状は依然として重く、日中の倦怠感が強いため、就労は困難な状態。

【診断書の目的】
勤務先に提出し、病気療養のための休職を申請する。

【出力形式】
一般的な診断書のフォーマットに従って、丁寧な言葉遣いで記述してください。特に、診断書の目的に関連する「今後の見通し」や「必要な配慮」については、具体的に記述してください。
```

### AIの出力例

```
診断書

氏名: 山田 太郎 様
生年月日: 1985年4月10日 (満40歳)
住所: （患者の住所）

【傷病名】
うつ病 (ICD-10コード: F32.1)

【診断根拠および経過】
上記患者は、2025年10月頃より、持続的な抑うつ気分、興味・喜びの喪失、不眠、食欲不振、思考力・集中力の低下といった症状を自覚され、当院を受診されました。
診察所見および心理検査の結果から、2025年11月15日、うつ病と診断いたしました。
現在、抗うつ薬（SSRI）による薬物療法、および支持的精神療法による加療を行っております。

【現在の状況および今後の見通し】
現在も上記精神症状は持続しており、特に日中の強い倦怠感や意欲の低下が顕著です。このため、現状では集中力や判断力を要する業務に従事することは極めて困難な状態と判断いたします。

つきましては、十分な休養と治療に専念するため、本日より約3ヶ月間の休職が必要と考えられます。
休職期間中は、定期的な通院加療により、症状の改善と職場復帰に向けた調整を行っていく予定です。

職場におかれましては、まずは治療に専念できる環境を整えていただけますよう、ご配慮のほどお願い申し上げます。

以上、診断いたします。

2025年12月13日

医療機関名: 〇〇病院 精神科
所在地: （病院の所在地）
医師氏名: (あなたの名前)  ㊞
```

**【重要】**
この出力はあくまでドラフトです。必ず、**ご自身の専門的判断に基づき、内容の事実確認、医学的妥当性の検証、そして患者一人ひとりの状況に合わせた修正**を加えてから、最終的な文書として完成させてください。

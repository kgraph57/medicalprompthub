
# ステップ3: 想定問答集を作成する

## なぜ想定問答集が必要か？

インフォームド・コンセントの場では、患者さんやご家族から様々な質問が寄せられます。事前にこれらの質問を予測し、的確で分かりやすい回答を準備しておくことは、以下の点で非常に重要です。

*   **信頼関係の構築:** その場でよどみなく回答できることで、患者は医師への信頼を深めます。
*   **不安の軽減:** 患者が抱える具体的な疑問や不安を解消できます。
*   **説明の効率化:** よくある質問への回答を準備しておくことで、説明時間を短縮できます。
*   **法的リスクの回避:** 「説明が不十分だった」という後のトラブルを防ぎます。

AIは、多様な視点から質問を生成するのに非常に長けています。この能力を活用して、網羅的な想定問答集を作成しましょう。

## 想定問答集を生成するプロンプト

### 📋 基本プロンプト（コピペ用）

```
あなたは経験豊富な医師であり、同時に様々な背景を持つ患者や家族の気持ちも深く理解しています。以下の状況設定に基づき、患者さんやご家族から出そうな質問を、できるだけ多く、多様な視点からリストアップしてください。そして、それぞれの質問に対して、共感的で分かりやすい回答例を作成してください。

【状況設定】
*   疾患名: [例: 大腸がん]
*   治療法: [例: 腹腔鏡下結腸切除術]
*   患者背景: [例: 50代、働き盛り、インターネットで情報をよく調べる]

【出力形式】
以下のカテゴリーに分けて、Q&A形式で記述してください。

1.  **病気そのものに関する質問** (原因、進行度、再発率など)
2.  **治療法に関する質問** (手術の成功率、副作用、代替治療など)
3.  **入院・術後の生活に関する質問** (入院期間、食事、仕事復帰、後遺症など)
4.  **費用に関する質問** (医療費、高額療養費制度など)
```

## 実践例: 化学療法の想定問答集

### 入力プロンプト

```
あなたは経験豊富な腫瘍内科医であり、同時に様々な背景を持つ患者や家族の気持ちも深く理解しています。以下の状況設定に基づき、患者さんやご家族から出そうな質問を、できるだけ多く、多様な視点からリストアップしてください。そして、それぞれの質問に対して、共感的で分かりやすい回答例を作成してください。

【状況設定】
*   疾患名: 進行胃がん術後
*   治療法: 術後補助化学療法（S-1＋ドセタキセル併用療法）
*   患者背景: 68歳男性。手術を乗り越えたが、抗がん剤治療への不安が強い。

【出力形式】
以下のカテゴリーに分けて、Q&A形式で記述してください。

1.  **治療の必要性に関する質問**
2.  **副作用に関する具体的な質問**
3.  **日常生活に関する質問**
4.  **精神的な不安に関する質問**
```

### AIの出力例

```
【化学療法の想定問答集】

#### 1. 治療の必要性に関する質問

**Q: 手術でがんは全部取れたはずなのに、なぜ抗がん剤治療が必要なのですか？**
**A:** はい、おっしゃる通り、目に見えるがんは手術で全て取りきることができました。しかし、画像検査では見つけられないほど小さな、ミクロのがん細胞が、血液やリンパの流れに乗って体のどこかに隠れている可能性があります。この「見えない敵」を叩いて、将来の再発のリスクをできるだけ下げるために行うのが、術後の抗がん剤治療です。いわば、「念のための治療」ですね。

**Q: この治療を受ければ、絶対に再発しませんか？**
**A:** 残念ながら、「絶対に再発しない」と100%保証することはできません。しかし、多くの研究から、この治療を行うことで、再発のリスクを明らかに下げられることが分かっています。何もしない場合と比べて、再発率を約10-15%程度減らすことができると期待されています。この数字をどう捉えるかですが、私たちは、この可能性のために治療を受ける価値は十分にあると考えています。

#### 2. 副作用に関する具体的な質問

**Q: 髪の毛は抜けますか？**
**A:** はい、この治療では、高い確率で脱毛が起こります。治療開始から2-3週間ほどで抜け始め、治療中は脱毛した状態が続きます。これは多くの患者さんにとって非常につらい副作用だと思います。事前に帽子やウィッグ（医療用かつら）を準備される方も多いです。自治体によっては助成金が出る場合もありますので、相談員にお繋ぎしますね。治療が終われば、また必ず生えてきますから、安心してください。

**Q: 吐き気はどのくらい続きますか？食事はとれますか？**
**A:** 吐き気は、最も心配な副作用の一つですよね。最近は、吐き気を抑える非常に良いお薬がたくさんあります。点滴の日とその後の数日間、予防的に吐き気止めを使いますので、以前と比べてずっと楽に過ごせるようになっています。もちろん個人差はありますが、全く食事がとれなくなる方は稀です。もし食事が難しいときは、点滴で栄養を補うこともできますので、無理しないでくださいね。

**Q: 副作用が辛かったら、治療を休むことはできますか？**
**A:** もちろんです。治療のスケジュールは大切ですが、副作用が強すぎて日常生活が送れないほどであれば、お薬の量を減らしたり、治療を一時的にお休みしたりします。絶対に無理はしないで、辛いときはすぐに私たち医療スタッフに伝えてください。患者さんの体調が最優先です。

#### 3. 日常生活に関する質問

**Q: 治療中でも、散歩や趣味のゲートボールはできますか？**
**A:** はい、体調が良い日であれば、ぜひ続けてください。無理のない範囲での散歩や軽い運動は、体力を維持し、気分転換にもなるので、むしろお勧めします。ただし、治療中は感染症にかかりやすくなるので、人混みを避け、手洗いやうがいをしっかり行うようにしてください。

**Q: 孫に会っても大丈夫ですか？うつしたりしませんか？**
**A:** 大丈夫ですよ。抗がん剤治療は、ご自身の体に作用するもので、風邪のように他の人にうつることはありません。お孫さんと会うことは、何よりの心の支えになりますよね。ただし、先ほどお話ししたように、外から感染をもらわないように、お孫さんが風邪をひいているときなどは、少し接触を控えた方が安心かもしれません。

#### 4. 精神的な不安に関する質問

**Q: 治療のことを考えると、夜も眠れなくなります。どうしたらいいですか？**
**A:** これから始まる未知の治療に対して、不安で眠れなくなるのは当然のことです。一人で抱え込まないでください。私たち医師や看護師に、いつでもそのお気持ちを話してください。また、同じような経験をした他の患者さんとお話しできる「患者サロン」のような場もあります。必要であれば、心の専門家である臨床心理士や、睡眠を助けるお薬の相談もできます。利用できるサポートは全て利用して、一緒に乗り越えていきましょう。
```

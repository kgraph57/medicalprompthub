
# 患者説明シナリオ作成：AIでわかりやすいインフォームド・コンセント

## はじめに

「先生、説明が難しくてよく分かりませんでした…」

インフォームド・コンセント（IC）は、医療の基本です。しかし、多忙な臨床現場で、患者さん一人ひとりの理解度に合わせて、病状や治療方針を分かりやすく説明する時間を確保するのは、多くの医師にとって大きな課題です。

このガイドでは、**AIを活用して、患者説明のシナリオ（台本）を効率的に作成する方法**を紹介します。病名や治療法を入力するだけで、患者さんの背景に合わせた、分かりやすく、共感的な説明シナリオをAIが生成します。

---

## なぜAI支援が必要なのか？

### 患者説明の課題

| 医師側の課題 | 患者側の課題 |
| :--- | :--- |
| **時間がない** | 専門用語が分からない |
| **説明が画一的になりがち** | 不安で頭が真っ白になる |
| **どこまで説明すべきか悩む** | 何を質問していいか分からない |
| **法的リスクへの懸念** | 家族と意見が違う |

これらの課題は、医師と患者の間のコミュニケーションギャップを生み、信頼関係の構築を妨げる原因となります。

### AI支援の効果

AIを活用することで、これらの課題を解決し、より質の高い患者説明を実現できます。

1.  **時間短縮:** 説明シナリオ作成時間を大幅に削減（例: 15分 → 3分）
2.  **質の標準化:** 重要な説明項目（診断、治療選択肢、リスク、予後）の漏れを防ぐ
3.  **個別化:** 患者の年齢、理解度、背景に合わせた説明を生成
4.  **共感的な表現:** 不安を和らげるような、配慮のある言葉遣いを提案
5.  **多言語対応:** 外国人の患者さんへの説明資料作成も可能

---

## このガイドで学べること

1.  **患者説明の基本構造（SPIKESモデルなど）**を理解する
2.  **AIプロンプトの実践的な使い方**を習得し、質の高い説明シナリオを生成する
3.  **様々な疾患・治療**（例: 癌告知、手術説明、生活習慣病指導）に対応したシナリオを作成する
4.  **難しい状況**（例: 悪い知らせ、治療中止）でのコミュニケーション方法を学ぶ
5.  **患者からの想定質問**とその回答を準備する方法

---

## 対象となる説明の場面

このガイドは、以下のような様々な臨床場面で活用できます。

*   **病状説明:** 診断の告知、検査結果の説明
*   **治療方針の説明:** 手術、化学療法、放射線治療のインフォームド・コンセント
*   **生活習慣病指導:** 糖尿病、高血圧の食事・運動療法
*   **悪い知らせを伝える場面 (Breaking Bad News):** 癌の告知、再発・転移の説明
*   **DNAR（蘇生措置拒否）に関する話し合い**

## このガイドの使い方

### ステップ1: 基本編（10分）
*   患者説明の基本モデル（SPIKES）を学ぶ
*   AIプロンプトの基本を理解する

### ステップ2: 実践編（20分）
*   疾患別の説明シナリオを生成する（癌告知、生活習慣病など）
*   患者の背景に合わせたカスタマイズ方法を学ぶ
*   想定問答集を作成する

### ステップ3: 応用編（10分）
*   「悪い知らせ」を伝えるシナリオの作成
*   説明用資料（図やイラスト）のアイデア出し

## 注意事項

AIが生成したシナリオは、あくまで**「下書き」**です。必ずご自身の言葉で、患者さんの表情や反応を見ながら、心を込めて説明してください。AIはコミュニケーションの補助ツールであり、医師自身の対話能力を代替するものではありません。

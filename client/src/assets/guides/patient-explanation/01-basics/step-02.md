# ステップ2: AIプロンプトで説明シナリオを生成する

## AIを共感的な対話パートナーにする

AIに患者説明シナリオを作成させる際の最も重要なコツは、AIに**「共感的で経験豊富な医師」という役割を与える**ことです。これにより、単なる事実の羅列ではない、温かみのある対話形式のシナリオを生成させることができます。

---

## 基本プロンプトテンプレート

このテンプレートは、あらゆる疾患・場面で応用可能です。SPIKESモデルの要素が組み込まれており、患者の背景に合わせてカスタマイズできます。

### 📋 プロンプトテンプレート（コピペ用）

```
あなたは共感的で経験豊富な医師です。以下の情報に基づき、患者さんへの説明シナリオを、SPIKESモデルに沿って作成してください。

【患者背景】
*   年齢・性別: [例: 65歳・男性]
*   理解度: [例: 高い / 普通 / 低い / 専門用語は避けてほしい]
*   性格・感情: [例: 不安が強い / 前向き / 家族の意見を気にしている]
*   同席者: [例: 配偶者と長男]

【説明内容】
*   診断名: [例: 2型糖尿病]
*   病状: [例: HbA1c 9.5%、軽度の腎症を合併]
*   治療方針: [例: 食事療法、運動療法、経口血糖降下薬の開始]
*   予後・リスク: [例: 放置した場合の合併症（網膜症、神経障害、心筋梗塞など）]

【出力形式】
*   医師と患者の対話形式で記述してください。
*   患者の感情に寄り添い、共感的な言葉かけを含めてください。
*   専門用語は避け、平易な言葉で説明してください。
*   最後に、患者からの想定質問を3つと、それに対する回答例を挙げてください。
```

## 実践例1: 2型糖尿病の新規診断

### 入力プロンプト

```
あなたは共感的で経験豊富な医師です。以下の情報に基づき、患者さんへの説明シナリオを、SPIKESモデルに沿って作成してください。

【患者背景】
*   年齢・性別: 55歳・男性
*   理解度: 普通
*   性格・感情: 仕事が忙しく、生活習慣を変えることに抵抗がある
*   同席者: なし

【説明内容】
*   診断名: 2型糖尿病
*   病状: 健康診断で高血糖を指摘。HbA1c 8.5%。自覚症状はなし。
*   治療方針: 食事療法、運動療法、メトホルミンの開始
*   予後・リスク: 放置した場合の合併症（網膜症、腎症、神経障害）

【出力形式】
*   医師と患者の対話形式で記述してください。
*   患者の感情に寄り添い、共感的な言葉かけを含めてください。
*   専門用語は避け、平易な言葉で説明してください。
*   最後に、患者からの想定質問を3つと、それに対する回答例を挙げてください。
```

### AIの出力例

```
【患者説明シナリオ：2型糖尿病】

**医師:** 「〇〇さん、こんにちは。先日の検査結果が出ましたので、お話しさせてください。少しお時間をいただけますか？」 (S: Setting up)

**患者:** 「はい、お願いします。」

**医師:** 「結果についてお話しする前に、〇〇さんご自身は、今回の検査結果について何か心配なことや、ご自身で感じていらっしゃることはありますか？」 (P: Perception)

**患者:** 「いや、特に自覚症状はないんですけどね。健康診断で血糖が高いって言われたくらいで…」

**医師:** 「そうですよね、自覚症状がないと、なかなかピンとこないですよね。これから検査結果について、詳しくご説明させていただいてもよろしいでしょうか？」 (I: Invitation)

**患者:** 「はい、お願いします。」

**医師:** 「ありがとうございます。結論から申し上げますと、先日の検査の結果、〇〇さんは『2型糖尿病』という状態であることが分かりました。」 (K: Knowledge - 予告と告知)

**患者:** 「糖尿病…ですか。やっぱり…。」

**医師:** 「はい。そう診断されたことを聞いて、今、どんなお気持ちですか？」 (E: Emotions)

**患者:** 「うーん、ショックというか…。仕事も忙しいし、食事制限とか運動とか、正直できる自信がないですね…。」

**医師:** 「お仕事がお忙しい中で、生活を変えるのは大変だと感じられるお気持ち、よく分かります。多くの方が同じように感じられますよ。でも、ご安心ください。今からしっかり対策をすれば、合併症を防ぎ、これまで通りの生活を続けることは十分可能です。これから一緒に、〇〇さんに合ったやり方を見つけていきましょう。」 (E: 共感と希望)

**医師:** 「まず、なぜ治療が必要かをご説明しますね。糖尿病は、血液の中の糖分が多くなりすぎる病気です。これを放置すると、将来的に目の病気（網膜症）や腎臓の病気（腎症）、手足のしびれ（神経障害）といった合併症が出てくる可能性があります。でも、これはあくまで『何もしなかった場合』の話です。」 (K: 知識の伝達)

**医師:** 「そこで、今後の計画ですが、まずは3つのことに取り組んでいきたいと考えています。一つは食事、二つ目は運動、そして三つ目はお薬の助けを借りることです。お仕事が忙しい〇〇さんの生活に合わせて、無理なく続けられる方法を一緒に探していきましょう。例えば、食事ならまず夕食の白米を半分にしてみるとか、運動なら通勤で一駅手前で降りて歩いてみるとか、小さなことからで大丈夫です。」 (S: Strategy & Summary)

**患者:** 「なるほど…。少しなら、できるかもしれません。」

**医師:** 「素晴らしいです。まずはそのお気持ちが一番大切です。今日のお話で、何か分からなかったことや、不安なことはありますか？」 (S: 要約と確認)

---

【想定質問と回答例】

**Q1: 糖尿病は一生治らないんですか？**
**A1:** 糖尿病は「治る」というより、「うまく付き合っていく」病気です。食事や運動、お薬で血糖値を良い状態にコントロールし続けることができれば、合併症を防ぎ、健康な方と変わらない生活を送ることができます。

**Q2: 薬はすぐに始めないといけないんですか？**
**A2:** 〇〇さんの場合、少し血糖値が高めなので、食事や運動だけだと下がるのに時間がかかる可能性があります。お薬の助けを少し借りて、早く良い状態にすることで、将来の合併症のリスクをぐっと減らすことができます。一番負担の少ないお薬から始めてみましょう。

**Q3: 家族に何と説明すればいいでしょうか？**
**A3:** 「健康診断で糖尿病と診断された。でも、これから食事や運動に気をつけて、先生と一緒に治療を頑張っていくから大丈夫だよ」と伝えてみてはいかがでしょうか。ご希望であれば、次回ご家族と一緒に説明を聞いていただくこともできますよ。
```

## プロンプトのカスタマイズ

### 1. 患者の理解度に合わせる

*   **子供向け:** `【患者背景】理解度: 小学生にも分かるように、例え話をたくさん使ってください。`
*   **高齢者向け:** `【患者背景】理解度: 高齢者向けに、ゆっくり、一つずつ、大きな文字で書くように説明してください。`

### 2. 患者の感情に合わせる

*   **不安が強い患者:** `【患者背景】性格・感情: 非常に不安が強く、最悪の事態を心配している。安心させる言葉を多めに含めてください。`
*   **否認傾向の患者:** `【患者背景】性格・感情: 自分は病気ではないと思っており、治療に協力的でない。治療の必要性を、脅すのではなく、本人のメリットとして伝えてください。`

### 3. 特定の状況に合わせる

*   **多忙なビジネスパーソン:** `【患者背景】性格・感情: 仕事が最優先で、治療に時間をかけたくない。効率的で、仕事と両立できる治療法を強調してください。`

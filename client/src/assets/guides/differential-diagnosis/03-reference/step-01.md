
# ステップ1: 複雑な症例での鑑別診断

## 複雑な症例とは？

実臨床では、単一の主訴だけでなく、複数の問題が絡み合った複雑な症例に遭遇します。

*   **複数の主訴:** 例）発熱と腹痛と意識障害
*   **複数の既往歴:** 例）糖尿病、腎不全、心不全を合併
*   **非典型的な症状:** 教科書通りではない症状

このような場合、鑑別診断は困難を極めます。AIを壁打ち相手として活用し、思考を整理しましょう。

---

## プロブレムリストを作成する

まず、症例の情報を**プロブレムリスト**として整理します。これは、診断の手がかりとなる陽性所見や異常値をリストアップする作業です。

### プロブレムリスト作成のプロンプト

```
あなたは経験豊富な臨床医です。以下の症例情報から、プロブレムリストを作成してください。

【症例情報】
[ここに症例情報を貼り付け]

【出力形式】
重要な順にプロブレムをリストアップし、それぞれに簡単な解説を加えてください。
```

### 実践例: 発熱と意識障害の症例

**症例情報:**
> 75歳男性。3日前からの発熱と食欲不振を主訴に来院。来院時、JCS II-10と意識障害を認める。既往に糖尿病、高血圧、2年前に前立腺癌で手術歴あり。バイタルは体温38.5℃、血圧90/50mmHg、脈拍110/分、SpO2 92%。身体所見で項部硬直はなし。血液検査でWBC 18,000, CRP 25, Cre 2.5。尿検査で混濁あり。

**入力プロンプト:**
```
あなたは経験豊富な臨床医です。以下の症例情報から、プロブレムリストを作成してください。

【症例情報】
75歳男性。3日前からの発熱と食欲不振を主訴に来院。来院時、JCS II-10と意識障害を認める。既往に糖尿病、高血圧、2年前に前立腺癌で手術歴あり。バイタルは体温38.5℃、血圧90/50mmHg、脈拍110/分、SpO2 92%。身体所見で項部硬直はなし。血液検査でWBC 18,000, CRP 25, Cre 2.5。尿検査で混濁あり。

【出力形式】
重要な順にプロブレムをリストアップし、それぞれに簡単な解説を加えてください。
```

**AIの出力例:**
```
【プロブレムリスト】

#1. 敗血症性ショック (Sepsis with Shock)
   - 解説: 発熱、頻脈、低血圧、意識障害、高炎症反応から、SIRS基準を満たし、臓器障害（意識障害、腎機能障害）を伴う敗血症であり、循環不全（ショック）を合併している状態です。最も緊急性の高い問題です。

#2. 意識障害 (Disturbance of Consciousness)
   - 解説: JCS II-10。敗血症による脳症、中枢神経系感染症、代謝性脳症など、多岐にわたる原因を考慮する必要があります。

#3. 急性腎障害 (Acute Kidney Injury, AKI)
   - 解説: Cre 2.5。敗血症による腎前性腎不全が最も考えられますが、薬剤性や腎後性も鑑別が必要です。

#4. 感染巣 (Infection Focus)
   - 解説: 敗血症の原因となっている感染巣の特定が急務です。尿混濁があることから尿路感染症が最も疑われますが、肺炎、胆管炎、カテーテル関連血流感染症なども鑑別します。

#5. 糖尿病 (Diabetes Mellitus)
   - 解説: 感染症の易感染性や、高血糖による意識障害（HHSなど）のリスク因子となります。

#6. 前立腺癌術後 (Post-prostatectomy state)
   - 解説: 尿路の構造変化により、複雑性尿路感染症のリスク因子となります。
```

## 複数の問題を統合して鑑別診断を考える

プロブレムリストを基に、それらを**一元的に説明できる疾患**は何か、AIに問いかけます。

### 鑑別診断の統合プロンプト

```
あなたは経験豊富な臨床医です。先ほどのプロブレムリストを基に、これらの問題を一元的に説明できる鑑別診断を5つ、優先順位の高い順に挙げてください。

【プロブレムリスト】
[ここにプロブレムリストを貼り付け]

【出力形式】
診断名と、その診断が各プロブレムをどのように説明できるかを記述してください。
```

### AIの出力例

```
【統合的鑑別診断】

1.  **尿路性敗血症 (Urosepsis) ★**
    *   **説明:** 尿混濁・前立腺癌術後という背景から**尿路感染症**が先行し、菌血症から**敗血症性ショック**に至ったと考えれば、#1, #2, #3, #4, #5, #6の全てのプロブレムを合理的に説明できます。最も可能性の高い診断です。

2.  **肺炎 (Pneumonia) に伴う敗血症**
    *   **説明:** 高齢者・糖尿病患者は肺炎のリスクが高く、肺炎から敗血症性ショックに至ることも多いです。ただし、呼吸器症状の記載がないため、尿路感染症よりは優先度が下がります。

3.  **胆管炎 (Cholangitis) に伴う敗血症**
    *   **説明:** 胆管炎も敗血症の原因として重要です。しかし、腹痛や黄疸の記載がないため、可能性は低いと考えられます。

4.  **髄膜炎 (Meningitis)**
    *   **説明:** 発熱と意識障害をきたす疾患として重要です。しかし、項部硬直がないこと、他の臓器障害（腎障害）を説明しにくいことから、優先度は低いです。

5.  **感染性心内膜炎 (Infective Endocarditis)**
    *   **説明:** 亜急性の経過で発熱、塞栓による脳梗塞（意識障害）、免疫複合体による腎障害をきたす可能性があり、鑑別には挙がります。しかし、心雑音や塞栓を示唆する皮膚所見がないため、積極的には疑いにくいです。
```

## まとめ

複雑な症例では、まず**プロブレムリストを作成**し、思考を整理することが重要です。その上で、AIに**複数の問題を統合する鑑別診断**を提案させることで、診断の精度を高めることができます。

このプロセスは、自身の臨床推論能力を鍛える上でも非常に有効なトレーニングとなります。

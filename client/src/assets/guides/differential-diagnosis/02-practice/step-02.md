# ステップ2: 鑑別診断の優先順位付け

## 優先順位付けの重要性

AIが生成した鑑別診断リストは、網羅的である一方、玉石混交です。**臨床的に重要な疾患を見逃さない**ために、優先順位付けが不可欠です。

### 優先順位付けの3つの基準

1.  **緊急性 (Urgency)**: 見逃すと生命に関わるか？
2.  **頻度 (Frequency)**: よくある疾患か？ (Common things are common)
3.  **所見の一致度 (Fit)**: 患者の所見とどれだけ一致するか？

---

## AIを使った優先順位付け

AIに優先順位付けをさせることで、客観的かつ迅速にリストを整理できます。

### 優先順位付けのプロンプト

```
あなたは経験豊富な臨床医です。以下の鑑別診断リストを、3つの基準（緊急性、頻度、所見の一致度）に基づいて優先順位付けしてください。

【鑑別診断リスト】
[ここにAIが生成したリストを貼り付け]

【出力形式】
以下の表形式で整理してください：

| 診断名 | 緊急性 (高/中/低) | 頻度 (高/中/低) | 所見の一致度 (高/中/低) | 総合優先度 (1-5位) |
| :--- | :---: | :---: | :---: | :---: |

最も可能性の高い診断を★で示してください。
```

## 実践例: 胸痛の鑑別診断の優先順位付け

### 入力プロンプト

```
あなたは経験豊富な臨床医です。以下の鑑別診断リストを、3つの基準（緊急性、頻度、所見の一致度）に基づいて優先順位付けしてください。

【鑑別診断リスト】
1. 急性冠症候群（ACS）
2. 大動脈解離
3. 肺塞栓症（PE）
4. 胃食道逆流症（GERD）
5. 筋骨格系の痛み

【症例情報】
- 60歳男性、突然発症の締め付けられるような胸痛、左肩への放散痛、冷や汗、喫煙歴、高血圧、脂質異常症

【出力形式】
以下の表形式で整理してください：

| 診断名 | 緊急性 (高/中/低) | 頻度 (高/中/低) | 所見の一致度 (高/中/低) | 総合優先度 (1-5位) |
| :--- | :---: | :---: | :---: | :---: |

最も可能性の高い診断を★で示してください。
```

---

### AIの出力例

| 診断名 | 緊急性 | 頻度 | 所見の一致度 | 総合優先度 |
| :--- | :---: | :---: | :---: | :---: |
| **急性冠症候群（ACS）** ★ | 高 | 中 | 高 | 1位 |
| **大動脈解離** | 高 | 低 | 中 | 2位 |
| **肺塞栓症（PE）** | 高 | 低 | 中 | 3位 |
| 胃食道逆流症（GERD） | 低 | 高 | 低 | 4位 |
| 筋骨格系の痛み | 低 | 高 | 低 | 5位 |

**解説:**
*   **ACS**は、緊急性・所見の一致度が高く、最も優先されます。
*   **大動脈解離**と**PE**は、頻度は低いものの、緊急性が非常に高いため、次点で考慮すべき疾患です。
*   **GERD**と**筋骨格系の痛み**は、頻度は高いですが、緊急性が低く、所見の一致度も低いため、優先度は下がります。

## 思考プロセスを可視化する

AIに優先順位付けの**思考プロセス**を説明させることで、臨床推論の学習にも役立ちます。

### 思考プロセス説明のプロンプト

```
上記の優先順位付けについて、あなたの思考プロセスを説明してください。なぜその順位になったのか、各疾患を比較しながら解説してください。
```

### AIの出力例

```
【優先順位付けの思考プロセス】

今回の症例は「60歳男性、リスク因子多数、突然発症の典型的な胸痛」です。この情報から、まず生命を脅かす疾患を最優先に考えます。

1.  **急性冠症候群（ACS）を1位とした理由:**
    *   **所見の一致度が最も高い:** 締め付けられる胸痛、放散痛、冷や汗、そして多数のリスク因子（年齢、性別、喫煙、高血圧、脂質異常症）は、典型的なACSの症状です。
    *   **緊急性が高い:** 見逃せば心筋梗塞に至り、生命に関わります。
    *   **頻度も中程度:** 胸痛の原因として決して稀ではありません。
    以上の理由から、ACSを最も疑い、迅速な心電図・心筋逸脱酵素の確認が必要です。

2.  **大動脈解離を2位とした理由:**
    *   **緊急性が非常に高い:** ACS同様、見逃しは致死的です。
    *   **所見の一部が一致:** 「突然発症の胸痛」という点は一致します。
    *   **頻度が低い:** ACSと比較すると頻度は低いです。また、典型的な背部痛や血圧左右差の記載がないため、ACSよりは優先度を下げました。しかし、常に念頭に置くべき疾患です。

3.  **肺塞栓症（PE）を3位とした理由:**
    *   **緊急性が高い:** これも見逃しは致死的です。
    *   **所見の一致度が低い:** 「突然発症の胸痛」は一致しますが、呼吸困難や下肢腫脹といったPEを示唆する典型的な所見がありません。そのため、大動脈解離よりもさらに優先度を下げました。

4.  **GERDと筋骨格系の痛みを下位とした理由:**
    *   **緊急性が低い:** 生命を脅かす可能性は低いです。
    *   **所見の一致度が低い:** 症状は非典型的です。
    *   **除外診断:** これらは、上記の致死的疾患を否定した後に考慮すべき診断です。

**結論として、** この症例では、まずACSを最優先に考え、同時に大動脈解離とPEの可能性も念頭に置きながら、検査を進めるべきです。
```

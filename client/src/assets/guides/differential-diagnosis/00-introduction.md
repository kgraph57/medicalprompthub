# 鑑別診断リスト生成：AIで見落としを防ぐ

## はじめに

臨床現場で最も重要なスキルの一つが、**網羅的な鑑別診断**を挙げる能力です。しかし、経験の浅い研修医にとって、主訴から全ての可能性を思い浮かべるのは容易ではありません。

> 「見落としてはいけない疾患（Don't miss diagnosis）を見逃してしまった...」

このような経験は、多くの研修医が抱える悩みです。このガイドでは、**AIを活用して網羅的な鑑別診断リストを生成する方法**を紹介します。

---

## なぜAI支援が必要なのか？

### 鑑別診断の課題

1. **経験不足**: 教科書的な知識はあっても、実際の症例で思い出せない
2. **時間的制約**: 救急外来では、ゆっくり考える時間がない
3. **認知バイアス**: 最初に思いついた診断に固執してしまう（アンカリングバイアス）
4. **稀な疾患**: 頻度の低い疾患は思い浮かばない

---

### AI支援の効果

AIを活用することで、以下のメリットがあります：

| 従来の方法 | AI支援 |
| :--- | :--- |
| 経験に依存 | 網羅的なリスト生成 |
| 時間がかかる | 数秒で生成 |
| バイアスがかかる | 客観的な提案 |
| 稀な疾患を見落とす | 稀な疾患も含める |

---

## このガイドで学べること

1. **主訴別の鑑別診断**を網羅的にリストアップする方法
2. **優先順位付け**の基準（緊急性・頻度・所見の一致度）
3. **除外診断**の根拠を明確にする方法
4. **AIプロンプトの実践的な使い方**
5. **臨床推論**のプロセスを可視化する方法

## 鑑別診断の重要性

### ケーススタディ: 見落としを防いだ例

**症例**: 50歳男性、胸痛で来院

**初期診断**: 急性冠症候群を疑い、心電図・トロポニン測定

**結果**: 心電図正常、トロポニン陰性

**問題**: 「急性冠症候群は除外できた」と判断し、帰宅させそうになった

**AI支援**: 鑑別診断リストを生成
- 急性冠症候群
- **大動脈解離** ← AIが提案
- 肺塞栓症
- 気胸

**追加検査**: 造影CTを施行 → **大動脈解離を発見**

**結果**: 緊急手術で救命

## 注意事項

### AIの限界を理解する

AIは強力なツールですが、**完璧ではありません**。以下の点に注意してください：

⚠️ **AIの提案を鵜呑みにしない**: 必ず医学的な妥当性を確認  
⚠️ **患者背景を考慮する**: AIは個別の患者背景を完全には理解できない  
⚠️ **最終判断は医師が行う**: AIはサポートツールであり、診断の責任は医師にある

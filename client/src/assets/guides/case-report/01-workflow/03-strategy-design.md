# ステップ3: 執筆戦略の立案とプロンプト設計
![執筆戦略フレームワーク](/Helix/assets/guides/case-report/images/step03_writing_strategy_framework.png)

## このステップの目的

論文の骨子となる詳細なアウトラインを設計し、各セクションをAIに高品質で執筆させるための一連の「マスタープロンプト」を構築します。
このステップは、AIを単なる文章作成ツールではなく、「執筆エージェント」として機能させるための司令塔を設計するプロセスです。

## 達成基準

- ターゲットジャーナルの投稿規定とCAREガイドラインに基づいた、詳細な論文アウトラインが完成している。
- 論文の各セクション（Introduction, Case Presentation, Discussion, Conclusion）を執筆させるための、文脈情報が豊富に含まれたマスタープロンプトが設計されている。
- 論文全体で伝えたい核心的なメッセージ（Take-Home Message）が明確に定義されている。

## 詳細な手順

### 3-1. ターゲットジャーナルの選定と投稿規定の分析

まず、論文を投稿する雑誌（ターゲットジャーナル）を決定します。
ジャーナルの格（インパクトファクター）、読者層、そしてあなたの症例の新規性がマッチするかを考慮して選びます。

決定後、そのジャーナルの「投稿規定（Instructions for Authors）」を熟読します。
特に、症例報告の文字数制限、構造、引用スタイル、図表の要件などを確認します。
この作業は、Claude 3.5 Sonnetに要約させることで効率化できます。

プロンプト例（Claude 3.5 Sonnet）:

```
# 指示
あなたは医学雑誌の編集アシスタントです。以下の投稿規定のテキスト（またはURL）を読み、症例報告を執筆する上で遵守すべき最も重要な要件を10個、箇条書きで要約してください。
特に文字数、構造、図表、参考文献のフォーマットに焦点を当ててください。

# 投稿規定
[ここに投稿規定のテキストを貼り付けるか、URLを記載]
```

### 3-2. 詳細アウトラインの構築

次に、ステップ1、2で得られた情報と、ターゲットジャーナルの規定を基に、論文の詳細なアウトラインを構築します。
これは単なるセクションの見出しではなく、各パラグラフで何を述べるか、どの図表をどこに挿入するかまで具体的に記述した「設計図」です。

アウトラインの構成要素:

-   Title: 論文の核心を突く魅力的なタイトル（仮）
-   Abstract: 構造化抄録の各項目で記述すべき内容
-   Introduction: 背景 → 問題提起 → 本報告の目的、という論理の流れ
-   Case Presentation: 臨床経過、検査所見、画像所見の提示順序
-   Discussion: 本症例の要約 → 文献比較 → メカニズム考察 → 臨床的意義 → 限界 → 結論
-   Conclusion: 2〜3文で凝縮したTake-Home Message
-   Figures & Tables: 各図表のタイトルと内容

### 3-3. マスタープロンプトの設計

ここがAACR Methodの核心です。
AIに論文を執筆させるための、包括的で多層的な「マスタープロンプト」を設計します。
このプロンプトは、AIに「何を」「どのように」書かせるかを正確に指示する司令塔の役割を果たします。

マスタープロンプトの構成要素:

1.  ペルソナ設定 (Persona): AIに特定の役割を付与します。
    > 例: 「あなたはThe New England Journal of Medicineに論文を多数掲載している、経験豊富な臨床研究者です。あなたの文章は、明快、簡潔、かつ論理的で、常に読者へのインパクトを意識しています。」

2.  コンテキスト注入 (Context Injection): AIが執筆に必要なすべての情報をインプットします。
    -   ステップ1で作成した「構造化サマリー」
    -   ステップ2で作成した「文献比較分析レポート」と「引用すべき論文リスト」
    -   ステップ3-1で分析した「ターゲットジャーナルの投稿規定」
    -   ステップ3-2で作成した「詳細アウトライン」

3.  タスク指示 (Task Instruction): 具体的で段階的な指示を与えます。
    > 例: 「以上のコンテキストに基づき、詳細アウトラインに従って、症例報告のIntroductionセクションのドラフトを作成してください。文字数は約300ワードです。CAREガイドラインを遵守し、提供された引用すべき論文リストから最低3つの文献を引用してください。」

プロンプト例（Claude 3.5 Sonnet）:

```
# マスタープロンプト: 症例報告執筆エージェント

## 1. ペルソナ
あなたはThe New England Journal of Medicineに論文を多数掲載している、経験豊富な臨床研究者です。あなたの文章は、明快、簡潔、かつ論理的で、常に読者へのインパクトを意識しています。

## 2. コンテキスト

### 2-1. 症例サマリー
[ステップ1で作成したサマリー]

### 2-2. 文献分析
[ステップ2で作成した文献比較分析と新規性のポイント]

### 2-3. 引用すべき論文リスト
[ステップ2で特定した論文リスト]

### 2-4. ターゲットジャーナル規定
[ステップ3-1で要約した投稿規定]

### 2-5. 詳細アウトライン
[ステップ3-2で作成したアウトライン]

## 3. タスク

以上のすべてのコンテキストを完全に理解した上で、以下の指示に従ってください。

指示1: まず、詳細アウトラインの【Introduction】セクションの記述に基づき、症例報告の導入部分のドラフトを作成してください。ターゲットジャーナルの規定に従い、文字数は約300ワード、引用は[1]のように番号で行ってください。
```

このマスタープロンプトをベースに、「指示」の部分を「Case Presentation」「Discussion」と変えていくことで、論文全体を体系的にAIに執筆させることが可能になります。

次のステップでは、いよいよこのマスタープロンプトを使い、AIによる執筆を開始します。

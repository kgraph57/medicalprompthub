# Step 17: 編集者の心を掴むカバーレターの書き方

### 📋 このステップの目的

論文の「ラブレター」とも言える、カバーレターを作成します。
これは、あなたが編集長に送る最初の手紙。
論文の第一印象を決め、「お、この論文は面白そうだ。読んでみたい」と思わせるための、超重要なマーケティングツールです。

![カバーレターの書き方](https://files.manuscdn.com/user_upload_by_module/session_file/90346460/SspdgZpwxvWxkzDF.png)

### ⏱️ 所要時間

1時間

### 🎯 達成基準

- 論文の「売り」と、なぜそのジャーナルにふさわしいかが、簡潔かつ説得的に書かれている。
- 必要な「お約束」（他誌への未投稿、全著者の同意など）がすべて含まれている。

### 📝 詳細手順

![カバーレターの書き方](./cover_letter_creation.png)

カバーレターは、以下の4つの要素で構成された、短い手紙です。

1.  宛名: 編集長の名前と役職を正確に。
「Dear Editor」はNG。
ジャーナルの「Editorial Board」ページで名前を確認しましょう。

2.  セールスポイント: なぜこの論文が重要で、このジャーナルの読者にとって面白いのか。
論文の「売り」を2〜3文で熱烈にアピールする、最重要パートです。

3.  ジャーナルへの適合性: なぜ、数あるジャーナルの中から「あなた」を選んだのか。
論文の内容とジャーナルの読者層がマッチしていることを示します。

4.  倫理的な宣言: 「この論文はオリジナルです」「他の雑誌には投稿していません」「著者全員が投稿に同意しています」といった、お決まりの宣言をします。

### 💡 プロンプト集

#### プロンプト1: カバーレターの草案をAIで一発作成

```
# 指示
あなたは経験豊富な研究者です。以下の情報に基づき、医学雑誌の編集長に宛てた、プロフェッショナルで説得力のあるカバーレターの草案を作成してください。

# 情報
- 投稿先ジャーナル名: [ジャーナル名]
- 編集長名: [編集長の名前]
- 論文タイトル: [論文のタイトル]
- 論文のセールスポイント: [論文の最も重要な発見や新規性を2-3文で]
- 責任著者: [あなたの名前、所属、連絡先]

# 制約
- 標準的なビジネスレターのフォーマットで。
- 倫理的な宣言（他誌への未投稿、全著者の同意など）を必ず含めること。
```

活用例: このプロンプトを使えば、数分で質の高い草案が完成。
あとは自分の言葉で少し手直しするだけです。

### ⚠️ よくある失敗と対処法

- 失敗: カバーレターで、論文の内容を長々と要約してしまう。
  - 対処法: 編集者は多忙です。
短く、要点を絞るのが鉄則。
役割は、抄録を読んでもらうための「きっかけ作り」です。

- 失敗: 宛名が「Dear Editor」になっている。
  - 対処法: 編集長の個人名を正確に記載することで、本気度が伝わります。
一手間を惜しまないことが、良い第一印象に繋がります。

# Step 17: カバーレターの作成

### 📋 このステップの目的

論文の「送り状」であるカバーレターを作成します。カバーレターは、あなたがジャーナルの編集長（Editor-in-Chief）に宛てて書く最初の手紙です。論文の第一印象を決定し、編集者に「この論文は面白そうだ、査読に回す価値がある」と思わせるための、重要なマーケティングツールです。

### ⏱️ 所要時間

**1時間**

### 🎯 達成基準

- カバーレターが、標準的なビジネスレターの形式で作成されている。
- 論文の新規性と、なぜそのジャーナルに適しているかが、簡潔かつ説得的に述べられている。
- 投稿規定で要求される宣言（例: 他誌への未投稿、全著者の同意）がすべて含まれている。

### 📝 詳細手順

カバーレターは、通常、以下の要素で構成されます。

1.  **宛名**: ジャーナルの編集長（Editor-in-Chief）の名前と役職を正確に記載します。編集長の名前は、ジャーナルのウェブサイトにある「Editorial Board」のページで確認できます。
    > `To: [Editor-in-Chief's Name], M.D., Ph.D.`
    > `Editor-in-Chief, [Journal Name]`

2.  **第1パラグラフ（導入）**: どの論文を、どのセクション（例: Case Report）に投稿したいのかを明確に述べます。
    > `Dear Dr. [Editor's Last Name],`
    > `Please find enclosed our manuscript, entitled "[Manuscript Title]," which we would like to submit for consideration for publication as a Case Report in [Journal Name].`

3.  **第2パラグラフ（セールスポイント）**: なぜこの論文が重要なのか、この論文の最も重要な発見（新規性）は何か、そして、なぜそれがこのジャーナルの読者にとって興味深いのかを、2〜3文で簡潔にアピールします。論文の「売り」を最も効果的に伝える、カバーレターの核心部分です。

4.  **第3パラグラフ（お決まりの宣言）**: 論文投稿における倫理的なお約束事を宣言します。これは定型文ですが、非常に重要です。
    - この論文がオリジナルであること。
    - 他の雑誌に投稿中または出版済みでないこと（二重投稿の禁止）。
    - すべての著者が最終稿を読み、投稿に同意していること。
    - 利益相反（COI）の有無。

5.  **結び**: 責任著者（Corresponding Author）の連絡先を明記し、結びの言葉で締めくくります。
    > `Thank you for your time and consideration.`
    > `Sincerely,`
    > `[Your Name], M.D., Ph.D.`
    > `[Your Affiliation]`
    > `[Your Address, Email, Phone Number]`

### 💡 プロンプト集

#### プロンプト1: カバーレターの草案を自動生成する

```
# 指示
あなたは経験豊富な研究者です。以下の情報に基づき、医学雑誌の編集長に宛てた、プロフェッショナルで説得力のあるカバーレターの草案を作成してください。

# 情報
- **投稿先ジャーナル名**: Internal Medicine
- **編集長名**: Hiroshi Itoh, M.D., Ph.D.
- **論文タイトル**: Early-onset Steroid-Resistant Organizing Pneumonia Successfully Treated with Cyclophosphamide: A Case Report
- **論文のセールスポイント**: 本症例は、中等量のステロイドに早期から抵抗性を示した稀な器質化肺炎(OP)であり、シクロホスファミドが著効したことを示しています。この知見は、難治性OPの治療戦略を考える上で、貴誌の読者である多くの内科医にとって非常に価値のある情報だと考えます。
- **責任著者**: [あなたの名前、所属、連絡先]

# 制約
- 標準的なビジネスレターのフォーマットに従うこと。
- 倫理的な宣言（他誌への未投稿、全著者の同意など）を必ず含めること。
```

**活用例**: このプロンプトを使えば、わずか数分で、形式的にも内容的にも質の高いカバーレターの草案が完成します。生成された草案を、自分の言葉で少し修正するだけで十分です。

### 📚 実践例

上記プロンプトで生成された草案を基に、Internal Medicine誌の伊藤裕編集長に宛てたカバーレターを作成。特に第2パラグラフの「セールスポイント」を、指導医と相談しながら、よりインパクトのある表現に磨き上げた。

### ⚠️ よくある失敗と対処法

- **失敗**: カバーレターで、論文の内容を長々と要約してしまう。
  - **対処法**: 編集者は多忙です。カバーレターは短く、要点を絞ることが鉄則です。論文の詳細は抄録を読めば分かります。カバーレターの役割は、抄録を読んでもらうための「きっかけ作り」です。

- **失敗**: 宛名が「Dear Editor」など、一般的な表現になっている。
  - **対処法**: 編集長の個人名を正確に記載することで、あなたがそのジャーナルに真剣に投稿しようとしている姿勢が伝わります。一手間を惜しまないことが、良い第一印象に繋がります。

### 💪 上級者向けTIPS

もし可能であれば、そのジャーナルの編集委員（Editorial Board）の中に、自分の研究分野に近い先生がいるか探してみましょう。そして、カバーレターの中で「特に、貴誌の編集委員である〇〇先生のご専門分野にも関連が深い内容かと存じます」のように言及することで、編集者が適切な査読者を選ぶ手助けとなり、好意的に受け取られる可能性があります。

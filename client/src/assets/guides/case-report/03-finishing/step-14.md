# Step 14: 参考文献リストの作成

### 📋 このステップの目的

論文の信頼性を支える根幹、参考文献リストを完成させます。このステップでは、引用したすべての文献を、投稿規定で定められた形式（引用スタイル）に従って正確にフォーマットし、リストを作成する技術を学びます。これは非常に手間がかかる作業ですが、文献管理ツールとAIを駆使することで、劇的に効率化できます。

### ⏱️ 所要時間

**1〜2時間**（文献管理ツール使用時）

### 🎯 達成基準

- 本文中の引用箇所と、参考文献リストの番号が過不足なく一致している。
- 参考文献リストが、投稿ジャーナルの指定する引用スタイル（例: Vancouver）で正確にフォーマットされている。
- 文献管理ツール（Zotero, Paperpileなど）を用いて、リストが体系的に管理されている。

### 📝 詳細手順

#### 手順1: 文献管理ツールの導入

まだ導入していないなら、今すぐ文献管理ツールを使い始めましょう。手作業での文献管理は、膨大な時間と労力を浪費し、ミスの温床となります。

- **Zotero**: 無料で高機能。多くの研究者に利用されているデファクトスタンダード。
- **Paperpile**: Googleドキュメントとの連携がスムーズ。シンプルで直感的な操作性が魅力（有料）。
- **Mendeley**: Elsevier社が提供。PDFの管理機能が強力。

これらのツールは、ブラウザ拡張機能を使って、ワンクリックでPubMedやジャーナルのウェブサイトから文献情報を取り込むことができます。

#### 手順2: 引用文献の整理とフォーマット

1.  **本文中への引用**: 論文を執筆しながら、引用したい箇所に文献管理ツールのプラグイン（WordやGoogleドキュメント用）を使って引用を挿入します。手動で「[1]」などと入力してはいけません。
2.  **引用スタイルの設定**: 文献管理ツール上で、投稿ジャーナルの引用スタイルを選択します。主要なジャーナルのスタイルはプリセットで用意されています。例えば、多くの医学雑誌で採用されている「**Vancouverスタイル**」は以下のような形式です。
    > `著者名. 論文タイトル. 雑誌名. 発行年;巻(号):ページ.`
    > `1. Hiraga A. How to write a case report effectively. Clin Neurol. 2023;63(5):305-313.`
3.  **リストの自動生成**: 本文の最後に、ツールを使って参考文献リストを自動生成します。本文中の引用番号とリストが自動でリンクされ、順番の入れ替えや追加・削除も瞬時に反映されます。

#### 手順3: 最終チェック

自動生成されたリストが完璧とは限りません。最後に、以下の点を自分の目でチェックします。

- **著者名**: 規定の人数（例: 6名まで記載し、それ以上は et al.）になっているか。
- **雑誌名**: 公式の略称（例: New England Journal of Medicine → N Engl J Med）になっているか。
- **巻・号・ページ**: 情報が正確か。
- **DOI**: 記載が必要な場合は、正しいDOIが付与されているか。

### 💡 プロンプト集

#### プロンプト1: 文献情報を指定の引用スタイルにフォーマットする (ChatGPT)

```
# 指示
あなたはメディカルライブラリアンです。以下の文献情報を、医学論文で標準的な「Vancouverスタイル」に正確にフォーマットしてください。

# 文献情報
- **著者**: Akiyuki Hiraga
- **論文タイトル**: How to make effective figures in case reports
- **雑誌名**: Clinical Neurology
- **発行年**: 2025
- **巻**: 65
- **号**: 2
- **ページ**: 150-156
- **DOI**: 10.5692/clinicalneurol.cn-002044

# Vancouverスタイルフォーマット例
著者名. 論文タイトル. 雑誌名. 発行年;巻(号):ページ.
```

**活用例**: 文献管理ツールがうまく機能しない場合や、数件の文献を手動で整形する必要がある場合に便利です。しかし、基本は文献管理ツールの使用を強く推奨します。

### 📚 実践例

- **執筆プロセス**: GoogleドキュメントとPaperpileを使用。緒言や考察を書きながら、関連文献を引用したい箇所でPaperpileのプラグインを起動し、ライブラリから文献を検索して挿入。
- **リスト生成**: 論文を書き終えた後、文末にカーソルを合わせ、Paperpileの「Generate Bibliography」ボタンをクリック。投稿先の「Internal Medicine」誌のスタイルを選択。
- **結果**: 本文中の引用番号[1], [2], [3]...と完全に連動した、正しいフォーマットの参考文献リストが10秒で自動生成された。

### ⚠️ よくある失敗と対処法

- **失敗**: すべての文献を手作業で管理し、投稿直前に番号のズレやフォーマットの間違いに気づき、徹夜で修正作業に追われる。
  - **対処法**: **今すぐ文献管理ツールを導入してください。** これに尽きます。ツールの初期設定に数時間かかったとしても、論文執筆全体で見れば、数十時間の節約とミスの劇的な減少に繋がります。

- **失敗**: 雑誌名の略称を自己流で書いてしまう。
  - **対処法**: 雑誌名の略称には、NLM（National Library of Medicine）による公式のルールがあります。文献管理ツールはこれを自動で適用してくれますが、手動で確認する場合は、PubMedでその雑誌を検索し、公式の略称を確認しましょう。

### 💪 上級者向けTIPS

Zoteroには「ZotFile」という拡張機能があり、これを使うと、収集した文献のPDFファイルを自動でリネームし、指定したクラウドストレージ（Dropbox, Google Driveなど）に整理して保存できます。これにより、複数のデバイスから自分の文献ライブラリにアクセスできるようになり、研究の効率がさらに向上します。

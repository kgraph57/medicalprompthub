# 01-basics/step-01.md: AIを活用した倫理審査申請の目的と初期情報の整理

## 倫理審査申請におけるAI活用のメリット

倫理審査申請書の作成は、研究の質を担保し、被験者の人権と安全を守る上で極めて重要ですが、その作成プロセスはしばしば時間と労力を要するボトルネックとなります。AI（特に大規模言語モデル, LLM）を活用することで、このプロセスを劇的に効率化し、申請書の質を向上させることができます。

| 課題 | AIによる解決策 | メリット |
| :--- | :--- | :--- |
| 論理構成の破綻 | 研究の背景、目的、方法、倫理的配慮の間の論理的な繋がりをチェックし、一貫性を保つための提案を行う。 | 審査員が理解しやすい、説得力のある構成を実現。 |
| 専門用語の不統一 | 申請書全体で使用される専門用語や略語の定義を一元管理し、統一された表現を提案する。 | 読み手が混乱しない、プロフェッショナルな文書を作成。 |
| 文献検索と引用 | 研究テーマに関連する最新の倫理指針や先行研究を迅速に検索し、適切な引用形式を提案する（ただし、最終確認は必須）。 | 倫理的・科学的妥当性の根拠を強化。 |
| 倫理的配慮の抜け漏れ | 研究デザインから想定される倫理的なリスクを洗い出し、それに対する具体的な対策案を提案する。 | 審査の差し戻しリスクを低減。 |

## 初期情報の整理とAIへのインプット準備

AIに高品質な申請書の下書きを作成させるためには、質の高いインプットが不可欠です。まずは、研究の核となる情報を整理し、AIに「倫理審査申請書作成の専門家」としての役割を与えるための準備を行います。

### 具体的な手順

1.  研究の基本情報の明確化:
    *   研究テーマ、研究責任者、実施期間、研究対象者（人数、属性）を明確にします。
2.  倫理審査の基本構造の理解:
    *   所属機関の倫理審査申請書の必須項目（例：研究の意義、方法、インフォームド・コンセントの手順、個人情報保護、利益相反）を把握します。
3.  AIへの役割設定の準備:
    *   AIに「あなたは厳格な倫理審査委員会の委員長であり、研究の科学的妥当性と倫理的配慮を厳しくチェックする」という役割を与えるための指示文を準備します。

### 実用的なプロンプト例

AIに倫理審査申請書の構成要素を理解させ、その後の作業の土台を築くためのプロンプトです。

```prompt
あなたは、日本の医療機関における研究倫理審査委員会の委員長です。
以下の研究計画の骨子案をレビューするために、まず倫理審査申請書に必須となる主要な構成要素を、その審査上の重要度が高い順に10項目リストアップしてください。
各項目について、審査員が特に注目するチェックポイントを簡潔に記述してください。

研究計画の骨子案:
[ここに、あなたの研究の簡単な概要（例：研究目的、方法、対象）を箇条書きで記述]
```

### 注意点とTips

*   機密情報の除外: AIに入力する情報には、絶対に機密性の高い患者データや未公開の個人情報を含めないでください。研究の骨子や抽象的な方法論に留めることが鉄則です。
*   AIの限界の理解: AIは情報処理と文章生成のツールであり、倫理的判断の最終責任は研究者自身にあります。AIが生成した内容は、必ず所属機関の指針と照らし合わせて確認してください。

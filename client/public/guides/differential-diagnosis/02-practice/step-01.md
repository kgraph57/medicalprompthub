# ステップ1: 主訴別の鑑別診断リストを生成する

## 実践：AIで鑑別診断リストを生成しよう

このステップでは、救急外来でよく遭遇する主訴について、実際にAIを使って鑑別診断リストを生成します。

---

## 1. 胸痛の鑑別診断

### 症例シナリオ

- 年齢・性別: 60歳、男性
- 主訴: 胸痛
- 重要な所見: 
  - 突然発症の締め付けられるような胸痛
  - 左肩への放散痛あり
  - 冷や汗を伴う
  - 喫煙歴: 30年
  - 既往歴: 高血圧、脂質異常症

---

### プロンプトテンプレート（コピペ用）

```
あなたは経験豊富な臨床医です。以下の症例について、鑑別診断を網羅的にリストアップしてください。

【症例情報】
- 年齢・性別: 60歳、男性
- 主訴: 胸痛
- 重要な所見: 
  - 突然発症の締め付けられるような胸痛
  - 左肩への放散痛あり
  - 冷や汗を伴う
  - 喫煙歴: 30年
  - 既往歴: 高血圧、脂質異常症

【出力形式】
以下の形式で鑑別診断をリストアップしてください：

1. 診断名
   - 支持する所見:
   - 否定的な所見:
   - 確定診断の方法:
   - 緊急性: 高/中/低

頻度と重症度を考慮して、優先順位の高い順に並べてください。
Don't miss diagnosisを必ず含めてください。
```

---

### AIの出力例

```
【鑑別診断リスト】

1. 急性冠症候群（ACS） ★
   - 支持する所見: 締め付けられる胸痛、放散痛、冷や汗、喫煙歴、高血圧、脂質異常症
   - 否定的な所見: なし
   - 確定診断の方法: 心電図、心筋逸脱酵素（トロポニン）、冠動脈造影
   - 緊急性: 高

2. 大動脈解離
   - 支持する所見: 突然発症の胸痛
   - 否定的な所見: 背部痛なし、血圧左右差なし（確認必要）
   - 確定診断の方法: 造影CT、経食道心エコー
   - 緊急性: 高

3. 肺塞栓症（PE）
   - 支持する所見: 突然発症の胸痛
   - 否定的な所見: 呼吸困難なし、下肢腫脹なし（確認必要）
   - 確定診断の方法: D-dimer、造影CT
   - 緊急性: 高

4. 胃食道逆流症（GERD）
   - 支持する所見: 胸痛
   - 否定的な所見: 食事との関連なし、胸焼けなし
   - 確定診断の方法: 上部消化管内視鏡、PPIテスト
   - 緊急性: 低

5. 筋骨格系の痛み（肋間神経痛など）
   - 支持する所見: 胸痛
   - 否定的な所見: 圧痛なし（確認必要）、体動との関連なし
   - 確定診断の方法: 身体所見、除外診断
   - 緊急性: 低
```

### AIの出力例

```
【鑑別診断リスト】

1. 急性心不全 ★
   - 支持する所見: 呼吸困難、coarse crackles、下腿浮腫、虚血性心疾患の既往
   - 否定的な所見: なし
   - 確定診断の方法: BNP、心エコー、胸部X線
   - 緊急性: 高

2. 肺炎
   - 支持する所見: 呼吸困難、coarse crackles
   - 否定的な所見: 発熱なし、炎症反応（確認必要）
   - 確定診断の方法: 血液検査、喀痰培養、胸部CT
   - 緊急性: 高

3. 肺塞栓症（PE）
   - 支持する所見: 突然発症の呼吸困難、低酸素血症
   - 否定的な所見: 下肢腫脹なし（確認必要）
   - 確定診断の方法: D-dimer、造影CT
   - 緊急性: 高

4. COPD急性増悪
   - 支持する所見: 呼吸困難
   - 否定的な所見: 喫煙歴なし（確認必要）、喘鳴なし
   - 確定診断の方法: 呼吸機能検査、動脈血ガス
   - 緊急性: 高

5. 間質性肺炎急性増悪
   - 支持する所見: 呼吸困難、coarse crackles
   - 否定的な所見: 間質性肺炎の既往なし（確認必要）
   - 確定診断の方法: 胸部HRCT、KL-6、SP-D
   - 緊急性: 高
```

## 3. 腹痛の鑑別診断

### 症例シナリオ

- 年齢・性別: 80歳、男性
- 主訴: 腹痛
- 重要な所見: 
  - 突然発症の激しい腹痛
  - 腹部全体の圧痛あり
  - 腹部所見に乏しい（pain out of proportion）
  - 既往歴: 心房細動

```
- 年齢・性別: 80歳、男性
- 主訴: 腹痛
- 重要な所見: 
  - 突然発症の激しい腹痛
  - 腹部全体の圧痛あり
  - 腹部所見に乏しい（pain out of proportion）
  - 既往歴: 心房細動

【出力形式】
以下の形式で鑑別診断をリストアップしてください：

1. 診断名
   - 支持する所見:
   - 否定的な所見:
   - 確定診断の方法:
   - 緊急性: 高/中/低

頻度と重症度を考慮して、優先順位の高い順に並べてください。
Don't miss diagnosisを必ず含めてください。
```

---

### AIの出力例

```
【鑑別診断リスト】

1. 上腸間膜動脈血栓症（SMA塞栓症） ★
   - 支持する所見: 突然発症の激しい腹痛、pain out of proportion、心房細動（塞栓源）
   - 否定的な所見: なし
   - 確定診断の方法: 造影CT、血液検査（Lactate、D-dimer）
   - 緊急性: 高

2. 腹部大動脈瘤破裂（AAA破裂）
   - 支持する所見: 突然発症の激しい腹痛、高齢男性
   - 否定的な所見: ショックバイタルなし（確認必要）、拍動性腫瘤なし
   - 確定診断の方法: 腹部超音波、造影CT
   - 緊急性: 高

3. 消化管穿孔
   - 支持する所見: 突然発症の激しい腹痛
   - 否定的な所見: 筋性防御なし（確認必要）、free airなし（確認必要）
   - 確定診断の方法: 腹部X線、CT
   - 緊急性: 高

4. 急性膵炎
   - 支持する所見: 激しい腹痛
   - 否定的な所見: 心窩部痛ではない、背部痛なし
   - 確定診断の方法: アミラーゼ、リパーゼ、造影CT
   - 緊急性: 高

5. 腸閉塞
   - 支持する所見: 腹痛
   - 否定的な所見: 嘔吐・便秘なし（確認必要）、腸蠕動音（確認必要）
   - 確定診断の方法: 腹部X線、CT
   - 緊急性: 中
```

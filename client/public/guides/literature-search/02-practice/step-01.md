
# ステップ1: 論文全体を構造化要約する

## アブストラクトの先へ：批判的吟味のための構造化

アブストラクト（抄録）の要約は、論文を読むべきか判断する「スクリーニング」には非常に有効です。しかし、その論文の結果を真に理解し、臨床に応用できるか判断する「批判的吟味」のためには、論文全体を精読する必要があります。

とは言え、長い英語の論文を最初から最後まで読むのは大変な作業です。そこで、AIに論文の全文を読み込ませ、PICOや研究デザイン、結果、限界などを構造的に整理させることで、読解の効率と精度を飛躍的に高めることができます。

## 論文全体構造化要約プロンプト

このプロンプトは、論文の全文（PDFからコピー＆ペーストしたもの）をAIに与え、その内容を体系的に整理・要約させるためのものです。

### 📋 構造化要約プロンプト（コピペ用）

```
あなたはEBMと臨床研究に精通した専門家です。以下の論文の全文を読み、その内容を下記の【出力形式】に従って、日本語で詳細に構造化要約してください。

【論文全文】
[ここに論文の全文をコピー＆ペーストする]

【出力形式】

### 論文の構造化要約

1. 書誌情報
*   論文名:
*   第一著者:
*   雑誌名:
*   発行年:

2. 研究の背景 (Background)
*   この研究が行われた背景や、臨床上のどのような課題（アンメット・メディカル・ニーズ）があったか？

3. PICO
*   P (Patient/Problem): どのような患者が対象か？（選択基準・除外基準）
*   I (Intervention): どのような介入が行われたか？
*   C (Comparison): 何と比較されたか？
*   O (Outcome): 主要評価項目（プライマリーエンドポイント）と副次評価項目は何か？

4. 研究デザイン (Study Design)
*   研究の種類は何か？ (例: ランダム化比較試験、コホート研究など)
*   ランダム化や盲検化はどのように行われたか？
*   対象患者数と追跡期間は？

5. 主要な結果 (Key Results)
*   主要評価項目の結果はどうだったか？（具体的な数値、ハザード比、p値など）
*   安全性の結果（主な有害事象）はどうだったか？

6. 結論 (Conclusion)
*   著者らは、この研究結果からどのような結論を導き出しているか？

7. 論文の限界 (Limitations)
*   著者らが述べている、この研究の限界点は何か？

8. 資金源 (Funding)
*   この研究の資金はどこから提供されたか？（利益相反の可能性）
```

## 実践例: SGLT2阻害薬の腎保護効果

テーマ: 慢性腎臓病（CKD）患者に対するSGLT2阻害薬の効果を調べた有名な論文「DAPA-CKD試験」を要約してみましょう。

*(ここでは論文全文の貼り付けは省略しますが、実際の使用時にはPDFなどからテキストを全てコピーして貼り付けてください)*

### AIの出力例

```
承知いたしました。ご指定の論文（DAPA-CKD試験）の全文を、指定された形式で構造化要約します。

### 論文の構造化要約

1. 書誌情報
*   論文名: Dapagliflozin in Patients with Chronic Kidney Disease.
*   第一著者: Heerspink HJL
*   雑誌名: N Engl J Med (The New England Journal of Medicine)
*   発行年: 2020

2. 研究の背景 (Background)
*   慢性腎臓病（CKD）は、末期腎不全や心血管疾患のリスクを高め、世界的な健康問題となっている。既存の標準治療（ACE阻害薬やARB）を行っても、多くの患者で腎機能の低下や心血管イベントが進行してしまう。SGLT2阻害薬は、糖尿病患者において腎保護効果や心血管保護効果が示されていたが、糖尿病の有無にかかわらず、CKD患者全般に対する効果は不明であった。

3. PICO
*   P (Patient/Problem): 糖尿病の有無にかかわらず、eGFRが25-75mL/min/1.73m²で、尿中アルブミン/クレアチニン比が200-5000mg/gの慢性腎臓病（CKD）患者。
*   I (Intervention): ダパグリフロジン（SGLT2阻害薬）10mgを1日1回投与。
*   C (Comparison): プラセボ（偽薬）を1日1回投与。
*   O (Outcome): 
    *   主要評価項目: eGFRの50%以上の持続的低下、末期腎不全への進行、腎臓病または心血管疾患による死亡、を合わせた複合エンドポイント。
    *   副次評価項目: 心不全による入院または心血管死、全死亡など。

4. 研究デザイン (Study Design)
*   研究の種類: 多国籍、多施設共同、ランダム化、二重盲検、プラセボ対照比較試験。
*   ランダム化・盲検化: 患者、治験担当医師、研究スタッフ、データ解析者など、全ての関係者が、誰がダパグリフロジンとプラセボのどちらに割り付けられたかを知らない「二重盲検」で行われた。
*   対象患者数と追跡期間: 4,304人の患者がランダムに割り付けられ、追跡期間の中央値は2.4年。

5. 主要な結果 (Key Results)
*   主要評価項目: ダパグリフロジン群はプラセボ群と比較して、主要評価項目の発生リスクを39%有意に減少させた (ハザード比 0.61, 95%信頼区間 0.51-0.72, p<0.001)。この効果は、糖尿病の有無にかかわらず一貫していた。
*   安全性: 重篤な有害事象の発生率は両群で同程度であった。糖尿病性ケトアシドーシスはダパグリフロジン群では報告されなかった。

6. 結論 (Conclusion)
*   糖尿病の有無にかかわらず、慢性腎臓病（CKD）患者において、ダパグリフロジンは標準治療に上乗せすることで、腎機能の悪化と死亡のリスクをプラセボに比べて有意に低下させた。

7. 論文の限界 (Limitations)
*   eGFRが25未満の、より進行したCKD患者は対象に含まれていない。
*   追跡期間が比較的短い。

8. 資金源 (Funding)
*   アストラゼネカ社（ダパグリフロジンの製造販売元）。
```

## まとめ

このステップで学んだこと:

✅ 論文全文をAIに読み込ませ、構造的に要約させるための詳細なプロンプト
✅ PICOだけでなく、研究デザイン、結論、限界、資金源といった、批判的吟味に不可欠な要素を網羅的に抽出する方法

この「構造化要約」を作成するだけで、論文の核心部分を深く、かつ体系的に理解することができます。これは、ジャーナルクラブでの発表資料作成や、自身の知識の整理に非常に役立ちます。

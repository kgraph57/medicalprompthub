
# ステップ1: 臨床の疑問をPICOで構造化する

## EBMの第一歩は「問い」を立てること

EBM（Evidence-Based Medicine）の実践は、日々の臨床現場で生まれる「疑問」から始まります。この漠然とした疑問を、検索可能な形に整理するためのフレームワークがPICOです。

適切な「問い」を立てることが、良質なエビデンスにたどり着くための最も重要なステップです。

---

![PICOフレームワーク](/Helix/assets/guides/literature-search/pico_framework.png)

## PICOとは？

PICOは、臨床上の疑問を4つの要素に分解し、構造化するためのツールです。

| 要素 | 名称 | 内容 | 例（糖尿病患者の心血管イベント予防） |
| :---: | :--- | :--- | :--- |
| P | Patient / Problem (患者・問題) | どのような患者が対象か？ | 2型糖尿病の成人患者 |
| I | Intervention (介入) | 検討したい治療法や検査は何か？ | SGLT2阻害薬の投与 |
| C | Comparison (比較) | 何と比較するか？（プラセボ、標準治療など） | プラセボまたは他の血糖降下薬 |
| O | Outcome (結果) | 評価したい結果は何か？ | 主要心血管イベント（心筋梗塞、脳卒中など）の発生率 |

このように疑問を構造化することで、検索キーワードが明確になり、論文検索の精度が格段に向上します。

---

## 様々なタイプの臨床疑問とPICO

PICOは、治療に関する疑問だけでなく、診断、予後、原因など、様々なタイプの臨床疑問に応用できます。

### 1. 治療 (Therapy)

*   疑問: 「高齢の高血圧患者に、サイアザイド系利尿薬とCa拮抗薬、どちらが脳卒中予防に有効か？」
*   P: 高齢の高血圧患者
*   I: Ca拮抗薬
*   C: サイアザイド系利尿薬
*   O: 脳卒中の発生率

### 2. 診断 (Diagnosis)

*   疑問: 「深部静脈血栓症（DVT）が疑われる患者に、D-dimer測定は診断の一助となるか？」
*   P: DVTが疑われる患者
*   I: D-dimer測定
*   C: D-dimer測定なし（または超音波検査のみ）
*   O: DVTの診断精度（感度・特異度）

### 3. 予後 (Prognosis)

*   疑問: 「急性心筋梗塞後、心不全を合併した患者の5年生存率は？」
*   P: 急性心筋梗塞後、心不全を合併した患者
*   I: (特定の介入なし)
*   C: (特定の比較なし)
*   O: 5年生存率

### 4. 原因 (Etiology / Harm)

*   疑問: 「経口避妊薬の服用は、静脈血栓塞栓症のリスクを増加させるか？」
*   P: 若年女性
*   I: 経口避妊薬の服用
*   C: 経口避妊薬の非服用
*   O: 静脈血栓塞栓症の発生率

## AIを使って臨床疑問をPICOに変換する

日常の臨床で浮かんだ漠然とした疑問を、AIに入力するだけで、構造化されたPICO形式に変換させることができます。

### 📋 PICO変換プロンプト（コピペ用）

```
あなたはEBMに精通した臨床医です。以下の日常的な臨床疑問を、PICO形式に構造化してください。

【臨床疑問】
[ここに、普段感じている疑問を自然な文章で入力]

【出力形式】
以下の形式で出力してください。

*   P (Patient/Problem):
*   I (Intervention):
*   C (Comparison):
*   O (Outcome):
```

### 実践例

入力プロンプト:
```
あなたはEBMに精通した臨床医です。以下の日常的な臨床疑問を、PICO形式に構造化してください。

【臨床疑問】
最近、外来に来た心房細動の患者さんに、新しい抗凝固薬（DOAC）と昔からのワルファリン、どっちを使うか迷っています。どっちが脳梗塞をよく防げて、安全なんですかね？
```

AIの出力例:
```
はい、承知いたしました。その臨床疑問をPICO形式に構造化します。

*   P (Patient/Problem): 非弁膜症性心房細動の患者
*   I (Intervention): 直接経口抗凝固薬（DOAC）の投与
*   C (Comparison): ワルファリンの投与
*   O (Outcome): 脳卒中または全身性塞栓症の発生率（有効性）、および重大な出血イベントの発生率（安全性）
```

このように、普段の言葉で問いかけるだけで、AIが的確なPICOを作成してくれます。これにより、次のステップである論文検索の準備が整います。

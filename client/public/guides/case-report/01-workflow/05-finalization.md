# ステップ5: 最終化と投稿準備

## 目的

AIが生成したドラフトを基に、図表、参考文献リスト、抄録（Abstract）を完成させ、ターゲットジャーナルへの投稿準備を整えます。この最終ステップでは、論文の「見た目」と「第一印象」をプロフェッショナルなレベルに引き上げます。

## 達成基準

- 論文の主張を視覚的に補強する高品質な図（Figure）と表（Table）が完成している。
- 参考文献リストがターゲットジャーナルの投稿規定に沿って正確にフォーマットされている。
- 論文の魅力を凝縮した、構造化抄録（Structured Abstract）とインパクトのあるタイトルが完成している。
- 投稿に必要なすべてのファイル（本文、図表、カバーレターなど）が準備できている。

## 詳細な手順

### 5-1. 図表の作成

「百聞は一見に如かず」の言葉通り、優れた図表は論文の理解度とインパクトを劇的に向上させます。臨床経過のグラフ、特徴的な画像所見、類似症例の比較表などを作成します。

- グラフ作成: 検査データの推移などは、Pythonの`matplotlib`や`seaborn`ライブラリを使うと、論文掲載レベルの高品質なグラフをコードで生成・再現できます。AI（Claude, Geminiなど）にコード生成を依頼することも可能です。
- イラスト作成: 疾患のメカニズムなどを視覚的に示すイラストは、BioRenderのような専門ツールが有用です。
- 表の作成: ステップ2でNotebookLMに作成させた類似症例の比較表などを、ターゲットジャーナルのフォーマットに合わせて整形します。

### 5-2. 参考文献リストのフォーマット

参考文献のフォーマットは、ジャーナルごとに異なり、非常に手間のかかる作業です。しかし、これもAIで自動化できます。

手順:

1.  文献管理ツールとの連携: ZoteroやMendeleyといった文献管理ツールに、引用したすべての論文を登録しておきます。
2.  フォーマット指示: ターゲットジャーナルの引用スタイル（例: Vancouver, AMA）をClaudeに指示し、Zoteroなどからエクスポートした文献リストをフォーマットさせます。

プロンプト例（Claude 3.5 Sonnet）:

```
# 指示
あなたは医学論文の編集者です。以下の文献リストを、Vancouverスタイルに従って正確にフォーマットしてください。

# 文献リスト
[Zoteroなどからエクスポートしたリストを貼り付け]
```

### 5-3. AbstractとTitleの最適化

Abstract（抄録）とTitle（タイトル）は、論文の「顔」です。読者や編集者が最初に目にする部分であり、ここで興味を引けなければ本文を読んでもらえません。論文の魅力を最大限に引き出すため、AIに複数のパターンを提案させ、最も優れたものを選びます。

プロンプト例（Claude 3.5 Sonnet）:

```
# 指示
あなたは優れたコピーライターであり、医学研究者でもあります。以下の完成した論文ドラフトを読み、読者の興味を強く引くようなタイトル案を5つ、構造化抄録（背景、症例提示、結論の3部構成）を1つ作成してください。

# 論文ドラフト
[ステップ4で完成した論文ドラフト全文を貼り付け]

# タイトル案の条件
- 15ワード以内
- 論文の最も重要な発見を含む
- 可能であれば、"A case report"という言葉を使わずに新規性を表現する

# 抄録の条件
- 250ワード以内
- ターゲットジャーナルの規定に準拠
```

### 5-4. 投稿準備

最後に、投稿に必要なすべての書類を準備します。

- 本文ファイル: Word形式またはLaTeX
- 図表ファイル: 高解像度（300 DPI以上）のTIFFまたはEPS形式
- タイトルページ: 著者情報、所属、連絡先、利益相反（COI）の開示など
- カバーレター: 論文の新規性と重要性を編集者にアピールする手紙。これもAIにドラフトを作成させることができます。

すべてのファイルが揃ったら、ターゲットジャーナルのオンライン投稿システム（例: ScholarOne, Editorial Manager）から投稿します。お疲れ様でした！

## AACR Methodのその先へ: 完全自動化の未来

今回紹介したAACR Methodは、現時点での最適解ですが、AIの進化は止まりません。将来的には、Claude ProjectsとMCP (Model Context Protocol) を組み合わせることで、このワークフローの大部分を自動化できる可能性があります。

- Elicit APIをMCP経由で呼び出し、文献検索を自動化
- Zotero APIと連携し、参考文献リストを自動生成
- CARE guidelinesをAIエージェントの思考フレームワークに組み込み、執筆精度を向上

このような「自律型執筆エージェント」が実現すれば、症例報告の執筆時間は現在の10時間から、わずか2〜3時間にまで短縮されるかもしれません。医学知識の共有は、新たな次元へと突入します。このガイドが、その未来への第一歩となることを願っています。

# Step 9: 緒言(Introduction)の執筆

### 📋 このステップの目的

論文の導入部である緒言（Introduction）を執筆します。緒言の役割は、読者を論文の世界に引き込み、「なぜこの症例報告を読む必要があるのか」を説得することです。背景知識の提供から問題提起、そして本論文の目的提示へと至る、論理的でスムーズな導入を構築する技術を学びます。

### ⏱️ 所要時間

**2〜3時間**

### 🎯 達成基準

- 緒言が、標準的な「3パラグラフ構成」で執筆されている。
- 背景 → 問題提起（知識のギャップ） → 目的、という論理的な流れが明確である。
- 緒言を読んだだけで、この症例報告が医学界にどのような貢献をするのかが理解できる。

### 📝 詳細手順

緒言は、漏斗（ファンネル）のように、広い話題から狭い話題へと絞り込んでいくイメージで構成します。

#### 手順1: 第1パラグラフ - 一般的背景

まず、テーマとなる疾患や病態についての一般的な情報を提供します。読者が専門家でなくても理解できるよう、基本的な知識から書き始めます。

- 疾患の定義、疫学、病態生理
- 標準的な診断法と治療法
- 予後

この部分は、Step 4でレビューした総説論文やガイドラインが主な情報源となります。

#### 手順2: 第2パラグラフ - 問題提起（知識のギャップ）

次に、一般的な知識だけではカバーできない「問題点」や「未解決の謎」を提示します。これが、あなたの症例報告が存在する意義、すなわち「知識のギャップ（Knowledge Gap）」です。

- 標準治療が効かない稀なケースの存在
- 未知の副作用や合併症
- 診断における落とし穴
- 確立されていない治療法

Step 4で作成した類似症例の比較表が、このギャップを浮き彫りにするための強力な証拠となります。

#### 手順3: 第3パラグラフ - 目的の提示

最後に、提示した「知識のギャップ」を、我々の症例がどのように埋めるのかを明確に宣言します。これは緒言の結論であり、論文全体の目的を示す最も重要な部分です。

- 「そこで我々は、〜という稀な臨床経過をたどった症例を経験したため、ここに報告する。」
- 「本稿では、〜という治療法が著効した症例を報告し、その有効性と安全性について考察する。」

### 💡 プロンプト集

#### プロンプト1: 3パラグラフ構成の緒言の草案を作成する

```
# 指示
あなたはメディカルライターです。以下の情報に基づき、症例報告の緒言（Introduction）を、標準的な3パラグラフ構成（背景→問題提起→目的）で執筆してください。

# 情報
- **疾患**: 器質化肺炎 (Organizing Pneumonia, OP)
- **背景情報 (パラグラフ1)**: OPは特発性間質性肺炎の一種で、通常はステロイド治療によく反応する。予後は比較的良好。
- **問題提起/知識のギャップ (パラグラフ2)**: しかし、稀にステロイドに抵抗性を示す症例が存在する。そのような難治性OPに対する最適な治療法は確立されていない。シクロホスファミドなどの免疫抑制薬が試されることがあるが、その有効性や安全性に関する報告は限定的である。
- **本症例の概要と目的 (パラグラフ3)**: 我々は、中等量のステロイド治療に早期から抵抗性を示したOPに対し、シクロホスファミド間欠療法が著効した稀な症例を経験した。本稿では、この症例を報告し、難治性OPの治療戦略について考察する。

# 制約
- 全体で200〜300ワード程度
- 学術論文として適切な、客観的でフォーマルな文体で記述すること
```

**活用例**: このプロンプトを使えば、文献レビューで得た知見を整理し、論理的な緒言の骨子を迅速に作成できます。生成された草案を基に、引用文献を追加し、表現を洗練させていきましょう。

### 📚 実践例

**症例**: ステロイド抵抗性OP

- **パラグラフ1 (背景)**: 器質化肺炎（OP）の定義、疫学、およびステロイドが第一選択薬であることを記述。参考文献として主要な総説を引用。
- **パラグラフ2 (問題提起)**: しかし、約10-20%の症例がステロイドに抵抗性を示すこと、その場合の次の一手が確立していないことを記述。シクロホスファミドなどの代替治療に関する既報告が数例しかないことに触れ、「知識のギャップ」を強調。
- **パラグラフ3 (目的)**: 「Herein, we report a case of steroid-resistant OP that showed rapid deterioration despite moderate-dose prednisolone but was successfully treated with intermittent intravenous cyclophosphamide therapy.（そこで我々は、中等量プレドニゾロンに抵抗し急速に増悪したが、シクロホスファミド間欠静注療法により寛解に至ったOPの1例を報告する。）」と締めくくる。

### ⚠️ よくある失敗と対処法

- **失敗**: 緒言が長すぎて、教科書のようになってしまう。
  - **対処法**: 緒言はミニレビューではありません。背景説明は必要最小限に留め、論文の核心である「問題提起」と「目的」に焦点を絞りましょう。3〜4パラグラフ、引用文献は5〜10本程度が目安です。

- **失敗**: 緒言で論文の結論まで書いてしまう。
  - **対処法**: 結論は考察（Discussion）の最後や結論（Conclusion）セクションで述べるものです。緒言では「何をしようとしているか（目的）」を述べるに留め、読者の期待感を煽りましょう。

### 💪 上級者向けTIPS

緒言の最後の一文（目的を提示する文）は、論文全体で最も重要な一文（The Most Important Sentence）と言っても過言ではありません。この一文を読めば、論文全体の価値がわかる、というレベルまで磨き上げましょう。指導医や共著者と何度も推敲を重ねる価値があります。

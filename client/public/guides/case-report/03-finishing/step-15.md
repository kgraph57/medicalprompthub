# Step 15: 英文校正

### 📋 このステップの目的

完成した論文原稿の英語の質を、プロフェッショナルなレベルに引き上げることが目的です。非ネイティブスピーカーにとって、自然で流暢な学術英語を書くことは非常に困難です。文法的な誤りや不自然な表現は、論文の内容以前に、査読者に悪い印象を与え、リジェクトの原因となりえます。このステップでは、AIと人間の専門家を組み合わせ、論文の英語を磨き上げます。

### ⏱️ 所要時間

**AI校正: 1〜2時間**
**プロの校正: 3日〜1週間**

### 🎯 達成基準

- 文法、スペル、句読点の誤りがゼロになっている。
- 不自然な表現や専門用語の誤用がなくなり、自然で読みやすい学術英語になっている。
- プロの英文校正サービスによる校正証明書が発行されている（ジャーナルによっては提出を求められる）。

### 📝 詳細手順

英文校正は、セルフチェック → AIチェック → プロのチェック、という3段階で行うのが理想的です。

#### 手順1: セルフチェックとAIによる一次校正

まずは、自分自身で原稿を音読し、おかしな点がないかを確認します。その後、AIライティングツールを使って、基本的な文法ミスやスペルミスを修正します。

- **Grammarly**: 文法・スペルチェックツールの定番。無料版でも基本的なミスは十分に検出できます。
- **DeepL Write**: DeepL翻訳の姉妹サービス。文章をより自然で洗練された表現に書き換えることを得意とします。
- **ChatGPT / Claude**: プロンプトを使って、特定のパラグラフをリライトさせたり、よりアカデミックな表現を提案させたりできます。

#### 手順2: プロの英文校正サービスへの依頼

AIによる校正は万能ではありません。文脈に応じた微妙なニュアンスや、専門分野特有の表現については、人間の専門家によるチェックが不可欠です。特に、トップジャーナルへの投稿を目指す場合は、プロの英文校正は必須と考えましょう。

- **サービスの種類**: 
  - **英文校正 (Editing)**: 文法、スペル、句読点、表現の自然さを修正。
  - **論文投稿支援 (Publishing Support)**: 校正に加え、論文の構成や論理展開にまで踏み込んでアドバイス。
- **主要なサービス**: 
  - **Edanz (エダンズ)**: 日本の研究者に広く利用されている。医学・歯学・薬学分野に強い。
  - **Editage (エディテージ)**: 24時間365日対応。幅広い分野をカバー。
  - **Enago (イーナゴ)**: 質の高い校正者と、きめ細かいサービスが特徴。

依頼する際は、論文の専門分野を指定し、投稿予定のジャーナル名を伝えると、そのジャーナルのスタイルに合わせた、より質の高い校正が期待できます。

### 💡 プロンプト集

#### プロンプト1: アカデミックな表現に書き換える (DeepL Write / ChatGPT)

```
# 指示
以下の英文を、よりフォーマルで、学術論文に適したアカデミックな表現に書き換えてください。

# 元の英文
We did a biopsy and found out it was OP. So, we gave the patient steroids. But it didn't work well. The patient got worse. Then, we tried IVCY, and it was great.

# 期待する出力（例）
A diagnosis of organizing pneumonia (OP) was established based on biopsy findings. Although corticosteroid therapy was initiated, the patient's condition deteriorated. Consequently, treatment with intravenous cyclophosphamide was administered, which resulted in a remarkable improvement.
```

**活用例**: 自分で書いた少し口語的な英語を、AIを使って瞬時に洗練された学術英語に変換できます。

### 📚 実践例

1.  **一次校正**: 完成した原稿全体をDeepL Writeにかけ、不自然な表現を修正。さらにGrammarlyで最終的な文法・スペルチェックを実施。
2.  **指導医レビュー**: AIで校正した原稿を指導医にレビューしてもらい、内容の最終確認。
3.  **プロ校正依頼**: 指導医の承認後、Edanzの英文校正サービスに依頼。専門分野を「呼吸器内科」、投稿予定誌を「Internal Medicine」と指定。3日後、修正履歴付きの原稿と校正証明書が納品された。

### ⚠️ よくある失敗と対処法

- **失敗**: 無料の自動翻訳ツール（例: Google翻訳）だけで英文を作成し、校正をしない。
  - **対処法**: 自動翻訳はあくまで下訳です。生成された英文には、文法的な誤りや不自然な表現が多く含まれます。必ずAI校正ツールやプロの校正を通して、文章の質を担保してください。

- **失敗**: 英文校正サービスから返ってきた修正を、確認せずにそのまま受け入れてしまう。
  - **対処法**: 校正者の修正が、必ずしもあなたの意図を正確に反映しているとは限りません。修正履歴を一つ一つ確認し、意図と異なる修正が加えられていないか、内容が変わってしまっていないかを吟味することが重要です。

### 💪 上級者向けTIPS

プロの校正者から返ってきた修正は、最高の英語学習教材です。なぜそのように修正されたのか、どのような表現がより自然なのかを分析し、次回の執筆に活かしましょう。修正箇所をまとめた自分だけの「表現ノート」を作成するのもお勧めです。

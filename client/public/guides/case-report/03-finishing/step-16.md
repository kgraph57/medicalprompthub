# Step 16: 投稿規定チェックリスト：門前払いを防ぐ最後の砦

### 📋 このステップの目的

いよいよ投稿直前！最後の関門、投稿規定の最終チェックです。どんなに素晴らしい論文も、ルール違反の服装ではパーティー会場に入れてもらえません。ここでは、「門前払いモンスター」を撃退し、確実に査読のリングに上がるための最終チェックリストを伝授します。

![投稿規定チェックリスト](https://files.manuscdn.com/user_upload_by_module/session_file/90346460/NXvbdkllmByWPjKb.png)

### ⏱️ 所要時間

1〜2時間

### 🎯 達成基準

- 論文のすべてが、投稿規定のフォーマットに完璧に準拠している。
- 投稿に必要なすべてのファイルが、正しく命名され、手元に揃っている。

### 📝 詳細手順

![投稿規定チェックリスト](/Helix/assets/guides/case-report/03-finishing/submission_guidelines_checklist.png)

投稿ジャーナルの「Instruction for Authors」を片手に、探偵になったつもりで、自分の原稿がすべての項目をクリアしているか、一つ一つ指差し確認していきましょう。

#### 手順1: 全体フォーマットの確認

- ファイル形式: Word (.docx)？ PDF？
- フォントと文字サイズ: Times New Romanの12ポイントが一般的。
- 行間: ダブルスペース？
- ページ番号と行番号: 必要？
- セクションの順序: タイトルページ → 抄録 → 本文…の順番は正しい？

#### 手順2: 各セクションの規定確認

- タイトルページ: 記載すべき情報（著者全員の氏名・所属、連絡先など）は全部入ってる？
- 抄録: 文字数制限（例: 250ワード）は守れてる？
- 本文: 文字数制限は大丈夫？
- 参考文献: 引用スタイルは完璧？文献数に上限はない？
- 図表: ファイル形式（TIFF, EPSなど）と解像度（300 dpi以上）はOK？ファイル名はルール通り？

#### 手順3: その他書類の準備

- カバーレター: 論文の「送り状」。次のステップで作成します。
- 利益相反(COI)申告書: 指定フォームはある？
- 著作権譲渡同意書: 署名は済んだ？

### 💡 プロンプト集

#### プロンプト1: 投稿規定から自分専用チェックリストを作成

```
# 指示
あなたは優秀なアシスタントです。以下の医学雑誌の投稿規定のテキストから、症例報告を投稿する際に確認すべき重要なフォーマット要件を抽出し、チェックリスト形式で日本語で要約してください。

# 投稿規定テキスト
[ここに投稿規定のウェブページからコピーしたテキストを貼り付ける]
```

活用例: 長い投稿規定を読む手間を省き、自分専用のチェックリストを瞬時に作成。確認漏れを劇的に減らせます。

### ⚠️ よくある失敗と対処法

- 失敗: 古い投稿規定を信じ込み、最新版を見逃す。
  - 対処法: 投稿規定は頻繁に改訂されます。投稿直前に、必ずもう一度サイトで最新情報を確認しましょう。

- 失敗: 細かい不備で、投稿システムから何度もエラー通知が来て、半日を無駄にする。
  - 対処法: このステップを軽視せず、時間をかけて丁寧に行うことが、結果的にスムーズな投稿に繋がります。「神は細部に宿る」です。

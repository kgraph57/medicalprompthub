# 感染対策マニュアル作成ガイド：基本編2

## アウトラインの生成と構成の最適化

ステップ1でマニュアルの目的と範囲が明確になりました。このステップでは、その情報を基にAIを活用してマニュアル全体の骨子（アウトライン）を生成し、医療現場での使いやすさを考慮して最適化します。

### 1. AIによる論理的なアウトラインの生成

AIは、入力された目的と対象者、そして「感染対策マニュアル」というキーワードから、論理的かつ網羅性の高い目次案を瞬時に生成できます。

具体的な手順:

1.  初期情報のインプット: ステップ1で整理したマニュアルの目的、対象者、そして含めるべき主要なトピック（例：標準予防策、経路別予防策、職業感染対策、サーベイランス）をAIに伝えます。
2.  目次案の生成指示: AIに対し、「医療従事者が緊急時にも参照しやすい構造」を意識した目次案を提案するよう指示します。
3.  階層構造の評価: 生成された目次案が、大項目、中項目、小項目と適切に階層化されているかを確認します。特に、現場での行動に直結する項目（例：針刺し事故発生時の対応）は、最上位の階層に配置するなど、アクセスのしやすさを重視します。

### 2. 実用的なプロンプト例：目次案の生成と修正

プロンプト例1：目次案の生成

```
あなたは、200床の急性期病院向け感染対策マニュアルの構成案を作成します。
マニュアルの対象者は、看護師、研修医、清掃スタッフです。
以下の要件を満たす、3階層（大項目、中項目、小項目）の目次案を提案してください。

[要件]
1.  標準予防策と経路別予防策を明確に区別すること。
2.  「緊急時の対応」（例：針刺し事故、感染性廃棄物の漏洩）を独立した章として設けること。
3.  清掃スタッフ向けの「環境整備」に関する項目を具体的に含めること。
4.  既存マニュアル（5年前作成）にはない、最新の「COVID-19を含む新興・再興感染症への対応」を含めること。
```

プロンプト例2：現場視点での修正指示

```
生成された目次案について、以下の点を修正してください。
「標準予防策」の章に、「個人防護具（PPE）の着脱手順」を、写真や図解を挿入しやすいように、ステップ形式（例：1. 手指衛生、2. ガウン着用...）で小項目として追加してください。
また、「職業感染対策」の章に、「職員の健康管理と予防接種」に関する項目を、具体的なチェックリスト形式で追記してください。
```

### 3. 注意点とTips

*   Tips: 現場の声を反映させる: AIが生成した論理的な目次案は、必ず現場のキーパーソン（ICD、ICN、各部署のリーダー）に見てもらい、「この構成で本当に使いやすいか」のフィードバックを得ましょう。AIは論理構造は得意ですが、現場の「動線」や「緊急性」の優先順位は人間が判断する必要があります。
*   注意点: 冗長な項目を避ける: 網羅性を追求するあまり、マニュアルが分厚くなりすぎると、誰も読まなくなります。AIが提案した項目の中で、病院の特性上、重要度が低いものや、他の文書で代替できるものは、思い切って削除または簡略化することを検討してください。

# Visual Abstractの作成

Visual Abstractは、研究論文の内容を1枚の図で要約したインフォグラフィックです。主要医学ジャーナルがSNSでの共有や学会発表で活用しています。

## Visual Abstractとは

**Visual Abstract（ビジュアルアブストラクト）**は、研究論文の主要な発見や結論を1枚のインフォグラフィックとして視覚的に要約したものです。BMJ、JAMA、NEJM、Lancet、Nature Medicineなどの主要医学ジャーナルが、論文のSNS共有や学会発表で積極的に使用しています。

### Visual Abstractの定義

Visual Abstractは**研究論文の要約を1スライドにまとめたインフォグラフィック**であり、以下の要素を含みます：

1. **研究デザイン**：RCT、コホート研究、症例対照研究など
2. **対象集団**：患者数、年齢、性別、疾患など
3. **介入/曝露**：治療法、薬剤、リスク因子など
4. **比較対照**：プラセボ、標準治療、対照群など
5. **主要アウトカム**：主要評価項目と結果データ
6. **結論**：研究の核心的発見

**重要:** Visual Abstractは、診断フローチャートや疾患メカニズム図ではなく、**特定の研究論文の内容を要約したもの**です。

### Visual Abstractの利点

1. **高い共有率**：テキストのみの投稿と比較して3〜5倍の共有率
2. **理解の促進**：複雑な研究内容を直感的に理解できる
3. **アクセシビリティ**：言語の壁を超えて情報を伝達
4. **プロフェッショナルな印象**：研究の質と信頼性を視覚的に示す

## 主要ジャーナル別Visual Abstractスタイル

各医学ジャーナルは独自のVisual Abstractスタイルを持っています。以下では、5つの主要ジャーナルのスタイルと特徴を紹介します。

### 1. BMJ Style（British Medical Journal）

**特徴:**
- **レイアウト**: 左サイドバー（20%）+ メインコンテンツ（80%）
- **カラー**: BMJブルー（#005EB8）、ティール（#00A9CE）、マゼンタ（#DC004E）
- **構成**: Summary → Study design → Population → Comparison → Outcomes
- **強み**: アイコン重視、データ可視化が充実

**適した研究:**
- 大規模RCT
- 介入研究
- 明確な比較対照がある研究

![BMJ Style Visual Abstract](/Helix/assets/guides/visual-abstract/bmj_style.png)

**この例の研究内容（架空）:**
- **試験名**: CEREBRO-PROTECT試験
- **デザイン**: ランダム化比較試験（24カ国、多施設共同研究）
- **対象**: 高血圧患者1,850名（55-80歳、過去2年以内にTIA既往）
- **介入**: 新規降圧薬XR-2847 200mg/日 + 標準治療 vs 標準治療のみ
- **主要評価項目**: 3年間の脳卒中発生率
- **結果**: 介入群8.2% vs 対照群14.1%（HR 0.58, 95% CI 0.44-0.76）
- **結論**: XR-2847は脳卒中発生率を42%減少させた

### 2. JAMA Style（Journal of the American Medical Association）

**特徴:**
- **レイアウト**: Question/Conclusion形式、セクション別ボックス
- **カラー**: JAMAレッド（#D22630）、グレー、ベージュ、ゴールド
- **構成**: Question → Conclusion → Population → Location → Intervention → Findings
- **強み**: シンプル、明確な問いと答え

**適した研究:**
- 臨床的に重要な問いを扱う研究
- 診断精度研究
- スクリーニング研究

![JAMA Style Visual Abstract](/Helix/assets/guides/visual-abstract/jama_style.png)

**この例の研究内容（架空）:**
- **試験名**: DEEP-LUNG試験
- **研究疑問**: AI支援CT読影は肺癌早期発見を改善するか？
- **デザイン**: ランダム化比較試験（米国、カナダ、英国、ドイツの48施設）
- **対象**: 喫煙歴≥30 pack-yearの50-75歳成人3,240名
- **介入**: AI支援読影 vs 標準読影
- **主要評価項目**: 24ヶ月以内のStage I肺癌検出率
- **結果**: AI支援群5.5%（89例） vs 標準群4.0%（64例）、RR 1.38（95% CI 1.15-1.67）
- **結論**: AI支援スクリーニングは早期肺癌検出率を38%改善し、偽陽性率を24%減少させた

### 3. NEJM Style（New England Journal of Medicine）

**特徴:**
- **レイアウト**: 3カラム構成（Trial Design | Endpoints | Findings）
- **カラー**: ネイビーブルー（#003366）、バーガンディ、グレー
- **構成**: クラシックで上品、データ重視
- **強み**: メカニズム図解、詳細なデータ表示

**適した研究:**
- 新薬の臨床試験
- メカニズムが重要な研究
- 複数のエンドポイントがある研究

![NEJM Style Visual Abstract](/Helix/assets/guides/visual-abstract/nejm_style.png)

**この例の研究内容（架空）:**
- **試験名**: SAFEGUARD試験
- **デザイン**: Phase 3ランダム化二重盲検実薬対照試験
- **対象**: 非弁膜症性心房細動患者2,680名（CHA2DS2-VASc≥2）
- **介入**: 新規抗凝固薬ZT-4501 75mg 1日2回 vs ワルファリン（INR調整）
- **主要評価項目**: 
  - 有効性：脳卒中または全身性塞栓症
  - 安全性：大出血イベント
- **結果**: 
  - 脳卒中/塞栓症：ZT-4501 2.1% vs ワルファリン 2.4%（HR 0.87, 非劣性達成）
  - 大出血：ZT-4501 1.8% vs ワルファリン 3.6%（HR 0.49, 優越性達成）
  - 頭蓋内出血：ZT-4501 0.3% vs ワルファリン 1.2%（HR 0.24）
- **結論**: ZT-4501は脳卒中予防で非劣性を示し、出血リスクを有意に低減

### 4. Lancet Style（The Lancet）

**特徴:**
- **レイアウト**: Background → Methods → Intervention → Results → Interpretation
- **カラー**: Lancetレッド（#E31E24）、オレンジ、グレー
- **構成**: モダン、公衆衛生重視
- **強み**: 社会的インパクト、グローバルヘルス

**適した研究:**
- ワクチン研究
- 公衆衛生介入
- グローバルヘルス研究

![Lancet Style Visual Abstract](/Helix/assets/guides/visual-abstract/lancet_style.png)

**この例の研究内容（架空）:**
- **研究名**: VARIANT-SHIELD研究
- **背景**: 新興SARS-CoV-2変異株が既存ワクチンの有効性を脅かす
- **デザイン**: 前向きコホート研究（6カ国、12ヶ月追跡）
- **対象**: 18,500名（18-49歳45%、50-64歳35%、65歳以上20%）
- **ワクチン群**: 
  - 更新型mRNAワクチン（2価、6,200名）
  - 従来型mRNAワクチン（1価、6,150名）
  - 未接種対照群（6,150名）
- **主要評価項目**: PCR確定症候性COVID-19感染
- **結果**: 
  - Alpha株：更新型94%、従来型91%
  - Delta株：更新型89%、従来型78%
  - Omicron BA.5：更新型76%、従来型52%
  - 感染率：更新型2.8%、従来型5.4%、未接種18.2%
- **解釈**: 更新型2価mRNAワクチンは新興変異株、特にOmicron系統に対して優れた防御効果を示し、変異株適応型ワクチン戦略を支持

### 5. Nature Medicine Style

**特徴:**
- **レイアウト**: Disease Mechanism → Approach → Results → Significance
- **カラー**: Natureグリーン（#00695C）、ブルー、オレンジ
- **構成**: 科学的詳細、分子メカニズム図解
- **強み**: 基礎研究と臨床の橋渡し、メカニズム重視

**適した研究:**
- トランスレーショナル研究
- 遺伝子治療
- 分子メカニズムが重要な研究

![Nature Medicine Style Visual Abstract](/Helix/assets/guides/visual-abstract/nature_medicine_style.png)

**この例の研究内容（架空）:**
- **試験名**: CRISPR-Cas9遺伝子治療 Phase 1/2試験
- **疾患**: Duchenne型筋ジストロフィー（DMD）
- **メカニズム**: 
  - 正常：ジストロフィンが細胞膜を安定化
  - DMD：exon 51欠失によりジストロフィン欠損、筋線維変性
- **対象**: exon 51欠失を持つDMD男児12名（6-12歳、歩行可能）
- **遺伝子治療アプローチ**: 
  - AAVベクターでCRISPR-Cas9を筋肉細胞に導入
  - sgRNAがCas9をexon 50/52接合部に誘導
  - Exon skippingにより短縮型だが機能的なジストロフィンを産生
  - 単回筋肉内注射（2×10^13 vg/kg）
- **結果**: 
  - ジストロフィン発現：治療前0.2% → 3ヶ月後28.4% → 12ヶ月後42.7%
  - 6分間歩行距離：平均+52メートル改善
  - North Star評価：+3.8ポイント改善
  - クレアチンキナーゼ：68%減少
- **意義**: 遺伝性筋疾患において、in vivo CRISPR遺伝子編集が機能的タンパク質発現を回復させた初の実証。精密医療アプローチの概念実証を確立

## Visual Abstractの10のルール

以下は、PLoS Computational Biologyで発表された「Ten Simple Rules for Visual Abstracts」に基づく原則です。

### Rule 1: 主要なメッセージを1つに絞る

**原則:** Visual Abstractは論文全体ではなく、最も重要な発見1つに焦点を当てる

❌ **悪い例:** 研究の全ステップを詰め込む  
✅ **良い例:** 「治療Xは疾患Yの予後を50%改善した」という核心的発見のみ

### Rule 2: シンプルなレイアウトを使用

**推奨レイアウト:**
- **横向き（Landscape）**：Twitter/SNS投稿に最適（16:9または4:3）
- **縦向き（Portrait）**：学会ポスターに最適（3:4または9:16）

**構成要素:**
1. タイトル（上部）
2. 主要な図解（中央）
3. 結論（下部）
4. 著者情報・所属（最下部）

### Rule 3: 色を戦略的に使用

**色の使用原則:**
- メインカラー：2〜3色に限定
- 強調色：重要な発見を示すために1色
- 背景色：白または明るいグレー（読みやすさ優先）

**色覚異常への配慮:**
- 赤と緑の組み合わせを避ける
- 色だけでなく、形状やパターンでも区別する

### Rule 4: テキストを最小限に

**テキストの原則:**
- タイトル：15語以内
- 本文：50語以内
- 箇条書き：3〜5項目まで

**フォント選択:**
- サンセリフ体（Arial, Helvetica, Calibri）
- 最小フォントサイズ：14pt（印刷時）、18pt（スクリーン表示）

### Rule 5: アイコンとシンボルを活用

**効果的なアイコン使用:**
- 患者：人型アイコン
- 治療：薬のアイコン、注射器
- 結果：グラフ、矢印（上昇/下降）
- 統計：パーセンテージ、p値

### Rule 6: データを視覚化

**効果的なデータ表示:**
- 棒グラフ：グループ間比較
- 折れ線グラフ：時系列変化
- Forest plot：ハザード比、リスク比
- インフォグラフィック：統計的発見

### Rule 7: ロジカルな流れを作る

**視線の流れ:**
1. 左上から右下（西洋文化）
2. 矢印で流れを明示
3. 番号付けで順序を示す

### Rule 8: ブランディング要素を含める

**必須要素:**
- 論文タイトル
- 著者名
- 所属機関
- ジャーナル名（掲載後）
- DOI（掲載後）

**オプション要素:**
- 機関ロゴ
- QRコード（論文へのリンク）
- ハッシュタグ

### Rule 9: ターゲットオーディエンスを考慮

**オーディエンス別の調整:**
- **専門家向け**：専門用語OK、詳細なデータ
- **一般向け**：平易な言葉、シンプルな概念
- **SNS向け**：キャッチーなタイトル、視覚的インパクト

### Rule 10: フィードバックを得て改善

**レビュープロセス:**
1. 同僚からのフィードバック
2. 非専門家の理解度テスト
3. 複数のバージョンを作成して比較

## Nanobananaで Visual Abstract を作成するコツ

### 1. ジャーナルスタイルを明確に指定

```
Create a professional BMJ-style visual abstract infographic for a randomized controlled trial.
Include the characteristic BMJ blue sidebar with icons for: Summary, Study design, Population, Comparison, Outcomes.
Main content area should show intervention vs control comparison with clear data visualization.
Use BMJ brand colors: blue #005EB8, teal #00A9CE, magenta #DC004E.
```

### 2. 研究データを具体的に記載

```
Study details:
- Title: [Your study title]
- Design: Randomized controlled trial, double-blind, multicentre
- Population: [N] patients with [condition], aged [age range]
- Intervention: [Treatment A] vs Control: [Treatment B]
- Primary outcome: [Outcome measure]
- Results: Intervention [X%] vs Control [Y%], HR [value] (95% CI [range])
- Conclusion: [Key finding in one sentence]
```

### 3. レイアウトと要素配置を詳細に指定

```
Layout structure:
- Top: Title in [journal] style
- Left section (30%): Study design and population with icons
- Center section (40%): Intervention comparison boxes and mechanism illustration
- Right section (30%): Results with data visualization (bar chart, forest plot)
- Bottom: Conclusion banner and citation
```

### 4. カラースキームとブランディング

```
Color scheme:
- Primary: [Journal brand color]
- Secondary: [Accent colors]
- Background: White or light gray
- Data visualization: Use color-blind friendly palette

Include:
- Journal logo in appropriate position
- Author names and affiliations
- DOI or citation information
```

## スタイル選択のガイドライン

| ジャーナル | 適した研究タイプ | 主な特徴 | 推奨する場合 |
|:---|:---|:---|:---|
| **BMJ** | 大規模RCT、介入研究 | アイコン重視、データ可視化充実 | 明確な比較対照、複数アウトカム |
| **JAMA** | 診断精度、スクリーニング | Question/Conclusion形式、シンプル | 臨床的問いが明確、一般向け |
| **NEJM** | 新薬臨床試験、メカニズム研究 | クラシック、詳細データ表示 | 複数エンドポイント、メカニズム重要 |
| **Lancet** | ワクチン、公衆衛生介入 | モダン、社会的インパクト | グローバルヘルス、政策的意義 |
| **Nature Medicine** | トランスレーショナル研究 | 分子メカニズム図解、科学的詳細 | 基礎と臨床の橋渡し、遺伝子治療 |

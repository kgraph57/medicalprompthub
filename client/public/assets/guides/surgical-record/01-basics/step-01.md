# ステップ1: 手術記録の必須要素

## 手術記録に含めるべき項目

手術記録には、法的・医学的に必要な情報を漏れなく記載する必要があります。

### 1. 基本情報

| 項目 | 内容 | 例 |
| :--- | :--- | :--- |
| 患者情報 | ID、氏名、年齢、性別 | 12345678 山田太郎 70歳 男性 |
| 手術日時 | 開始・終了時刻 | 2025年1月15日 9:00-12:30 |
| 術者・助手 | 執刀医、第一・第二助手 | 術者: 〇〇 助手: △△, □□ |
| 麻酔科医 | 担当麻酔科医名 | 麻酔科 〇〇 |

---

### 2. 診断名

| 項目 | 内容 |
| :--- | :--- |
| **術前診断** | 手術適応となった診断名 |
| **術後診断** | 術中所見に基づく最終診断 |

#### 記載のポイント
- ICD-10に準拠した病名
- 術前・術後で診断が異なる場合は明記

---

### 3. 術式

#### 記載形式

```
【術式】
胃幽門側切除術（D2郭清）
胆嚢摘出術（腹腔鏡下）
Billroth-I法再建
```

#### 注意点
- 正確な術式名を記載
- 郭清度、アプローチ（開腹/腹腔鏡）を明記
- 併施手術は全て記載

---

### 4. 麻酔方法

| 麻酔方法 | 例 |
| :--- | :--- |
| 全身麻酔 | GOF（気管内挿管全身麻酔） |
| 区域麻酔 | 硬膜外麻酔、脊椎麻酔 |
| 局所麻酔 | 1%キシロカイン局所浸潤 |
| 併用 | GOF + 硬膜外麻酔 |

---

### 5. 手術所見・経過

#### 必須記載事項

- **皮膚切開**: 部位、長さ、種類
- **腹腔内所見**: 癒着、腹水、転移など
- **腫瘍所見**: サイズ、局在、浸潤範囲
- **リンパ節**: 郭清範囲、転移の有無
- **剥離操作**: 重要な手順の詳細
- **吻合・再建**: 方法、使用器材
- **止血確認**: 最終止血確認
- **閉腹**: 層ごとの閉鎖方法

---

### 6. 出血量・輸血

| 項目 | 記載内容 |
| :--- | :--- |
| **出血量** | 推定出血量（mL） |
| **輸血** | 輸血製剤の種類と単位数 |
| **輸液** | 術中投与量 |

```
【出血量】約450mL
【輸血】RCC-LR 4単位
【輸液】乳酸リンゲル液 2500mL
```

---

### 7. 使用物品・標本

#### 使用物品

| カテゴリ | 例 |
| :--- | :--- |
| 自動縫合器 | Endo GIA 60mm x 3発 |
| 吸収糸 | 3-0 VICRYL |
| ドレーン | 10Fr J-VAC ダグラス窩 |
| メッシュ | ポリプロピレンメッシュ 10×15cm |

#### 摘出標本

```
【摘出標本】
1. 幽門側胃（腫瘍含む）: 病理提出
2. 大網（部分）: 病理提出
3. リンパ節 #1, 3, 7, 8a, 9, 11p: 個別に病理提出
```

# ステップ4: 術式別テンプレート

## テンプレート活用のメリット

術式別のテンプレートを使用することで、**記載漏れを防ぎ**、**作成時間を短縮**できます。

## 消化器外科

### 腹腔鏡下胆嚢摘出術

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
術式: 腹腔鏡下胆嚢摘出術
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【手術所見および経過】

1. ポート挿入
臍部にオープン法でカメラポートを挿入した。
気腹し（10mmHg）、心窩部および右側腹部に
ワーキングポートを計3本挿入した。

2. 腹腔内所見
肝表面・腹膜に異常所見なし。
胆嚢は[正常大/腫大/萎縮]であった。
周囲との癒着は[なし/軽度/高度]であった。

3. Calot三角の展開
胆嚢頸部を把持し、Calot三角を展開した。
胆嚢動脈を同定し、クリッピング後切離した。
胆嚢管を同定し、クリッピング後切離した。
Critical View of Safety（CVS）を確認した。

4. 胆嚢剥離
肝床から胆嚢を電気メスにて剥離した。
止血を確認した。

5. 標本回収・閉創
胆嚢を回収バッグに収納し、臍部創から摘出した。
腹腔内を観察し、出血・胆汁漏れがないことを確認した。
ポート孔を閉鎖した。

【出血量】少量（10mL未満）
【輸血】なし

【使用物品】
・5mmクリップ × __個
・回収バッグ × 1

【摘出標本】
胆嚢（結石含む/含まない）: 病理提出

【合併症】なし

【術後指示】
・翌日より飲水・食事開始
・ドレーンなし
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 虫垂切除術（腹腔鏡下）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
術式: 腹腔鏡下虫垂切除術
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【手術所見および経過】

1. ポート挿入
臍部にオープン法でカメラポートを挿入した。
左下腹部および恥骨上にワーキングポートを挿入した。

2. 腹腔内所見
腹水: [なし/少量/中等量]（[漿液性/膿性]）
虫垂所見: [発赤・腫大/壊疽/穿孔]
周囲膿瘍: [なし/あり]
回盲部: [正常/炎症波及]

3. 虫垂切除
虫垂間膜を超音波凝固切開装置にて処理した。
虫垂根部をEndo loop × 2本で結紮した。
虫垂を根部で切離した。
断端を電気メスで焼灼した。

4. 洗浄・ドレナージ
右下腹部を生理食塩水で洗浄した。
[ドレーン留置/ドレーンなし]

5. 閉創
虫垂を回収バッグに収納し摘出した。
ポート孔を閉鎖した。

【出血量】少量
【使用物品】
・Endo loop × 2
・超音波凝固切開装置

【摘出標本】
虫垂: 病理提出

【術後指示】
・抗菌薬: CTRX 2g/日 × __日間
・飲水・食事: [翌日/腸蠕動回復後]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 整形外科

### 骨接合術（大腿骨頸部骨折）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
術式: 観血的骨接合術（大腿骨頸部骨折）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【手術所見および経過】

1. 体位・展開
仰臥位にて牽引台に固定。
透視下に整復位を確認した。

2. アプローチ
大転子外側に約5cmの皮膚切開を加えた。
大腿筋膜を切開し、外側広筋を分けて
大腿骨外側皮質に達した。

3. 整復
透視下に徒手整復を行った。
正面像: 頸部軸はGarden alignment index [内反/正常/外反]
側面像: [前方/正常/後方]傾斜

4. 固定
ガイドピンを刺入し、位置を確認した。
[使用インプラント名]を刺入した。
Tip-apex distance: __mm
透視下に固定位置を確認した。

5. 閉創
洗浄後、ドレーンを留置した。
層々閉鎖した。

【出血量】__mL
【輸血】[なし/RCC-LR __単位]

【使用インプラント】
[CHS/γネイル/ハンソンピン] __mm

【術後X線】
正面・側面像で良好な固定位置を確認。

【術後指示】
・荷重: [免荷/部分荷重/全荷重]
・DVT予防: [弾性ストッキング/フットポンプ/抗凝固薬]
・リハビリ: 術後__日目から開始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

6. 合併症チェック

## テンプレートのカスタマイズ

### 施設ごとの調整ポイント

- 使用物品の品名
- 電子カルテのフォーマット
- 施設固有の確認事項
- 保険請求に必要な記載

### 定期的な更新

- 新しいデバイス・術式への対応
- 法改正・診療報酬改定への対応

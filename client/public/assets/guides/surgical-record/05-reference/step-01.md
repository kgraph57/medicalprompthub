# ステップ5: 法的・保険請求上の注意点

## 手術記録の法的位置づけ

手術記録は**法的文書**であり、医療訴訟の際に重要な証拠となります。

---

## 法的要件

### 医療法・医師法の規定

| 法律 | 内容 |
| :--- | :--- |
| 医師法第24条 | 診療録の記載義務 |
| 医療法施行規則 | 診療に関する諸記録の保存（5年間） |
| 保険医療機関及び保険医療養担当規則 | 診療録等の記載要件 |

### 手術記録の保存期間

- **最低5年間**の保存義務
- ただし、医療訴訟の時効（一般的に10-20年）を考慮し、**長期保存が推奨**

#### 避けるべき記載

| 悪い例 | 問題点 | 良い例 |
| :--- | :--- | :--- |
| 「出血は少なめ」 | 主観的 | 「出血量約100mL」 |
| 「普通に剥離」 | 曖昧 | 「電気メスにて鋭的に剥離」 |
| 「問題なく終了」 | 不明確 | 「合併症なく予定術式を完遂」 |
| 「いつもどおり」 | 不明確 | 具体的な手順を記載 |

---

### 2. 時系列の明確化

```
✓ 良い例
「9:00 皮膚切開
 9:30 腹腔鏡挿入、腹腔内観察
 10:00 胆嚢動脈処理
 10:30 胆嚢摘出
 10:45 閉創」

✗ 悪い例
「手術を行った」（時間経過不明）
```

---

### 3. インフォームドコンセントとの整合性

- 術前説明と**実際の術式の一致**を確認
- 術式変更があった場合は**理由を明記**
- 追加手術が必要だった場合は**判断根拠を記載**

---

### 4. 修正・追記のルール

| ルール | 内容 |
| :--- | :--- |
| **訂正** | 二重線で消し、日時・署名を付記 |
| **追記** | 追記日時を明記、なぜ追記したか記載 |
| **電子カルテ** | 修正履歴が残るシステムを使用 |

#### 絶対にしてはいけないこと

❌ 白紙部分を削除して再作成
❌ 修正液で消す
❌ 日付を遡っての記載

## 保険請求上の注意点

### 術式名の記載

- **正確な術式名**を使用（Kコード対応）
- 保険収載されている術式名を使用
- 複数術式の場合は**主たる術式を明確に**

#### 例

```
✓ 正確な記載
「腹腔鏡下胃切除術（悪性腫瘍手術）」

✗ 不正確な記載
「胃の手術」「胃オペ」
```

---

### 必要な記載事項（診療報酬請求）

| 項目 | 理由 |
| :--- | :--- |
| **手術時間** | 時間加算の根拠 |
| **麻酔方法** | 麻酔料算定の根拠 |
| **使用物品** | 特定保険医療材料の請求 |
| **術式の詳細** | 点数の妥当性確認 |
| **病理標本** | 病理検査料の算定 |

---

### 査定・返戻を防ぐポイント

1. **術前診断と術式の整合性**
   - 診断名と手術の適応が合致しているか

2. **医学的必要性の記載**
   - なぜその術式を選択したか
   - 追加術式が必要だった理由

3. **合併症・偶発症の記載**
   - 追加処置が必要だった場合の理由

## 医療事故・訴訟対策

### 手術記録が重要となる場面

| 場面 | 手術記録の役割 |
| :--- | :--- |
| **医療事故調査** | 経緯の確認 |
| **民事訴訟** | 過失の有無の判断材料 |
| **刑事事件** | 業務上過失の評価 |
| **院内調査** | 原因分析 |

---

### 記載のポイント

#### 1. 予見・対応の記録

```
「癒着が高度であり、剥離に注意を要した。
腸管損傷を避けるため、鋭的剥離から鈍的剥離に変更した」
```

#### 2. 判断根拠の明確化

```
「腫瘍が膵臓に浸潤していたため、
膵合併切除の方針としたが、ご家族への術前説明で
合併切除の可能性についても同意を得ていた」
```

#### 3. 困難な状況の記録

```
「出血点の同定に時間を要したが、
圧迫止血と凝固止血を併用し、
最終的に確実な止血を確認した」
```

## まとめ

### 手術記録作成の原則

1. **正確性**: 客観的事実を記載
2. **完全性**: 必要項目を漏れなく記載
3. **即時性**: 可能な限り術後すぐに記載
4. **整合性**: 術前説明・麻酔記録との一致

### AIは補助ツール

- AI出力は必ず医師が確認
- 最終責任は記載した医師にある
- テンプレートは施設に合わせてカスタマイズ

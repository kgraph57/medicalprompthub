# 手術記録・処置記録作成ガイド

## はじめに

手術記録は**法的文書**であり、**診療報酬請求の根拠**でもあります。正確で漏れのない記録を効率的に作成することは、すべての外科系医師にとって重要なスキルです。

> 「手術記録の作成に時間がかかりすぎて、次の手術に遅れそうになった...」

このガイドでは、AIを活用して**質を維持しながら作成時間を短縮**する方法を紹介します。

---

## 手術記録の重要性

### 法的・倫理的観点

| 観点 | 内容 |
| :--- | :--- |
| **法的責任** | 医療訴訟の際の重要な証拠 |
| **説明義務** | 患者への説明内容の記録 |
| **継続性** | 術後管理・再手術時の情報源 |
| **教育** | 研修医教育の素材 |

### 診療報酬の観点

- 手術記録は**保険請求の根拠**
- 不備があると**査定・返戻の原因**に
- 術式・所見の正確な記載が必須

## 手術記録作成の課題

### よくある問題

| 問題 | 影響 |
| :--- | :--- |
| 記載漏れ | 術後管理に支障、訴訟リスク |
| 曖昧な記載 | 再手術時の情報不足 |
| 時間不足 | 記録が後回しに、記憶が薄れる |
| 標準化の欠如 | 質のばらつき |

## AI活用のメリット

### 時間短縮と質の向上を両立

| 従来の方法 | AI支援 |
| :--- | :--- |
| 白紙から記載 | 構造化されたテンプレート |
| 記載漏れリスク | チェックリストで漏れ防止 |
| 医学用語の揺れ | 標準的な用語に統一 |
| 術後に長時間かかる | 術中メモから自動生成 |

## 注意事項

### 個人情報の取り扱い

⚠️ **重要**: AIツールに入力する際は**必ず匿名化**してください

- 患者名 → 「症例」「患者」
- ID番号 → 削除
- 日付 → 一般化（「術後〇日目」など）

### AIの限界

- AIは**補助ツール**であり、**最終確認は医師が行う**
- 術式の詳細は**自身の記憶・術中メモ**に基づく
- 合併症・出血量などは**正確な値**を記載

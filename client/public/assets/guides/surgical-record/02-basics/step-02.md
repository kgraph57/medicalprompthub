# ステップ2: 処置記録の構造化

## 処置記録とは

手術室外で行われる**各種処置**（CVカテーテル挿入、胸腔ドレナージ、創傷処置など）の記録です。手術記録同様、正確な記載が求められます。

---

## 処置記録の重要性

| 観点 | 内容 |
| :--- | :--- |
| **法的** | 処置の正当性・適切性の証明 |
| **医学的** | 合併症発生時の参照資料 |
| **教育的** | 研修医教育・技術向上 |
| **保険** | 診療報酬請求の根拠 |

## 処置記録のポイント

### 1. リアルタイム記録

- 処置直後に記載（記憶が鮮明なうちに）
- 術中メモの活用

### 2. 客観的記載

- 主観を排除
- 数値で記録（サイズ、量など）

### 3. 合併症の記載

- 発生した場合は詳細に
- 発生しなかった場合も「合併症なし」と明記

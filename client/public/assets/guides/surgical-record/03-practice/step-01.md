## プロンプト集

### プロンプト1: 術中メモからの構造化

```
以下の術中メモを、正式な手術記録形式に構造化してください。

【術式】{術式名}
【術中メモ】
{術中に記録したメモを貼り付け}

以下の項目を含む手術記録を作成してください：
1. 手術所見および経過（時系列で詳細に）
2. 出血量・輸血
3. 使用物品
4. 摘出標本
5. 特記事項

医学的に正確で、法的文書として適切な表現を使用してください。
```

---

### プロンプト2: 術式別テンプレート生成

```
以下の術式の手術記録テンプレートを作成してください。

【術式】{術式名を入力}
【アプローチ】開腹/腹腔鏡/ロボット支援下
【再建法】（該当する場合）

テンプレートには以下を含めてください：
1. その術式に特有の所見・操作項目
2. 確認すべきポイント
3. よく使用する物品のリスト
4. 合併症リスクの項目

穴埋め形式で、実際の手術時に記入しやすい形式にしてください。
```

---

### プロンプト3: 手術記録のレビュー

```
以下の手術記録をレビューし、改善点を指摘してください。

【手術記録】
{作成した手術記録を貼り付け}

以下の観点でチェックしてください：
1. 必須項目の記載漏れはないか？
2. 記載内容は具体的で客観的か？
3. 時系列は論理的か？
4. 医学用語の使用は適切か？
5. 法的文書として問題ないか？
6. 保険請求上問題となりそうな点はないか？

改善案があれば具体的に提示してください。
```

## 実践例

### Before: 術中メモ

```
胃ca、幽門側胃切
腹腔鏡で開始、癒着なし
大網切離
右胃動静脈処理、#6
十二指腸切離 Endo60
左胃動静脈、#7郭清
胃切離、小弯側マーキング
小開腹で吻合、Billroth-I
#8a、9、11pも郭清
ドレーン×2
出血200くらい
```

### After: AI支援で構造化

```
【手術所見および経過】

1. 体位・アプローチ
砕石位にて手術開始。臍部および4ポートにて腹腔鏡を挿入した。
腹腔内を観察し、腹水なく、肝表面転移・腹膜播種を認めなかった。

2. 授動・郭清操作
大網を肝弯曲部より剥離し、胃大弯側を授動した。
右胃大網動静脈を根部で処理し、#4dリンパ節を郭清した。
続いて幽門上リンパ節（#5）、幽門下リンパ節（#6）を郭清した。
右胃動静脈を根部で処理した。

3. 十二指腸切離
十二指腸球部をEndo GIA 60mmにて切離した。

4. 左側郭清
左胃動静脈を根部で処理し、#7リンパ節を郭清した。
#8a、#9、#11pリンパ節を郭清した。

5. 胃切離・再建
小弯側のマーキングに従い、Endo GIAにて胃体部を切離した。
上腹部正中に約5cm小開腹し、標本を摘出した。
Billroth-I法にて胃十二指腸吻合を行った
（器械吻合: CDH 25mm）。

6. 止血確認・閉創
腹腔内を洗浄し、止血を確認した。
ウィンスロー孔および吻合部近傍にドレーンを各1本留置した。
ポート孔および開腹創を層々閉鎖した。

【出血量】約200mL
【輸血】なし

【使用物品】
・Endo GIA 60mm × 3発
・CDH 25mm × 1
・3-0 VICRYL
・10Fr J-VACドレーン × 2本

【摘出標本】
幽門側胃（病変含む）: 病理提出
郭清リンパ節（#4d, 5, 6, 7, 8a, 9, 11p）: 個別に提出
```

## 効率化のコツ

### 1. 術中メモを充実させる

- 重要なステップごとにメモ
- 時間、出血、使用器材を記録
- 音声入力の活用も検討

### 2. テンプレートの準備

- よく行う術式はテンプレート化
- 電子カルテにマスター登録

### 3. ルーティン化

- 手術終了後すぐに記録開始
- AIで下書き → 確認・修正

## 注意点

### 個人情報の取り扱い

⚠️ AIに入力する際は**必ず匿名化**

| 元の情報 | 匿名化後 |
| :--- | :--- |
| 山田太郎 70歳男性 | 70代男性 |
| 2025年1月15日 | 削除 |
| ID: 12345678 | 削除 |

### AI出力の確認

- 必ず医師が最終確認
- 事実と異なる記載がないか確認
- 術式特有の表現の適切性を確認

## 診療科別テンプレート

### 内科病棟用

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[🟡] 70代男性 [302号室]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【病名】市中肺炎 Day3 / DM, HT

【Summary】
CTRX継続中、発熱改善傾向だが食欲不振続く

【バイタル傾向】
BT: 37.2℃(↓) HR: 78 BP: 128/76 SpO2: 96%(RA)

【Action】
□ 明朝採血結果確認 → 改善あればde-escalation検討
□ 経口摂取不良続けば補液追加検討
□ 血糖チェック 眠前

【If-Then】
• 発熱再燃(38.5℃以上) → 血培再検、抗菌薬変更検討
• SpO2 93%以下 → O2開始、胸部X線考慮

【注意】
⚠️ PCNアレルギー（蕁麻疹既往）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 循環器病棟用

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[🟡] 60代女性 [CCU-2]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【病名】急性心不全（HFrEF）/ ACS後

【Summary】
利尿剤で体液管理中、IN-OUTバランス改善傾向

【血行動態】
BP: 110/68 HR: 72(SR) CVP: 8 体重: 62kg(−1.5kg/入院時)
尿量: 2400mL/24h IN-OUT: −800mL

【Action】
□ 尿量 100mL/4h以下 → Lasix追加投与検討
□ 体重測定 6時 → 目標62kg維持
□ 電解質チェック 明朝 → K<3.5でKCL補充

【If-Then】
• 呼吸困難増悪 → 酸素・Lasix追加、レントゲン
• 胸痛出現 → 12誘導心電図、循環器当直コール
• BP < 90 → 利尿剤中止、カテコラミン準備

【モニタリング】
✅ 心電図モニタリング継続
✅ 酸素飽和度連続測定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 外科病棟用（術後患者）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[🟡] 50代男性 [401号室]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【病名】胃癌術後（幽門側胃切除）POD2

【Summary】
術後経過良好、ドレーン排液は漿液性に変化

【術後チェック項目】
✅ バイタル: 安定（BT 37.5℃まで許容）
✅ ドレーン: 80mL/24h 漿液性
✅ 創部: 発赤なし
✅ 腸蠕動: +（排ガスあり）

【Action】
□ 発熱38℃以上持続 → 血液検査、創部・ドレーン確認
□ 疼痛時 → アセリオ1g静注（6時間以上あけて）
□ 明朝 → 飲水開始可否判断

【If-Then】
• ドレーン排液血性/混濁 → 外科当直に報告
• 腹痛増強+腹膜刺激徴候 → 緊急CT、外科当直コール
• SpO2 94%以下 → 深呼吸指導、改善なければ酸素

【注意】
⚠️ DVT予防 フットポンプ継続
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 救急外来用テンプレート

### 観察患者用

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[🟡] 観察患者: 40代男性
来院時刻: 22:30  予定退院判断: 翌朝6:00
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【主訴・診断】
胸痛 → 非心原性胸痛疑い（ACS除外目的で観察）

【施行済み検査】
✅ 心電図: 有意なST変化なし
✅ トロポニンI: 陰性（来院時）
□ トロポニンI再検: 02:30予定（6時間後）

【Action】
□ 02:30 トロポニン再検 → 陰性なら朝帰宅可
□ 胸痛再発 → 心電図再検、循環器コンサルト
□ バイタル2時間毎チェック

【If-Then】
• トロポニン陽性 → 循環器緊急コール、入院
• 胸痛持続+心電図変化 → 循環器緊急コール
• 問題なく経過 → 6:00に帰宅判断

【連絡先】
主治医Dr.____ 携帯:______
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## AIプロンプト集

### テンプレート自動生成プロンプト

```
以下の患者情報から、I-PASS形式の引き継ぎサマリーを作成してください。

【診療科】{内科/外科/循環器/etc}
【患者情報】
{匿名化した患者情報}

以下の形式で出力してください：
1. 重症度アイコン（🔴🟡🟢）と理由
2. 2文以内のサマリー
3. 具体的なアクションリスト（タイミング付き）
4. If-Then形式の予測と対応（最低2つ）
5. 特記事項・注意点

簡潔かつ実用的に記載してください。
```

## チェックリスト

### 引き継ぎ前の最終確認

- [ ] 全患者の重症度を分類したか？
- [ ] アクションリストは具体的か？
- [ ] If-Then条件は明確か？
- [ ] アレルギー・DNARは確認したか？
- [ ] 緊急連絡先は共有したか？
- [ ] 受け手が内容を復唱したか？

## まとめ

### I-PASS活用のポイント

1. **構造化**することで漏れを防ぐ
2. **AIを補助ツール**として活用
3. **確認のプロセス**を必ず入れる
4. **テンプレート**で効率化

### 継続的な改善

- チーム内でテンプレートを共有・改善
- ヒヤリハット事例からの学び
- 定期的なフィードバック

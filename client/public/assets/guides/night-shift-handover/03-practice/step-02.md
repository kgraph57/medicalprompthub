# ステップ3: AIを活用した効率化

## AIを使う目的

AIを活用することで、引き継ぎサマリー作成の**時間を短縮**しながら、**漏れを防ぐ**ことができます。

## 基本プロンプト

### プロンプト1: カルテ情報からI-PASS形式への変換

```
以下の患者情報をI-PASS形式の引き継ぎサマリーに変換してください。

【患者情報】
{ここにカルテ情報を貼り付け（匿名化済み）}

【出力形式】
- Illness severity: 安定/要注意/不安定から選択し、理由を1文で
- Patient summary: 2文以内で患者の概要
- Action list: 具体的なタスクをタイミングとともに箇条書き
- Situation awareness: If-Then形式で予測と対応
- Synthesis: 確認すべきポイント

簡潔かつ具体的に記載してください。
```

---

### プロンプト2: 引き継ぎ内容の漏れチェック

```
以下の引き継ぎサマリーをレビューし、不足している情報や改善点を指摘してください。

【引き継ぎサマリー】
{作成したサマリーを貼り付け}

以下の観点でチェックしてください：
1. 重症度は適切か？
2. サマリーは簡潔で十分な情報を含んでいるか？
3. アクションリストに漏れはないか？
4. 予測と対応計画は具体的か？
5. 曖昧な表現はないか？
```

## 注意点

### 個人情報の取り扱い

⚠️ **必須**: AIに入力する前に**必ず匿名化**してください

| 元の情報 | 匿名化後 |
| :--- | :--- |
| 鈴木太郎（ID: 12345678） | 70代男性 |
| 住所: 東京都〇〇区 | 削除 |
| 電話番号: 090-xxxx-xxxx | 削除 |
| 特定可能な既往歴 | 一般化 |

---

### AIの限界

- AIは**補助ツール**であり、最終判断は医師が行う
- **臨床判断**はAIに依存しない
- 出力は必ず**確認・修正**する

---

## 効率化のコツ

### 1. テンプレート化

よく使うプロンプトを保存しておく

### 2. バッチ処理

複数患者の情報をまとめて構造化

### 3. 確認の習慣化

AI出力 → 医師確認 → 引き継ぎのフローを確立

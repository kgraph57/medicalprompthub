# Step 1: どの論文を読む？〜全ての始まりは「目的」から〜

### 📋 このステップの目的

効率的な論文読解の第一歩、それは「読む価値のある論文」を選ぶこと。全ての論文が、あなたの貴重な時間を費やす価値があるわけではありません。このステップでは、大海原のような論文の海から、あなたの「お宝」を見つけ出し、冒険の準備を整える方法を学びます。

![論文の選定と準備](https://files.manuscdn.com/user_upload_by_module/session_file/90346460/iZZLhUBjQRdmcUQy.png)

### ⏱️ 所要時間

**5〜10分**

### 🎯 達成基準

- 「なぜ読むのか？」（臨床疑問、研究テーマ、後輩への指導など）が、自分の言葉で言える。
- 選んだ論文が、その「なぜ」に答えてくれる一本だと確信が持てる。
- 論文の「カルテ情報」（タイトル、著者、ジャーナル、発行年、DOI）を控えている。

### 📝 詳細手順

![論文の選定と準備](./paper_selection_prep.png)

#### 手順1: 「なぜ？」を問う

論文を開く前に、少しだけ立ち止まって、自分に問いかけてみましょう。「なぜ、私はこの論文を読もうとしているんだろう？」と。この「なぜ」が、あなたの羅針盤になります。

| 目的 | 重点ポイント | 時間配分 |
|---|---|---|
| **臨床での疑問解決** | Abstract, Results, Discussion | 30-45分 |
| **研究デザインの勉強** | Methods, Results | 45-60分 |
| **最新情報のキャッチアップ** | Abstract, Introduction, Conclusion | 20-30分 |
| **批判的吟味のトレーニング** | 全体（特にMethods, Results） | 60-90分 |
| **抄読会の準備** | 全体 + 批判的吟味 | 60-90分 |

#### 手順2: 論文の「顔」を見る

選んだ論文の基本情報をチェックします。これは、初対面の患者さんの第一印象を掴むようなもの。

- **タイトル**: 内容を的確に表しているか？
- **著者・所属**: 信頼できる専門家か？
- **ジャーナル**: 信頼性は？（Impact Factorだけでなく、査読の質も）
- **発行年**: 最新情報か、古典か？
- **DOI**: 後で迷子にならないための「ID番号」

#### 手順3: 論文の「種類」を見極める

論文の種類によって、注目すべきポイントは全く異なります。

- **RCT**: Methods（ランダム化）とResults（主要アウトカム）
- **観察研究**: Methods（バイアス対策）とResults（交絡因子の調整）
- **レビュー**: Methods（検索戦略）とResults（統合結果）
- **症例報告**: Case PresentationとDiscussion（新規性）

### 💡 AIへの「話し方」

#### プロンプト1: 「この論文、読む価値ある？」

```
# 指示
あなたは僕の指導医です。以下の論文、読む価値ありますかね？

# 評価してほしいこと
1. **ジャーナルの信頼性**: この雑誌、ちゃんとしてますか？
2. **研究の新規性**: 新しいこと言ってますか？
3. **臨床での価値**: 現場で役立ちますか？
4. **研究デザイン**: 研究のやり方、イケてますか？
5. **読むべき？**: ぶっちゃけ、読むべきですか？（高/中/低）

# 論文情報
- **タイトル**: [論文タイトル]
- **ジャーナル**: [ジャーナル名]
- **発行年**: [年]
- **研究デザイン**: [RCT/観察研究など]
- **僕の目的**: [臨床での疑問解決/最新情報のキャッチアップなど]
```

#### プロンプト2: 「どこから読めばいい？」

```
# 指示
この論文、効率的に読みたいです。どこを重点的に、どんな順番で読めばいいか教えてください。

# 論文情報
- **タイトル**: [論文タイトル]
- **研究デザイン**: [RCT/観察研究など]
- **僕の目的**: [臨床での疑問解決/最新情報のキャッチアップなど]
```

### ⚠️ よくある「落とし穴」

- **失敗**: タイトルだけで「これだ！」と飛びついてしまう。
  - **対処法**: Abstract（抄録）は必ず読むこと。ここでミスマッチを防ぐ。

- **失敗**: 古い情報を掴んでしまう。
  - **対処法**: 発行年を確認。ただし、分野の基礎を築いた「古典」は別腹。

- **失敗**: 「ハゲタカジャーナル」に捕まる。
  - **対処法**: AIに「この雑誌、大丈夫？」と聞いてみるか、信頼できるジャーナルリストで確認。

### 💪 上級者への道

- **引用数をチェック**: たくさん引用されている論文は、それだけ重要だということ。（Google Scholarで確認）
- **著者の「過去」を深掘り**: 同じ著者の過去論文を読むと、研究の文脈が立体的に見えてくる。
- **参考文献を「芋づる式」に**: 参考文献リストは、次なるお宝への地図。

# 第0章: イントロダクション

## このガイドについて

このガイドは、「忙しい臨床医が効率的に論文を読んで理解する」ための実践的なワークフローを提供します。従来、1本の論文を深く理解するのに数時間かかっていた作業を、AIツールを活用することで**30分〜1時間**に短縮することを目指します。

各ステップでは、**ChatGPT**、**Claude**、**Gemini**、**NotebookLM**、**Manus**といった最新のAIツールを最大限に活用するための**具体的なプロンプト**と**実践例**を豊富に紹介します。これにより、論文抄読会（Journal Club）の準備時間を大幅に短縮し、より深い理解と批判的吟味に時間を割けるようになります。

## 論文読解の重要性

医学は日々進歩しており、新しいエビデンスが次々と発表されています。臨床医として、最新の知見を自分の診療に反映させるためには、効率的な論文読解スキルが不可欠です。

| 項目 | 説明 |
|---|---|
| **エビデンスに基づいた診療（EBM）** | 最新の研究結果を臨床判断に活用する |
| **継続的な学習** | 専門知識を常にアップデートする |
| **批判的思考** | 論文の質を評価し、適切に解釈する |
| **教育・指導** | 後輩や学生に最新知見を伝える |

## なぜ効率化が必要か？

従来の論文読解プロセスには、以下のような課題があります：

- **時間がかかりすぎる**: 1本の論文を完全に理解するのに2-3時間
- **情報の取捨選択が難しい**: 重要な部分とそうでない部分の区別がつかない
- **批判的吟味が不十分**: 時間が足りず、表面的な理解で終わってしまう
- **記憶に残らない**: 読んだ内容を後で思い出せない

このガイドでは、AIツールを活用することで、これらの課題を解決し、**質の高い読解を短時間で実現**する方法を解説します。

## 完成までの全体像（タイムライン）

本ガイドでは、論文読解のプロセスを5つのステップに分けて解説します。以下は、標準的なタイムラインの目安です。

| ステップ | 内容 | 所要時間（目安） |
|---|---|---|
| **Step 1** | 論文の選定と準備 | 5-10分 |
| **Step 2** | 論文の構造理解と効率的な読み方 | 10-15分 |
| **Step 3** | AIツールを使った要約と分析 | 10-15分 |
| **Step 4** | 批判的吟味（Critical Appraisal） | 15-20分 |
| **Step 5** | まとめと記録 | 5-10分 |
| **合計** | | **45-70分** |

従来の2-3時間から、**約1時間**に短縮できます。

## 📚 推奨参考資料

本ガイドと併せて、以下のリソースを参照することで、論文読解のスキルがさらに向上します。

### 批判的吟味（Critical Appraisal）

- **CASP (Critical Appraisal Skills Programme)**: 様々な研究デザインの評価ツール
  - [CASP公式サイト](https://casp-uk.net/)
- **JAMA Evidence**: エビデンスの評価と活用に関する実践的なガイド
  - [JAMA Evidence](https://jamanetwork.com/collections/44068/evidence-based-medicine)

### 論文の読み方

- **How to Read a Paper**: Trisha Greenhalgh著の古典的なガイド
- **Users' Guides to the Medical Literature**: JAMA Evidenceシリーズ

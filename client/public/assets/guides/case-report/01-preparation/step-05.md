# Step 5: 症例データの整理

### 📋 このステップの目的

本格的な執筆を始める前に、膨大な電子カルテの情報の中から、症例報告に必要なデータを抽出し、構造化することが目的です。
時系列に沿って臨床経過を整理し、検査データや画像所見を一覧化することで、後の「Case Presentation」セクションの執筆が飛躍的にスムーズになります。
この地道な作業が、抜け漏れのない正確な症例記述の土台となります。

![症例データ整理のプロセス](./data_organization_process.png)

### ⏱️ 所要時間

2〜4時間

### 🎯 達成基準

- 臨床経過が時系列で整理されたサマリー（経過表）が作成されている。
- 主要な検査データ（血液検査、画像所見など）が表形式でまとめられている。
- CAREガイドラインのチェックリスト項目が、カルテのどこに記載されているかマッピングできている。

### 📝 詳細手順

#### 手順1: 臨床経過の時系列サマリー作成

電子カルテの情報を時系列に沿って整理します。
手作業でも可能ですが、AIを活用すると効率的です。
ただし、患者の個人情報をAIに直接入力することは絶対に避けてください。
匿名化した上で、以下のように整理します。

1.  カルテ情報の匿名化: 患者名、ID、生年月日などを削除または仮名に置き換えます。
2.  時系列データの抽出: 受診日、入院日、症状出現日、検査日、治療開始日、退院日などの重要な日付と、その日に起こったイベント（症状、所見、治療内容など）をテキストとしてコピーします。
3.  AIによる構造化: 匿名化したテキストをChatGPTやClaudeにインプットし、時系列の表を作成させます。

#### 手順2: 検査・画像データの表形式化

臨床経過で変動した重要な検査データ（炎症反応、腫瘍マーカー、疾患特異的抗体など）や、画像所見の推移を表にまとめます。
これにより、読者は治療による変化を一目で理解できます。

- 検査データ: ExcelやGoogle Sheetsを使い、日付ごとに行を追加し、各検査項目を列にして入力します。
グラフ化も念頭に置いて作成すると良いでしょう。
- 画像データ: 主要な画像（CT、MRI、病理写真など）を選び、撮影日、所見を簡潔に記述したリストを作成します。
後のFigure作成の準備になります。

#### 手順3: CAREガイドラインとのマッピング

症例報告の国際的な標準であるCAREガイドラインのチェックリストを用意し、各項目に必要な情報がカルテのどこにあるかを確認・マッピングします。
これにより、記述の抜け漏れを防ぎます。

### 🤖 推奨AIツール

| ツール | 用途 | 優先度 |
|---|---|---|
| ChatGPT / Claude | 匿名化したテキストデータからの臨床経過サマリー（時系列の表）作成 | ⭐⭐⭐⭐ |
| Excel / Google Sheets | 検査データの整理とグラフ化 | ⭐⭐⭐ |

### 💡 プロンプト集

#### プロンプト1: 匿名化されたカルテ情報から臨床経過のサマリーを作成する

```
# 指示
あなたは優秀な病棟クラークです。以下の匿名化された診療録のテキストから、時間経過に沿った臨床経過サマリーをマークダウン形式の表で作成してください。

# 抽出項目
- 日付 (Day): 発症日をDay 1とする
- イベント: 症状、診察所見、検査、治療など
- メモ: 特記事項

# 匿名化済み診療録テキスト
[ここに匿名化したカルテ情報を貼り付ける]
例：
2024/01/10 (Day 1): 乾性咳嗽が出現。
2024/01/15 (Day 6): 近医受診。胸部X線で両側肺に浸潤影を指摘され、当科紹介。
2024/01/17 (Day 8): 当科初診。SpO2 95%。血液検査でCRP 5.8。気管支鏡検査を予定。
...
```

活用例: カルテのコピー＆ペースト（匿名化後）から、構造化された経過表を瞬時に作成できます。
これにより、複雑な経過を客観的に整理し、執筆の準備を整えることができます。

### 📚 実践例

症例: ステロイド抵抗性OP

1.  データ抽出: 電子カルテから、患者背景、現病歴、入院後経過、検査データ、画像レポートをテキストファイルにコピー。
2.  匿名化: 患者情報を仮名「鈴木一郎、45歳男性」などに置換。
3.  経過サマリー作成: 匿名化したテキストをChatGPTにインプットし、プロンプト1を実行。
入院日から退院後までの経過表が生成された。
4.  データ整理: 生成された経過表をもとに、ExcelでCRP、KL-6、WBCなどの推移をまとめた検査データ表を作成。
主要な胸部CT画像を時系列に並べたリストを作成。
5.  CAREマッピング: CAREチェックリストの「Clinical findings」「Diagnostic assessment」などの項目が、作成したサマリーのどこに対応するかを確認。

### ⚠️ よくある失敗と対処法

- 失敗: 患者の個人情報（氏名、IDなど）をそのままAIツールに入力してしまう。
  - 対処法: 絶対に避けてください。
これは重大なプライバシー侵害であり、法的な問題に発展する可能性があります。
AIを利用する際は、必ず個人情報を削除・仮名に置換するなどの匿名化処理を徹底してください。

- 失敗: データを整理せず、カルテを見ながら直接書き始めてしまう。
  - 対処法: 一見遠回りに見えますが、このステップを丁寧に行うことで、後の執筆効率が格段に上がります。
特に、共著者との情報共有や、査読者からの修正依頼に対応する際に、整理されたデータが非常に役立ちます。

### 💪 上級者向けTIPS

作成した検査データ表は、そのまま論文のTable（表）として利用できます。
また、時系列データをグラフ化することで、読者の理解を助けるFigure（図）を作成できます。
Pythonの`matplotlib`や`seaborn`ライブラリを使えば、論文掲載レベルの高品質なグラフをプログラムで作成することも可能です。

# Step 16: 投稿規定の最終確認

### 📋 このステップの目的

いよいよ投稿を目前にして、最後の関門である投稿規定（Instruction for Authors / Author Guidelines）の最終確認を行います。どんなに素晴らしい内容の論文でも、フォーマットが規定に従っていなければ、査読の土俵にすら上がれず、門前払い（デスクリジェクト）されてしまいます。このステップでは、細部にまで注意を払い、完璧な形式で論文を提出するための最終チェックリストを学びます。

### ⏱️ 所要時間

**1〜2時間**

### 🎯 達成基準

- 論文原稿のすべての項目（テキスト、図表、参考文献）が、投稿規定のフォーマットに完全に準拠している。
- 投稿に必要なすべてのファイル（カバーレター、本文ファイル、図表ファイルなど）が揃っており、正しく命名されている。

### 📝 詳細手順

投稿ジャーナルのウェブサイトにある「Instruction for Authors」を印刷またはPDFで保存し、マーカーを片手に、自分の原稿がすべての項目をクリアしているか、一つ一つ指差し確認していきます。

#### 手順1: 原稿全体のフォーマット確認

- **ファイル形式**: Word (.docx) か、PDFか、LaTexか。
- **フォントと文字サイズ**: Times New Romanの12ポイントが一般的。
- **行間**: ダブルスペース（2行間隔）が指定されていることが多い。
- **ページ番号と行番号**: 連続したページ番号や行番号が必要か。
- **セクションの順序**: タイトルページ → 抄録 → 本文 → 謝辞 → 参考文献 → 図の説明文 → 表 → 図、というように、ジャーナル指定の順序になっているか。

#### 手順2: 各セクションの規定確認

- **タイトルページ**: 
  - 記載すべき情報（全著者の氏名・所属、責任著者の連絡先、ランニングタイトル、文字数、図表の数など）がすべて含まれているか。
- **抄録 (Abstract)**: 
  - 文字数制限（例: 250ワード）を厳守しているか。
  - 構造化抄録の各セクション名（Background, Case, etc.）は正しいか。
- **本文 (Main Text)**: 
  - 文字数制限を超えていないか。
- **参考文献 (References)**: 
  - 引用スタイルのフォーマットは完璧か。
  - 引用文献の数に上限はないか。
- **図表 (Figures and Tables)**: 
  - ファイル形式（TIFF, EPS, JPEGなど）と解像度（300 dpi以上）の規定を満たしているか。
  - 図と表は、それぞれ別のファイルとしてアップロードするのか、本文の末尾に含めるのか。
  - ファイル名（例: Figure1.tiff, Table1.docx）の命名規則は正しいか。

#### 手順3: 投稿に必要なその他書類の準備

- **カバーレター (Cover Letter)**: 論文の「送り状」。後述のStep 17で作成します。
- **利益相反 (COI) 申告書**: ジャーナル指定のフォームがある場合は、ダウンロードして記入します。
- **著作権譲渡同意書 (Copyright Transfer Agreement)**: 同様に、指定フォームに署名します。
- **英文校正証明書**: プロの校正サービスを利用した場合。

### 💡 プロンプト集

#### プロンプト1: 投稿規定からチェックリストを作成する

```
# 指示
あなたは優秀なアシスタントです。以下の医学雑誌の投稿規定（Instruction for Authors）のテキストから、症例報告（Case Report）を投稿する際に確認すべき重要なフォーマット要件を抽出し、チェックリスト形式で日本語で要約してください。

# 投稿規定テキスト
[ここに投稿規定のウェブページからコピーしたテキストを貼り付ける]

# 出力形式
## 投稿規定チェックリスト

### 全体フォーマット
- ファイル形式: [...]
- フォント: [...]

### タイトルページ
- 必要な項目: [...]

### 抄録
- 文字数制限: [...]

(以下、各項目について続く)
```

**活用例**: 長くて複雑な投稿規定を読む手間を省き、自分専用のチェックリストを瞬時に作成できます。これにより、確認漏れを劇的に減らすことができます。

### 📚 実践例

- 投稿予定の「Internal Medicine」誌のInstruction for Authorsをウェブで開き、上記プロンプトを使ってChatGPTにチェックリストを作成させる。
- 作成されたチェックリストを印刷し、完成した原稿と照らし合わせながら、一つずつチェックマークを付けていく。
- 「図は本文とは別に、TIFF形式でアップロードすること」という項目を見つけ、PowerPointで作成した図をTIFF形式で書き出し、"Fig1.tiff", "Fig2.tiff"とリネームする。
- 「タイトルページに全著者のORCID iDを記載すること」という指示に従い、共著者からORCID iDを聞き出し、追記する。

### ⚠️ よくある失敗と対処法

- **失敗**: Step 6で確認した古い投稿規定を信じ込み、改訂された最新の規定を見逃す。
  - **対処法**: 投稿規定は頻繁に改訂されます。投稿直前に、必ずもう一度ジャーナルのウェブサイトにアクセスし、最新の情報を確認する習慣をつけましょう。

- **失敗**: 細かいフォーマットの不備で、投稿システムから何度もエラー通知が来て、投稿作業だけで半日を費やしてしまう。
  - **対処法**: このステップを軽視せず、時間をかけて丁寧に行うことが、結果的にスムーズな投稿に繋がります。「神は細部に宿る」を肝に銘じましょう。

### 💪 上級者向けTIPS

投稿に成功した先輩や同僚に、投稿時のファイル一式（もちろん個人情報をマスクした上で）を見せてもらうのも非常に参考になります。実際のファイル構成や命名規則など、規定を読むだけでは分かりにくい「暗黙知」を学ぶことができます。

# 段階7: 倫理・コンプライアンス確認

## 目的

本段階は、特定のプロセスではなく、研究の全段階（アイデア創出から出版後まで）にわたって継続的に実施される、包括的な監視・検証プロセスです。その目的は、研究と論文執筆のすべての活動が、最高の倫理基準、法規制、および学術的誠実性の原則に準拠していることを保証することです。医療研究において、倫理とコンプライアンスは、科学的妥当性と同等、あるいはそれ以上に重要です。

## 主要な確認領域

この段階は、研究プロセス全体に組み込まれるべき一連のチェックリストとして機能します。

### 7.1 データプライバシーとセキュリティ

患者の信頼は医療研究の基盤です。その信頼を維持するため、個人情報の保護は最優先事項です。

**確認事項:**
-   [ ] **倫理委員会の承認**: 研究計画は、着手前に所属機関の倫理審査委員会（IRB）の承認を得ているか？
-   [ ] **インフォームドコンセント**: すべての参加者から、研究の目的、リスク、利益について十分な説明の上で、自由意思による同意が文書で得られているか？
-   [ ] **データの匿名化**: 患者データは、各国の個人情報保護法（日本の個人情報保護法、米国のHIPAA、欧州のGDPRなど）に従い、適切に匿名化または非識別化されているか？
-   [ ] **安全なデータ保管**: 生データおよび解析データは、不正アクセスから保護された、所属機関が承認する安全な環境に保管されているか？
-   [ ] **AIツールへの入力制限**: **【最重要】** 承認されていない外部のAIツールに、個人を特定できる可能性のある情報を一切入力していないか？

### 7.2 AI使用の透明性と責任

AIの利用は、研究の透明性と再現性を損なうものであってはなりません。

**確認事項:**
-   [ ] **AI使用の記録**: 研究の各段階で使用したすべてのAIツール（名称、バージョン、製造元）とその使用目的が記録されているか？（AI使用記録テンプレート参照）
-   [ ] **プロンプトの保管**: 主要な結果や文章生成に使用したプロンプトは、再現性を確保するために保管されているか？
-   [ ] **人間による検証**: AIが生成したすべてのコンテンツ（テキスト、コード、データ、引用）は、人間によってその正確性と妥当性が検証されたか？
-   [ ] **Methodsセクションへの記述**: 論文のMethodsセクションに、ICMJEの推奨に従い、AIの使用方法が詳細かつ正確に記述されているか？
-   [ ] **著者の最終責任**: AIは共著者ではなく、あくまでツールとして使用され、論文内容に対する全責任を著者が負うことを理解しているか？

### 7.3 学術的誠実性

科学の信頼性は、研究者の誠実な行動にかかっています。

**確認事項:**
-   [ ] **盗作（Plagiarism）の防止**: 自身の論文に、他者のアイデア、文章、データを適切な引用なしに使用していないか？（盗作チェックツールの使用を推奨）
-   [ ] **引用の正確性**: AIが生成した引用を含め、すべての参考文献が実在し、原著論文の内容を正確に反映していることを確認したか？（**AIハルシネーションの徹底検証**）
-   [ ] **著者資格（Authorship）**: 著者リストに含まれる全員が、ICMJEが定める4つの基準（1. 研究への実質的貢献、2. 論文の起草または批判的レビュー、3. 最終版の承認、4. 全内容への説明責任）をすべて満たしているか？
-   [ ] **利益相反（Conflict of Interest）の開示**: 研究の資金源、著者個人の金銭的関係など、研究の客観性に影響を与えうるすべての利益相反が、論文および投稿システムで適切に開示されているか？
-   [ ] **データの完全性**: 研究結果を支持しないデータを選択的に除外したり、データを不適切に加工したりしていないか？（p-hacking, cherry-pickingの回避）

### 7.4 ICMJE Recommendationsへの準拠

多くの主要医学ジャーナルが採用する国際的な標準への準拠を確認します。

**確認事項:**
-   [ ] **報告ガイドラインの使用**: 研究デザインに適した報告ガイドライン（例: CONSORT, STROBE, PRISMA）に準拠して論文が執筆されているか？
-   [ ] **臨床試験の登録**: 臨床試験は、着手前に公開臨床試験登録サイト（例: UMIN-CTR, ClinicalTrials.gov）に登録されているか？
-   [ ] **データ共有**: ジャーナルのポリシーに従い、研究の生データを共有する準備ができているか、あるいは共有できない正当な理由があるか？
-   [ ] **最終承認**: 投稿前に、すべての共著者が論文の最終版を読み、その内容に同意しているか？

## ワークフローの反復性と結論

MARWは、一方通行の直線的なプロセスではありません。研究の過程で新たな発見があれば、前の段階に戻って研究質問を修正したり、追加の文献レビューを行ったりすることが頻繁にあります。この**反復的な性質**こそが、研究を深化させ、質を高める原動力となります。

AIを責任ある形で活用することで、研究者は定型的な作業から解放され、最も重要な「思考」のプロセス、すなわち、批判的吟味、創造的統合、そして倫理的判断により多くの時間を費やすことができます。MARWが、日本の医療専門家の皆様にとって、質の高いエビデンスを創出し、世界の医学の進歩に貢献するための一助となることを心から願っています。

## AI使用記録テンプレート（最終確認）

論文投稿前に、すべての段階のAI使用記録をレビューし、一貫性と網羅性を確認します。

```
【最終確認: 全段階のAI使用記録レビュー】

レビュー日: [日付]
レビュー担当者: [氏名]

確認事項:
- [ ] すべての段階で使用したAIツールが記録されているか？
- [ ] Methodsセクションの記述と使用記録に矛盾はないか？
- [ ] AI生成コンテンツの検証プロセスは適切に記録されているか？
- [ ] 保管されているプロンプトは、主要な結果の再現に十分か？

総括コメント: [例: 全段階を通じてAIの使用は適切に記録・検証されており、論文の透明性と再現性は確保されていると判断する。]
```

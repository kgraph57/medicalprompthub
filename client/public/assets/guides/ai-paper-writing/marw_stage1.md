# 段階1: 研究課題の設定と文献探索

## 目的

本段階の目的は、研究の方向性を定め、既存の知識におけるギャップを特定し、臨床的に重要かつ実現可能な研究課題を明確に定義することです。質の高い研究は、質の高い問いから始まります。AIを戦略的に活用することで、この探索プロセスを大幅に加速し、深化させることが可能です。

## 主要タスク

### 1.1 研究トピックの探索

臨床現場での疑問や関心を起点に、AIを用いて広範な知識の中から最新のトレンド、未解決の問題、そして研究の機会（リサーチギャップ）を探ります。

-   **人間の役割**: 臨床経験や専門知識に基づき、初期の関心領域を選定する。
-   **AIの役割**: 選定されたトピックに関する膨大な文献を迅速に分析し、最新の動向、議論の的となっている論点、そして研究が不足している領域を客観的に提示する。

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **Perplexity AI** | 会話形式の検索エンジン | 広範なトピックの概要把握、初期の質問生成に最適。 |
| **Consensus** | 科学的根拠に基づく回答 | 特定の研究質問に対して、文献から直接的な「Yes/No」形式の回答や要約を迅速に得る。 |
| **Undermind** | ディープサーチ機能 | 表面的な検索では見つからない、より深い関連性を持つ文献や概念を発見する。 |

**プロンプト例1: トピック探索**

```
私は[専門分野]の医師です。[臨床状況や特定の問題]に強い関心を持っています。
この分野における過去5年間の主要な研究動向、現在活発に議論されている論点、そして臨床的に重要でありながらエビデンスが不足している研究ギャップを包括的に分析してください。
特に以下の点に焦点を当ててください:
1. 主要なランドマーク研究とその影響
2. 新たな治療法や診断技術の出現
3. 未解決の臨床的疑問
```

### 1.2 初期文献検索と研究質問の洗練

探索したトピックから、具体的で検証可能な研究質問を作成します。ここでは、臨床研究で広く用いられる**PICO**フレームワークを活用します。

-   **人間の役割**: AIが生成した研究質問の臨床的意義、新規性、実現可能性を評価し、最終的な研究質問を決定する。
-   **AIの役割**: PICOフレームワークに基づき、具体的で検証可能な研究質問の候補を複数生成する。また、それらの質問に対するPubMedなどのデータベース検索戦略を立案する。

**PICOフレームワーク**

-   **P (Patient/Population)**: 対象となる患者または集団
-   **I (Intervention)**: 評価する介入（治療、検査など）
-   **C (Comparison)**: 比較対象（標準治療、プラセボなど）
-   **O (Outcome)**: 評価する結果（有効性、安全性など）

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **Elicit** | 研究質問に基づく論文検索 | PICO形式の質問を入力し、関連する論文を構造化された形式で抽出する。 |
| **Semantic Scholar** | AI駆動型学術検索 | 論文の意図を理解し、より関連性の高い文献を提示する。 |

**プロンプト例2: 研究質問の洗練（PICO形式）**

```
以下の研究トピックについて、PICO形式で検証可能な研究質問を3つ提案してください:

トピック: [関心のあるトピック]

PICOの各要素を明確に定義し、それぞれの研究質問について、以下の点を評価してください:
1. 臨床的重要性
2. 新規性（既存研究との差別化）
3. 研究の実現可能性（倫理、コスト、時間）
```

**プロンプト例3: 検索戦略の作成**

```
以下の研究質問について、PubMedで再現可能な検索戦略を作成してください:

研究質問: [PICO形式で定義された研究質問]

検索戦略には以下を含めてください:
1. 適切なMeSH（Medical Subject Headings）用語とキーワードの組み合わせ
2. 検索の感度と特異度のバランスを考慮した論理演算子（AND, OR, NOT）の使用
3. 適切なフィルター（例: 研究デザイン、出版年、言語、対象年齢）

作成した検索式が、関連文献を網羅しつつ、ノイズを最小限に抑えるように設計されている理由を説明してください。
```

## 成果物

本段階の完了時には、以下の成果物が得られます。

-   **明確な研究質問**: PICO形式で定義され、臨床的重要性、新規性、実現可能性が確認されたもの。
-   **初期文献リスト**: 研究質問に直接関連する10〜20本の重要論文リスト。
-   **研究ギャップの記述**: 本研究が既存の知識体系においてどのような貢献を果たすかを明確にした記述。

## 検証チェックポイント

次に進む前に、以下の項目を確認してください。

-   [ ] 研究質問はPICO形式で明確に定義されているか？
-   [ ] 研究は臨床的に重要な問題に取り組んでいるか？
-   [ ] 既存の研究との差別化は明確か？
-   [ ] 倫理的に許容され、実現可能な研究デザインが想定できるか？
-   [ ] AIの出力（特に引用）は人間によって検証されたか？

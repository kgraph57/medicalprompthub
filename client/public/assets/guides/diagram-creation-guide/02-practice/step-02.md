# 実践編 - ステップ2: 治療アルゴリズムの図解化

## はじめに

このステップでは、治療アルゴリズムを図解化する方法を学びます。
治療アルゴリズムは、治療方針の決定プロセスを視覚的に表示する図解です。

---

## 治療アルゴリズム図の基本構造

治療アルゴリズム図には以下の要素が含まれます：

- **開始点**: 患者の状態や診断
- **判断点**: 治療方針の分岐点
- **治療**: 具体的な治療方法
- **評価点**: 治療効果の評価

---

## 効果的なプロンプト例

### 例1: 高血圧の治療アルゴリズム

```
以下の高血圧の治療アルゴリズムを図解化してください：

【開始】
高血圧の診断

【判断1】
生活習慣の改善で十分？
- Yes → 生活習慣改善（食事、運動）
- No → 次の判断へ

【判断2】
第一選択薬の選択
- ACE阻害薬/ARB
- カルシウム拮抗薬
- 利尿薬

【判断3】
治療効果の評価
- 目標血圧達成？
- Yes → 継続
- No → 次の判断へ

【判断4】
併用療法の検討
- 2剤併用
- 3剤併用

【視覚的な要素】
- フローチャート形式で表示
- 各判断点を明確に区別
- 治療方法を四角形で表示
- 矢印でアルゴリズムの流れを示す
```

### 例2: 糖尿病の治療アルゴリズム

```
以下の2型糖尿病の治療アルゴリズムを図解化してください：

【開始】
2型糖尿病の診断

【判断1】
HbA1cの値は？
- <7.0% → 生活習慣改善
- ≥7.0% → 次の判断へ

【判断2】
第一選択薬の選択
- メトホルミン
- SGLT2阻害薬
- GLP-1受容体作動薬

【判断3】
治療効果の評価
- 目標HbA1c達成？
- Yes → 継続
- No → 次の判断へ

【判断4】
併用療法の検討
- 2剤併用
- 3剤併用
- インスリン療法

【視覚的な要素】
- アルゴリズム形式で表示
- 各判断点を明確に区別
- 治療方法を色分けで表示
- 矢印でアルゴリズムの流れを示す
```

---

## プロンプトのポイント

### 1. 判断基準を明確にする

各判断点での判断基準を明確にします。

### 2. 治療方法を具体的にする

具体的な治療方法を記載します。

### 3. 評価点を明確にする

治療効果の評価方法を明確にします。

### 4. 視覚的な構造を指定

アルゴリズムの形式を指定します。

---

## 実践例

実際にNanobananaで治療アルゴリズム図を作成してみましょう：

1. 上記のプロンプト例をコピー
2. Nanobananaに貼り付け
3. 「生成」ボタンをクリック
4. 生成された図解を確認
5. 必要に応じてプロンプトを修正して再生成

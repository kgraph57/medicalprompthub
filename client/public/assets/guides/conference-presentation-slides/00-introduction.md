# 医療AIワークフローガイド：学会発表スライド作成支援

**著者:** Manus AI
**最終更新日:** 2025年12月13日

## イントロダクション

学会発表スライドの作成は、研究成果を効果的に伝えるための重要なプロセスですが、構成の検討、内容の要約、デザインの調整など、多大な時間を要します。本ガイドは、AIを活用してこのプロセスを効率化し、発表の質を向上させるための具体的なワークフローを提示します。AIは、発表の骨子作成から、聴衆に合わせた表現の調整、質疑応答の準備に至るまで、多岐にわたるサポートを提供します。

**AI活用のメリット**

| メリット | 詳細 |
| :--- | :--- |
| **時間短縮** | 構成案やスクリプトの初稿を瞬時に生成し、作成時間を大幅に短縮します。 |
| **論理構成の強化** | 研究内容を客観的に分析し、聴衆に伝わりやすい論理的なストーリーラインを提案します。 |
| **伝達効果の向上** | 専門用語の平易化や、効果的な図表のアイデアを提供し、メッセージの伝達力を高めます。 |
| **準備の充実** | 予想される質疑応答を事前にシミュレーションし、発表への自信を高めます。 |

## 基本編：発表の骨子と設計図の作成（2ステップ）

この段階では、発表する研究内容をAIにインプットし、発表時間と聴衆に合わせたスライドの**骨子（構成案）**と、各スライドの**設計図（キーメッセージと図表案）**を作成します。

### ステップ 1: 発表原稿の要約と構成案の作成

**目的:** 論文や研究データを基に、発表時間（例：5分、7分、10分）に合わせた要約と、論理的なスライド構成案を迅速に生成します。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. 発表したい研究の論文要旨、結果、結論などのテキストデータを準備します。2. 発表時間と聴衆のレベルを指定し、AIに構成案の作成を依頼します。 |
| **プロンプト例** | `以下の論文要旨と研究結果に基づき、7分間の学会発表用のスライド構成案（タイトル、COI、目的、方法、結果、考察、結論）を、各セクションの要点を含めて作成してください。聴衆は専門分野の医師とします。` |
| **AIの出力例** | **スライド構成案（7分発表用）**<br>1. タイトル・COI (30秒): 研究テーマ、発表者名、COI開示<br>2. 目的 (45秒): 研究の背景と、本研究で解決したい課題、明確な目的<br>3. 方法 (1分30秒): 対象患者、介入方法、評価項目、統計解析（簡潔に）<br>4. 結果 (2分30秒): 主要評価項目の結果、重要な副次評価項目の結果（図表の提示を推奨）<br>5. 考察 (1分): 結果の解釈、先行研究との比較、本研究の限界<br>6. 結論 (45秒): 研究から得られた最も重要な結論と今後の展望 |
| **まとめ** | AIは、発表の制限時間と聴衆の専門性を考慮し、必要な要素を網羅した論理的なストーリーラインを構築します。これにより、発表の骨子を迅速に確立できます。 |

### ステップ 2: 各スライドのキーメッセージと図表案の生成

**目的:** ステップ1で作成した構成案に基づき、各スライドで伝えるべき具体的な**キーメッセージ**と、そのメッセージを効果的に視覚化するための**図表のアイデア**を生成します。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. ステップ1の構成案と、対応する詳細な研究データをAIに提供します。2. 特定のセクション（例：結果）を指定し、各スライドのキーメッセージと推奨される図表の形式を提案させます。 |
| **プロンプト例** | `構成案の「結果」セクション（スライド3枚分）について、各スライドのキーメッセージと、そのメッセージを視覚的に伝えるための図表（グラフの種類、データポイントの強調点）のアイデアを提案してください。データは以下の通りです：[具体的なデータや統計値]。` |
| **AIの出力例** | **結果セクション スライド設計図**<br>・**スライド 1 (主要評価項目):**<br>　- **キーメッセージ:** 「新規治療群は標準治療群と比較して、主要評価項目であるイベント発生率を25%有意に低下させた（p < 0.01）。」<br>　- **推奨図表:** カプランマイヤー曲線（イベント発生までの時間経過を視覚化し、ハザード比とP値を明記）。<br>・**スライド 2 (副次評価項目):**<br>　- **キーメッセージ:** 「特に高齢者サブグループにおいて、治療効果が顕著であった。」<br>　- **推奨図表:** 森林プロット（サブグループ解析の結果を一覧表示し、高齢者群のオッズ比を強調）。 |
| **まとめ** | AIが提供する具体的なメッセージと図表の設計図は、スライド作成の具体的な方向性を定め、視覚的に説得力のある発表資料の作成を可能にします。 |

## 実践編：内容の洗練と発表準備の強化（3ステップ）

この段階では、作成したスライドの原稿を、聴衆への伝達効果を最大化するように洗練させ、発表本番に向けた準備をAIの助けを借りて強化します。

### ステップ 3: 専門用語の平易化と聴衆に合わせた表現の調整

**目的:** 専門性の高い内容を、ターゲットとする聴衆（例：専門外の医師、コメディカル、一般聴衆）に合わせて分かりやすく調整し、メッセージの理解度を高めます。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. 作成したスライドの原稿（特に「方法」や「結果」の記述）をAIに提供します。2. ターゲットとする聴衆を指定し、専門用語の平易化や表現の調整を依頼します。 |
| **プロンプト例** | `以下のスライド原稿を、循環器内科以外の一般内科医にも理解できるように、専門用語「心筋バイオマーカーの動態」を平易な表現に修正し、より直感的な言葉遣いに調整してください。` |
| **AIの出力例** | **修正された原稿**<br>（修正前）「我々は、高感度トロポニンTおよびNT-proBNPといった心筋バイオマーカーの動態を評価した。」<br>（修正後）「私たちは、心臓のダメージを示す血液中の物質（高感度トロポニンTやNT-proBNPなど）が、治療によってどのように変化したかを調べました。」 |
| **まとめ** | AIは、聴衆の知識レベルを推測し、専門的な内容を正確性を保ちつつ、より多くの人に伝わる表現に変換する能力に優れています。 |

### ステップ 4: 質疑応答（Q&A）の想定と回答案の作成

**目的:** 発表内容に基づき、予想される質問とそれに対する適切な回答案を事前に準備することで、発表本番での対応力を高めます。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. スライド構成案全体、または特に議論を呼びそうなセクション（例：研究の限界、結果の解釈）をAIに提供します。2. AIに、発表内容の弱点や疑問点に焦点を当てた質問を想定させ、それに対する回答案を作成させます。 |
| **プロンプト例** | `発表スライドの構成案全体を考慮し、最も鋭い質問を3つ想定してください。特に「本研究の限界」に関する質問を含めてください。また、それぞれの質問に対する簡潔で論理的な回答案を作成してください。` |
| **AIの出力例** | **想定Q&A**<br>・**質問 1 (限界):** 「本研究は単施設での検討ですが、多施設共同研究と比較して結果の一般化可能性についてどうお考えですか？」<br>　- **回答案:** 「ご指摘の通り、単施設研究である点は限界です。しかし、本施設は地域の中核病院であり、患者背景は比較的均一です。今後は、多施設での検証を計画しています。」<br>・**質問 2 (方法):** 「なぜこの評価項目を選択されたのですか？他の指標では不十分でしたか？」<br>　- **回答案:** 「この評価項目は、先行研究において最も感度・特異度が高いと報告されており、本研究の目的に最も適していると判断しました。」 |
| **まとめ** | AIによるシミュレーションは、発表者が盲点となりがちな部分を浮き彫りにし、発表への自信と準備を劇的に強化します。 |

### ステップ 5: 発表用スクリプトの作成と時間調整

**目的:** スライドの内容に基づき、発表時間内に収まるように詳細な口頭発表用スクリプトを作成し、発表練習の効率を最大化します。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. 各スライドの最終的なキーメッセージと図表をAIに提供します。2. 発表時間（例：7分）と、各スライドに割り当てる時間（例：タイトル30秒、結果2分30秒）を指定し、スクリプトの作成を依頼します。 |
| **プロンプト例** | `スライド構成案とキーメッセージに基づき、7分間の発表に収まるように詳細な発表スクリプトを作成してください。特に、スライドの切り替えタイミングと、強調すべきキーワードを明記してください。` |
| **AIの出力例** | **スライド 4: 主要評価項目の結果**<br>（発表時間：1分30秒）<br>「次に、主要評価項目の結果をご説明します。（**スライド切り替え**）ご覧ください。これは、新規治療群と標準治療群におけるイベント発生率を比較したカプランマイヤー曲線です。新規治療群（青線）は、標準治療群（赤線）と比較して、イベント発生までの期間が有意に延長していることが分かります。（**強調：有意な延長**）具体的には、イベント発生率は25%低下し、ハザード比は0.75（95%CI: 0.65-0.85）でした。この結果は、統計学的に**極めて重要**です。（**次のスライドへ**）」 |
| **まとめ** | AIが作成したスクリプトは、発表練習の際の「台本」として機能し、時間配分の感覚を養うのに役立ちます。 |

## 応用編：デザインと国際化のサポート（2ステップ）

この段階では、AIをデザイン補助や国際的な発表に向けた言語サポートに活用し、スライドの最終的な完成度を高めます。

### ステップ 6: スライドデザインの原則と視覚的改善の提案

**目的:** 視覚的に魅力的で、メッセージが際立つスライドデザインの原則をAIに提案させ、デザインの改善に役立てます。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. 作成したスライドのレイアウトや使用している図表の概要をAIに伝えます。2. 医療系学会発表に特化した「伝わるデザインの原則」や、改善点を提案させます。 |
| **プロンプト例** | `以下のスライド原稿と図表の概要に基づき、医療系学会発表で聴衆の注意を引くためのデザイン原則を3つ提案してください。特に、文字情報が多いスライドの改善策に焦点を当ててください。` |
| **AIの出力例** | **デザイン改善の原則**<br>1. **「One Slide, One Message」の徹底:** 1枚のスライドには最も重要なメッセージを1つだけ記載し、それをタイトルまたはキーメッセージとして大きく表示します。文字情報は極力減らし、口頭で補足します。<br>2. **コントラストと色覚バリアフリー:** 背景色と文字色のコントラストを明確にし、特にグラフでは色覚異常の方にも配慮した配色（例：青とオレンジの組み合わせ）を推奨します。<br>3. **視線の誘導:** 図表の最も重要なデータポイントや、結論に直結する部分に、矢印やハイライト、異なる色を用いて聴衆の視線を誘導します。 |
| **まとめ** | AIは、デザインの専門知識に基づき、学術的な内容を損なうことなく、視覚的な説得力を高めるための具体的な指針を提供します。 |

### ステップ 7: 英語発表用スライドの作成と自然な表現への調整

**目的:** 国際学会での発表に備え、日本語で作成したスライド原稿を、学術的かつ自然な英語表現に翻訳・調整します。

| 項目 | 詳細 |
| :--- | :--- |
| **具体的なステップ** | 1. 日本語で完成したスライドのタイトル、キーメッセージ、スクリプトをAIに提供します。2. 英語での発表であることを指定し、学術的なトーンを保ちつつ、ネイティブが聞いても自然な表現に翻訳・調整を依頼します。 |
| **プロンプト例** | `以下の日本語のスライド原稿を、国際学会での発表に適した学術的かつ自然な英語に翻訳してください。特に、結論部分の表現を力強く、自信に満ちたトーンに調整してください。` |
| **AIの出力例** | **英語スライド原稿（結論）**<br>（日本語原稿）「本研究の結果から、新規治療は既存治療よりも優れていることが示唆されました。今後、大規模な臨床試験が必要です。」<br>（英語翻訳）"In conclusion, our findings strongly suggest the superiority of the novel treatment over the conventional approach. **Further large-scale, multi-center clinical trials are warranted to confirm these results.**" |
| **まとめ** | AIは単なる直訳ではなく、学術的な文脈や発表のトーンを理解した上で、最も適切な英語表現を選択します。これにより、国際的な舞台での発表の質が向上します。 |

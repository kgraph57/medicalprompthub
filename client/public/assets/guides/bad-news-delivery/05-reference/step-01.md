## シナリオ1: がん告知

### 設定

```
患者: 55歳男性、会社員
状況: 健診で肺に異常陰影、精査でStage IIIA肺がんと診断
家族: 妻が同席
```

### 対応例

```
【Setting】
医師: 「本日は検査結果をお伝えするため、
       時間を取っていただきありがとうございます。
       奥様にも同席いただいてよろしいですか？」

【Perception】
医師: 「これまでの検査について、
       どのように説明を受けていらっしゃいますか？」

患者: 「CT検査をして、影があると言われました。
       でも、タバコはやめていたので、
       きっと違うと思っています」

【Invitation】
医師: 「そうでしたか。今日は詳しい結果をお伝えしたいのですが、
       よろしいでしょうか」

【Knowledge】
医師: 「残念ながら、お伝えしなければならないことがあります」
       （Warning shot）

      「検査の結果、肺にあった影は、
       がんであることが分かりました」

患者: 「え...がん...」

【Emotions】
医師: 「突然のことで、とてもショックですよね」
       （Naming）

      「このような状況で動揺されるのは当然のことです」
       （Understanding）

妻:  （涙）

医師: 「奥様も辛いですよね。
       私たちは最後までサポートします」
       （Supporting）

【Strategy】
医師: 「今後についてですが、いくつか治療法があります。
       詳しくは次回、改めてお話ししたいと思います。
       今日はショックも大きいと思いますので、
       まずはお二人でゆっくりお話しください」
```

## シナリオ2: 予後の説明

### 設定

```
患者: 72歳女性、膵臓がん Stage IV
状況: 化学療法効果なく、BSC（Best Supportive Care）への移行を相談
家族: 長女が同席
```

### 対応例

```
【Setting・Perception】
医師: 「今日は、今後の治療方針について
       お話ししたいと思います。
       今の治療について、どのようにお感じですか？」

患者: 「正直、副作用が辛くて...
       でも、頑張らないといけないと思っています」

【Invitation・Knowledge】
医師: 「頑張っていらっしゃいますね」（Respecting）

      「今日は、治療の効果と今後についてお話しします。
       残念ながら、この数ヶ月の治療で、
       腫瘍は小さくなっていません」

患者: 「...効いていないんですか」

医師: 「はい。このまま同じ治療を続けても、
       お体への負担が大きくなるばかりで、
       効果は期待しにくい状況です」

【Emotions・感情対応】
長女: 「他に方法はないんですか？」

医師: 「お気持ちは分かります。
       残念ながら、今の状態で効果が期待できる
       抗がん剤治療は難しい状況です」

      「ただ、治療の選択肢がないわけではありません」

【Strategy・今後の方針】
医師: 「今後は、痛みや苦しさを和らげることに
       集中することをお勧めします。
       それによって、残された時間を
       少しでも穏やかに過ごしていただけます」

患者: 「あとどのくらい...」

医師: 「正確なことは申し上げられませんが、
       月単位で考えていく必要があります。
       ただ、個人差がありますし、
       大切なのは、今をどう過ごすかです」
```

## シナリオ3: 急変時の家族対応

### 設定

```
患者: 80歳男性、肺炎で入院中に急変
状況: 心肺停止、蘇生処置を行ったが死亡確認
家族: 息子が病院に駆けつけた直後
```

### 対応例

```
【Setting】
医師: 「〇〇さんのご家族でいらっしゃいますね。
       こちらの部屋でお話しさせてください」
       （個室に案内）

看護師: 「お茶をお持ちしました。
        座ってお待ちください」

【Knowledge・Warning shot】
医師: 「急なご連絡となり、申し訳ありません。
       大変残念なお知らせをしなければなりません」

息子: 「...」

医師: 「お父様は、先ほど心臓が止まり、
       懸命に蘇生処置を行いましたが、
       残念ながら、お亡くなりになりました」

【Emotions】
息子: 「そんな...今朝は元気だったのに...」

医師: 「突然のことで、信じられないお気持ちですよね」
      （Naming）

      「驚きと悲しみでいっぱいだと思います」
      （Understanding）

       （沈黙、待つ）

息子: 「なぜ...どうして...」

医師: 「経過についてご説明してもよろしいですか？」
      （間を置いてから、簡潔に説明）

【Supporting】
医師: 「今は、お父様のそばにいてあげてください。
       お顔を見て、お別れをしていただけます。
       必要なことは、看護師がご説明します」
```

## AIプロンプト: シナリオ練習

### ロールプレイ用プロンプト

```
Bad News Deliveryのロールプレイをしたいです。
あなたは患者（または家族）役を演じてください。

【状況設定】
疾患: {例: 乳がん Stage II}
患者: {例: 45歳女性、主婦、子供が2人}
同席者: {例: 夫が同席}
患者の性格: {例: 楽観的、詳しく知りたがる}

私は医師として、SPIKESプロトコルに従って
悪い知らせを伝えます。

患者役として、以下の点を意識して反応してください:
1. 感情的な反応（涙、怒り、沈黙など）
2. 質問（治療法、予後、仕事のことなど）
3. 家族としての反応（同席者がいる場合）

会話を始めます。
```

## 医師のセルフケア

### 燃え尽き予防

Bad News Deliveryは医師にとっても辛い経験です。

```
✓ 自分の感情を認識する
✓ 同僚と経験を共有する
✓ 休息を取る
✓ 必要に応じて専門家に相談
✓ 「完璧」を求めすぎない
```

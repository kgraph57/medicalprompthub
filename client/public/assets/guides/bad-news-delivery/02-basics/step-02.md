# ステップ2: 事前準備（Setting）

## なぜ準備が重要か

Bad News Deliveryの成否は、**事前準備で8割決まる**と言われています。

---

## 準備のチェックリスト

### 1. 情報の準備

```
□ 診断名・検査結果を正確に把握
□ 病期（ステージ）の確認
□ 治療選択肢の整理
□ 予後に関するデータ
□ 患者の背景情報（家族、仕事、価値観）
□ これまでの治療経過
```

### 2. 場所の準備

```
□ プライバシーが確保された個室
□ 座って話せる椅子の配置
□ ティッシュを手の届く場所に
□ 時計は患者から見えない位置に
□ 電話・PHSをサイレントモードに
□ 「面談中」の表示を出す
```

### 3. 時間の準備

```
□ 十分な時間を確保（最低30分）
□ 中断されない時間帯を選ぶ
□ 患者・家族の都合を確認
□ 緊急呼び出しへの対応を同僚に依頼
```

### 4. 人の準備

```
□ 患者の希望する同席者を確認
□ 看護師の同席を調整
□ 必要に応じてMSW・心理士に連絡
□ 通訳が必要な場合は手配
```

## 心の準備

### 自分自身の準備

Bad News Deliveryは医師にとっても辛い経験です。

#### 自己チェック

```
□ 自分の感情を認識しているか
□ この患者に伝えることへの覚悟はあるか
□ 感情的になりすぎていないか
□ 時間的・精神的な余裕はあるか
```

### 困難な場合

- 経験豊富な上級医への相談
- 同僚医師との情報共有
- 面談の同席を依頼

## 予測される質問への準備

### よく聞かれる質問

| 質問 | 準備すべき回答の要素 |
| :--- | :--- |
| 「治るんですか？」 | 正直かつ希望を残す表現 |
| 「余命は？」 | 数字の出し方、不確実性 |
| 「なぜもっと早く分からなかった？」 | 事実に基づく説明 |
| 「セカンドオピニオンは？」 | 肯定的な対応 |
| 「仕事は続けられますか？」 | 個別の状況に応じて |

---

### 回答の準備例

#### 「治りますか？」への対応

```
🚫 避ける回答:
「治りません」（断定的すぎる）
「分かりません」（無責任に聞こえる）

✅ 推奨する回答:
「完全に治すことは難しい状況ですが、
 治療によって進行を抑えることは可能です。
 できることを一緒に考えていきましょう」
```

#### 「余命は？」への対応

```
🚫 避ける回答:
「3ヶ月です」（根拠なく断定）
「分かりません」（患者の不安を助長）

✅ 推奨する回答:
「正確な予測は難しいのですが、
 月単位で考えていく必要があります。
 ただ、個人差が大きく、
 治療の効果によっても変わります」
```

## 事前シミュレーション

### イメージトレーニング

```
1. 面談室に入る場面を想像
2. 最初の言葉を考える
3. 核心を伝える部分を練習
4. 患者の反応をいくつか想定
5. それぞれへの対応を準備
```

### AIを活用したロールプレイ

```
【プロンプト例】
以下の状況でBad News Deliveryのロールプレイをしてください。
あなたは患者役を演じてください。

状況:
- 60代男性、会社経営者
- 大腸がん、肝転移あり（Stage IV）
- 手術は困難、化学療法が選択肢

私が医師役で説明を始めます。
患者としてリアルに反応してください。
質問や感情的な反応も含めてください。
```

## 当日の流れ

### 面談開始前（15分前）

```
□ カルテ・検査結果を再確認
□ 面談室の環境を確認
□ 同席者（看護師など）と打ち合わせ
□ 電話をサイレントに
□ 深呼吸をして気持ちを整える
```

### 面談開始時

```
□ 患者・家族を迎え入れる
□ 座る位置を調整（目線の高さを合わせる）
□ 全員の名前と関係性を確認
□ 「お時間をとっていただきありがとうございます」
```

# ステップ1: SPIKESプロトコルの理解

## SPIKESの全体像

SPIKESは、Buckman博士らによって開発された、Bad News Deliveryのための6ステップのプロトコルです。

---

## 各ステップの詳細

### Step 1: Setting（環境設定）

**適切な環境を整える**ことから始めます。

#### チェックリスト

- [ ] プライバシーが確保された場所
- [ ] 座って話せる環境
- [ ] 十分な時間の確保
- [ ] 電話・ポケベルをサイレントに
- [ ] 必要なスタッフの同席（看護師など）
- [ ] 患者の希望する同席者の確認

#### 注意点

```
✗ 避けるべき場所:
- 廊下やナースステーション
- ベッドサイドでカーテン越し
- 時間に追われている状況

✓ 望ましい場所:
- 面談室・説明室
- 静かで落ち着いた空間
- 座って目線を合わせられる場所
```

---

### Step 2: Perception（認識の確認）

**患者が現状をどう理解しているか**を確認します。

#### 使える質問例

```
「これまでの検査や治療について、どのように理解されていますか？」

「主治医からどのような説明を受けていらっしゃいますか？」

「ご自身のお体の状態について、どのようにお考えですか？」
```

#### 確認のポイント

| 確認事項 | 理由 |
| :--- | :--- |
| 現状の理解度 | 情報のギャップを把握 |
| 期待・希望 | 現実との乖離を確認 |
| 誤解の有無 | 訂正が必要か判断 |

---

### Step 3: Invitation（招待）

**情報を聞く準備ができているか**を確認します。

#### 使える質問例

```
「検査結果について、詳しくお話ししてもよろしいですか？」

「今日は検査結果をお伝えしたいのですが、どの程度詳しく
 お知りになりたいですか？」

「結果をお聞きになるのは、どなたと一緒がよろしいですか？」
```

#### 情報量の確認

患者によって知りたい情報量は異なります。

| タイプ | 対応 |
| :--- | :--- |
| 詳細を知りたい | 検査結果・治療選択肢を詳しく |
| 概要だけでよい | 大まかな方針を中心に |
| 知りたくない | 信頼できる家族への説明を提案 |

---

### Step 4: Knowledge（知識・情報）

**悪い知らせを伝える**ステップです。

#### 伝え方の原則

<!-- 図解挿入ポイント -->
<!--
図解プロンプト:
「悪い知らせの伝え方の原則を示す図。
4つのポイントを四角形で囲み、中央の『伝え方』から放射状に配置:
1. 『Warning shot』- 警告の予告
2. 『Simple language』- 平易な言葉
3. 『Small chunks』- 小分けに
4. 『Check understanding』- 理解確認
各ポイントに具体的なフレーズ例を付記。」
-->

1. **Warning shot（予告）**
   - いきなり核心を伝えない
   - 「残念なお知らせがあります」と前置き

2. **Simple language（平易な言葉）**
   - 専門用語を避ける
   - 「悪性」→「がん」と明確に

3. **Small chunks（小分けに）**
   - 一度に多くの情報を伝えない
   - 患者の反応を見ながら進める

4. **Check understanding（理解確認）**
   - 「ここまでで、ご質問はありますか？」
   - 必要に応じて繰り返す

---

#### 言葉の選び方

| 避ける表現 | 推奨する表現 |
| :--- | :--- |
| 「悪性新生物」 | 「がん」 |
| 「予後不良」 | 「治療が難しい状態」 |
| 「何もできません」 | 「できることを一緒に考えましょう」 |
| 「余命3ヶ月」 | 「月単位で考える必要があります」 |

---

### Step 5: Emotions（感情への対応）

**患者の感情に共感的に対応**します。

#### よく見られる感情反応

- 否認：「そんなはずはない」
- 怒り：「なぜ私が」
- 悲嘆：涙、沈黙
- 不安：「これからどうなるの」

#### NURSE（共感の技術）

| 要素 | 例文 |
| :--- | :--- |
| **N**aming | 「とても辛いお気持ちですよね」 |
| **U**nderstanding | 「そのようなお気持ちになるのは当然です」 |
| **R**especting | 「今まで頑張ってこられましたね」 |
| **S**upporting | 「私たちは最後までサポートします」 |
| **E**xploring | 「今、一番心配なことは何ですか？」 |

---

### Step 6: Strategy/Summary（方針と要約）

**今後の方針を共有**し、会話をまとめます。

#### 含めるべき内容

```
1. 今日話した内容の要約
2. 今後の治療選択肢
3. 次のステップ（いつ、何を）
4. サポート体制
5. 次回の面談予定
```

#### 例

```
「今日は検査結果についてお話ししました。
 今後の治療について、いくつかの選択肢があります。
 次回、ご家族も一緒に詳しくお話ししましょう。
 いつでもご連絡ください。」
```

---

## まとめ: SPIKESの流れ

```
S: 環境を整える（場所・時間・人）
    ↓
P: 患者の認識を確認
    ↓
I: 情報提供の許可を得る
    ↓
K: 悪い知らせを伝える（warning shot → 平易な言葉 → 小分けに）
    ↓
E: 感情に共感的に対応（NURSE）
    ↓
S: 今後の方針を共有、要約
```

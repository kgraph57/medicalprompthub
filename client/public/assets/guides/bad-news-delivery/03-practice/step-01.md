# ステップ3: 患者の認識確認（Perception）

## なぜ認識確認が重要か

患者の現状理解を確認することで、**適切な情報提供の出発点**が分かります。

## 認識を確認する質問

### 開かれた質問（Open-ended Questions）

```
「これまでの検査について、どのような説明を受けていらっしゃいますか？」

「ご自身の体の状態について、どのようにお考えですか？」

「この検査を受けることになった経緯を教えていただけますか？」
```

### 確認の質問（Clarifying Questions）

```
「〇〇という説明を受けられたとのことですが、
 それをどのように理解されていますか？」

「少し詳しくお聞かせいただけますか？」
```

---

## よくある認識パターン

### パターン1: 適切に理解している

```
患者: 「前回の検査で影があると言われて、
       もしかしたらがんかもしれないと思っています」

対応: 理解を認め、追加情報を提供
「はい、その通りです。今日は詳しい結果をお伝えします」
```

### パターン2: 楽観的すぎる

```
患者: 「影があるって言われましたが、
       きっと良性だと思います」

対応: 急に否定せず、段階的に現実に近づける
「そうお考えなのですね。今日は詳しい結果が出ましたので、
 一緒に確認していきましょう」
```

### パターン3: 理解していない

```
患者: 「よく分かっていないんです。
       何のための検査だったんでしょうか」

対応: 基本から説明する必要があることを認識
「分かりました。最初から順番にご説明しますね」
```

### パターン4: 否認している

```
患者: 「検査結果なんて見なくていいです。
       私は元気です」

対応: 無理に進めず、感情に対応
「今はお聞きになりたくないお気持ちなのですね。
 それは自然なことです」
```

## 認識確認の技術

### アクティブリスニング

| 技術 | 方法 | 例 |
| :--- | :--- | :--- |
| **うなずき** | 非言語で傾聴を示す | |
| **反復** | 患者の言葉を繰り返す | 「〇〇と感じていらっしゃるのですね」 |
| **言い換え** | 内容を確認 | 「つまり、△△ということでしょうか」 |
| **沈黙** | 考える時間を与える | （待つ） |

---

### 非言語コミュニケーション

```
✓ 目を見て話す（適度に）
✓ 前傾姿勢で関心を示す
✓ 腕を組まない（開かれた姿勢）
✓ うなずきで傾聴を示す
✓ 適切な距離を保つ
```

---

## 認識確認後の進め方

### 認識に応じた情報提供

```
適切に理解 → そのまま次のステップへ
楽観的 → Warning shotを入れてから情報提供
理解不足 → 基本から丁寧に説明
否認 → 感情対応を優先、無理に進めない
```

### 移行のフレーズ

```
「ありがとうございます。
 今日は詳しい検査結果が出ましたので、
 お伝えしたいのですが、よろしいでしょうか？」

「お気持ちを聞かせていただきました。
 これから結果についてお話ししますね」
```

## 練習: 認識確認のロールプレイ

### 場面設定

```
状況: 65歳女性、乳房のしこりで検査
結果: 乳がん（Stage II）
患者は「良性だと信じている」と述べている
```

### 医師役の練習

```
1. 開かれた質問で認識を確認
2. 楽観的な認識に対して共感
3. Warning shotを入れる
4. 情報提供への許可を得る
```

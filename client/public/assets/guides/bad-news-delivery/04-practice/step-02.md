# ステップ4: 情報提供と感情対応

## Knowledge: 悪い知らせを伝える

認識確認と許可取得の後、**悪い知らせを伝える**ステップです。

---

## 伝え方の4原則

### 1. Warning Shot（予告）

いきなり核心を伝えず、**心の準備**を促します。

```
✓ 良い例:
「残念ながら、良くないお知らせがあります」
「検査結果について、少し心配な内容をお伝えしなければなりません」

✗ 避ける例:
「がんでした」（いきなり核心）
「心配しないでほしいのですが...」（矛盾したメッセージ）
```

---

### 2. Simple Language（平易な言葉）

専門用語を避け、**分かりやすい言葉**で伝えます。

| 専門用語 | 平易な言葉 |
| :--- | :--- |
| 悪性新生物 | がん |
| 転移 | がんが広がっている |
| 予後不良 | 治療が難しい状態 |
| 根治的治療 | 完全に治すことを目指す治療 |
| 緩和的治療 | 症状を和らげる治療 |
| BSC | つらい症状を和らげることに集中する |

---

### 3. Small Chunks（小分けに）

**情報を段階的に**伝え、患者の反応を確認します。

```
段階1: 診断を伝える
「検査の結果、残念ながら、がんでした」
　　↓ （反応を確認、質問を受ける）

段階2: 病状を伝える
「現在の状態についてお話しします...」
　　↓ （反応を確認、質問を受ける）

段階3: 治療選択肢を伝える
「今後の治療についてですが...」
```

---

### 4. Check Understanding（理解確認）

患者の**理解度を確認**しながら進めます。

```
「ここまでで、ご質問はありますか？」

「今お話ししたことを、どのように受け止めていらっしゃいますか？」

「もう一度説明が必要なところはありますか？」
```

---

## Emotions: 感情への対応

悪い知らせを聞いた患者には、**様々な感情反応**が生じます。

### よく見られる感情反応

| 反応 | 表れ方 | 対応のポイント |
| :--- | :--- | :--- |
| **否認** | 「そんなはずは...」 | 無理に説得しない、時間をかける |
| **怒り** | 「なぜもっと早く...」 | 怒りを受け止める、弁解しない |
| **取引** | 「〇〇すれば治る？」 | 希望を否定しない |
| **抑うつ** | 沈黙、涙 | そばにいる、沈黙を共有 |
| **受容** | 「分かりました」 | 今後の方針を共有 |

---

## NURSE: 共感的対応の技術

### Naming（感情の言語化）

患者の感情を**言葉にして**返します。

```
「とても辛いお気持ちですよね」

「ショックを受けていらっしゃるのが伝わります」

「怒りを感じていらっしゃるのは当然のことです」
```

### Understanding（理解を示す）

その感情が**当然であること**を伝えます。

```
「このような状況で、そのようなお気持ちになるのは
 ごく自然なことです」

「誰でもそう感じると思います」
```

### Respecting（敬意を示す）

患者の**これまでの努力**を認めます。

```
「これまで、本当に頑張ってこられましたね」

「ご家族のことを一番に考えていらっしゃるのですね」
```

### Supporting（サポートを約束）

**一人ではない**ことを伝えます。

```
「私たちは最後までサポートします」

「一緒に考えていきましょう」

「何でもご相談ください」
```

### Exploring（さらに探る）

患者の**気持ちをさらに聴く**姿勢を示します。

```
「今、一番心配なことは何ですか？」

「他に気になることはありますか？」

「ご家族のことで心配なことはありますか？」
```

## 困難な場面への対応

### 患者が泣いている時

```
✓ すべきこと:
- 沈黙を共有する
- ティッシュを差し出す
- そばにいる
- 「泣いていいんですよ」と伝える

✗ 避けること:
- 急かす
- 話題を変える
- 励ましすぎる
```

### 患者が怒っている時

```
✓ すべきこと:
- 怒りを受け止める
- 「怒りを感じるのは当然です」と認める
- 弁解せずに聴く

✗ 避けること:
- 反論する
- 責任を転嫁する
- 感情的になる
```

### 患者が沈黙している時

```
✓ すべきこと:
- 沈黙を許容する（最低10秒は待つ）
- 「ゆっくりでいいですよ」と伝える
- 非言語で寄り添う

✗ 避けること:
- すぐに言葉で埋めようとする
- 質問攻めにする
```

## 実践例: 会話のスクリプト

### 場面: 大腸がん Stage IV の告知

```
医師: 「検査結果が出ました。
       残念ながら、良くないお知らせをしなければなりません」
       （Warning shot）

患者: 「...」

医師: 「大腸の検査で見つかった腫瘍は、
       残念ながら、がんでした」
       （Simple language）

患者: 「がん...ですか...」

医師: 「はい。とてもショックですよね」
       （Naming）

患者: （涙を流す）

医師: （沈黙、ティッシュを差し出す）
      「泣いていいんですよ。
       このような状況で涙が出るのは当然のことです」
       （Understanding）

　　　 （間を置く）

医師: 「少し落ち着かれましたら、
       今後のことについてお話ししてもよろしいですか？」
       （Check / Invitation）
```

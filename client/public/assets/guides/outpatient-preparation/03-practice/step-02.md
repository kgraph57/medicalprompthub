# ステップ3: AIを活用した予習メモ作成

## AIで予習を効率化

AIを活用することで、**情報の整理**と**方針の確認**を効率化できます。

## プロンプト集

### プロンプト1: カルテ情報からの予習メモ作成

```
以下の患者情報から、外来予習メモを作成してください。

【患者情報】
{匿名化したカルテ情報を貼り付け}

以下のPREP形式で整理してください：
- Problem（今回の受診目的）
- Review（前回からの経過・変化）
- Evaluation（検査結果の解釈、異常値の意味）
- Plan（今回の診療で確認・実施すべきこと）

特に以下の点に注目してください：
1. 前回からの変化（改善/悪化/横ばい）
2. 異常値とその臨床的意義
3. 治療目標との乖離
4. 患者に確認すべき事項
```

---

### プロンプト2: 検査異常値の解釈

```
以下の検査結果について、臨床的な解釈と推奨される対応を教えてください。

【患者背景】
年齢・性別: {例: 70代女性}
疾患: {例: 2型糖尿病、高血圧、CKD G3b}
現在の処方: {例: メトホルミン、ARB}

【検査結果】
{検査項目と値を記載}
前回値: {比較値があれば}

以下について簡潔に回答してください：
1. この異常値の臨床的意義
2. 考えられる原因
3. 推奨される対応（処方変更、追加検査等）
4. 患者に確認すべきこと
```

---

### プロンプト3: 複数問題の優先順位付け

```
以下の患者は複数の問題を抱えています。
外来診療での優先順位付けを提案してください。

【患者背景】
{匿名化した患者情報}

【問題リスト】
1. {問題1}
2. {問題2}
3. {問題3}

以下の観点から優先順位を付けてください：
- 緊急性（早急な対応が必要か）
- 重要性（患者の予後への影響）
- 実現可能性（今日の外来で対応可能か）

また、各問題について今日の外来でどこまで対応すべきか、
次回以降に回してよいものは何かを提案してください。
```

## 実践例

### 入力情報（匿名化済み）

```
70代女性
疾患: 2型糖尿病、高血圧、CKD G3b、骨粗鬆症
処方:
- メトホルミン500mg 2T2x
- アジルサルタン20mg 1T1x
- アムロジピン5mg 1T1x
- アレンドロン酸35mg 週1回

前回(3ヶ月前):
HbA1c 7.3%, BP 142/88, Cr 1.4, K 4.5, eGFR 36

今回検査:
HbA1c 7.5%, Cr 1.6, K 5.1, eGFR 32

前回のPlan:
「腎機能に注意、次回eGFR確認してメトホルミン継続可否判断」
```

### AI出力例

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
外来予習メモ: 70代女性 DM/HT/CKD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【Problem: 今回の受診目的】
✓ 定期フォロー
✓ 腎機能評価（前回からの懸念事項）
✓ 糖尿病薬の継続可否判断

【Review: 前回からの変化】
⚠️ 腎機能悪化: Cr 1.4→1.6, eGFR 36→32
⚠️ K上昇: 4.5→5.1（要注意域）
↗️ HbA1c微増: 7.3%→7.5%

【Evaluation: 検査結果の解釈】
🔴 重要: eGFR 32でCKD G3b継続
   → メトホルミンはeGFR 30未満で禁忌
   → 現時点で継続は可能だが要注意

🟡 注意: K 5.1
   → ARBの影響を考慮
   → さらなる上昇で減量・中止検討

🟡 注意: HbA1c 7.5%（高齢者目標7.0-8.0%）
   → 目標範囲内だが上昇傾向

【Plan: 今日の診療方針】

1. メトホルミンについて（優先度高）
   □ 継続可能だが、eGFR 30未満で中止方針を説明
   □ 代替薬（DPP4阻害薬）への変更を検討
   □ 現状維持の場合は次回1-2ヶ月後にeGFR再検

2. 高K血症対策（優先度高）
   □ 食事でのK摂取状況を確認
   □ ARB減量（アジルサルタン20mg→10mg）を検討
   □ 改善なければCaポリスチレン追加検討

3. 腎臓内科紹介の検討（優先度中）
   □ eGFR 30未満になる前に紹介を検討
   □ 患者の希望を確認

4. 血圧管理（優先度中）
   □ 前回142/88→今回の値を確認
   □ ARB減量時はCa拮抗薬増量で対応

【患者への確認事項】
□ 服薬アドヒアランス
□ 低血糖症状の有無
□ 食事摂取状況（K制限の理解）
□ 腎臓内科紹介の意向
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 効率化のコツ

### 1. 定型プロンプトの保存

よく使うプロンプトをテンプレートとして保存しておく

### 2. 複数患者の一括処理

```
以下の3名の患者について、それぞれ簡潔な予習メモを作成してください。

【患者1】...
【患者2】...
【患者3】...

各患者について3-5行で、
異常値・方針変更の要否・確認事項を記載してください。
```

### 3. 出力の確認

- AI出力は必ず医師が確認
- 検査値の解釈が正しいか確認
- 施設のガイドライン・方針と照合

---

## 注意点

### 個人情報の匿名化

| 元の情報 | 匿名化後 |
| :--- | :--- |
| 山田花子 | 70代女性 |
| ID: 12345678 | 削除 |
| 住所・電話 | 削除 |

### AIの限界

- 最新のガイドラインに対応していない可能性
- 施設固有の方針は反映されない
- 患者の個別事情は考慮されない

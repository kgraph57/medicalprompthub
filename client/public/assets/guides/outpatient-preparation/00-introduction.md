# 外来予習ワークフロー

## はじめに

外来診療の質は**事前準備**に大きく左右されます。限られた診察時間の中で、適切な診療を行うためには、効率的な予習が不可欠です。

> 「次の患者さん、何で来院されるんだっけ...」

このような状況を防ぎ、**患者さんに安心感を与える診療**を実現するためのワークフローを紹介します。

---

## なぜ外来予習が重要か？

### 外来診療の課題

| 課題 | 影響 |
| :--- | :--- |
| 診察時間の制約 | 1人5-10分で判断が必要 |
| 複数の問題 | 複数の疾患を同時に管理 |
| 前回からの変化 | 経過を把握する必要 |
| 検査結果の確認 | 見落としのリスク |

## 対象読者

- 外来診療を担当する研修医
- 複数の患者を効率的に診る必要がある医師
- 時間管理に悩む外来担当医
- 新しい科をローテートする研修医

## 注意事項

### 個人情報の取り扱い

⚠️ **重要**: AIツールに患者情報を入力する際は**必ず匿名化**してください

### 予習の限界

- 予習で全てをカバーできるわけではない
- 診察時に新たな情報が出てくることを想定
- 柔軟な対応を心がける

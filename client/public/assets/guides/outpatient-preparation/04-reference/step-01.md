# ステップ4: 診療科別の予習ポイント

## 診療科別チェックリスト

各診療科で特に注目すべき予習ポイントを紹介します。

---

## 内科系

### 一般内科・総合内科

```
【予習チェックリスト】

□ 問題リストの確認（Active/Inactive）
□ 各疾患のコントロール状況
□ 処方内容と残薬の確認
□ 直近の検査結果（代謝・腎・肝）
□ 他科受診の情報
□ 健診結果の確認

【重要検査項目】
- 血糖・HbA1c（糖尿病）
- 腎機能（eGFR, Cr, K）
- 肝機能（AST, ALT, γGTP）
- 脂質（LDL, TG, HDL）
- 血圧トレンド
```

---

### 循環器内科

```
【予習チェックリスト】

□ 心機能（EF, BNP/NT-proBNP）
□ 不整脈の有無・ペースメーカー設定
□ 抗凝固薬の管理（PT-INR, DOAC）
□ 血圧・脈拍のトレンド
□ 心エコー・ホルターの結果
□ CAG/PCIの既往・ステント情報

【重要検査項目】
- BNP/NT-proBNP（心不全）
- PT-INR（ワーファリン）
- Cr, K（利尿剤・ACEI/ARB使用時）
- 心電図変化
```

---

### 呼吸器内科

```
【予習チェックリスト】

□ 呼吸機能検査（FEV1, %FVC）
□ 画像所見の変化（胸部CT/X線）
□ 酸素療法の有無・設定
□ 吸入薬の使用状況
□ 喫煙歴・禁煙状況
□ ワクチン接種状況

【重要検査項目】
- SpO2
- 呼吸機能検査
- KL-6, SP-D（間質性肺炎）
- 腫瘍マーカー（悪性腫瘍フォロー）
```

---

### 消化器内科

```
【予習チェックリスト】

□ 肝機能・肝予備能（Child-Pugh）
□ 腫瘍マーカー（AFP, CEA, CA19-9）
□ 内視鏡検査の結果・次回予定
□ 画像検査（CT/MRI/US）の結果
□ 便潜血・大腸カメラの予定
□ H. pylori除菌の既往

【重要検査項目】
- 肝機能（AST, ALT, Alb, Bil, PT）
- 腫瘍マーカー
- ウイルス肝炎関連（HBV-DNA, HCV-RNA）
```

---

### 腎臓内科

```
【予習チェックリスト】

□ 腎機能トレンド（eGFR, Cr）
□ 尿検査（蛋白, 血尿）
□ 電解質バランス（K, Na, Ca, P）
□ 貧血管理（Hb, Fe, フェリチン, TSAT）
□ 血圧コントロール
□ 透析導入の検討状況

【重要検査項目】
- eGFR, Cr
- K, Ca, P
- Hb, フェリチン
- 尿蛋白/Cr比
- intact PTH
```

---

### 糖尿病内科

```
【予習チェックリスト】

□ 血糖コントロール（HbA1c, GA）
□ 低血糖の有無
□ 合併症評価（眼科・腎機能・神経）
□ 体重変化
□ CGMデータ（使用している場合）
□ フットケア・足病変

【重要検査項目】
- HbA1c
- 尿アルブミン/Cr比
- eGFR
- LDL
- 眼科受診状況
```

---

## 外科系

### 一般外科

```
【予習チェックリスト】

□ 術後フォローの場合: 術式・術後経過
□ 腫瘍マーカーのトレンド
□ 画像検査結果（再発の有無）
□ 化学療法の有無・副作用
□ 栄養状態（体重、Alb）
□ 再発リスクの評価

【重要検査項目】
- 腫瘍マーカー（CEA, CA19-9, AFP等）
- 画像検査結果
- 栄養指標（Alb, 体重）
```

---

### 整形外科

```
【予習チェックリスト】

□ 術後の場合: 術式・X線所見
□ リハビリの進捗
□ 疼痛コントロール
□ 骨粗鬆症治療の状況
□ 骨密度検査結果
□ 装具・固定具の状況

【重要検査項目】
- X線所見（骨癒合・アライメント）
- 骨密度（DXA）
- Ca, ALP, ビタミンD
```

---

## 小児科

```
【予習チェックリスト】

□ 成長曲線（身長・体重）
□ 発達マイルストーン
□ 予防接種歴
□ アレルギー（食物・薬物）
□ 学校・保育園の状況
□ 家族歴

【重要確認事項】
- 年齢に応じた発達評価
- 予防接種スケジュール
- 食事・睡眠の状況
```

---

## 精神科

```
【予習チェックリスト】

□ 現在の精神状態・症状
□ 処方薬と副作用
□ 希死念慮の有無
□ 社会機能（就労・就学）
□ 家族のサポート状況
□ 心理検査・評価結果

【重要確認事項】
- 服薬アドヒアランス
- 副作用（錐体外路症状、代謝異常）
- 生活リズム・睡眠
```

## まとめ

### 外来予習の原則

1. **PREP法**で情報を構造化
2. **優先順位**を付けて効率的に
3. **AIを補助ツール**として活用
4. **診療科特性**に応じた確認

### 継続的な改善

- 自分なりのチェックリストを作成
- 見落としがあったらリストに追加
- チームで予習のコツを共有

## 情報抽出のショートカット

### 1. 検査結果の効率的確認

```
確認のコツ:
✓ 異常値にフラグが立っているものを優先
✓ 前回値との比較機能を活用
✓ トレンドグラフで傾向を把握
✓ 新規オーダーの結果を優先確認
```

### 異常値チェックリスト

| 検査項目 | 要注意基準 | アクション |
| :--- | :--- | :--- |
| HbA1c | >8.0% or 変動>0.5% | 治療見直し |
| eGFR | <45 or 低下傾向 | 腎臓内科相談検討 |
| K | <3.5 or >5.5 | 薬剤調整 |
| LDL | >140（高リスク>100） | スタチン検討 |

---

### 2. カルテサマリーの読み方

#### 読むべき箇所

1. **Assessment/Plan**（前回の方針）
2. **次回予定**
3. **申し送り事項**

#### 読まなくてよい箇所

- 毎回同じ定型文
- 詳細なバイタル履歴（トレンドで確認）
- 変化のない所見

---

### 3. 処方内容の確認

```
確認ポイント:
✓ 前回からの変更有無
✓ 残薬の有無（Do処方か調整処方か）
✓ 相互作用チェック
✓ 腎機能・年齢に応じた用量
```

---

## 効率化テクニック

### 1. バッチ処理

同じ種類の情報をまとめて確認します。

```
効率的な順序:
1. 全患者の検査結果を一括確認（異常値をマーク）
2. 異常値のある患者から詳細確認
3. 定期フォローの患者は簡潔に
```

---

### 2. 色分け・マーキング

```
マーキングシステム例:
🔴 要対応（検査異常、方針変更要）
🟡 確認事項あり（質問したい、追加検査）
🟢 定型（変更なし、処方継続）
```

---

### 3. テンプレートの活用

#### 簡易予習メモ

```
[患者番号] [病名]
検査: [異常値/正常]
方針: [継続/変更/要相談]
確認: [特記事項]
```

#### 例

```
#3 DM/HT
検査: HbA1c 7.5%(↓), Cr 0.9(→)
方針: 継続
確認: 運動状況
---
#7 CKD/HT/DM
検査: Cr 2.1(↑), K 5.2
方針: 要相談（腎臓内科）
確認: ACEI減量検討
```

## よくある落とし穴

### 避けるべきパターン

| パターン | 問題点 | 解決策 |
| :--- | :--- | :--- |
| 全員を同じ深さで予習 | 時間不足 | 優先順位付け |
| カルテを最初から読む | 非効率 | サマリーから読む |
| 検査結果を全て確認 | 情報過多 | 異常値のみ抽出 |
| 予習内容を覚えようとする | 負担大 | メモを活用 |

## 実践: 情報抽出の練習

### 例題: 以下の情報から予習メモを作成

**電子カルテ情報（抜粋）:**

```
患者: 72歳女性
病名: 2型糖尿病、高血圧、慢性腎臓病G3b
処方: メトホルミン500mg 2T2x
      アムロジピン5mg 1T1x
      アジルサルタン20mg 1T1x
前回(3ヶ月前): HbA1c 7.3%, Cr 1.4, K 4.5
       「次回、腎機能に注意してメトホルミン継続可否判断」
今回検査: HbA1c 7.5%, Cr 1.6, K 5.1, eGFR 32
```

### 解答例

```
#__ 72F DM/HT/CKD
検査: 🔴Cr↑1.6(1.4→), K↑5.1, eGFR32, HbA1c7.5%
方針: 要変更（メトホルミン中止検討）
確認:
□ メトホルミン→DPP4阻害薬へ変更
□ ARB減量または変更検討
□ 腎臓内科紹介検討
```

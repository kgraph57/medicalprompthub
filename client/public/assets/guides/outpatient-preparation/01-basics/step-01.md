# ステップ1: 外来予習の基本フレームワーク

## PREP法による予習

外来予習には**PREP法**を活用します。これは効率的な情報整理のためのフレームワークです。

## 各要素の詳細

### 1. Problem（受診目的）

**今回の受診目的**を明確にします。

#### 確認ポイント

- 定期フォロー？新規症状？検査結果説明？
- 前回の診察で予定されていたこと
- 患者からの連絡・相談事項

```
【確認例】
・2型糖尿病の定期フォロー
・3ヶ月毎のHbA1c確認
・前回、薬の調整を検討予定だった
```

---

### 2. Review（経過確認）

**前回からの経過**を確認します。

#### 確認ポイント

- 前回の診療内容（サマリー）
- 処方内容の変更有無
- 患者の訴え・症状の変化
- 他科受診の情報

```
【確認例】
・前回（3ヶ月前）: HbA1c 7.8%
・メトホルミン 500mg 2T2x → 1000mg 2T2x に増量
・副作用（消化器症状）の確認が必要
```

---

### 3. Evaluation（評価）

**検査結果・バイタル**を確認します。

#### 確認ポイント

- 新しい検査結果
- 異常値のチェック
- 前回との比較
- 他院からの情報

```
【確認例】
・HbA1c: 7.5%（前回7.8%→改善）
・Cr: 1.0 mg/dL（横ばい）
・LDL: 140 mg/dL（やや高値、要検討）
```

---

### 4. Plan（方針）

**今回の診療方針**を立てます。

#### 確認ポイント

- 継続か変更か
- 追加検査の必要性
- 専門科への紹介
- 次回予約の間隔

```
【確認例】
・HbA1c改善→現行処方継続
・LDL高値→スタチン追加を検討
・腎機能確認のため次回Cr再検
・次回3ヶ月後
```

## 予習時間の目安

### 患者1人あたりの予習時間

| 患者の複雑さ | 予習時間 |
| :--- | :--- |
| シンプル（単一疾患） | 1-2分 |
| 中等度（複数疾患） | 3-5分 |
| 複雑（多疾患・問題あり） | 5-10分 |

### 外来開始前の予習時間

| 患者数 | 必要時間 |
| :--- | :--- |
| 10人 | 20-30分 |
| 20人 | 40-60分 |
| 30人以上 | 1時間以上 |

## 優先順位付け

### トリアージの考え方

外来予習でも**優先順位**を付けましょう。

| 優先度 | 患者の特徴 | 予習の深さ |
| :--- | :--- | :--- |
| 🔴 高 | 新患、検査結果異常、複雑な問題 | 詳細に |
| 🟡 中 | 定期フォロー、複数疾患 | 標準 |
| 🟢 低 | 安定した単一疾患、処方のみ | 簡潔に |

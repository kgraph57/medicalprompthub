# ステップ3: 効果的なフィードバック技法

## フィードバックの重要性

フィードバックは、研修医の成長を促す**最も重要な教育活動**の一つです。

---

### 2. タイムリーであること

```
最適なタイミング:
✓ 観察直後（24時間以内）
✓ 記憶が鮮明なうちに
✓ プライバシーが確保できる場所で

避けるべきタイミング:
✗ 数週間後（記憶が薄れる）
✗ 人前で（恥をかかせる）
✗ 研修医が疲れきっている時
```

---

### 3. 行動に焦点を当てること

人格ではなく、**具体的な行動**についてコメントします。

```
✗ 「あなたは不注意だ」（人格への評価）
✓ 「今回、電解質の確認が抜けていました」（行動への評価）

✗ 「やる気がない」（人格への評価）
✓ 「今日のカンファレンスで発言が少なかったですね」（行動への評価）
```

## フィードバックの手法

### 1. サンドイッチ法

**ポジティブ → 改善点 → ポジティブ**の順で伝えます。

```
「今日のプレゼンテーション、
 患者さんの背景がよく整理されていました（+）。

 ただ、アセスメントの部分がやや長くなっていたので、
 鑑別診断を3つに絞るともっと分かりやすくなります（改善）。

 全体的に自信を持って話せていたのは良かったですね（+）」
```

---

### 2. ASK-TELL-ASK法

**質問で始め、伝え、質問で終わる**方法です。

```
【ASK】自己評価を聞く
「今日の手技、自分ではどうでしたか？」

【TELL】フィードバックを伝える
「私が見たところ、穿刺は上手くいきましたが、
 固定のところでステップを一つ飛ばしていました」

【ASK】理解を確認する
「次に気をつけることは何だと思いますか？」
```

---

### 3. Pendleton法

研修医の**自己評価を中心**にする方法です。

```
ステップ1: 研修医「良かった点」を述べる
ステップ2: 指導医「良かった点」を追加
ステップ3: 研修医「改善したい点」を述べる
ステップ4: 指導医「改善点」について提案
ステップ5: 次のステップを合意
```

## フィードバックの実践例

### 例: 症例プレゼンテーション後

```
指導医:「今のプレゼンテーション、自分ではどうでしたか？」
        （ASK）

研修医:「主訴から現病歴までは順序よく話せたと思いますが、
        身体所見のところで何を言うか迷ってしまいました」

指導医:「そうですね。主訴と現病歴は簡潔で分かりやすかったです。
        身体所見は、異常所見を先に言うと聞き手が分かりやすいですよ。
        例えば『心雑音あり。他の所見は正常』という形式です」
        （TELL）

指導医:「次のプレゼンで試してみますか？」（ASK）
```

---

### 例: 手技の失敗後

```
（プライベートな場所で）

指導医:「今回の穿刺、うまくいきませんでしたね。
        辛かったと思いますが、大丈夫ですか？」
        （共感）

研修医:「はい...何がいけなかったのか分からなくて」

指導医:「見ていて気づいたのですが、エコーのプローブと
        針の角度が合っていなかったようです。
        次回は、まずプローブと針を同じ平面に
        合わせるところから意識してみましょう」
        （具体的な改善点）

指導医:「穿刺自体のテクニックは良くなっていますよ。
        あと数回で上手くいくと思います」
        （励まし）
```

## 困難なフィードバック

### ネガティブなフィードバックの伝え方

```
【準備】
- 具体的な事実を整理
- プライベートな場所を確保
- 十分な時間を確保
- 自分の感情を整理

【伝え方】
- 事実から始める（「〇〇ということがありましたね」）
- 影響を説明する（「それによって△△が起きました」）
- 相手の視点を聞く（「どう思いますか？」）
- 改善策を一緒に考える（「次はどうしましょうか」）
```

---

### 研修医が defensive になった時

```
対応:
- 感情を受け止める（「そう感じるのは分かります」）
- 事実に焦点を戻す（「今回の具体的な状況について話しましょう」）
- 責めるのではなく、改善を一緒に考える姿勢を示す
- 必要なら時間を置く（「少し考えてから話しましょうか」）
```

## プロンプト集

### プロンプト1: ミニレクチャー資料の作成

```
研修医向けのミニレクチャー資料を作成してください。

【トピック】{例: 高カリウム血症の対応}
【対象者】初期研修医（1年目）
【時間】15分程度

以下を含めてください：
1. 学習目標（3つ程度）
2. 導入（臨床的重要性、よくあるシナリオ）
3. 本論（体系的な内容、キーポイント）
4. まとめ（覚えるべきこと）
5. 確認クイズ（3問）

実践的で分かりやすい内容でお願いします。
```

---

### プロンプト2: 教育用症例の作成

```
研修医教育用の症例を作成してください。

【学習目標】{例: 急性腹症の鑑別診断を学ぶ}
【難易度】研修医1年目レベル
【形式】症例提示 → 質問 → 解説

以下を含めてください：
1. 患者背景・主訴
2. 現病歴・既往歴
3. バイタル・身体所見
4. 初期検査結果

質問:
- この患者の鑑別診断は？
- 次に行うべき検査は？
- 治療方針は？

各質問に詳細な解説を付けてください。
```

---

### プロンプト3: ロールプレイシナリオ

```
研修医のコミュニケーション訓練用のロールプレイ
シナリオを作成してください。

【場面】{例: 困難な患者への説明}
【学習目標】{例: 怒っている患者への対応}

以下を含めてください：
1. 場面設定（患者背景、状況）
2. 患者役の指示（どのように振る舞うか）
3. 医師役が達成すべきポイント
4. 評価のポイント
5. ディスカッションのガイド
```

---

### プロンプト4: 評価ルーブリックの作成

```
以下の臨床能力の評価ルーブリックを作成してください。

【評価対象】{例: 退院サマリーの作成}
【対象者】研修医

以下の形式でお願いします：
- 評価項目（5-7項目）
- 各項目の4段階評価基準
  （要改善 / 基準を満たす / 標準的 / 模範的）
- 各レベルの具体的な行動記述
```

---

### プロンプト5: フィードバックコメント

```
以下の研修医の観察結果から、
フィードバックコメントを作成してください。

【観察結果】
{具体的な観察内容を記載}

以下を含むコメントを作成してください：
1. 強みのコメント（具体的に）
2. 改善点のコメント（建設的に）
3. 次の目標設定への提案
4. 励ましの言葉

サンドイッチ法を使い、
モチベーションを維持しながら成長を促す内容で。
```

## 実践例: ミニレクチャー資料

### 入力

```
研修医向けのミニレクチャー資料を作成してください。
トピック: 抗菌薬の選択（市中肺炎）
対象: 初期研修医
時間: 10分
```

### AI出力例（一部）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
市中肺炎の抗菌薬選択 〜迷わない処方のために〜
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【学習目標】
1. 市中肺炎の主要な原因菌を3つ挙げられる
2. 重症度に応じた抗菌薬選択ができる
3. 非定型肺炎のカバーが必要な場面を理解する

【導入】（1分）
救急外来で最もよく遭遇する感染症の一つが市中肺炎です。
「この患者さん、どの抗菌薬を出せばいいですか？」
と迷う場面、ありませんか？

【本論】（6分）

1. 原因菌を知る
   - 肺炎球菌（最多）
   - インフルエンザ菌
   - マイコプラズマ
   - モラキセラ
   - クラミジア

2. 重症度評価（A-DROP）
   ...

【まとめ】（2分）
✓ 外来治療: AMPC or レスピラトリーキノロン
✓ 入院治療: ABPC/SBT ± マクロライド
✓ 重症: PIPC/TAZ + マクロライド or キノロン

【確認クイズ】
Q1. A-DROPスコア2点の患者の治療場所は？
...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 注意点

### AIの限界

```
⚠️ 医学的正確性の確認が必要
- ガイドラインと照合する
- 最新の推奨を確認する

⚠️ 施設に合わせた調整
- 使用できる薬剤・検査
- 施設のプロトコル

⚠️ 著作権への配慮
- 図表は自作または許諾取得
```

## まとめ

### 研修医指導の原則

```
1. マイルストーンで目標を明確に
2. 多角的な評価で成長を把握
3. 効果的なフィードバックで改善を促す
4. 困難な場面も適切に対応
5. AIを活用して効率的に教育資料を作成
```

### 指導医として大切なこと

```
✓ 研修医の成長を信じる
✓ 患者安全を最優先に
✓ 自らも学び続ける姿勢
✓ チームで研修医を育てる
```

# 研修医指導計画作成ガイド

## はじめに

研修医の教育は、次世代の医療を担う人材を育てる重要な仕事です。しかし、多忙な臨床業務の中で、**効果的な指導計画を立てる**ことは容易ではありません。

> 「研修医にどこまで任せていいのか分からない...」
> 「どうやって評価すればいいのか...」

このガイドでは、**マイルストーンに基づいた指導計画**の作成方法と、AIを活用した教育資料の効率的な作成方法を紹介します。

---

## なぜ体系的な指導計画が必要か

### 研修医教育の課題

| 課題 | 影響 |
| :--- | :--- |
| 場当たり的な指導 | 学びにムラが生じる |
| 評価基準の曖昧さ | 成長の実感が得られない |
| 時間不足 | 振り返りができない |
| 指導者間の連携不足 | 一貫性のない教育 |

## 主要な概念

### マイルストーン（Milestones）

研修医が**段階的に習得すべき能力**を定義したものです。

```
例: 中心静脈カテーテル挿入のマイルストーン

Level 1: 手技の見学、解剖の理解
Level 2: 指導下での手技実施
Level 3: 監督下での独立した手技
Level 4: 合併症対応を含む独立した実施
Level 5: 後輩への指導が可能
```

# ステップ2: 到達度評価の設計

## なぜ評価が重要か

適切な評価は、研修医の成長を促し、**何ができて何が課題か**を明確にします。

---

## 評価の種類

### 形成的評価 vs 総括的評価

| | 形成的評価 | 総括的評価 |
| :--- | :--- | :--- |
| **目的** | 成長を促す | 到達度を判定 |
| **頻度** | 頻繁（日々） | 定期的（月・クール毎） |
| **方法** | フィードバック、観察 | 評価シート、面談 |
| **結果の活用** | 学習計画の修正 | 進級・修了判定 |

## 評価方法

### 1. 直接観察評価（Mini-CEX）

臨床場面を**直接観察**して評価します。

```
【Mini-CEX評価項目】

□ 病歴聴取
□ 身体診察
□ コミュニケーション能力
□ 臨床判断
□ プロフェッショナリズム
□ 効率性・組織化

各項目を9段階で評価:
1-3: 期待以下
4-6: 期待通り
7-9: 期待以上
```

---

### 2. 手技評価（DOPS）

臨床手技を**直接観察**して評価します。

```
【DOPS評価項目】（例: CV挿入）

□ 適応の理解と説明
□ インフォームドコンセント
□ 準備（物品、患者体位）
□ 清潔操作
□ 技術的能力
□ 合併症への対応
□ 記録

評価: 要改善 / 最低限 / 標準 / 模範的
```

---

### 3. 多職種評価（360度評価）

看護師、薬剤師、他のスタッフからのフィードバックを収集します。

```
【360度評価の対象者】

- 上級医
- 同僚の研修医
- 看護師
- 薬剤師
- 患者・家族

【評価項目】
- コミュニケーション
- チームワーク
- プロフェッショナリズム
- 責任感
```

---

### 4. ポートフォリオ評価

研修医が自ら**学びを記録・振り返る**評価方法です。

```
【ポートフォリオの内容例】

□ 経験症例のサマリー
□ 症例の振り返り（リフレクション）
□ 学習目標と達成状況
□ 手技ログ
□ 学会発表・論文
□ フィードバックの記録
```

## 評価のタイミング

### 定期評価のスケジュール

| タイミング | 内容 |
| :--- | :--- |
| **毎日** | 日々のフィードバック |
| **毎週** | 症例カンファレンスでの評価 |
| **月1回** | 形式的な振り返り面談 |
| **ローテーション終了時** | 総括的評価 |

---

## 評価のポイント

### 良い評価の特徴

```
✓ 具体的な行動に基づく
✓ 観察した事実を記載
✓ 改善のための提案を含む
✓ 強みも伝える（サンドイッチ法）
✓ 研修医と共有する
```

### 避けるべき評価

```
✗ 曖昧な表現（「まあまあ」「頑張っている」）
✗ 人格への評価（能力・行動への評価が原則）
✗ 他の研修医との比較
✗ フィードバックなしの評価
```

## AIを活用した評価

### プロンプト例: 評価コメントの作成

```
以下の研修医の観察結果から、
評価コメントを作成してください。

【観察結果】
・病歴聴取は丁寧だが時間がかかる
・身体診察は基本的な手技は問題なし
・プレゼンテーションは情報が多く、要点が不明確
・患者への説明は分かりやすい
・カルテ記載は適切

【作成してほしい内容】
1. 強みのコメント（2-3点）
2. 改善点のコメント（具体的な提案付き）
3. 次の1ヶ月の目標設定
4. 5段階評価の提案
```

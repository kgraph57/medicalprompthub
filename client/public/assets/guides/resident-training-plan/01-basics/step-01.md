# ステップ1: 研修目標の設定（マイルストーン）

## マイルストーンとは

マイルストーンは、研修医が**段階的に習得すべき能力**を明文化したものです。「何ができるようになればよいか」を具体的に示します。

## 臨床手技のマイルストーン例

### 中心静脈カテーテル挿入

```
【マイルストーン: 中心静脈カテーテル挿入】

Level 1:
□ 解剖学的構造を説明できる
□ 適応と禁忌を述べられる
□ 必要物品を準備できる
□ 手技を見学した

Level 2:
□ 指導医の直接監督下で手技を実施
□ エコーガイド下の穿刺が可能
□ 清潔操作を遵守できる
□ 穿刺後の確認手順を理解

Level 3:
□ 指導医の監督下で独立して実施
□ 一般的な合併症を認識・報告できる
□ 困難例を認識し、助けを求められる
□ 適切な固定・ドレッシングができる

Level 4:
□ 独立して安全に実施
□ 合併症（気胸など）に初期対応できる
□ 困難例（解剖学的変異など）に対応可能
□ 適応の妥当性を判断できる

Level 5:
□ 後輩への指導ができる
□ 施設のプロトコル改善に貢献
□ シミュレーション教育を実施
```

### 問診・病歴聴取

```
【マイルストーン: 問診・病歴聴取】

Level 1:
□ 病歴の構成要素を理解
□ 基本的な問診項目を挙げられる
□ 見学を通じて問診の流れを理解

Level 2:
□ 指導下で一般的な主訴の病歴聴取
□ 系統的レビューを実施
□ 基本的なコミュニケーション

Level 3:
□ 監督下で独立して包括的な病歴聴取
□ 重要な情報を適切に抽出
□ 患者の訴えを傾聴

Level 4:
□ 複雑な症例での包括的な病歴聴取
□ 困難な患者への対応
□ 病歴から適切な鑑別診断を立案

Level 5:
□ 問診技術を後輩に指導
□ 高度なコミュニケーション技術
□ 教育セッションの実施
```

## マイルストーン設定のプロセス

### 1. 到達目標の明確化

```
質問:
- この研修期間で何を習得すべきか？
- 終了時に何ができるようになるべきか？
- 最低限の能力は何か？
```

### 2. 段階的な目標設定

```
プロセス:
1. 最終目標（Level 4-5）を設定
2. 逆算して各レベルを定義
3. 具体的な行動目標を記述
4. 観察可能・測定可能な表現に
```

### 3. 時間軸との対応

```
例: 初期研修2年間

1年目前半: Level 1-2 を目指す
1年目後半: Level 2-3 へ
2年目前半: Level 3 を達成
2年目後半: Level 3-4 へ
```

## AIを活用したマイルストーン作成

### プロンプト例

```
以下の臨床能力について、研修医向けのマイルストーン
（5段階のレベル設定）を作成してください。

臨床能力: {例: 腹腔穿刺}

以下を含めてください：
1. Level 1-5 の段階的な能力記述
2. 各レベルで達成すべき具体的な行動目標
3. 観察可能・測定可能な表現
4. 次のレベルへ進むための基準

研修医が自己評価でき、指導医が評価しやすい
形式でお願いします。
```

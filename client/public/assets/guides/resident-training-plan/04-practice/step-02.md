# ステップ4: 困難な研修医への対応

## 困難な研修医とは

一定数の研修医は、**期待通りに成長しない**、あるいは**問題行動がある**場合があります。適切な対応が必要です。

---

## 困難な状況のパターン

### 1. 学習面の問題

| 問題 | 特徴 | 対応の方向性 |
| :--- | :--- | :--- |
| 知識不足 | 基本的な知識が欠如 | 学習計画の見直し |
| 応用力不足 | 知識はあるが活用できない | 症例ベースの学習 |
| 学習障害 | 読み書き・処理速度の問題 | 専門家への相談 |

---

### 3. 健康・メンタルヘルスの問題

```
注意が必要なサイン:
- 遅刻・欠勤の増加
- 以前と比べてのパフォーマンス低下
- 過度の疲労・ストレスの訴え
- 感情の不安定さ
- 孤立傾向

対応:
- プライベートな場所での面談
- 傾聴と共感
- 専門家（産業医、メンタルヘルス）への紹介
- 研修プログラム責任者への報告
```

---

## 対応のステップ

### ステップ1: 問題の特定

```
確認すべきこと:
□ 具体的な行動・事実は何か？
□ 複数の場面で観察されるか？
□ 他の指導医も同じ問題を認識しているか？
□ 患者安全に影響があるか？
□ 研修医自身は問題を認識しているか？
```

---

### ステップ2: 原因の分析

```
考えられる原因:
- 学習機会の不足
- 学習方法の問題
- モチベーションの問題
- 健康上の問題（うつ、バーンアウト）
- 個人的な問題（家庭、人間関係）
- 学習障害
- 研修環境の問題
```

---

### ステップ3: 面談の実施

```
面談の準備:
□ プライベートな場所を確保
□ 十分な時間を確保（最低30分）
□ 具体的な事実・データを用意
□ 感情を整理しておく
□ 必要に応じて研修責任者に事前相談

面談の進め方:
1. 現状認識の共有
2. 研修医の視点を聞く（傾聴）
3. 原因を一緒に探る
4. 改善計画を立てる
5. フォローアップの約束
```

---

### ステップ4: 改善計画の策定

```
改善計画に含めるべき項目:

1. 具体的な目標（SMART目標）
   - 具体的（Specific）
   - 測定可能（Measurable）
   - 達成可能（Achievable）
   - 関連性がある（Relevant）
   - 期限がある（Time-bound）

2. 必要な支援
   - 追加の学習機会
   - メンター・チューターの配置
   - 専門家への紹介

3. モニタリング方法
   - 定期的な面談
   - 進捗の評価

4. 期限
   - 評価のタイミング
   - 次のステップの判断時期
```

## エスカレーション

### 報告が必要な場合

```
以下の場合は、研修プログラム責任者に報告:

□ 患者安全に影響がある行動
□ 繰り返されるプロフェッショナリズムの問題
□ 改善計画が効果を示さない
□ 健康上の深刻な問題が疑われる
□ ハラスメントの訴えがある
```

## 支援リソース

### 活用できるリソース

| リソース | 役割 |
| :--- | :--- |
| 研修プログラム責任者 | 全体的な管理、重大問題への対応 |
| メンター | 日常的な相談、キャリア支援 |
| 産業医 | 健康問題への対応 |
| 心理士・カウンセラー | メンタルヘルス支援 |
| 学習支援専門家 | 学習障害への対応 |

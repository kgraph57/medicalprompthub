# ステップ1: 実際の症例でAIを使って資料を作成する

## 実践：AIで症例発表資料を作成しよう

このステップでは、実際の症例を使ってAIで発表資料を作成します。**ChatGPT**または**Claude**を使用します。

---

## 準備：症例情報を整理する

まず、手元の症例情報を整理します。カルテや検査結果から、以下の情報を抽出してください。

### 症例情報チェックリスト

- [ ] 年齢・性別
- [ ] 主訴
- [ ] 現病歴（時系列）
- [ ] 既往歴・内服薬
- [ ] 身体所見（バイタル、重要な所見）
- [ ] 検査所見（血液検査、画像検査）
- [ ] 診断
- [ ] 治療内容
- [ ] 経過・転帰

## ステップ3: 実践例 - 肺塞栓症の症例

### 入力する症例情報

```
あなたは経験豊富な臨床医です。以下の症例情報から、カンファレンス発表用のスライド構成を提案してください。

【症例情報】
- 年齢・性別: 65歳、男性
- 主訴: 呼吸困難
- 現病歴: 来院2時間前から突然の呼吸困難が出現。胸痛はなし。1週間前に右下肢の腫脹を自覚していた。
- 既往歴: 糖尿病、高血圧。3ヶ月前に右膝の人工関節置換術を施行。
- 身体所見: 
  - バイタル: 体温36.8°C、血圧130/80mmHg、脈拍110/分（整）、SpO2 88%（室内気）
  - 重要な所見: 右下肢の腫脹・圧痛あり、Homans徴候陽性
- 検査所見:
  - 血液検査: WBC 8,500、CRP 2.1、D-dimer 8.5μg/mL、BNP 150pg/mL
  - 心電図: S1Q3T3パターン
  - 胸部X線: 明らかな異常なし
  - 造影CT: 右肺動脈に血栓、右下肢深部静脈血栓症
- 診断: 肺塞栓症（深部静脈血栓症に伴う）
- 治療: ヘパリン持続静注、その後DOACへ切り替え
- 経過: 酸素化改善、第7病日に退院

【出力形式】
以下の構成でスライドの内容を提案してください：
1. タイトルスライド
2. 症例サマリー
3. 現病歴
4. 身体所見
5. 検査所見
6. 鑑別診断（優先順位付き、各診断の根拠と除外理由を含む）
7. 診断根拠
8. 治療方針（選択肢と選択理由）
9. 経過
10. Take-home message（3つ以内）

各スライドには、箇条書きで具体的な内容を記載してください。
発表時間は15分を想定しています。
```

## ステップ4: AIの出力を確認する

AIが生成したスライド構成を確認します。以下のポイントをチェックしてください。

### ✅ チェックリスト

- [ ] **網羅性**: 必要な情報が全て含まれているか
- [ ] **論理性**: 診断プロセスが論理的か
- [ ] **正確性**: 医学的に正確か（必ず確認！）
- [ ] **簡潔性**: 冗長な部分はないか
- [ ] **時間配分**: 15分で発表できる量か

---

## ステップ5: 出力を修正・改善する

AIの出力に不足や誤りがあれば、追加のプロンプトで修正します。

### 修正プロンプトの例

#### 例1: 鑑別診断を追加する

```
鑑別診断に「急性冠症候群」も追加してください。除外した理由も含めてください。
```

#### 例2: Take-home messageを強調する

```
Take-home messageをより具体的にしてください。特に「術後患者のVTE予防」について強調したいです。
```

#### 例3: スライドを簡潔にする

```
スライド5（検査所見）が長すぎます。最も重要な所見のみに絞ってください。
```

---

## ステップ6: PowerPoint/Keynoteに転記する

AIが生成した内容を、PowerPoint/Keynoteに転記します。

### 効率的な転記方法

1. **スライドマスターを活用**
   - タイトル、本文のフォーマットを統一
   - 病院のテンプレートがあれば使用

2. **箇条書きをそのままコピペ**
   - AIの出力は箇条書きなので、そのまま貼り付け可能

3. **画像・グラフを追加**
   - CT画像、心電図などを挿入
   - AIは画像を生成できないので、手動で追加

## 実践例：完成したスライド（抜粋）

### スライド1: タイトル

```
65歳男性、突然の呼吸困難で来院
診断: 肺塞栓症

発表者: 山田太郎（内科研修医）
2025年12月13日
```

### スライド6: 鑑別診断

```
【鑑別診断】

1. 肺塞栓症 ★
   - 術後、下肢腫脹、D-dimer上昇、S1Q3T3
   
2. 急性冠症候群
   - 呼吸困難、胸部不快感
   → 胸痛なし、心電図でST変化なし、除外
   
3. 気胸
   - 突然の呼吸困難
   → 胸部X線で異常なし、除外
   
4. 心不全
   - 呼吸困難
   → BNP軽度上昇のみ、除外
```

### スライド10: Take-home message

```
【Take-home message】

1. 術後患者の呼吸困難では、
   肺塞栓症を必ず鑑別に挙げる

2. D-dimer上昇 + 下肢腫脹 = DVT/PEを強く疑う

3. 術後VTE予防（早期離床、抗凝固療法）の重要性
```

## 時間短縮のコツ

### 従来の方法
- **所要時間**: 2〜4時間
- **プロセス**: 
  1. スライド構成を考える（30分）
  2. 鑑別診断を調べる（30分）
  3. 各スライドを作成（1〜2時間）
  4. 推敲・修正（30分〜1時間）

### AI活用の方法
- **所要時間**: 30分〜1時間
- **プロセス**: 
  1. 症例情報を整理（10分）
  2. AIにプロンプト入力（5分）
  3. 出力を確認・修正（10分）
  4. PowerPointに転記（15〜30分）

**時間短縮効果**: **約60〜75%削減**

## よくある質問

### Q1: AIの出力をそのまま使っても大丈夫？

**A**: **必ず医学的な正確性を確認してください**。AIは時に誤った情報を生成することがあります（ハルシネーション）。特に以下の点は必ず確認：

- 診断基準
- 治療ガイドライン
- 薬剤の用法・用量

### Q2: 無料版のChatGPTで十分？

**A**: 基本的には**無料版で十分**です。ただし、以下の場合は有料版（ChatGPT Plus、Claude Pro）を検討：

- より長い症例（複雑な経過）
- より詳細な鑑別診断
- 文献検索も含めたい場合

### Q3: プロンプトは毎回入力する必要がある？

**A**: いいえ。**テンプレートを保存**しておけば、症例情報を入れ替えるだけで再利用できます。

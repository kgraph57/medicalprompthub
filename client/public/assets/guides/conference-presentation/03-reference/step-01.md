# ステップ1: 発表シナリオを作成する

## 発表シナリオの重要性

スライドを作成しただけでは、良い発表はできません。**発表シナリオ**（原稿）を作成することで、以下のメリットがあります：

✅ **時間配分**が明確になる  
✅ **話す内容**を事前に整理できる  
✅ **緊張を軽減**できる  
✅ **質疑応答**の準備ができる

---

## 発表シナリオ生成のプロンプト

### 基本プロンプト

```
あなたは経験豊富な臨床医です。以下のスライド構成から、カンファレンス発表用のシナリオ（原稿）を作成してください。

【スライド構成】
[ここにスライドの内容を貼り付け]

【発表条件】
- 発表時間: 15分
- 聴衆: 内科研修医・指導医
- 発表スタイル: フォーマル

【出力形式】
各スライドについて、以下の形式でシナリオを作成してください：

【スライド1: タイトル】
所要時間: 30秒
シナリオ: [話す内容]

【スライド2: 症例サマリー】
所要時間: 1分
シナリオ: [話す内容]

...

各スライドの所要時間を明記し、合計が15分になるように調整してください。
```

## 実践例: 肺塞栓症の発表シナリオ

### 入力プロンプト

```
あなたは経験豊富な臨床医です。以下のスライド構成から、カンファレンス発表用のシナリオ（原稿）を作成してください。

【スライド構成】

スライド1: タイトル
- 65歳男性、突然の呼吸困難で来院
- 診断: 肺塞栓症

スライド2: 症例サマリー
- 年齢・性別: 65歳、男性
- 主訴: 呼吸困難
- 現病歴: 来院2時間前から突然の呼吸困難が出現
- 診断: 肺塞栓症（深部静脈血栓症に伴う）
- 転帰: ヘパリン治療開始、第7病日に退院

スライド3: 現病歴
- X日 16:00: 突然の呼吸困難が出現
- X日 18:00: 症状増悪し救急搬送
- 1週間前から右下肢の腫脹を自覚
- 既往歴: 糖尿病、高血圧
- 3ヶ月前に右膝の人工関節置換術を施行

スライド4: 身体所見
- バイタル: 体温36.8°C、血圧130/80mmHg、脈拍110/分、SpO2 88%
- 右下肢の腫脹・圧痛あり
- Homans徴候陽性

スライド5: 検査所見
- WBC 8,500、CRP 2.1、D-dimer 8.5μg/mL
- 心電図: S1Q3T3パターン
- 造影CT: 右肺動脈に血栓、右下肢深部静脈血栓症

スライド6: 鑑別診断
1. 肺塞栓症 ★
2. 急性冠症候群 → 胸痛なし、除外
3. 気胸 → 胸部X線で異常なし、除外
4. 心不全 → BNP軽度上昇のみ、除外

スライド7: 診断根拠
- 術後、下肢腫脹、D-dimer上昇、S1Q3T3
- 造影CTで肺動脈血栓を確認

スライド8: 治療方針
- ヘパリン持続静注 → DOACへ切り替え
- 理由: 急性期は迅速な抗凝固が必要

スライド9: 経過
- 第1病日: ヘパリン開始、酸素化改善
- 第3病日: DOACへ切り替え
- 第7病日: 退院

スライド10: Take-home message
1. 術後患者の呼吸困難では、肺塞栓症を必ず鑑別に挙げる
2. D-dimer上昇 + 下肢腫脹 = DVT/PEを強く疑う
3. 術後VTE予防の重要性

【発表条件】
- 発表時間: 15分
- 聴衆: 内科研修医・指導医
- 発表スタイル: フォーマル

【出力形式】
各スライドについて、以下の形式でシナリオを作成してください：

【スライド1: タイトル】
所要時間: 30秒
シナリオ: [話す内容]

各スライドの所要時間を明記し、合計が15分になるように調整してください。
```

## AIの出力例

```
【発表シナリオ】

【スライド1: タイトル】
所要時間: 30秒
シナリオ:
「本日は、65歳男性、突然の呼吸困難で来院され、肺塞栓症と診断した症例を発表させていただきます。」

---

【スライド2: 症例サマリー】
所要時間: 1分
シナリオ:
「症例は65歳の男性です。主訴は呼吸困難です。来院2時間前から突然の呼吸困難が出現し、救急搬送されました。診断は肺塞栓症で、深部静脈血栓症に伴うものでした。ヘパリン治療を開始し、第7病日に退院されました。」

---

【スライド3: 現病歴】
所要時間: 2分
シナリオ:
「現病歴です。X日の16時、突然の呼吸困難が出現しました。症状は徐々に増悪し、18時に救急搬送されました。

重要な点として、1週間前から右下肢の腫脹を自覚されていました。

既往歴として、糖尿病と高血圧があります。また、3ヶ月前に右膝の人工関節置換術を施行されています。この術後という点が、今回の肺塞栓症のリスク因子として重要です。」

---

【スライド4: 身体所見】
所要時間: 1分30秒
シナリオ:
「身体所見です。バイタルサインは、体温36.8度、血圧130/80、脈拍110と頻脈、SpO2は室内気で88%と低酸素血症を認めました。

重要な所見として、右下肢の腫脹と圧痛を認め、Homans徴候が陽性でした。これは深部静脈血栓症を示唆する所見です。」

---

【スライド5: 検査所見】
所要時間: 2分
シナリオ:
「検査所見です。血液検査では、白血球とCRPは軽度上昇のみでしたが、D-dimerが8.5μg/mLと著明に上昇していました。これは血栓形成を強く示唆します。

心電図では、S1Q3T3パターンを認めました。これは肺塞栓症に特徴的な所見です。

造影CTでは、右肺動脈に血栓を認め、さらに右下肢の深部静脈にも血栓を認めました。これにより、深部静脈血栓症に伴う肺塞栓症と診断しました。」

---

【スライド6: 鑑別診断】
所要時間: 2分
シナリオ:
「鑑別診断です。突然の呼吸困難という主訴から、以下の疾患を鑑別に挙げました。

第一に、肺塞栓症です。術後、下肢腫脹、D-dimer上昇、S1Q3T3パターンという所見から、最も可能性が高いと考えました。

第二に、急性冠症候群を考えましたが、胸痛がなく、心電図でもST変化を認めなかったため、除外しました。

第三に、気胸を考えましたが、胸部X線で異常を認めず、除外しました。

第四に、心不全を考えましたが、BNPは軽度上昇のみで、心エコーでも明らかな心機能低下を認めず、除外しました。」

---

【スライド7: 診断根拠】
所要時間: 1分30秒
シナリオ:
「診断根拠です。本症例では、術後という背景、下肢腫脹、D-dimerの著明な上昇、心電図でのS1Q3T3パターンという臨床所見から、肺塞栓症を強く疑いました。

造影CTで肺動脈血栓を確認し、確定診断に至りました。」

---

【スライド8: 治療方針】
所要時間: 1分30秒
シナリオ:
「治療方針です。急性期はヘパリンの持続静注を開始しました。ヘパリンは効果発現が早く、急性期の治療に適しています。

その後、第3病日にDOAC、直接経口抗凝固薬に切り替えました。DOACはINRモニタリングが不要で、外来での管理が容易です。」

---

【スライド9: 経過】
所要時間: 1分
シナリオ:
「経過です。第1病日にヘパリンを開始し、酸素化は速やかに改善しました。第3病日にDOACに切り替え、第7病日に退院されました。

退院後もDOACを継続し、外来でフォローしています。」

---

【スライド10: Take-home message】
所要時間: 1分30秒
シナリオ:
「本症例から学んだことを3点にまとめます。

第一に、術後患者の呼吸困難では、肺塞栓症を必ず鑑別に挙げることです。特に整形外科手術後は高リスクです。

第二に、D-dimerの上昇と下肢腫脹があれば、深部静脈血栓症や肺塞栓症を強く疑うべきです。

第三に、術後のVTE予防の重要性です。早期離床や予防的抗凝固療法を適切に行うことで、VTEのリスクを減らすことができます。

以上で発表を終わります。ご清聴ありがとうございました。」

---

【合計所要時間】: 約15分
```

## シナリオのカスタマイズ

### カスタマイズ1: 発表時間を短縮する

```
上記のシナリオを10分に短縮してください。各スライドの所要時間を調整してください。
```

---

### カスタマイズ2: より詳細なシナリオにする

```
スライド6（鑑別診断）のシナリオをより詳細にしてください。各鑑別診断について、なぜ除外したかをより丁寧に説明してください。
```

---

### カスタマイズ3: 聴衆のレベルに合わせる

```
聴衆が医学生の場合、専門用語をより丁寧に説明するシナリオに修正してください。
```

## 発表練習のコツ

### 1. 声に出して練習する

- **目標**: 3回以上
- **方法**: シナリオを見ながら、実際に声に出して読む
- **効果**: 時間配分の確認、言いにくい箇所の修正

---

### 2. タイマーを使う

- **目標**: 各スライドの所要時間を守る
- **方法**: スマートフォンのタイマーを使用
- **効果**: 時間超過を防ぐ

---

### 3. 録音・録画する

- **目標**: 自分の発表を客観的に評価
- **方法**: スマートフォンで録音・録画
- **効果**: 話し方、間の取り方を改善

---

## 質疑応答の準備

発表後の質疑応答に備えて、想定質問とその回答を準備します。

### 想定質問生成のプロンプト

```
上記の症例発表について、カンファレンスで想定される質問を5つ挙げてください。

各質問について、適切な回答例も提示してください。
```

---

### AIの出力例

```
【想定質問と回答例】

Q1: なぜヘパリンからDOACに切り替えたのですか?
A1: 急性期はヘパリンで迅速な抗凝固を行い、安定後はDOACに切り替えました。DOACはINRモニタリングが不要で、外来管理が容易なためです。また、最近のガイドラインでもDOACが推奨されています。

Q2: Wells scoreは計算しましたか?
A2: はい、Wells scoreは7点で、肺塞栓症の可能性が高いと判定されました。DVTの臨床症状で3点、PEが最も疑わしいで3点、心拍数>100で1.5点です。

Q3: 下大静脈フィルターは検討しましたか?
A3: 本症例では、抗凝固療法が可能であったため、下大静脈フィルターは使用しませんでした。フィルターの適応は、抗凝固療法が禁忌の場合や、抗凝固療法中にも関わらず再発する場合です。

Q4: 退院後の抗凝固療法の期間は?
A4: ガイドラインでは、誘因のあるVTE（本症例では術後）の場合、3ヶ月間の抗凝固療法が推奨されています。本症例も3ヶ月間DOACを継続する予定です。

Q5: 術後のVTE予防はどのように行うべきですか?
A5: 術後VTE予防には、早期離床と薬物的予防が重要です。整形外科手術などの高リスク手術では、予防的抗凝固療法（ヘパリンやDOAC）が推奨されます。また、弾性ストッキングや間欠的空気圧迫法も有効です。
```

## 発表時の注意点

### 1. アイコンタクト

- スライドばかり見ない
- 聴衆を見て話す
- 特に指導医の反応を確認

---

### 2. 話すスピード

- **ゆっくり、はっきり**話す
- 早口にならないよう注意
- 重要なポイントは**間を取る**

---

### 3. 専門用語の説明

- 専門用語は必要に応じて説明
- 聴衆のレベルに合わせる

---

### 4. 時間管理

- 各スライドの所要時間を守る
- 時計を見ながら発表
- 時間が足りない場合は、詳細を省略

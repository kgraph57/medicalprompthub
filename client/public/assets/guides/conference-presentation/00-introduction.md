# カンファレンス発表資料作成支援：AIで効率化する症例発表

## はじめに

研修医・若手医師にとって、毎週のカンファレンス発表は避けて通れない業務です。症例発表、抄読会、M&Mカンファレンスなど、発表の機会は多岐にわたります。しかし、日常の診療業務に追われる中で、質の高い発表資料を準備する時間を確保するのは容易ではありません。

このガイドでは、AIを活用して発表資料作成を効率化する方法を紹介します。症例情報を入力するだけで、標準的なスライド構成、鑑別診断リスト、参考文献の提案まで、AIが自動的にサポートします。

## なぜAI支援が必要なのか？

### 研修医の現状

- 週平均労働時間: 56時間22分（法定労働時間を大幅超過）
- カンファレンス発表: 週1〜2回
- 資料作成時間: 1回あたり2〜4時間
- 結果: 睡眠時間を削って資料作成

### AI支援の効果

1. 時間短縮: 2〜4時間 → 30分〜1時間
2. 質の向上: 標準的な構成で見落としを防ぐ
3. 学習効果: AIの提案から学べる
4. ストレス軽減: 「何を書くべきか」の悩みを解消

## このガイドで学べること

1. 症例発表の標準的な構成を理解し、効率的に資料を作成できるようになる
2. AIプロンプトの使い方を習得し、質の高い発表資料を短時間で作成できる
3. 鑑別診断の網羅的なリストをAIで生成し、見落としを防ぐ
4. 参考文献の効率的な検索と引用方法を学ぶ
5. 発表シナリオの作成まで、一連のプロセスを習得する

## 対象となる発表タイプ

このガイドは以下のカンファレンスに対応しています：

| カンファレンスタイプ | 所要時間 | 主な目的 |
| :--- | :--- | :--- |
| 症例カンファレンス | 10-15分 | 診断・治療方針の検討 |
| M&Mカンファレンス | 15-20分 | 医療安全・改善策の検討 |
| 抄読会 | 10-15分 | 最新論文の紹介と批判的吟味 |
| 症例報告発表 | 5-10分 | 興味深い症例の共有 |

## このガイドの使い方

### ステップ1: 基本編（10分）
- カンファレンス発表の標準的な構成を学ぶ
- AIプロンプトの基本を理解する

### ステップ2: 実践編（20分）
- 実際の症例でAIを使って資料を作成する
- 鑑別診断リストを生成する
- 参考文献を効率的に検索する

### ステップ3: 応用編（10分）
- 発表シナリオを作成する
- 質疑応答の準備をする

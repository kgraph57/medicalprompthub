# ステップ1: 試験の傾向分析

## なぜ傾向分析が重要か

限られた時間で効率的に学習するには、**出題傾向を把握**することが不可欠です。

---

## 出題パターンの分析

### 問題形式の種類

| 形式 | 特徴 | 対策 |
| :--- | :--- | :--- |
| **症例問題** | 臨床シナリオから診断・治療を問う | 症例演習を重視 |
| **知識問題** | 純粋な知識を問う | 暗記・反復 |
| **画像問題** | X線、CT、病理などの読影 | 画像アトラスで練習 |
| **計算問題** | 薬用量、栄養計算など | 公式の暗記と演習 |

---

### よく出る出題形式

```
1. 「最も適切なものを選べ」
   → 標準治療・ガイドラインを確認

2. 「誤っているものを選べ」
   → 細かい知識が問われる、注意深く読む

3. 「次に行うべき検査は」
   → 診断フローチャートを理解

4. 「この患者に禁忌なのは」
   → 禁忌・副作用の暗記が重要
```

---

## 分野別の出題ポイント

### 循環器

```
高頻度トピック:
- 急性冠症候群の診断・治療
- 不整脈（Af、VT）の管理
- 心不全の分類と治療
- 高血圧の治療選択
- 感染性心内膜炎

重要ポイント:
- ガイドラインに基づく治療選択
- 禁忌薬の暗記
```

### 消化器

```
高頻度トピック:
- 消化管出血の対応
- 肝硬変の合併症管理
- 炎症性腸疾患の鑑別・治療
- 膵炎の重症度判定
- 肝機能障害の鑑別

重要ポイント:
- 内視鏡所見の読影
- 重症度スコア（APACHE II、CT Grade等）
```

---

## 学習優先度の決め方

### 優先度マトリックス

| | 出題頻度高 | 出題頻度低 |
| :--- | :--- | :--- |
| **得意分野** | 確認程度 | 後回し |
| **苦手分野** | 最優先 | 時間があれば |

---

### 優先順位の設定

```
【最優先】出題頻度高 × 苦手
  → 集中的に対策、毎日触れる

【高優先】出題頻度高 × 得意
  → 取りこぼし防止、定期的に確認

【中優先】出題頻度低 × 苦手
  → 基本事項のみ押さえる

【低優先】出題頻度低 × 得意
  → 試験前に軽く確認
```

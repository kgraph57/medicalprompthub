# ステップ2: 学習計画の立案

## 効率的な学習計画とは

限られた時間で最大の効果を得るには、**戦略的な学習計画**が不可欠です。

---

---

### 2. スペース学習

「一気に覚える」より「間隔を空けて繰り返す」方が効果的です。

| 学習法 | 効果 |
| :--- | :--- |
| 一夜漬け | 短期記憶のみ、すぐ忘れる |
| スペース学習 | 長期記憶に定着 |

#### スペース学習のスケジュール

```
Day 1: 新規学習
Day 2: 1回目の復習
Day 4: 2回目の復習
Day 7: 3回目の復習
Day 14: 4回目の復習
Day 30: 5回目の復習
```

---

### 3. アクティブラーニング

「読む」だけでなく「アウトプット」することで記憶が定着します。

| パッシブ学習 | アクティブ学習 |
| :--- | :--- |
| 教科書を読む | 要点をまとめる |
| 講義を聞く | 問題を解く |
| 解説を読む | 人に説明する |
| マーカーを引く | 自分で問題を作る |

---

## 具体的な計画立案

### 学習期間の設定

#### 6ヶ月計画（推奨）

| 期間 | 目標 | 主な活動 |
| :--- | :--- | :--- |
| 1-2ヶ月目 | 基礎固め | 教科書通読、基本用語の暗記 |
| 3-4ヶ月目 | 演習強化 | 過去問演習、分野別問題集 |
| 5ヶ月目 | 弱点克服 | 苦手分野の集中対策 |
| 6ヶ月目 | 最終調整 | 模試、総復習、直前対策 |

---

### 週間スケジュール例

```
【平日】（臨床業務と並行）
朝: 30分 - 暗記項目の確認（通勤中など）
昼: 30分 - 過去問1-2問（昼休み）
夜: 1-2時間 - 教科書・問題集

【休日】
午前: 2-3時間 - 分野別の集中学習
午後: 2-3時間 - 過去問演習
夜: 1時間 - 復習・弱点確認

週間目標: 20-25時間
```

### モチベーション維持

```
✓ 小さな目標を設定し、達成感を得る
✓ 同じ試験を受ける仲間と情報共有
✓ 定期的に模試で実力を確認
✓ 適度な休息（燃え尽き防止）
✓ 合格後のイメージを持つ
```

---

## 計画修正のタイミング

### 見直しが必要なサイン

- [ ] 予定通り進んでいない
- [ ] 過去問の正答率が上がらない
- [ ] 特定分野で大きく点を落とす
- [ ] 勉強時間が確保できていない
- [ ] モチベーションが低下している

### 修正のポイント

```
1. 週次で小さな調整
   → 翌週の計画を微調整

2. 月次で中程度の調整
   → 学習分野の優先度を見直し

3. 中間地点で大きな調整
   → 全体計画の見直し、目標の再設定
```

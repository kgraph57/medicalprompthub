# 専門医試験対策ガイド

## はじめに

専門医試験は、医師としてのキャリアにおける重要なマイルストーンです。多忙な臨床業務と並行して試験勉強を進めることは、容易ではありません。

> 「何から手をつければいいのか分からない...」
> 「過去問を解いても点数が上がらない...」

このガイドでは、AIを活用した**効率的な学習戦略**を紹介します。

---

## なぜAI活用が有効か？

### 従来の学習法の限界

| 課題 | 影響 |
| :--- | :--- |
| 膨大な出題範囲 | 網羅的な学習が困難 |
| 弱点の把握が困難 | 効率の悪い勉強 |
| 暗記の限界 | 知識の定着が不十分 |
| 時間の制約 | 勉強時間の確保が困難 |

## 注意事項

### AI活用の注意点

⚠️ **AIの回答は100%正確ではない**
- 医学的内容は必ず教科書・ガイドラインで確認
- 最新のガイドラインに対応していない可能性

⚠️ **著作権への配慮**
- 過去問の原文をそのままAIに入力しない
- 問題の趣旨を活用した練習問題を生成

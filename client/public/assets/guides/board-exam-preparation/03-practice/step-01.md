# ステップ3: AIを活用した問題演習

## AIで問題演習を効率化

AIを活用することで、**オリジナルの練習問題**を生成し、効率的に演習できます。

## プロンプト集

### プロンプト1: 基本的な問題生成

```
{分野名}に関する専門医試験レベルの問題を5問作成してください。

【条件】
- 難易度: 専門医試験レベル
- 形式: 5択問題
- トピック: {具体的なトピック}

各問題に以下を含めてください：
1. 問題文
2. 選択肢（5つ）
3. 正解
4. 解説（なぜ正解か、なぜ他の選択肢は不正解か）
```

---

### プロンプト2: 症例ベースの問題生成

```
以下の症例に基づいた試験問題を作成してください。

【症例の設定】
年齢・性別: {例: 65歳男性}
主訴: {例: 呼吸困難}
現病歴: {例: 3日前からの呼吸困難}
既往歴: {例: 高血圧、糖尿病}
検査結果: {例: 胸部X線で心拡大}

【作成する問題】
1. 最も考えられる診断は？
2. 次に行うべき検査は？
3. 最も適切な治療は？

各問題に5択と詳細な解説を付けてください。
```

---

### プロンプト3: 解説の補足依頼

```
以下の問題の解説をより詳しく教えてください。

【問題】
{問題文を入力}

【選択肢】
{選択肢を入力}

【正解】
{正解を入力}

以下について詳しく解説してください：
1. なぜこの選択肢が正解なのか（機序・エビデンス）
2. 他の選択肢がなぜ不正解なのか（それぞれについて）
3. この問題で問われている重要なポイント
4. 関連する類似トピック
5. 覚え方・語呂合わせ（あれば）
```

---

### プロンプト4: 類題の作成

```
以下の問題を間違えました。
類題を3問作成してください。

【間違えた問題】
{問題文}

【私の誤答と正解】
誤答: {選択した答え}
正解: {正解}

【類題作成の条件】
- 同じ分野・トピックで
- 難易度は同程度から少し難しく
- 異なる角度からの出題
- 詳細な解説付き
```

## 効果的な演習方法

### 1. 分野別集中演習

```
1週間の演習スケジュール例:

月: 循環器 - 不整脈
火: 循環器 - 心不全
水: 消化器 - 肝疾患
木: 消化器 - 消化管
金: 腎臓 - 電解質異常
土: 週の総復習
日: 模試形式の演習
```

---

### 2. 時間を測って演習

試験本番を想定して、**時間制限**を設けて演習します。

```
目標時間:
- 1問あたり: 約2分
- 50問: 約100分

練習の進め方:
1. タイマーをセット
2. 制限時間内に解答
3. 終了後に答え合わせ
4. 間違えた問題を重点復習
```

---

## 実践例

### 問題生成の例

```
【入力】
心房細動の抗凝固療法に関する問題を作成してください。

【AI出力例】
問題:
70歳男性。持続性心房細動で来院。高血圧、糖尿病の既往あり。
腎機能は正常（eGFR 75）。脳梗塞・出血の既往なし。
この患者の抗凝固療法として最も適切なものはどれか。

a) ワルファリン（PT-INR 2.0-3.0）
b) ダビガトラン 150mg 1日2回
c) リバーロキサバン 15mg 1日1回
d) エドキサバン 60mg 1日1回
e) アスピリン 100mg 1日1回

正解: b

解説:
この患者のCHADS2スコアは3点（高血圧1点、糖尿病1点、75歳以上1点）
であり、抗凝固療法の適応がある。

DOACはワルファリンと同等以上の有効性があり、
頭蓋内出血のリスクが低いため、第一選択となる。
腎機能正常であり、標準用量のDOACが適切。

e) アスピリンは心房細動の脳梗塞予防には不十分。
```

## 注意点

### AI活用の限界

⚠️ **情報の正確性**
- AIの回答は100%正確ではない
- 重要な情報は教科書・ガイドラインで確認

⚠️ **最新情報**
- ガイドラインの更新に対応していない可能性
- 最新の推奨を確認

⚠️ **著作権**
- 過去問の原文をそのまま使用しない
- 問題の趣旨を参考にオリジナル問題を生成

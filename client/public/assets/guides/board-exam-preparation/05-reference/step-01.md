# ステップ5: 暗記効率化テクニック

## 効率的な暗記とは

医学知識は膨大ですが、**効率的な暗記法**を使えば、記憶の定着率を大幅に向上できます。

---

| 20分後 | 58% |

---

### 最適な復習タイミング

```
学習日 → 翌日 → 3日後 → 1週間後 → 2週間後 → 1ヶ月後

このタイミングで復習すると、
長期記憶に定着しやすくなります。
```

## 暗記テクニック

### 1. 語呂合わせ（ゴロ）

覚えにくい項目を**リズムのある言葉**に変換します。

#### AIによる語呂合わせ生成

```
【プロンプト例】
以下の内容を覚えるための語呂合わせを作成してください。

覚えたい内容:
褐色細胞腫の症状（5H）
- Hypertension（高血圧）
- Headache（頭痛）
- Hyperhidrosis（発汗過多）
- Hypermetabolism（代謝亢進）
- Hyperglycemia（高血糖）

条件:
- 日本語で覚えやすく
- 5つの症状を網羅
- 複数の候補を提示
```

---

### 2. 記憶の宮殿（場所法）

**馴染みのある場所**にイメージを配置して覚えます。

```
例: 自分の家の玄関からリビングまでの道順に配置

玄関: CKDステージ1（eGFR ≥90）
廊下: CKDステージ2（eGFR 60-89）
リビング入口: CKDステージ3（eGFR 30-59）
ソファ: CKDステージ4（eGFR 15-29）
窓際: CKDステージ5（eGFR <15）

各場所に「イメージ」を配置して記憶
```

---

### 3. 比較表・マトリックス

類似項目を**表形式で整理**し、違いを明確にします。

```
【ループ利尿薬 vs サイアザイド利尿薬】

         | フロセミド   | ヒドロクロロチアジド
-------------------------------------------------
作用部位  | ヘンレ上行脚 | 遠位尿細管
効力      | 強力        | 中等度
Na排泄   | +++        | ++
K排泄    | ++         | ++
Ca作用   | Ca排泄↑    | Ca再吸収↑
適応      | 浮腫、心不全 | 高血圧、浮腫
```

---

### 4. 連想法（ストーリー法）

**ストーリーを作って**関連付けて覚えます。

```
例: 心不全の病態生理をストーリー化

「心臓が弱った（収縮力低下）ため、
 体が助けようとRAASを活性化した。
 でもNaと水が溜まって（浮腫）、
 心臓はさらに疲れてしまった（悪循環）。
 そこでACE阻害薬とβ遮断薬が助けに来た。」
```

---

### 5. フラッシュカード

**反復学習**に最適なツールです。

```
【フラッシュカードの作り方】

表: 質問
「CKD患者でACE阻害薬を使用する際の注意点は？」

裏: 答え
「・高K血症のモニタリング
 ・eGFR 30未満で減量検討
 ・Cr上昇30%以内なら許容
 ・妊婦には禁忌」

デジタルツール:
- Anki（スペース学習に対応）
- Quizlet
```

## AIを活用した暗記支援

### プロンプト1: 重要事項のまとめ

```
{分野名}の専門医試験で頻出の暗記項目をまとめてください。

以下の形式で整理してください：
1. 絶対に覚えるべき項目（Top 10）
2. 数字が重要な項目（基準値、カットオフ値など）
3. 分類・分類（ステージ、グレードなど）
4. 禁忌事項
5. 覚え方のコツ（語呂合わせがあれば）
```

---

### プロンプト2: 暗記用の質問集

```
{トピック}について、一問一答形式の暗記用質問を20問作成してください。

条件:
- 専門医試験レベル
- 簡潔な質問と答え
- 重要度の高い順に並べる
- 答えは1-2文で

例:
Q: CKDステージ3のeGFR範囲は？
A: 30-59 mL/min/1.73m²
```

---

### プロンプト3: 比較表の作成

```
以下の項目について、暗記用の比較表を作成してください。

比較する項目:
{例: 1型糖尿病 vs 2型糖尿病}

含める項目:
- 発症年齢
- 病態
- 自己抗体
- インスリン
- 治療
- 合併症

表形式で分かりやすく整理してください。
```

## 暗記の時間帯

### 効率的な学習時間

| 時間帯 | 適した学習 |
| :--- | :--- |
| 朝 | 新しい内容の暗記（脳がリフレッシュ） |
| 昼 | 問題演習、復習 |
| 夜 | 軽い復習（睡眠中に記憶が定着） |

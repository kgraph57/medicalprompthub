# ステップ4: 弱点の分析と克服

## 弱点分析の重要性

効率的な学習には、**自分の弱点を客観的に把握**し、集中的に対策することが必要です。

---

### 2. 間違いパターンの分析

| パターン | 特徴 | 対策 |
| :--- | :--- | :--- |
| **知識不足** | 知らない内容 | 教科書で学習 |
| **記憶混乱** | 似た内容の混同 | 比較表を作成 |
| **理解不足** | 機序が分からない | 解説を熟読 |
| **読み違い** | 問題文の誤読 | 丁寧に読む練習 |
| **計算ミス** | 計算間違い | 公式の確認、練習 |

## 弱点克服の戦略

### 1. 集中学習法

弱点分野に**集中的に時間を投下**します。

```
弱点克服の2週間プラン例:

Week 1: 腎臓分野
Day 1-2: 電解質異常（Na, K, Ca）
Day 3-4: 酸塩基平衡
Day 5-6: CKD/AKI
Day 7: 週の総復習

Week 2: 内分泌・代謝分野
Day 1-2: 糖尿病の基本
Day 3-4: 糖尿病合併症
Day 5-6: 甲状腺疾患
Day 7: 模試で確認
```

---

### 2. 比較表の作成

似たような内容を**比較表で整理**し、混同を防ぎます。

#### 例: 電解質異常の比較

```
【低Na血症の原因鑑別】

         | 尿Na | 血漿浸透圧 | 代表疾患
-----------------------------------------
腎性喪失 | >20  | 低        | 利尿薬、腎疾患
腎外喪失 | <20  | 低        | 下痢、嘔吐
希釈性   | <20  | 低        | 心不全、肝硬変
SIADH    | >40  | 低        | 肺癌、脳疾患
```

---

### 3. 語呂合わせ・記憶術

覚えにくい内容は**語呂合わせ**で記憶します。

#### AIによる語呂合わせ生成

```
【プロンプト例】
以下の項目を覚えるための語呂合わせを作成してください。

覚えたい内容:
- CKDステージ分類（G1-5）
- 各ステージのeGFR値
- 治療介入のタイミング

条件:
- 日本語で
- 覚えやすく、医学生・研修医が使いやすい
- 複数の候補を提示
```

---

### 4. 問題を作る側に回る

**自分で問題を作成**することで、深い理解が得られます。

```
【自作問題のプロセス】

1. 教科書で重要ポイントを確認
2. 出題されそうな切り口を考える
3. 問題文と選択肢を作成
4. 解説を書く
5. 同僚と問題を交換して解き合う
```

## 克服度の確認

### 定期的な実力チェック

```
【弱点克服チェックリスト】

□ 1週間後: ___% （目標: 65%以上）
□ 2週間後: ___% （目標: 70%以上）
□ 1ヶ月後: ___% （目標: 75%以上）

目標達成 → 維持モードへ

```

## 実践例: 弱点克服計画

```

- 電解質異常の鑑別
- CKDステージ別の治療

  - 過去問10問で確認

  - 導入基準の暗記
  - 合併症の対策
  - 症例問題の演習

Day 11-13: 総復習
  - 間違いノートの見直し
  - 弱点トピックの再演習

Day 14: 最終チェック
  - 模試形式で確認

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

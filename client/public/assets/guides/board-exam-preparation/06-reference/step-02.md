# ステップ6: 試験直前の最終確認

## 直前期の過ごし方

試験直前（1-2週間前）は、**新しいことを詰め込むより、これまでの復習**に集中しましょう。

## 最終確認チェックリスト

### 知識面

```
□ 高頻度分野の重要事項
□ 弱点分野の基本事項
□ 間違いノートの内容
□ 暗記項目（数値、分類など）
□ ガイドラインの最新推奨
```

### 持ち物

```
□ 受験票
□ 身分証明書（写真付き）
□ 筆記用具（鉛筆・消しゴム複数）
□ 時計（音が出ないもの）
□ 昼食・飲み物
□ 薬（必要な場合）
□ 防寒具（冷房対策）
□ 最終確認用ノート
```

## 試験直前の心理的準備

### 不安への対処

```
よくある不安:
- 「全然覚えていない気がする」
- 「落ちたらどうしよう」
- 「周りの人が賢そうに見える」

対処法:
- これまでの努力を思い出す
- 深呼吸でリラックス
- 自分のペースを守る
- 「できる問題から解く」と決める
```

---

### ポジティブな自己対話

```
✓ 「十分に準備してきた」
✓ 「できる問題を確実に取る」
✓ 「分からない問題は皆も分からない」
✓ 「最後まで諦めない」
```

---

## 試験テクニック

### 問題の読み方

```
1. 最後の1文（質問）を最初に読む
   → 何を問われているか把握

2. 選択肢をざっと確認
   → 解答の方向性を予測

3. 問題文を丁寧に読む
   → キーワードに注目

4. 消去法を活用
   → 明らかに違う選択肢を除外
```

---

### 時間配分

```
【100問・200分の場合】

目標ペース: 1問2分

最初の1時間: 40問（ペースを掴む）
次の1時間: 40問（ペースを維持）
残り40分: 20問（難問も対応）

見直し: 10-15分

⚠️ 1問に3分以上かけない
   → 飛ばして後で戻る
```

---

### 迷った時の対処

```
2択で迷った時:
- 最初の直感を信じることが多い
- 変更は「明確な根拠がある時のみ」

全く分からない時:
- 消去法で2-3択に絞る
- 統計的に多い選択肢（真ん中など）を選ぶ
- 空欄は絶対に作らない
```

## 試験後

### 自己採点

```
試験後すぐに:
- 解答速報で自己採点（公開されれば）
- 一喜一憂しすぎない

合格発表まで:
- 結果を待つ
- 次のステップを考える
```

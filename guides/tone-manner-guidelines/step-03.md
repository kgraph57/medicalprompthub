# ステップ3: 避けるべき表現を確認する

## 📋 このステップの目的

良い文章を書くには、「何を書くか」だけでなく、「何を書かないか」も重要です。このステップでは、避けるべき4つの表現パターンを学びます。

## ⏱️ 所要時間

10分

## 🎯 達成基準

- 4つの避けるべき表現パターンを理解している
- 実際のドキュメントで、これらの表現を見つけて修正できる

## 📝 詳細手順

### NG表現1: 過度な専門用語

**なぜダメ？**: 専門用語は、読者の理解を妨げます。必要最小限にし、必ず説明を添えましょう。

❌ **悪い例**:
```markdown
エビデンスレベルの高いRCTを参照する。
```

✅ **良い例**:
```markdown
信頼性の高い臨床試験（RCT）を参照します。
```

**修正のポイント**:
- 専門用語には、必ず説明を添える
- 略語は初出時に正式名称を併記する
- 可能であれば、平易な言葉に置き換える

### NG表現2: 命令口調

**なぜダメ？**: 命令口調は、読者に圧迫感を与えます。柔らかい表現を使いましょう。

❌ **悪い例**:
```markdown
必ず実施せよ。
データを収集しなさい。
論文を読め。
```

✅ **良い例**:
```markdown
必ず実施してください。
データを収集しましょう。
論文を読んでみてください。
```

**修正のポイント**:
- 「〜せよ」→「〜してください」
- 「〜しなさい」→「〜しましょう」
- 「〜しろ」→「〜してみてください」

### NG表現3: 曖昧な表現

**なぜダメ？**: 曖昧な表現は、読者を混乱させます。具体的なアクションを示しましょう。

❌ **悪い例**:
```markdown
適切に対応する。
必要に応じて確認する。
しっかりと準備する。
```

✅ **良い例**:
```markdown
指導医に相談して、次のステップを決めます。
倫理委員会の承認が必要な場合は、申請書を提出します。
データを整理し、関連論文を3本以上読んでおきます。
```

**修正のポイント**:
- 「適切に」「必要に応じて」「しっかりと」などの曖昧な表現を避ける
- 具体的なアクションを示す
- 数値や期限を明示する

### NG表現4: 長すぎる文章

**なぜダメ？**: 長い文章は、読者の理解を妨げます。短く分割しましょう。

❌ **悪い例**:
```markdown
症例報告は、臨床現場で遭遇した稀な疾患や予期せぬ経過、新しい治療法の効果など、医学的に価値のある情報を学術コミュニティと共有するための重要な手段であり、一人の患者さんの経験が、未来の医療の発展に貢献する可能性を秘めています。
```

✅ **良い例**:
```markdown
症例報告は、臨床現場の貴重な経験を学術コミュニティと共有する手段です。稀な疾患、予期せぬ経過、新しい治療法の効果など、医学的に価値のある情報を伝えます。

一人の患者さんの経験が、未来の医療の発展に貢献する可能性を秘めています。
```

**修正のポイント**:
- 1文は2行以内に収める
- 複数の情報を含む場合は、文を分割する
- 箇条書きやリストを活用する

## 🎯 実践例

実際のドキュメントで、これらのNG表現を修正してみましょう。

### 修正前

```markdown
エビデンスレベルの高いRCTを参照せよ。適切に対応し、必要に応じて倫理委員会の承認を得る。症例報告は、臨床現場で遭遇した稀な疾患や予期せぬ経過、新しい治療法の効果など、医学的に価値のある情報を学術コミュニティと共有するための重要な手段である。
```

### 修正後

```markdown
信頼性の高い臨床試験（RCT）を参照してください。指導医に相談して、次のステップを決めます。倫理委員会の承認が必要な場合は、申請書を提出します。

症例報告は、臨床現場の貴重な経験を学術コミュニティと共有する手段です。以下のような情報を伝えます：

- 稀な疾患
- 予期せぬ経過
- 新しい治療法の効果
```

### 適用したルール

- **NG表現1**: 専門用語に説明を追加（「RCT」→「臨床試験（RCT）」）
- **NG表現2**: 命令口調を修正（「参照せよ」→「参照してください」）
- **NG表現3**: 曖昧な表現を具体化（「適切に対応」→「指導医に相談」）
- **NG表現4**: 長い文章を分割（箇条書きを活用）

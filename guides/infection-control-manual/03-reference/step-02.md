# 感染対策マニュアル作成ガイド：応用編2

## マニュアルの自動更新とナレッジベース化

感染対策マニュアルは、一度作成したら終わりではなく、常に最新の知見やガイドラインに合わせて更新し続ける必要があります。このステップでは、AIをマニュアルの維持管理に活用し、改訂作業の負担を劇的に軽減する方法を解説します。

### 1. AIによる最新ガイドラインの自動追跡と差分検出

厚生労働省、CDC、WHOなどの公的機関は、頻繁に感染対策に関するガイドラインを更新します。AIは、これらの情報を定期的に監視し、既存のマニュアルとの差分（変更点）を自動で検出できます。

具体的な手順:

1.  監視対象のURLリスト化: 監視したい公的機関のガイドラインが掲載されているWebページのURLをリスト化します。
2.  定期的な比較指示: AIに対し、「リスト化されたURLのコンテンツを毎月1回チェックし、既存のマニュアル（PDFまたはテキストデータ）と比較して、変更された数値、推奨事項、手順を抽出せよ」と指示します。
3.  改訂提案の生成: 抽出された変更点に基づき、「マニュアルの〇〇ページ、〇〇項目を、△△から□□に改訂することを推奨する」という形式の改訂提案書を生成させます。

### 2. 実用的なプロンプト例：改訂提案書の生成

ここでは、AIに新しいガイドラインの変更点をインプットし、具体的な改訂提案書を生成させるプロンプト例を示します。

プロンプト:

```
あなたは、感染対策マニュアルの改訂担当者です。
以下の「新しいガイドラインの変更点」に基づき、既存マニュアルの改訂提案書を作成してください。

[新しいガイドラインの変更点]
- 変更点1: 従来の「アルコール手指消毒液の推奨濃度70%以上」が、「60%〜95%」に緩和された。
- 変更点2: 接触予防策の解除基準に、「C.difficile感染症の症状改善後、48時間以上の間隔をあけた2回の陰性検査結果」が追加された。

[既存マニュアルの該当箇所（架空）]
- 項目A: 手指衛生の基本 -> アルコール手指消毒液は、70%以上の濃度を使用すること。
- 項目B: 接触予防策の解除 -> 症状改善後、医師の判断により解除する。

[改訂提案書の要件]
1.  変更点ごとに、既存マニュアルの「項目名」と「現行の記述」を明記すること。
2.  「推奨される改訂後の記述」を具体的に提案すること。
3.  改訂の「理由」（例：最新の科学的根拠に基づく）を簡潔に記述すること。
4.  改訂提案書は、表形式で整理し、視認性を高めること。
```

### 3. 注意点とTips

*   Tips: マニュアルのナレッジベース化: マニュアル全体を一つの大きなテキストファイルとして扱うのではなく、各手順や項目を独立した小さなデータ（例：Notion、Wiki、データベース）として管理し、AIがアクセスしやすいナレッジベースとして構築します。これにより、AIは特定の項目だけをピンポイントで改訂提案できるようになり、改訂作業の粒度が向上します。
*   注意点: 最終決定は人間が: AIは改訂の「提案」はできますが、その変更が病院の現場の状況や文化に合致するかどうか、また、他の項目との整合性が取れているかどうかの最終的な判断は、必ず感染対策チーム（ICT）が行う必要があります。AIの提案はあくまで「下書き」として活用し、人間の専門家によるレビューを必須としてください。

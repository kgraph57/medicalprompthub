# 感染対策マニュアル作成ガイド：基本編1

## AI活用のメリットとマニュアル作成の目的設定

このステップでは、感染対策マニュアルの作成プロセスにAIを導入する基本的なメリットを理解し、マニュアル作成の目的と範囲を明確に設定します。AIは、情報収集の効率化、最新ガイドラインの反映、構造化されたアウトラインの生成において、医療従事者の強力なサポートツールとなります。

### 1. AI導入のメリット理解とマニュアルの目標設定

感染対策マニュアルは、職員の安全と患者のQOLを守るための最も重要な文書の一つです。AIを活用することで、従来の作成プロセスで課題となりがちだった「時間と労力」「情報の網羅性」「最新性の維持」といった問題を解決できます。

| 従来の課題 | AI活用による解決策 |
| :--- | :--- |
| 時間と労力 | 膨大なガイドラインや文献の迅速な要約・比較 |
| 情報の網羅性 | 見落としがちな項目や最新の知見の提案 |
| 構造化の難しさ | 目的と対象者に応じた論理的な目次・構成案の自動生成 |
| 表現の統一 | 専門用語の定義や文体の統一性のチェック |

具体的な手順:

1.  AI活用の目的を定義する: 「ゼロから完璧なマニュアルを作成する」のではなく、「既存マニュアルの改訂を効率化する」「特定の部署（例：透析室、手術室）に特化した補完マニュアルを作成する」など、具体的な目標を設定します。
2.  マニュアルの対象者と範囲を明確にする: 医師、看護師、事務職員、清掃スタッフなど、誰が主な読者かによって、記載すべき専門性や表現の平易さが変わります。
3.  AIに与えるべき初期情報を整理する: マニュアルの核となる情報（例：病院の規模、地域特性、既存の感染対策組織図）を箇条書きで準備します。

### 2. 実用的なプロンプト例：目的設定のための質問生成

AIにマニュアル作成の目的を深く理解させるため、まずAIに「質問」を生成させ、その質問に答える形で情報を整理します。

プロンプト:

```
あなたは、医療機関の感染対策マニュアル作成を支援する専門コンサルタントです。
以下の情報に基づき、マニュアルの目的と範囲を明確にするために、私（マニュアル作成担当者）に尋ねるべき最も重要な質問を5つ提案してください。

[初期情報]
- 医療機関の種別: 200床の急性期病院
- 主な課題: 新人職員の入れ替わりが多く、標準予防策（スタンダードプリコーション）の徹底が不十分
- 既存マニュアルの有無: あり（ただし5年前に作成されたもので、最新ガイドラインに対応できていない）
- マニュアルの主な読者: 看護師、研修医、清掃スタッフ

[質問の要件]
1. 質問は具体的で、回答がマニュアルの構成に直結する内容であること。
2. 質問は、対象者（看護師、研修医など）の行動変容を促す視点を含むこと。
3. 質問は、AIが次のステップ（目次作成）に進むために不可欠な情報に焦点を当てること。
```

### 3. 注意点とTips

*   Tips: AIを「秘書」として扱う: AIは万能な「専門家」ではなく、あなたの指示を忠実に実行する「優秀な秘書」と捉えましょう。初期の目的設定が曖昧だと、AIは一般的な情報しか提供できず、現場で使えるマニュアルにはなりません。
*   注意点: 著作権と情報の正確性: AIが生成した文章をそのまま使用する際は、必ず最新の公的ガイドライン（厚生労働省、CDCなど）と照らし合わせ、情報の正確性を確認してください。特に、数値や薬剤名、手順の順序は厳重なチェックが必要です。

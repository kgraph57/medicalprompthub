# Step 15: 論文の英語、プロのレベルに磨き上げよう

### 📋 このステップの目的

完成した論文の英語を、ネイティブの専門家レベルに引き上げます。不自然な英語は、内容以前にリジェクトの原因になりかねません。ここでは、AIとプロの力を借りて、英語の質を完璧に仕上げる方法を解説します。

![英文校正プロセス](https://files.manuscdn.com/user_upload_by_module/session_file/90346460/OMfKlwLsIzTjuCYG.png)

### ⏱️ 所要時間

AI校正: 1〜2時間
プロの校正: 3日〜1週間

### 🎯 達成基準

- 文法、スペル、句読点のミスがゼロになっている。
- 不自然な表現がなく、自然で読みやすい学術英語になっている。
- プロの英文校正サービスによる「校正証明書」が手元にある。

### 📝 詳細手順

![英文校正プロセス](/Helix/assets/guides/case-report/03-finishing/proofreading_process.png)

英文校正は、セルフチェック → AIチェック → プロのチェックの３段階で進めるのが最強です。

#### 手順1: セルフチェックとAIによる一次校正

まずは自分で音読し、違和感がないか確認。その後、AIツールで基本的なミスを修正します。

- Grammarly: 文法・スペルチェックの定番。無料版でも十分役立ちます。
- DeepL Write: 文章をより自然で洗練された表現に書き換えるのが得意。
- ChatGPT / Claude: 特定の文章をリライトさせたり、アカデミックな表現を提案させたりできます。

#### 手順2: プロの英文校正サービスに依頼

AIは万能ではありません。特にトップジャーナルを目指すなら、プロの英文校正は必須です。

- 主要なサービス:
  - Edanz (エダンズ): 日本の研究者に人気。医学分野に強い。
  - Editage (エディテージ): 24時間365日対応。幅広い分野をカバー。
  - Enago (イーナゴ): 質の高い校正者が揃っています。

依頼する際は、論文の専門分野と投稿予定のジャーナル名を伝えると、より質の高い校正が期待できます。

### 💡 プロンプト集

#### プロンプト1: アカデミックな表現に書き換える (DeepL Write / ChatGPT)

```
# 指示
以下の英文を、よりフォーマルで、学術論文に適したアカデミックな表現に書き換えてください。

# 元の英文
We did a biopsy and found out it was OP. So, we gave the patient steroids. But it didn't work well. The patient got worse. Then, we tried IVCY, and it was great.
```

活用例: 自分で書いた少し口語的な英語を、AIで瞬時に洗練された学術英語に変換できます。

### ⚠️ よくある失敗と対処法

- 失敗: 無料の自動翻訳だけで英文を作成し、校正しない。
  - 対処法: 自動翻訳は下訳です。必ずAI校正ツールやプロの校正を通しましょう。

- 失敗: 校正サービスからの修正を、確認せずにそのまま受け入れる。
  - 対処法: 修正が自分の意図と合っているか、必ず一つ一つ確認しましょう。

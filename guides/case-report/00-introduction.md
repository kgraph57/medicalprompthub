# 第0章: イントロダクション

## このガイドについて

このガイドは、「これさえ読めば、この通りやれば症例報告が書ける」をコンセプトに、構想から投稿までの一連のプロセスを18のステップに分解し、具体的な手順とAI活用法を網羅した実践的なマニュアルです。

各ステップでは、目的、達成基準、詳細な手順に加え、**ChatGPT**、**NotebookLM**、**Perplexity**といった最新のAIツールを最大限に活用するための**具体的なプロンプト**と**実践例**を豊富に紹介します。これにより、従来数ヶ月かかっていた症例報告の執筆プロセスを、大幅に効率化し、質を高めることを目指します。

## 症例報告とは何か？

症例報告は、一人または少数の患者の経過を詳細に記述する医学論文の一形式です。新しい疾患の発見、稀な疾患の臨床像、予期せぬ治療効果や副作用など、臨床現場での貴重な「気づき」を世界中の医療者と共有するための重要な手段です [1]。

| 項目 | 説明 |
|---|---|
| **目的** | 新しい知見の共有、医学教育、将来の研究仮説の提供 |
| **対象** | 1人または少数の患者 |
| **特徴** | 詳細な臨床経過、画像、検査データの記述 |
| **重要性** | 新疾患の発見、稀な病態の理解、治療法の開発に貢献 |

## なぜ症例報告を書くべきか？

症例報告の執筆は、単なる義務ではありません。自身の臨床経験を深く省察し、科学的思考を鍛える絶好の機会です。また、自身の名前で医学の発展に貢献し、キャリア形成にも繋がる重要な活動です。

症例を深く掘り下げることで、診断・治療への理解が深まり、臨床能力が向上します。同時に、批判的吟味や論理的思考が身につき、科学的思考の訓練にもなります。さらに、症例報告は業績として評価され、専門医取得や昇進に有利になるという実利的な側面もあります。何より、あなたの経験が未来の医療を変えるかもしれないという、医学への貢献という大きな意義があります。

## 完成までの全体像（タイムライン）

本ガイドでは、症例報告の執筆プロセスを3つの章、18のステップに分けて解説します。以下は、標準的なタイムラインの目安です。

| 章 | ステップ | 内容 | 所要時間（目安） |
|---|---|---|---|
| **第1章: 準備編** | 1-5 | 症例選定、文献レビュー、データ整理 | 1-2週間 |
| **第2章: 執筆編** | 6-12 | 考察構成、各セクション執筆 | 2-3週間 |
| **第3章: 仕上げ編** | 13-18 | 参考文献整理、英文校正、投稿 | 1-2週間 |
| **合計** | | | **4-7週間** |

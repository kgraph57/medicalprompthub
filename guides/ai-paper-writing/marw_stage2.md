# 段階2: 体系的文献レビュー

## 目的

本段階の目的は、段階1で設定した研究課題に基づき、関連する既存の知識を体系的に収集、評価、統合することです。これは、研究の新規性と文脈を確立し、自身の研究を学術的な対話の中に位置づけるための基盤となります。AIを活用することで、膨大な文献の中から重要な知見を効率的に抽出し、深い洞察を得ることが可能になります。

## 主要タスク

### 2.1 重要論文の発見と優先順位付け

初期検索で見つけた論文を起点に、引用ネットワークをたどり、分野内で最も影響力のある論文や最新の重要論文を特定します。

このプロセスでは、人間はAIが可視化した文献マップを解釈し、研究質問との関連性や文脈に基づき、読むべき論文の最終的な優先順位を決定する役割を担います。一方、AIは引用関係を分析し、文献間のつながりを視覚的にマッピングし、被引用数や発表年などの指標に基づき、影響力の高い論文や新しい論文をハイライトする役割を果たします。この協働により、効率的な文献探索が可能になります。

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **Litmaps** | 視覚的な文献マップ作成 | 少数の「種論文」から関連論文を自動で検索し、引用ネットワークを可視化。時間軸に沿った研究の進化を把握するのに有効。 |
| **Connected Papers** | 論文間の関係性可視化 | ある論文に類似した論文群をグラフで表示し、分野の主要なクラスターを特定する。 |
| **Research Rabbit** | 文献コレクションの構築 | 論文を追加していくと、関連論文を継続的に推薦。インタラクティブな文献探索が可能。 |

**プロンプト例4: 論文の優先順位付け**

```
以下の論文リストを、私の研究質問「[研究質問]」との関連性に基づいて優先順位付けしてください:

[論文リスト]

各論文について、以下の多角的な観点から評価し、総合スコアを算出してください:
1. 研究質問との直接的関連性（1-5点）
2. 研究デザインの質とエビデンスレベル（1-5点）
3. 引用数とジャーナルのインパクトファクター（1-5点）
4. 出版年の新しさ（1-5点）

総合スコアに基づき、読むべき上位10論文を推奨し、その選定理由を論文ごとに簡潔に説明してください。
```

### 2.2 論文の読解と批判的吟味

選定した論文を深く読み込み、その内容を批判的に評価します。AIは、論文の要約や複雑な概念の説明を補助し、読解の効率と質を高めます。

このプロセスでは、人間は論文の内容を批判的に吟味し、研究の強み、限界、そして自身の研究への示唆を深く考察する役割を担います。一方、AIは論文の要約を作成し、キーポイントを抽出し、複雑な統計手法や専門用語を平易な言葉で解説する役割を果たします。この協働により、効率的かつ深い論文読解が可能になります。

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **SciSummary** | AIによる論文要約 | 論文PDFをアップロードすると、構造化された要約を生成。複数の論文を比較する機能も持つ。 |
| **Anara** | 深い論文分析 | 論文の各セクションを詳細に分析し、キーポイントや図表の説明を抽出する。 |
| **ChatGPT/Claude** | 対話型AI | 論文中の特定の概念や手法について、対話形式で質問し、理解を深める。 |

**プロンプト例5: 論文の批判的要約**

```
以下の論文について、研究者を対象とした批判的要約を作成してください:

[論文のタイトル、著者、抄録、またはPDFを添付]

以下の構造で、各項目を詳細に記述してください:
1. **研究目的と背景**: この研究が解決しようとしている臨床的疑問は何か？
2. **研究デザインと方法**: 研究デザイン（例: RCT, コホート研究）とエビデンスレベルは？主要な方法論の妥当性は？
3. **主要な結果**: 主要な発見は何か？統計的有意性と臨床的意義は？
4. **結論と臨床的示唆**: 著者の結論は何か？臨床実践にどのような影響を与えるか？
5. **研究の強みと限界**: この研究の信頼性を高める要因と、結果の解釈に注意を要する点は何か？
6. **自身の研究への関連性**: この論文が私の研究質問にどのように貢献するか？
```

### 2.3 参考文献管理とノート作成

読解した内容を整理し、後で容易に参照できるように管理します。ここでは、事実と自身の考察を区別して記録することが重要です。

このプロセスでは、人間は論文から得られた洞察や批判的なコメントを自身の言葉で記録し、「アトミック・ステートメント」として抽出された事実を組み合わせて論理的なつながりを構築する役割を担います。一方、AIは論文から「アトミック・ステートメント」（検証可能な単一の事実とその引用源）を抽出し、参考文献情報を自動でフォーマットして管理ツールにインポートする役割を果たします。この協働により、体系的な知識管理が可能になります。

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **Zotero / Paperpile** | 参考文献管理ソフト | 論文のメタデータ、PDF、ノートを一元管理。引用リストの自動生成。 |
| **Obsidian** | ノートアプリ | Zotero/Paperpileと連携し、論文ノートをリンクさせながら管理。知識のネットワークを構築。 |

**プロンプト例6: アトミック・ステートメントの抽出**

```
以下の論文から、私の研究質問「[研究質問]」に直接関連する「アトミック・ステートメント」（単一の事実 + 引用）を抽出してください:

[論文のテキストまたはPDF]

各ステートメントは以下の形式で出力してください:
「[客観的な事実の記述]。 (著者, 年)」

例: 「高血圧患者において、ACE阻害薬はプラセボと比較して心血管イベントのリスクを20%有意に減少させた。 (Smith, 2020)」

抽出したステートメントを、以下のテーマ別に分類してください:
- 疫学・有病率
- 病態生理
- 診断・検査
- 治療・介入
- 予後・アウトカム
```

### 2.4 文献の統合とナラティブ構築

収集したアトミック・ステートメントやノートを統合し、文献レビューの草稿を作成します。これは、単なる要約の寄せ集めではなく、一貫した物語（ナラティブ）として構築する必要があります。

-   **人間の役割**: AIが整理した情報に基づき、論理的な流れを持つナラティブを構築する。研究の文脈と重要性を明確にし、研究ギャップを説得力をもって提示する。
-   **AIの役割**: テーマ別に分類されたアトミック・ステートメントを論理的な順序で並べ替え、文献レビューの構成案（アウトライン）を提案する。

**プロンプト例7: 文献レビューのナラティブ構築**

```
以下のアトミック・ステートメント群を使用して、文献レビューの導入部分のナラティブ（物語的な草稿）を作成してください:

[テーマ別に分類されたアトミック・ステートメントのリスト]

以下の構造で、論理的で説得力のある文章を作成してください:
1. **現在の知識の状態**: トピックの重要性と確立されたコンセンサスを概説する。
2. **矛盾する発見と論争点**: 文献間で結果が一致しない点や、現在議論されている点を提示する。
3. **研究ギャップの特定**: 上記の議論から導かれる、まだ解明されていない重要な問い（リサーチギャップ）を明確にする。
4. **本研究の位置づけ**: この研究が、特定されたギャップをどのように埋めるのかを簡潔に述べる。

各段落は3〜5文で構成し、先行研究への敬意を払いながら、本研究の必要性を強調してください。
```

## システマティックレビュー・メタアナリシスのための特別考慮事項

PRISMA声明などに準拠した厳密なシステマティックレビューやメタアナリシスを行う場合、追加のツールと手順が必要です。

-   **推奨ツール**: **Rayyan**, **Covidence** などのスクリーニング支援ツールを使用し、複数の査読者による独立した論文選択プロセスを管理します。
-   **検索戦略の再現性**: AIによる検索は便利ですが、現時点では再現性に課題があるため、PubMed, Embase, Cochrane Libraryなどの主要データベースでのキーワードと統制語（MeSHなど）を組み合わせた、**再現可能な検索戦略**を併用することが必須です。
-   **プロセスの記録**: PRISMAフローダイアグラムに従い、文献検索、スクリーニング、適格性評価、採用の各段階での論文数を正確に記録します。

## 成果物

-   **体系的文献レビューの草稿**: 2000〜3000語程度の、研究の背景、既存知識、研究ギャップをまとめた文章。
-   **参考文献リスト**: 30〜50本の主要な論文リスト（参考文献管理ソフトで管理）。
-   **アトミック・ステートメント集**: 論文から抽出した事実と引用源のデータベース。
-   **文献マップ**: 論文間の関係性を視覚化した図（Litmapsなどで作成）。

## 検証チェックポイント

-   [ ] 分野の主要な論文やランドマーク研究を網羅しているか？
-   [ ] エビデンスレベル（RCT, 観察研究など）を考慮して文献を評価しているか？
-   [ ] AIが生成した引用や事実は、すべて原著論文で確認したか？（**AIハルシネーションの検証**）
-   [ ] 矛盾する結果や少数意見も公平に扱っているか？
-   [ ] 文献レビューの結果、研究ギャップがより明確に定義されたか？

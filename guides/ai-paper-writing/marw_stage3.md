# 段階3: 研究計画とアウトライン作成

## 目的

本段階の目的は、研究の具体的な設計図である研究計画（プロトコル）を策定し、論文全体の論理的な骨格となるアウトラインを作成することです。強固な計画は、研究の質と再現性を保証し、後の執筆プロセスを円滑に進めるための鍵となります。AIは、研究デザインの選択肢を提示し、統計的妥当性を確保し、論文の構造を体系化する上で強力な支援を提供します。

## 主要タスク

### 3.1 研究デザインの選択

研究質問に最も効果的に答えるための研究デザインを決定します。これは、研究のエビデンスレベルを決定する最も重要なステップです。

このプロセスでは、人間は研究の目的、倫理的制約、利用可能なリソース（時間、予算、人員）を総合的に考慮し、最適な研究デザインを最終的に決定する役割を担います。一方、AIは研究質問に基づき、ランダム化比較試験（RCT）、コホート研究、症例対照研究、横断研究などの複数の研究デザイン案を提示し、それぞれの長所、短所、バイアスのリスクを比較分析する役割を果たします。この協働により、最適な研究デザインの選択が可能になります。

**プロンプト例8: 研究デザインの比較検討**

```
以下の研究質問に対して、最も適切な研究デザインを3つ提案し、それぞれの長所と短所を詳細に比較してください:

研究質問: [PICO形式で定義された研究質問]

各デザインについて、以下の観点から評価してください:
1. **因果関係の証明力**: エビデンスの階層における位置づけは？
2. **実現可能性**: 必要な時間、コスト、リソースは？
3. **倫理的考慮事項**: 倫理的な障壁は何か？
4. **バイアスのリスク**: 選択バイアス、情報バイアス、交絡のリスクは？
5. **臨床的妥当性**: 結果は実臨床にどの程度一般化できるか？

最終的に、最も推奨するデザインを1つ選び、その選定理由を明確に説明してください。
```

### 3.2 サンプルサイズ計算

研究が統計的に意味のある結果を検出するために必要な参加者数を科学的根拠に基づいて算出します。これは、研究の妥当性と効率性を保証するために不可欠です。

このプロセスでは、人間は既存の文献や臨床経験に基づき、検出したい「臨床的に意味のある最小差（MCID）」や予想される効果量を決定する役割を担います。一方、AIは指定されたパラメータ（効果量、有意水準、検出力）に基づき、必要なサンプルサイズを計算し、計算式やその根拠を説明し、異なる条件下でのシミュレーション結果を提示する役割を果たします。この協働により、科学的根拠に基づいたサンプルサイズ計算が可能になります。

**プロンプト例9: サンプルサイズの計算**

```
以下の研究パラメータに基づいて、必要なサンプルサイズを計算してください:

- **研究デザイン**: [例: 2群間比較のランダム化比較試験]
- **主要アウトカム**: [例: 連続変数（血圧値）/ 二値変数（イベント発生率）]
- **予想される効果量**: [例: 平均値の差、オッズ比、ハザード比など具体的な数値]
- **有意水準 (α)**: 0.05 (両側検定)
- **検出力 (1-β)**: 0.80
- **予想される脱落率**: [例: 15%]

計算の根拠となる式を示し、結果を平易な言葉で説明してください。
また、効果量の見積もりが変化した場合にサンプルサイズがどのように変動するかを示す感度分析も行ってください。
```

### 3.3 論文アウトラインの作成

論文全体の論理的な流れを設計します。IMRAD（Introduction, Methods, Results, and Discussion）構造は、医学論文の標準的なフォーマットです。

このプロセスでは、人間はAIが提案したアウトラインを基に、論文の核となる主張（コアメッセージ）を明確にし、各セクションがその主張をどのように補強するかを考え、論理的な流れを構築する役割を担います。一方、AIはICMJE（国際医学雑誌編集者委員会）の推奨に準拠した、詳細なIMRAD構造のアウトライン案を作成し、各セクションに含めるべきキーポイントや論理展開を提案する役割を果たします。この協働により、論理的で説得力のある論文構造が構築されます。

**プロンプト例10: IMRAD構造のアウトライン作成**

```
以下の研究計画に基づき、ICMJE Recommendationsに準拠した詳細な論文アウトラインを作成してください:

- **研究質問**: [PICO形式]
- **研究デザイン**: [デザイン名]
- **主要な発見（予想）**: [研究の核となる予想結果]

以下のIMRAD構造に従い、各セクションで記述すべき内容を箇条書きで具体的に示してください:

1.  **Introduction (序論)**: なぜこの研究が必要なのか？ (3段落構成: 背景→ギャップ→目的)
2.  **Methods (方法)**: 何をどのように行ったのか？ (再現可能な詳細さで)
3.  **Results (結果)**: 何が見つかったのか？ (客観的な事実のみ)
4.  **Discussion (考察)**: それが何を意味するのか？ (解釈と示唆)

各セクションが、論文全体の物語の中でどのような役割を果たすかを念頭に置いて構成してください。
```

## 成果物

本段階の完了時には、4つの主要な成果物が得られます。まず、研究の目的、デザイン、対象、方法、統計解析計画をまとめた研究プロトコル（簡易版）です。次に、IMRAD構造に基づいた、各セクションのキーポイントを含む詳細な骨子である論文アウトラインです。さらに、計算の根拠と結果をまとめたサンプルサイズ計算書です。最後に、研究計画に基づいた倫理審査委員会への申請書ドラフトです。これらの成果物は、次の段階であるデータ収集と分析への基礎となります。

## 検証チェックポイント

次に進む前に、以下の項目を確認してください。研究デザインが研究質問に答えるために最も適切か、サンプルサイズの計算が統計的に妥当で臨床的に意味のある差を検出できる設定か、研究計画において潜在的な倫理的問題が特定され対処法が検討されているか、論文アウトラインが論理的に一貫しており説得力のある物語を構成しているか、そしてすべての計画がICMJEの推奨事項に準拠しているかを確認することが重要です。これらのチェックポイントをすべて満たすことで、質の高い研究計画が完成します。

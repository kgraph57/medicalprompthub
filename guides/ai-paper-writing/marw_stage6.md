# 段階6: 編集と査読対応

## 目的

本段階の目的は、完成した論文初稿を洗練させ、投稿準備を整えること、そして学術出版の核心的プロセスである査読（ピアレビュー）に建設的に対応することです。AIは、言語的な質の向上、潜在的な弱点の特定、そして査読者との対話を円滑に進めるためのコミュニケーション支援において、強力なパートナーとなります。

## 主要タスク

### 6.1 原稿の編集と洗練

投稿前の最終仕上げとして、論文の明確性、簡潔性、そして学術的なトーンを向上させます。

このプロセスでは、人間は論文全体の論理的な流れと科学的な主張の妥当性を最終確認し、AIの提案を取捨選択して著者の意図が正確に伝わるように調整する役割を担います。一方、AIは文法、スペル、句読点の誤りを修正し、より明確で簡潔な学術的表現を提案し、冗長な表現や専門的でない言い回しを特定する役割を果たします。この協働により、言語的な質を向上させながら、科学的な内容の正確性も保つことができます。

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **Grammarly** | 英文校正ツール | 文法、スタイル、トーンをリアルタイムでチェックし、改善案を提示。 |
| **Paperpal** | AI学術執筆アシスタント | 学術論文に特化した言語チェックと表現の提案。投稿規定への適合性も一部確認。 |
| **thesify** | AI駆動型学術レビュー | 文章の構造、論証の明確さ、議論の説得力など、より高次の視点から改善点を提案。 |

**プロンプト例19: 学術英語の改善**

```
以下の段落を、インパクトの高い医学論文に適した、より専門的で洗練された英語に改善してください:

[改善したい段落のテキスト]

以下の点に特に注意して、具体的な改善案を提示してください。まず、簡潔性を重視し、冗長なフレーズを削除して一文を短くします。次に、明確性を高めるため、曖昧な代名詞や表現をより具体的な言葉に置き換えます。さらに、客観性を保つため、「我々は信じる」などの主観的な表現を、データに基づいた客観的な記述に修正します。最後に、より専門的で力強い動詞を選択することで、学術的なトーンを向上させます。

改善前と改善後のテキストを並べて提示し、各変更の理由を簡潔に説明してください。
```

### 6.2 投稿ジャーナルの選定

研究内容に最も適したジャーナルを選ぶことは、論文が適切な読者に届き、正当な評価を受けるために極めて重要です。

このプロセスでは、人間は研究の専門性、インパクト、ターゲットオーディエンスを考慮し、最終的な投稿先を決定する役割を担います。一方、AIは論文のタイトルと抄録に基づき、内容が合致するジャーナルの候補リストを作成し、各ジャーナルのインパクトファクター、採択率、査読期間などの客観的データを提供する役割を果たします。この情報を基に、研究者は最適な投稿先を選択できます。

**推奨AIツール**

| ツール | 特徴 | 利用シナリオ |
|:---|:---|:---|
| **JournalGuide** | ジャーナル検索エンジン | 論文のタイトルや抄録を貼り付けると、適合するジャーナルを推薦。 |
| **Scimago Journal Rank** | ジャーナル評価指標 | ジャーナルの影響力や専門分野内でのランキングを比較検討。 |

**プロンプト例20: 投稿ジャーナルの選定**

```
以下の論文タイトルと抄録に基づき、投稿に適した医学ジャーナルを5誌提案してください:

- **タイトル**: [論文タイトル]
- **抄録**: [論文の抄録]

各ジャーナルについて、以下の情報を表形式でまとめてください:
1. ジャーナル名
2. 専門分野とスコープ
3. 最新のインパクトファクターと分野内ランキング
4. 採択率と平均査読期間
5. オープンアクセスオプションの有無と費用

最終的に、本研究の性質（例: 基礎研究、臨床応用、症例報告）を考慮し、最も推奨するジャーナルを1誌選び、その理由を説明してください。
```

### 6.3 査読コメントへの対応

査読は、論文の質を向上させるための建設的な対話の機会です。査読者のコメントに敬意を払い、誠実に対応することが採択への鍵となります。

このプロセスでは、人間は査読者のコメントの意図を深く理解し、対応方針（修正、反論、追加実験など）を決定し、最終的な返答レターと改訂原稿に責任を持つ役割を担います。一方、AIは査読コメントを論点ごとに整理し、返答レター（Rebuttal Letter）の構成案と丁寧な表現の草稿を作成し、原稿のどの部分をどのように修正したかを明確に示すための記述を補助する役割を果たします。この協働により、効率的かつ丁寧な査読対応が可能になります。

**【重要】査読中の原稿や査読コメントを、機密保持契約に違反して外部のAIツールに入力してはいけません。所属機関が承認した、セキュリティが確保された環境でのみ使用してください。**

**プロンプト例21: 査読コメントへの返答レター作成**

```
以下の査読コメントに対して、丁寧かつ建設的な返答レター（Point-by-point response）の草稿を作成してください:

[査読者1のコメント]
[査読者2のコメント]

各コメントに対して、以下の構造で返答を作成してください:
1. **査読者のコメントの引用**: 査読者の指摘を正確に引用する。
2. **感謝と同意**: 「ご指摘ありがとうございます。この点は非常に重要だと考えます。」など、敬意を示す。
3. **対応内容**: コメントにどのように対応したかを具体的に説明する。（例: 「ご指摘に基づき、統計解析に感度分析を追加しました。」）
4. **原稿の変更箇所**: 変更が加えられた原稿のページ番号と行番号を正確に示す。

もし査読者の意見に同意できない場合は、その科学的根拠を丁寧に、かつ非対立的なトーンで説明する文章案も作成してください。
```

## 成果物

本段階の完了時には、3つの主要な成果物が得られます。まず、編集・校正が完了し、ターゲットジャーナルの投稿規定に合わせてフォーマットされた最終版原稿である投稿準備完了原稿です。次に、各査読者のコメントに逐一対応した、丁寧で論理的な文書である査読コメントへの返答レターです。最後に、査読コメントに基づき修正され、変更箇所がハイライトされた改訂原稿です。これらの成果物により、論文の質が向上し、採択の可能性が高まります。

## 検証チェックポイント

次に進む前に、以下の項目を確認してください。論文がターゲットジャーナルの投稿規定（フォーマット、語数制限、引用スタイルなど）を完全に満たしているか、すべての査読コメントに対して誠実かつ完全に対応しているか、返答レターが敬意を払った建設的なトーンで書かれているか、原稿の修正によって新たな論理的矛盾や不整合が生じていないか、そして査読プロセスにおけるAIの使用がジャーナルおよび所属機関のポリシーに準拠しているかを確認することが重要です。これらのチェックポイントをすべて満たすことで、質の高い査読対応が実現します。

## 統計解析計画書（SAP）の作成支援

### 1. SAP作成におけるAIの活用範囲

統計解析計画書（SAP: Statistical Analysis Plan）は、研究の透明性と客観性を保証するために不可欠な文書です。AIは、SAPの構成案の作成、解析手法の提案、サンプルサイズ計算の補助など、初期段階でのブレインストーミングと文書化を強力にサポートします。

| SAPの要素 | AIの具体的な支援内容 |
| :--- | :--- |
| 研究デザイン | 適切な統計モデルの選択肢（例：RCTならANCOVA、コホートならCox回帰）を提示。 |
| 主要・副次評価項目 | 各評価項目に対する適切な要約統計量（例：平均値±標準偏差、中央値と四分位範囲）を提案。 |
| 解析手法 | 交絡因子、欠損値、多重比較などの問題に対する具体的な処理方法の提案。 |
| サンプルサイズ | 必要なパラメータ（検出力、有意水準、効果量）を入力することで、計算式と結果を提示。 |

### 2. 実用的なプロンプト例：SAPの構成案と手法の提案

AIにSAPの支援を求める際は、研究の目的、デザイン、主要評価項目を明確に伝えることが重要です。

プロンプト例 1: SAPの構成案作成

```
あなたは臨床研究の統計家です。
以下の情報に基づき、臨床試験の統計解析計画書（SAP）の構成案をMarkdown形式で作成してください。

- 研究デザイン: 多施設共同ランダム化比較試験（RCT）
- 対象疾患: 2型糖尿病
- 介入: 新規薬剤A vs 標準治療B
- 主要評価項目: 6ヶ月後のHbA1cの変化量
- 副次評価項目: 低血糖イベントの発生率、体重の変化

特に、解析対象集団（ITT, PP, Safety）の定義と、欠損値処理の戦略について具体的に記述するセクションを含めてください。
```

プロンプト例 2: サンプルサイズ計算の補助

```
以下の条件で、両側t検定を用いる場合のサンプルサイズを計算するためのRコード（またはPythonコード）と、計算に必要なパラメータを教えてください。

- 主要評価項目: HbA1cの変化量（連続変数）
- 有意水準（α）: 0.05
- 検出力（1-β）: 0.80
- 期待される群間差（効果量）: 0.5%
- 標準偏差（SD）: 1.0%
```

### 3. 注意点とTips：AIの提案を専門家としてレビューする

*   統計的妥当性の確認: AIが提案した解析手法が、あなたの研究デザインやデータの特性（例：非正規分布、繰り返し測定）に本当に適しているか、統計学の教科書や専門家の意見と照らし合わせて確認してください。
*   多重比較の調整: 複数の副次評価項目やサブグループ解析を行う場合、AIに多重比較の調整方法（例：Bonferroni法、FDR法）の提案を求め、その適用理由を明確にしてください。
*   プロトコルとの一貫性: SAPは、事前に作成された研究プロトコルと完全に一致している必要があります。AIが生成した内容がプロトコルから逸脱していないか、厳密にチェックしてください。

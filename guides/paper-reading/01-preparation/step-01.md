# Step 1: 論文の選定と準備

### 📋 このステップの目的

効率的な論文読解の第一歩は、**読む価値のある論文を選ぶこと**です。すべての論文が同じ価値を持つわけではありません。このステップでは、自分の目的に合った論文を選び、読解の準備を整える方法を学びます。

### ⏱️ 所要時間

**5〜10分**

### 🎯 達成基準

- 読む目的（臨床疑問、研究テーマ、教育目的など）が明確になっている
- 選んだ論文が自分の目的に適していることを確認できている
- 論文の基本情報（タイトル、著者、ジャーナル、発行年、DOI）を記録している

### 📝 詳細手順

#### 手順1: 読む目的を明確にする

論文を読む前に、**なぜこの論文を読むのか**を明確にしましょう。目的によって、読むべき部分や深さが変わります。

| 目的 | 重点を置くべき部分 | 所要時間の目安 |
|---|---|---|
| **臨床疑問への回答** | Abstract, Results, Discussion | 30-45分 |
| **研究デザインの理解** | Methods, Results | 45-60分 |
| **最新知見の把握** | Abstract, Introduction, Conclusion | 20-30分 |
| **批判的吟味の練習** | 全体（特にMethods, Results） | 60-90分 |
| **抄読会の準備** | 全体 + 批判的吟味 | 60-90分 |

#### 手順2: 論文の基本情報を確認する

選んだ論文について、以下の基本情報を確認・記録します：

- **タイトル**: 論文の内容を端的に表しているか
- **著者・所属**: 信頼できる研究者・機関か
- **ジャーナル**: Impact Factor、査読プロセスの質
- **発行年**: 最新の知見か、それとも古典的な論文か
- **DOI**: 後で参照する際の識別子

#### 手順3: 論文の種類を判断する

論文の種類によって、読むべき部分や評価方法が異なります：

- **ランダム化比較試験（RCT）**: Methods（特にランダム化方法）、Results（主要アウトカム）
- **観察研究（コホート研究、ケースコントロール研究）**: Methods（バイアス対策）、Results（交絡因子の調整）
- **システマティックレビュー・メタアナリシス**: Methods（検索戦略、選択基準）、Results（統合結果）
- **症例報告**: Case Presentation、Discussion（新規性、臨床的意義）

### 🤖 推奨AIツール

| ツール | 用途 | 優先度 |
|---|---|---|
| **Perplexity** | ジャーナルの信頼性、Impact Factorの確認 | ⭐⭐⭐ |
| **Consensus** | 類似研究の探索、エビデンスレベルの確認 | ⭐⭐⭐ |
| **ChatGPT** | 論文の種類の判断、読むべき部分のアドバイス | ⭐⭐ |

### 💡 プロンプト集

#### プロンプト1: 論文の読む価値を評価する

```
# 指示
あなたは経験豊富な医学研究者です。以下の論文情報を読み、この論文を読む価値があるかどうかを評価してください。

# 評価観点
1. **ジャーナルの信頼性**: このジャーナルは信頼できる査読プロセスを持っていますか？
2. **研究の新規性**: この研究は新しい知見を提供していますか？
3. **臨床的意義**: この研究の結果は、実際の臨床現場にどの程度役立ちますか？
4. **研究デザインの適切性**: 研究デザインは、研究質問に適していますか？
5. **読むべき優先度**: この論文を読むべき優先度はどの程度ですか？（高/中/低）

# 論文情報
- **タイトル**: [論文タイトル]
- **ジャーナル**: [ジャーナル名]
- **発行年**: [年]
- **研究デザイン**: [RCT/観察研究/システマティックレビューなど]
- **読む目的**: [臨床疑問への回答/研究デザインの理解/最新知見の把握など]
```

**活用例**: 限られた時間の中で、どの論文に優先的に時間を割くべきかを判断する際に使用します。

#### プロンプト2: 論文の種類に応じた読むべき部分を提案する

```
# 指示
以下の論文情報に基づき、この論文を効率的に読むために、どの部分を重点的に読むべきか、そして各セクションで何に注目すべきかを教えてください。

# 論文情報
- **タイトル**: [論文タイトル]
- **研究デザイン**: [RCT/観察研究/システマティックレビュー/症例報告など]
- **読む目的**: [臨床疑問への回答/研究デザインの理解/最新知見の把握など]

# 出力形式
以下の形式で回答してください：
1. **重点的に読むべきセクション**: [セクション名]
2. **各セクションでの注目ポイント**: 
   - Abstract: [注目すべき点]
   - Introduction: [注目すべき点]
   - Methods: [注目すべき点]
   - Results: [注目すべき点]
   - Discussion: [注目すべき点]
3. **読む順序の推奨**: [推奨する読む順序]
```

**活用例**: 初めて読む論文で、どこに注目すべきかわからない場合に使用します。

### 📚 実践例

**シナリオ**: ステロイド抵抗性の器質化肺炎（OP）の治療について、最新のエビデンスを知りたい

1. **目的の明確化**: 「ステロイド抵抗性OPに対する代替治療のエビデンスを探す」
2. **論文検索**: Consensusで「steroid-resistant organizing pneumonia treatment」を検索
3. **論文選定**: 2020年以降のシステマティックレビューまたはRCTを優先
4. **基本情報の確認**:
   - タイトル: "Cyclophosphamide for steroid-resistant organizing pneumonia: a systematic review"
   - ジャーナル: Chest (Impact Factor: 9.6)
   - 発行年: 2023年
   - DOI: 10.1016/j.chest.2023.xx.xxx
5. **読むべき部分の決定**: システマティックレビューなので、Methods（検索戦略）、Results（統合結果）、Discussion（臨床的意義）を重点的に読む

### ⚠️ よくある失敗と対処法

- **失敗**: タイトルだけで判断して、不適切な論文を選んでしまう
  - **対処法**: Abstractを必ず読んで、自分の目的に合っているか確認しましょう。時間を無駄にしないためにも、この確認は重要です。

- **失敗**: 古い論文を選んでしまう
  - **対処法**: 発行年を必ず確認し、可能な限り最新のエビデンスを優先しましょう。ただし、古典的な論文やメタアナリシスで引用されている重要な論文は例外です。

- **失敗**: ハゲタカジャーナル（predatory journal）を選んでしまう
  - **対処法**: Perplexityや既知の信頼できるジャーナルリストで、ジャーナルの信頼性を確認しましょう。

### 💪 上級者向けTIPS

- **論文の引用数を確認**: 高く引用されている論文は、その分野で重要な知見を含んでいる可能性が高いです（Google Scholarで確認可能）
- **著者の過去の研究を確認**: 同じ著者の過去の研究を読むことで、研究の文脈をより深く理解できます
- **関連論文を事前にリストアップ**: 読む論文の参考文献から、関連する重要な論文を事前にリストアップしておくと、効率的に知識を深められます

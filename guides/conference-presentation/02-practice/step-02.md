# ステップ2: 鑑別診断リストをAIで生成する

## 鑑別診断の重要性

カンファレンスで最も重要なのは、網羅的な鑑別診断です。見落としを防ぎ、診断プロセスを明確にすることで、聴衆の理解が深まります。

しかし、鑑別診断を網羅的にリストアップするのは時間がかかります。AIを活用することで、この作業を大幅に効率化できます。

---

## 鑑別診断生成のプロンプト

### 基本プロンプト

```
あなたは経験豊富な臨床医です。以下の症例について、鑑別診断を網羅的にリストアップしてください。

【症例情報】
- 年齢・性別: [ここに入力]
- 主訴: [ここに入力]
- 重要な所見: [ここに入力]

【出力形式】
以下の形式で鑑別診断をリストアップしてください：

1. 診断名
   - 支持する所見:
   - 否定的な所見:
   - 確定診断の方法:
   - 緊急性: 高/中/低

頻度と重症度を考慮して、優先順位の高い順に並べてください。
最低5つ、最大10個の鑑別診断を提示してください。
```

  - Murphy徴候陽性
  - WBC 12,000、CRP 5.2
  - 腹部超音波: 胆嚢壁肥厚、胆石あり

【出力形式】
以下の形式で鑑別診断をリストアップしてください：

```
【鑑別診断リスト】

1. 急性胆嚢炎 ★

   - 確定診断の方法: 腹部超音波、造影CT
   - 緊急性: 高

2. 胆管炎
   - 支持する所見: 発熱、右上腹部痛、炎症反応
   - 否定的な所見: 黄疸なし、胆管拡張なし
   - 確定診断の方法: 血液検査（ビリルビン、肝酵素）、MRCP
   - 緊急性: 高

3. 急性肝炎
   - 支持する所見: 右上腹部痛、発熱
   - 否定的な所見: 肝腫大なし（超音波で確認必要）

4. 肝膿瘍
   - 支持する所見: 発熱、右上腹部痛
   - 否定的な所見: 超音波で膿瘍なし
   - 確定診断の方法: 造影CT、血液培養
   - 緊急性: 高

5. 急性膵炎
   - 支持する所見: 腹痛、炎症反応
   - 否定的な所見: 心窩部痛ではない、背部痛なし
   - 確定診断の方法: アミラーゼ、リパーゼ、造影CT
   - 緊急性: 高

6. 胃十二指腸潰瘍穿孔
   - 支持する所見: 腹痛
   - 否定的な所見: 筋性防御なし、free airなし
   - 確定診断の方法: 腹部X線、CT

   - 否定的な所見: 肋骨脊柱角叩打痛なし、尿所見異常なし
   - 確定診断の方法: 尿検査、腹部CT
   - 緊急性: 中

8. 虫垂炎（後腹膜型）
   - 支持する所見: 腹痛、発熱、炎症反応
   - 否定的な所見: 右下腹部痛ではない
   - 確定診断の方法: 腹部CT
   - 緊急性: 中
```

## 実践例2: 呼吸困難の鑑別診断

### 入力プロンプト

```
あなたは経験豊富な臨床医です。以下の症例について、鑑別診断を網羅的にリストアップしてください。

【症例情報】
- 年齢・性別: 70歳、男性
- 主訴: 呼吸困難
- 重要な所見:
  - 喫煙歴: 40年（1日20本）
  - SpO2 88%（室内気）
  - 両側下肺野にcoarse crackles
  - 下腿浮腫あり
  - BNP 850 pg/mL
  - 胸部X線: 両側肺野に浸潤影、心拡大

【出力形式】
以下の形式で鑑別診断をリストアップしてください：

1. 診断名
   - 支持する所見:
   - 否定的な所見:
   - 確定診断の方法:
   - 緊急性: 高/中/低

頻度と重症度を考慮して、優先順位の高い順に並べてください。
最低5つ、最大10個の鑑別診断を提示してください。
```

---

### AIの出力例

```
【鑑別診断リスト】

1. 急性心不全 ★
   - 支持する所見: 呼吸困難、下腿浮腫、BNP上昇、心拡大、両側浸潤影
   - 否定的な所見: なし
   - 確定診断の方法: 心エコー、胸部X線、BNP
   - 緊急性: 高

2. 肺炎
   - 支持する所見: 呼吸困難、両側浸潤影、coarse crackles
   - 否定的な所見: 発熱なし（確認必要）、炎症反応（確認必要）
   - 確定診断の方法: 血液検査、喀痰培養、胸部CT
   - 緊急性: 高

3. COPD急性増悪
   - 支持する所見: 喫煙歴、呼吸困難
   - 否定的な所見: 両側浸潤影（COPDでは通常なし）
   - 確定診断の方法: 呼吸機能検査、動脈血ガス、胸部CT
   - 緊急性: 高

4. 間質性肺炎
   - 支持する所見: 呼吸困難、両側浸潤影、coarse crackles
   - 否定的な所見: 急性発症（間質性肺炎は通常慢性）
   - 確定診断の方法: 胸部HRCT、KL-6、SP-D
   - 緊急性: 中

5. 肺水腫（心原性・非心原性）
   - 支持する所見: 呼吸困難、両側浸潤影、心拡大
   - 否定的な所見: なし
   - 確定診断の方法: 胸部X線、心エコー、動脈血ガス
   - 緊急性: 高

6. 肺塞栓症
   - 支持する所見: 呼吸困難、低酸素血症
   - 否定的な所見: 両側浸潤影（PEでは通常なし）、下肢腫脹なし
   - 確定診断の方法: D-dimer、造影CT、下肢静脈エコー
   - 緊急性: 高

7. 誤嚥性肺炎
   - 支持する所見: 高齢、両側浸潤影
   - 否定的な所見: 嚥下障害の既往なし（確認必要）
   - 確定診断の方法: 嚥下評価、胸部CT
   - 緊急性: 中
```

## 鑑別診断の優先順位付け

AIが生成した鑑別診断を、以下の基準で優先順位付けします。

### 優先順位の基準

1. 緊急性（見逃してはいけない疾患）
   - 致死的: 肺塞栓症、急性冠症候群、大動脈解離
   - 緊急治療が必要: 急性心不全、肺炎、敗血症

2. 頻度（Common disease）
   - よくある疾患を優先

3. 所見の一致度
   - 支持する所見が多い疾患を優先

### 優先順位付けのプロンプト

```
上記の鑑別診断を、以下の基準で優先順位付けしてください：

1. 緊急性（見逃してはいけない疾患）
2. 頻度（Common disease）
3. 所見の一致度

最も可能性の高い診断を★で示してください。
```

## 鑑別診断の除外根拠を明確にする

カンファレンスでは、「なぜこの診断を除外したか」を説明することが重要です。

### 除外根拠のプロンプト

```
鑑別診断リストの中で、除外した診断について、その根拠を明確にしてください。

特に以下の点を含めてください：
- どの所見が否定的だったか
- どの検査で除外できたか
- 除外診断の確実性（完全に除外 / 可能性は低い）
```

---

### 出力例

```
【除外した診断とその根拠】

1. COPD急性増悪
   - 除外根拠: 両側肺野に浸潤影あり（COPDでは通常なし）
   - 除外の確実性: 可能性は低い（完全除外ではない）
   - 追加検討: 呼吸機能検査で確認が望ましい

2. 肺塞栓症
   - 除外根拠: 
     - 両側浸潤影あり（PEでは通常なし）
     - 下肢腫脹なし
     - D-dimer測定していない（要確認）
   - 除外の確実性: 可能性は低い
   - 追加検討: D-dimer測定を推奨

3. 間質性肺炎
   - 除外根拠: 急性発症（間質性肺炎は通常慢性）
   - 除外の確実性: 可能性は低い
   - 追加検討: HRCT、KL-6で確認が望ましい
```

## 鑑別診断のマインドマップ

複雑な症例では、鑑別診断をマインドマップで整理すると理解しやすくなります。

### マインドマップのプロンプト

```
上記の鑑別診断を、主訴ごとに分類してマインドマップ形式で整理してください。

【分類軸】
- 呼吸器疾患
- 循環器疾患
- その他

各疾患について、キーワードを3つ以内で示してください。
```

---

### 出力例

```
【呼吸困難の鑑別診断マインドマップ】

呼吸困難
├─ 呼吸器疾患
│  ├─ 肺炎（発熱、浸潤影、炎症反応）
│  ├─ COPD急性増悪（喫煙歴、呼吸困難、喘鳴）
│  ├─ 間質性肺炎（慢性、crackles、HRCT）
│  └─ 肺塞栓症（突然発症、D-dimer、DVT）
│
├─ 循環器疾患
│  ├─ 急性心不全 ★（浮腫、BNP、心拡大）
│  └─ 肺水腫（浸潤影、心エコー、利尿薬反応）
│
└─ その他
   └─ 誤嚥性肺炎（高齢、嚥下障害、下葉優位）
```

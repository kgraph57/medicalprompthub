## 使い方
ChatGPTやClaudeなどのAIに論文のPDFをアップロードして、以下のプロンプトを入力するだけ。

---

## 基本的な理解のためのプロンプト

### 1. 論文の要約作成
```
この論文の要旨を簡単にまとめて
```

**用途**: 論文全体の概要を素早く把握する

---

### 2. 結論の抜粋
```
論文の結論部分だけを抽出して
```

**用途**: 論文の最終的な主張を確認する

---

### 3. 研究の目的を整理
```
この研究の目的を簡単に説明して
```

**用途**: 研究の動機と目標を理解する

---

### 4. 方法論の要約
```
この論文の研究方法を一文でまとめて
```

**用途**: 研究デザインを素早く把握する

---

### 5. 結果の要点整理
```
この論文の主要な結果を簡単にまとめてください
```

**用途**: 主要な発見を抽出する

---

## 深い理解のためのプロンプト

### 6. 新しい発見の強調
```
この論文で最も重要な発見は何ですか？
```

**用途**: 論文の核心的な貢献を特定する

---

### 7. 参考文献の確認
```
論文で使用された参考文献をリスト化して
```

**用途**: さらに読むべき文献を特定する

---

### 8. 研究の限界を指摘
```
この論文の限界や弱点を簡単に教えてください
```

**用途**: 批判的に論文を評価する

---

## 応用・発展のためのプロンプト

### 9. 文献調査の助け
```
このトピックに関連する最新の研究をいくつか挙げて
```

**用途**: 関連研究を探索する（※要ファクトチェック）

---

### 10. 論文の影響度を評価
```
この研究が医療業界に与える影響は何ですか？
```

**用途**: 臨床的・学術的意義を考察する

---

### 11. 他の研究との比較
```
この論文の結論は、他の研究とどう異なりますか？
```

**用途**: 研究の位置づけを理解する

---

### 12. 実際の臨床応用の可能性
```
この研究は臨床でどのように応用できますか？
```

**用途**: 実践への応用を検討する

---

## 論文執筆支援のためのプロンプト

### 13. キーワード抽出
```
論文における主要なキーワードを抽出して
```

**用途**: 検索キーワードや論文のタグ付けに活用

---

### 14. 批評ポイントの提案
```
この論文に対しての批評ポイントを提案して
```

**用途**: ジャーナルクラブや論文レビューの準備

---

### 15. 次の研究提案
```
この論文に基づいて次に研究すべきテーマは何ですか？
```

**用途**: 研究アイデアの発想

## 症例報告執筆での活用例

```

```

```

```

## 発展的な活用プロンプト

### 複数論文の比較
```
これら3つの論文について、以下の観点で比較表を作成してください：
1. 研究デザイン
2. 対象患者
3. 主要な結果
4. 限界
5. 臨床的意義
```

### 特定セクションの詳細分析
```
この論文のMethodsセクションについて、以下の点を詳しく説明してください：
1. 研究デザイン（コホート、RCT、症例報告など）
2. 対象患者の選択基準
3. 介入内容
4. アウトカム指標
5. 統計解析方法
```

### 批判的吟味（Critical Appraisal）
```
この論文をCASPチェックリストに基づいて評価してください：
1. 研究の焦点は明確か？
2. 研究デザインは適切か？
3. バイアスのリスクは？
4. 結果は信頼できるか？
5. 結果は臨床的に重要か？
6. 結果は自分の患者に適用できるか？
```

---

## 注意事項

1. **ファクトチェック必須**: AIが生成した情報は必ず原文で確認する
2. **文献検索の限界**: プロンプト9のような「関連研究を挙げて」は、AIが架空の論文を生成する可能性があるため、必ずPubMedで確認
3. **複数論文の比較**: NotebookLMと併用すると効果的
4. **言語**: 英語論文の場合、プロンプトも英語にすると精度が上がる場合がある

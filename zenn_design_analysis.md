# Zennの記事デザイン分析

## URL
https://zenn.dev/catnose99/articles/nani-translate

## レイアウト構造

### 全体構成
- **3カラムレイアウト**
  - 左: ソーシャルアクションボタン（いいね、ブックマーク等）- 固定
  - 中央: 記事本文 - スクロール可能
  - 右: 著者情報 + 目次 - 固定

### 記事本文エリアの特徴

1. **コンテンツ幅**
   - 最大幅: 約680px（推定）
   - 左右に十分な余白

2. **フォント**
   - 本文フォントサイズ: 16px（推定）
   - 行間（line-height）: 1.8〜2.0（推定）
   - 段落間のスペース: 1.5em〜2em

3. **余白**
   - 段落間: 広め（24px〜32px程度）
   - セクション間: さらに広め（40px〜48px程度）
   - 左右のパディング: 24px〜32px

4. **見出し**
   - H2: 大きめ（24px〜28px）、上下に余白
   - H3: 中程度（20px〜22px）
   - 見出しの上余白: 大きめ（40px〜48px）
   - 見出しの下余白: 中程度（16px〜24px）

5. **リスト**
   - リスト項目間のスペース: 8px〜12px
   - リスト全体の上下余白: 16px〜24px

6. **コードブロック**
   - 背景色: 薄いグレー
   - パディング: 16px〜24px
   - 角丸: 4px〜8px

7. **カラー**
   - 本文: 濃いグレー（#000000に近いが完全な黒ではない）
   - 背景: 白（#FFFFFF）
   - リンク: 青系

## 改善ポイント

現在の症例報告ガイドと比較して：

1. **行間が狭すぎる** → 1.8〜2.0に拡大
2. **段落間のスペースが不足** → 24px〜32pxに拡大
3. **コンテンツ幅が広すぎる** → 最大680px程度に制限
4. **左右の余白が不足** → 24px〜32pxに拡大
5. **見出しの余白が不足** → 上40px、下24px程度に拡大

# NotebookLM 活用方法（医学論文・症例報告向け）

## 基本情報

**開発元**: Google
**URL**: https://notebooklm.google/
**料金**: 無料

## 主な特徴

1. **複数文献の同時分析**: 最大49件の論文を同時にアップロード可能
2. **正確な情報抽出**: アップロードした文献のみを情報源とし、ハルシネーションを防止
3. **明確な引用**: どの箇所から回答を生成したかをマーカーで表示
4. **文献間の関連性分析**: 複数論文の関係性を自動分析

## NotebookLM vs 一般的なGen AI

| 項目 | NotebookLM | ChatGPT/Claude/Gemini |
|------|------------|----------------------|
| 情報源 | アップロードした文献のみ | 幅広いデータ |
| 正確性 | 高い（引用明示） | ハルシネーションのリスクあり |
| 用途 | 文献読解・分析 | 創造的タスク全般 |
| 文献数 | 最大49件 | 制限なし（但し精度低下） |

## 症例報告執筆での活用シーン

### 1. 文献レビュー（Step 4）
**用途**: 類似症例の論文を一括分析

**手順**:
1. PubMedで検索した類似症例のPDFを10-20件ダウンロード
2. NotebookLMにアップロード
3. 以下のプロンプトで分析

**プロンプト例**:
```
これらの論文から、器質化肺炎の症例報告を以下の観点で比較表を作成してください：
- 著者・年
- 年齢・性別
- 主要な臨床所見
- 画像所見
- 治療内容（薬剤名・用量）
- 転帰
- 特徴的な点
```

### 2. 考察の構成（Step 6）
**用途**: 複数論文から考察のポイントを抽出

**プロンプト例**:
```
これらの論文のDiscussionセクションで、以下の点がどのように議論されているか要約してください：
1. 器質化肺炎の診断基準
2. 鑑別診断のポイント
3. ステロイド治療の用量と期間
4. 予後因子
5. 再発リスク
```

### 3. Introduction執筆（Step 8）
**用途**: 背景情報の整理

**プロンプト例**:
```
これらの論文から、器質化肺炎のIntroductionに含めるべき以下の情報を抽出してください：
1. 疾患の定義
2. 疫学（有病率、好発年齢）
3. 病態生理
4. 臨床的重要性
5. 診断の課題

各項目について、引用すべき論文を明示してください。
```

### 4. 統計データの確認
**プロンプト例**:
```
これらの論文で報告されている器質化肺炎のステロイド治療の反応率を抽出し、表にまとめてください。
```

### 5. ガイドラインの要約
**用途**: CAREガイドラインや疾患ガイドラインの理解

**手順**:
1. CAREガイドラインのPDFをアップロード
2. 疾患ガイドラインのPDFをアップロード

**プロンプト例**:
```
CAREガイドラインに基づいて、私の症例報告に不足している要素をチェックリスト形式で示してください。
```

## 高度な活用法

### Audio Overview機能
**機能**: アップロードした文献を2人のAIホストが対話形式で解説する音声を生成

**活用シーン**:
- 通勤中に論文の内容を耳で学習
- 複雑な論文の全体像を掴む
- 英語論文の理解を深める

**使い方**:
1. 論文をアップロード
2. "Generate Audio Overview"をクリック
3. 10-15分の対話形式の音声が生成される

### Study Guide作成
**機能**: アップロードした文献から学習ガイドを自動生成

**プロンプト例**:
```
この論文の重要なポイントを学習ガイドとして、以下の形式でまとめてください：
1. Key Concepts（重要概念）
2. Main Findings（主な発見）
3. Clinical Implications（臨床的意義）
4. Limitations（限界）
5. Questions for Further Study（さらなる研究課題）
```

### FAQ生成
**プロンプト例**:
```
この論文について、読者が疑問に思いそうな質問とその回答を10個作成してください。
```

## NotebookLMの限界と対処法

### 限界1: 最大49件まで
**対処法**: 重要な論文を厳選してアップロード

### 限界2: 最新情報は含まれない
**対処法**: 最新論文を手動でアップロード

### 限界3: 画像の詳細分析は不得意
**対処法**: 画像所見はテキストで記述してアップロード

### 限界4: 日本語論文の精度
**対処法**: 英語論文を優先的に使用

## 実践例：器質化肺炎の症例報告

**アップロードファイル**:
1. CAREガイドライン（PDF）
2. 器質化肺炎の総説論文（3件）
3. 類似症例報告（10件）
4. 自分の症例のメモ（Googleドキュメント）

**プロンプトの流れ**:

1. **全体把握**
```
これらの論文から、器質化肺炎の症例報告に必要な情報を整理してください。
```

2. **比較表作成**
```
類似症例10件を比較表にまとめてください。
```

3. **新規性の特定**
```
私の症例（Googleドキュメント）と既報告の違いを明確にしてください。
```

4. **考察の骨子**
```
私の症例のDiscussionで議論すべきポイントを5つ挙げてください。
```

5. **引用文献の選定**
```
Discussionで引用すべき論文を、各ポイントごとに推奨してください。
```

# ChatGPTを使った症例報告執筆プロンプト集

## 出典
- 臨床研究論文作成マニュアル「ChatGPTと症例報告を書く」
- URL: http://gotoresearch.jp/wp/?page_id=2420
- 著者: 後藤匡啓先生（Duke-NUS岡田遥平先生監修）

## 前提：個人情報の取り扱い

### ChatGPTに入力してよい情報
- **匿名化した症例情報**（氏名・生年月日・検査値などを置き換える）
- 病歴は一部変更して入力する（要配慮個人情報のため）

### 避けるべき情報（HIPAA基準）
- 氏名、電話番号、FAX番号
- 社会保障番号、医療記録番号
- 生年月日・入退院日時（年以外）
- メールアドレス、IPアドレス
- 顔全体の写真
- 州より細かい住所
- その他固有の識別子

### 患者同意
投稿先雑誌によっては指定の同意書が必要（BMJ Case Reportsなど）。同意取得時に「論文作成時にChatGPTなどの生成AIを使用する」旨を含めることも検討。

## 基本的なプロンプトの流れ

### プロンプト1: 初期設定
```
英語で症例報告をしたいので手伝ってください。
症例は決まっていて、CARE Case Report Guidelinesに準じて書きたいと思います。
まず何をすれば良いですか？
```

**期待される回答**:
- CAREガイドラインの各セクション（Title, Abstract, Introduction, Patient Information, Diagnostic Assessment, Therapeutic Intervention, Follow-up and Outcomes, Discussion）の説明
- 各セクションで書くべき内容の概要

---

### プロンプト2: タイトルと構成の提案
```
症例報告したいのは次の症例です：
『[症例のタイトル（日本語）]』
次はどうすれば良いでしょうか？
```

**例**:
```
症例報告したいのは次の症例です：
『栄養補助食品の誤用に合併したビタミンD中毒と重症高カルシウム血症の一例』
次はどうすれば良いでしょうか？
```

**期待される回答**:
- 英語タイトルの提案
- 各セクションの具体的な書き方の例文

---

### プロンプト3: 各セクションの執筆依頼

#### Abstract
```
以下の情報をもとに、Abstractを200語以内で作成してください：

【患者背景】
- 年齢・性別: [例: 45歳男性]
- 主訴: [例: 全身倦怠感、食欲不振]

【経過】
- 来院時所見: [例: 血清カルシウム15.2 mg/dL、ビタミンD 180 ng/mL]
- 診断: [例: ビタミンD中毒による重症高カルシウム血症]
- 治療: [例: 補液、カルシトニン投与、ビスホスホネート投与]
- 転帰: [例: 治療開始後7日でカルシウム値正常化]

【結論】
- [例: 栄養補助食品の過剰摂取のリスクについて啓発が必要]
```

#### Introduction
```
以下の疾患について、症例報告のIntroductionセクションを作成してください：

【疾患名】
[例: ビタミンD中毒]

【含めるべき内容】
1. 疾患の定義と疫学
2. 一般的な原因
3. 臨床的重要性
4. 本症例の新規性（栄養補助食品の誤用という点）

文献引用が必要な箇所は[citation needed]と記載してください。
```

#### Case Presentation
```
以下の臨床情報をもとに、Case Presentationセクションを作成してください：

【患者情報】
- 年齢・性別: [例: 45歳男性]
- 既往歴: [例: 特記事項なし]
- 家族歴: [例: 特記事項なし]
- 内服歴: [例: ビタミンDサプリメント（自己判断で通常の10倍量を3ヶ月間服用）]

【現病歴】
- 主訴: [例: 全身倦怠感、食欲不振、多尿]
- 経過: [例: 1ヶ月前から症状出現、徐々に悪化]

【身体所見】
- バイタルサイン: [例: BP 130/80, HR 88, BT 36.8℃]
- その他所見: [例: 脱水所見あり]

【検査所見】
- 血液検査: [例: Ca 15.2 mg/dL, P 2.8 mg/dL, Cr 1.5 mg/dL, 25(OH)D 180 ng/mL]
- 画像検査: [例: 胸部X線で異常なし]

【診断】
[例: ビタミンD中毒による重症高カルシウム血症]

【治療】
[例: 大量補液、カルシトニン、ビスホスホネート投与]

【経過】
[例: 治療開始後7日でCa値正常化、症状改善]

CAREガイドラインに従って、時系列で記載してください。
```

#### Discussion
```
以下の症例について、Discussionセクションを作成してください：

【症例の要約】
[例: 栄養補助食品の誤用によるビタミンD中毒と重症高カルシウム血症]

【議論すべきポイント】
1. ビタミンD中毒の診断基準と本症例の特徴
2. 栄養補助食品の誤用という原因の特異性
3. 類似症例との比較（文献レビュー）
4. 治療選択の根拠
5. 臨床的教訓（栄養補助食品の適切な使用の重要性）
6. 本症例の限界

各ポイントについて、文献引用が必要な箇所は[citation needed]と記載してください。
```

## 高度な活用プロンプト

### 類似症例の検索支援
```
以下の症例について、PubMedで検索すべきキーワードを提案してください：

【症例】
[症例の概要]

【検索目的】
1. 類似症例の報告
2. 疾患の診断基準
3. 治療のエビデンス
```

### CAREチェックリストの確認
```
以下の症例報告がCAREガイドラインの全項目を満たしているか確認してください。不足している項目があれば指摘してください：

【Title】
[タイトル]

【Abstract】
[Abstract全文]

【Introduction】
[Introduction全文]

...（以下全セクション）
```

### 投稿先雑誌に合わせた調整
```
以下の症例報告を、[雑誌名]のフォーマットに合わせて調整してください：

- Abstract: 250語以内

- 図表: 3点以内

```

## 注意事項

1. **ファクトチェック必須**: ChatGPTが生成した医学的情報は必ず文献で確認する
2. **数値の確認**: 検査値や統計データは必ず元データと照合する
3. **引用文献**: ChatGPTは架空の文献を生成することがあるため、全ての引用を確認する
4. **個人情報**: 匿名化を徹底する
5. **雑誌規定**: 投稿先雑誌のAI使用に関する規定を確認する
6. **最終確認**: 上級医や共著者による確認を必ず受ける

# Cursor Learn UI/UX 実装計画

## 📋 概要

Cursor Learnと同じUI/UXを実現するための詳細な実装計画です。

## 🎯 目標

- Cursor Learnと視覚的に同じデザインを実現
- クリーンでミニマルなインターフェース
- 優れたユーザー体験

## 📐 現在の実装との比較

### 実装済み
- ✅ 基本的なページ構造（左サイドバー + メインコンテンツ）
- ✅ トピック一覧と詳細表示
- ✅ モバイル対応
- ✅ ルーティング

### 改善が必要な点

#### 1. トップナビゲーションバー
**現在**: Helixの標準ヘッダー（シンプルなロゴのみ）
**目標**: Cursor風のナビゲーションバー
- 左側: ロゴ + ナビゲーションリンク（ドキュメント、API、学習）
- 中央: 検索バー（⌘K）、AIに質問ボタン（⌘I）
- 右側: ダッシュボードリンク
- アクティブなページにはオレンジの下線

#### 2. 左サイドバー
**現在**: 基本的なスタイル
**目標**: Cursor風のシンプルなデザイン
- セクションタイトルをより目立たせる
- トピックリストの番号を強調
- ホバー時のアニメーション
- アクティブなトピックのハイライト

#### 3. メインコンテンツエリア
**現在**: Cardコンポーネントを使用
**目標**: Cursor風のクリーンなレイアウト
- 動画プレイヤー部分（プレースホルダー）
- タイトルと説明のスタイリング
- アナロジーセクションのデザイン改善

#### 4. カラースキーム
**現在**: デフォルトのテーマ
**目標**: Cursor風のカラーパレット
- ライト背景
- オレンジのアクセントカラー（アクティブ状態）
- 控えめなボーダーとシャドウ

## 🎨 デザイン仕様

### カラーパレット
- **アクセント**: `#FF6B35` (オレンジ) - アクティブ状態
- **背景**: `#FFFFFF` (ライト)
- **テキスト**: `#1A1A1A` (ダークグレー)
- **ボーダー**: `#E5E5E5` (ライトグレー)
- **ホバー**: `#F5F5F5` (ライトグレー背景)

### タイポグラフィ
- **タイトル**: 32px, font-weight: 700
- **サブタイトル**: 20px, font-weight: 400
- **本文**: 16px, line-height: 1.6
- **サイドバー**: 14px, font-weight: 500

### スペーシング
- **コンテナ**: max-width: 1200px, padding: 24px
- **セクション間**: 32px
- **要素間**: 16px
